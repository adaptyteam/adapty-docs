This article describes the data sent in [subscription events](events.md) and explains how it differs between event types.

## Subscription started

This event is triggered when a new subscription is started. It contains all the necessary information about the purchase, the user, and the product.

<details>
   <summary>Event specification</summary>

| **Profile ID** | String | The unique identifier of the user in Adapty. |
| **Profile event ID** | String | Unique identifier for the event generated by Adapty. |
| **Event datetime** | String | The date and time when the event occurred (ISO 8601). |
| **Environment** | String | The environment where the event occurred: `Sandbox` or `Production`. |
| **Product ID** | String | The unique identifier of the product in the store (e.g., `com.example.app.monthly`). |
| **Transaction ID** | String | The unique identifier of the transaction from the store. |
| **Original transaction ID** | String | The unique identifier of the original transaction. |
| **Purchase date** | String | The date and time when the purchase was made (ISO 8601). |
| **Original purchase date** | String | The date and time of the original purchase (ISO 8601). |
| **Subscription expires at** | String | The date and time when the current subscription period expires. |
| **Currency** | String | The 3-letter currency code of the purchase (e.g., `USD`, `EUR`). |
| **Price (Local)** | Float | The price of the product in the local currency. |
| **Price (USD)** | Float | The price of the product converted to USD. |
| **Proceeds (Local)** | Float | The amount of money you receive after the store commission is deducted, in the local currency. |
| **Proceeds (USD)** | Float | The proceeds converted to USD. |
| **Net revenue (Local)** | Float | The net revenue (proceeds minus tax) in the local currency. |
| **Net revenue (USD)** | Float | The net revenue (proceeds minus tax) in USD. |
| **Tax amount (Local)** | Float | The estimated tax amount in the local currency. |
| **Tax amount (USD)** | Float | The estimated tax amount in USD. |
| **Store** | String | The store where the purchase was made: `app_store` or `play_store`. |
| **Store country** | String | The two-letter country code of the store where the purchase was made (e.g., `US`, `DE`). |
| **Offer category** | String | The category of the offer used (e.g., `introductory`, `promotional`). Sent only if an offer was used. |
| **Offer discount type** | String | The type of the discount (e.g., `free_trial`, `pay_as_you_go`). Sent only if an offer was used. |
| **Promotional offer ID** | String | The identifier of the promotional offer used (iOS only). Sent only if a promotional offer was used. |
| **Cohort** | String | The name of the cohort the user belongs to. Sent only if the user is assigned to a cohort. |
| **Paywall** | String | The name of the paywall where the purchase was made. Sent only if the purchase was made from a paywall. |
| **Paywall revision** | String | The revision number of the paywall. Sent only if the purchase was made from a paywall. |
| **Variation ID** | String | The unique identifier of the paywall variation. Sent only if the purchase was made from a paywall. |
| **A/B test** | String | The name of the A/B test. Sent only if the user is in an A/B test. |
| **A/B test revision** | String | The revision of the A/B test. Sent only if the user is in an A/B test. |
| **Developer ID** | String | The custom developer ID associated with the paywall. Sent only if defined in the paywall configuration. |
| **Base plan ID** | String | The base plan ID (Android only). Sent only for Android subscriptions with a base plan. |
| **IP address** | String | The IP address of the user. |
| **Profile country** | String | The country determined from the user's IP address. |
| **Consecutive payments** | Integer | The number of consecutive payments made by the user. |
| **Rate after first year** | Boolean | Indicates if the subscription rate changes after the first year (e.g. for Apple subscriptions). |
| **Has access level** | Boolean | Indicates if the user currently has an active access level. |
| **Total revenue (USD)** | Float | The total revenue generated by the user in USD. |
| **Profiles sharing access level** | Array | A list of other profiles that share the same access level (e.g., via family sharing or restored purchases). |

</details>
