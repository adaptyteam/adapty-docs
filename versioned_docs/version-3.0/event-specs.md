This article describes the data sent in [subscription events](events.md) and explains how it differs between event types.

## Subscription started

This event is triggered when a new subscription is started. It contains all the necessary information about the purchase, the user, and the product.

<details>
   <summary>Event specification</summary>

| Field                             | Type    | Description                                                                                                                                                                                                                                                                                                                                       | Example                                                                                                                                                                |
|:----------------------------------|:--------|:--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Store**                         | String  | The store where the purchase was made: `app_store` or `play_store`.                                                                                                                                                                                                                                                                               | `app_store`                                                                                                                                                            |
| **Currency**                      | String  | The 3-letter currency code of the purchase (e.g., `USD`, `EUR`).                                                                                                                                                                                                                                                                                  | `USD`                                                                                                                                                                  |
| **Price (USD)**                   | Float   | The price of the product converted to USD.                                                                                                                                                                                                                                                                                                        | `1.99`                                                                                                                                                                 |
| **Profile ID**                    | String  | The unique identifier of the user in Adapty.                                                                                                                                                                                                                                                                                                      | `727eff32-0299-44b4-9462-6d5b97253bc8`                                                                                                                                 |
| **Cohort**                        | String  | The name of the cohort the user belongs to. Sent only for purchases made in paywalls created in the Adapty paywall builder.                                                                                                                                                                                                                       | `All Users`                                                                                                                                                            |
| **Environment**                   | String  | The environment where the event occurred: `Sandbox` or `Production`.                                                                                                                                                                                                                                                                              | `Sandbox`                                                                                                                                                              |
| **Price (Local)**                 | Float   | The price of the product in the local currency.                                                                                                                                                                                                                                                                                                   | `1.99`                                                                                                                                                                 |
| **Developer ID**                  | String  | The placement ID of the paywall. Sent only for purchases made in paywalls created in the Adapty paywall builder.                                                                                                                                                                                                                                  | `dev_id_123`                                                                                                                                                           |
| **Paywall**                       | String  | The name of the paywall where the purchase was made. Sent only for purchases made in paywalls created in the Adapty paywall builder.                                                                                                                                                                                                              | `main_paywall`                                                                                                                                                         |
| **Proceeds (USD)**                | Float   | The proceeds converted to USD.                                                                                                                                                                                                                                                                                                                    | `1.69`                                                                                                                                                                 |
| **Variation ID**                  | String  | The unique identifier of the paywall variation. Sent only if the purchase was made from a paywall.                                                                                                                                                                                                                                                | `ef022bcd-9fef-4512-9ab6-11b8b761204a`                                                                                                                                 |
| **Purchase date**                 | String  | The date and time when the purchase was made in the store (ISO 8601).                                                                                                                                                                                                                                                                             | `2025-11-25T14:42:16.000000+0000`                                                                                                                                      |
| **Store country**                 | String  | The two-letter country code of the store where the purchase was made (e.g., `US`, `DE`).                                                                                                                                                                                                                                                          | `US`                                                                                                                                                                   |
| **Profile event ID**              | String  | Unique identifier for the event generated by Adapty.                                                                                                                                                                                                                                                                                              | `a7f7e754-e04b-431b-a2a1-4023372a0ea4`                                                                                                                                 |
| **Event datetime**                | String  | The date and time Adapty recorded the event (ISO 8601).                                                                                                                                                                                                                                                                                           | `2025-11-25T14:42:16.000000+0000`                                                                                                                                      |
| **Proceeds (Local)**              | Float   | The amount of money you receive after the store commission is deducted, in the local currency.                                                                                                                                                                                                                                                    | `1.69`                                                                                                                                                                 |
| **Tax amount (USD)**              | Float   | The estimated tax amount in USD.                                                                                                                                                                                                                                                                                                                  | `0.33`                                                                                                                                                                 |
| **Transaction ID**                | String  | The unique identifier of the transaction from the store.                                                                                                                                                                                                                                                                                          | `2000001065003213`                                                                                                                                                     |
| **Net revenue (USD)**             | Float   | The net revenue (proceeds minus tax) in USD.                                                                                                                                                                                                                                                                                                      | `1.41`                                                                                                                                                                 |
| **Profile country**               | String  | The country determined from the user's IP address.                                                                                                                                                                                                                                                                                                | `US`                                                                                                                                                                   |
| **Paywall revision**              | String  | The revision number of the paywall. Sent only for purchases made in paywalls created in the Adapty paywall builder.                                                                                                                                                                                                                               | `1`                                                                                                                                                                    |
| **Tax amount (Local)**            | Float   | The estimated tax amount in the local currency.                                                                                                                                                                                                                                                                                                   | `0.33`                                                                                                                                                                 |
| **Net revenue (Local)**           | Float   | The net revenue (proceeds minus tax) in the local currency.                                                                                                                                                                                                                                                                                       | `1.41`                                                                                                                                                                 |
| **Product ID**                    | String  | The unique identifier of the product in the store (e.g., `com.example.app.monthly`).                                                                                                                                                                                                                                                              | `monthly.premium.999`                                                                                                                                                  |
| **IP address**                    | String  | The IP address of the user if provided by the SDK.                                                                                                                                                                                                                                                                                                | `109.228.106.147`                                                                                                                                                      |
| **Consecutive payments**          | Integer | The number of consecutive payments made by the user. It counts payments associated with the original transaction ID. For new subscribers, it is `1`. However, if a user churns and later resubscribes, and the store keeps the same transaction ID, this number will continue incrementing (e.g., it could be 5 if they had 4 previous payments). | `1`                                                                                                                                                                    |
| **Offer category**                | String  | The category of the offer used (e.g., `introductory`, `promotional`). Sent only if an offer was used.                                                                                                                                                                                                                                             | `introductory`                                                                                                                                                         |
| **Offer discount type**           | String  | The type of the discount (e.g., `free_trial`, `pay_as_you_go`). Sent only if an offer was used.                                                                                                                                                                                                                                                   | `pay_as_you_go`                                                                                                                                                        |
| **Rate after first year**         | Boolean | Indicates if the subscription rate changes after the first year (e.g. for Apple subscriptions).                                                                                                                                                                                                                                                   | `false`                                                                                                                                                                |
| **Original purchase date**        | String  | The date and time of the original purchase (ISO 8601).                                                                                                                                                                                                                                                                                            | `2025-11-25T14:42:17.000000+0000`                                                                                                                                      |
| **Original transaction ID**       | String  | The unique identifier of the original transaction.                                                                                                                                                                                                                                                                                                | `2000001065003213`                                                                                                                                                     |
| **Subscription expires at**       | String  | The date and time when the current subscription period expires.                                                                                                                                                                                                                                                                                   | `2025-11-25T14:47:16.000000+0000`                                                                                                                                      |
| **Has access level**              | Boolean | Indicates if the user currently has an active access level.                                                                                                                                                                                                                                                                                       | `true`                                                                                                                                                                 |
| **Profile total revenue (USD)**   | Float   | The total revenue generated by the user in USD.                                                                                                                                                                                                                                                                                                   | `1.99`                                                                                                                                                                 |
| **Promotional offer ID**          | String  | The identifier of the promotional offer used (iOS only). Sent only if a promotional offer was used.                                                                                                                                                                                                                                               | `promo_123`                                                                                                                                                            |
| **A/B test**                      | String  | The name of the A/B test. Sent only if the user is in an A/B test.                                                                                                                                                                                                                                                                                | `onboarding_test`                                                                                                                                                      |
| **A/B test revision**             | String  | The revision of the A/B test. Sent only if the user is in an A/B test.                                                                                                                                                                                                                                                                            | `2`                                                                                                                                                                    |
| **Base plan ID**                  | String  | The base plan ID (Android only). Sent only for Android subscriptions with a base plan.                                                                                                                                                                                                                                                            | `premium-base`                                                                                                                                                         |
| **Profiles sharing access level** | Array   | A list of other profiles that share the same access level (e.g., via family sharing or restored purchases).                                                                                                                                                                                                                                       | `[{"profile_id": "4a7bf852-c03f-4fb2-a54e-7c6e30b24776", "customer_user_id": null}, {"profile_id": "75083806-9d0c-4cd7-afc8-5818987ad8b3", "customer_user_id": null}]` |

</details>
