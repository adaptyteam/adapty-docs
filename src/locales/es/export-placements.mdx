Cuando trabajas con varios flows, paywalls y onboardings, es importante saber cuáles se muestran a cada usuario. Puedes exportar todos los ajustes de [placement](placements) a un archivo CSV para ver qué flow/paywall/onboarding aparece para cada audiencia y revisar tu configuración tras realizar cambios o ejecutar experimentos.

:::tip
Si te resulta más cómodo, puedes [exportar placements mediante la API server-side](api-export-analytics/operations/retrievePlacementInfo).
:::

Para exportar los placements de flows, paywalls u onboardings:
1. Ve a **[Placements](https://app.adapty.io/placements)** en el menú principal. Cambia a la pestaña **Flows**, **Paywalls** u **Onboardings** — los placements de cada tipo se exportan por separado.
2. Haz clic en **Export to CSV**.

El archivo CSV exportado contiene la siguiente información sobre tus placements:
- ID de placement
- Nombre del placement
- Nombre de audiencia
- Nombre de segmento
- Nombre de prueba A/B entre placements
- Nombre de prueba A/B
- Nombre de flow, nombre de paywall o nombre de onboarding (según la pestaña desde la que hayas exportado)

:::note
Las pruebas A/B entre placements no son compatibles con los placements de flow, por lo que esa columna estará vacía en las exportaciones de flows.
:::
