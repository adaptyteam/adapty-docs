Puedes controlar si un elemento aparece añadiéndole una condición. Un elemento condicional solo es visible para los usuarios que cumplen los criterios especificados.

:::important
Si muestras u ocultas un elemento mediante la [acción](onboarding-actions) **Show** o **Hide**, dicha acción tiene prioridad sobre la condición de **Visibility** establecida en ese elemento.
Usa las condiciones de **Visibility** para los elementos que siempre deben mostrarse u ocultarse según un criterio fijo. Usa acciones cuando la visibilidad deba cambiar en función de la interacción del usuario; por ejemplo, para mostrar un botón después de que el usuario responda una pregunta de un cuestionario.
:::

Para añadir una condición a un elemento:

1. Selecciona el elemento en el lienzo o en el panel de capas.
2. En la sección **Visibility** del panel derecho, selecciona **Conditional**.

<ZoomImage id="flow-conditional-visibility.webp" width="500px" alt="Configuración de visibilidad condicional" />
3. Configura la condición eligiendo un tipo de propiedad en una de las tres pestañas:
   - **Custom**: Variables que creas y gestionas; sus valores pueden actualizarse mediante las interacciones del usuario. Consulta [Variables](onboarding-variables) para más información.
   - **Products**: Propiedades de los productos de tu flow, como el precio o el nombre.
   - **Elements**: Estados de otros elementos del flow, como si un interruptor de prueba está activo.
4. Introduce el **Value** que deseas hacer coincidir.
5. Haz clic en el operador para cambiarlo si es necesario.
6. (Opcional) Haz clic en **Add condition** para añadir más condiciones. Usa el selector para requerir que se cumplan todas las condiciones o cualquiera de ellas.
