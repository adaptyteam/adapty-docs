Las **audiencias** en Adapty son grupos de usuarios basados en [segmentos](segments), que te ayudan a personalizar paywalls, onboardings o pruebas A/B para grupos de usuarios específicos. Puedes definir estos segmentos usando filtros para asegurarte de que los usuarios correctos vean el paywall o el onboarding adecuado en tu app.

En Adapty, un **Placement** es donde puedes mostrar paywalls, onboardings o pruebas A/B. Cuando añades una audiencia a un placement, estás dirigiendo contenido personalizado a grupos de usuarios concretos. Por ejemplo, puedes mostrar diferentes paywalls según la edad, el dispositivo o el estado de suscripción del usuario. Si un usuario pertenece a varios grupos, puedes elegir qué grupo tiene prioridad para decidir qué paywall verá.

En el ejemplo a continuación, tenemos un flujo de onboarding para mostrar tu placement con el identificador `Onboarding`. En el código de tu app, accederás al placement usando este identificador. Si el usuario pertenece a la audiencia "Yoga beginners", verá el primer paywall. Los que no encajen en la audiencia "Yoga beginners" verán el segundo paywall.

Para mostrar un paywall, onboarding o prueba A/B a una audiencia específica, sigue estos pasos:

1. [Crea un segmento de usuarios](segments#creation). Puedes omitir este paso si quieres mostrar el paywall o la prueba A/B a todos los usuarios. En ese caso, usa la audiencia "All users" creada por defecto.
2. [Añade este segmento como audiencia al placement y define qué paywall o prueba A/B se le mostrará](add-audience-paywall-ab-test). La audiencia "All users" se añade automáticamente a cada placement; solo necesitas especificar qué paywall o prueba A/B se debe mostrar.
3. [Establece las prioridades correctas](change-audience-priority) si tienes más de una audiencia en un placement. Esto garantiza que los usuarios que pertenezcan a más de una audiencia vean el contenido más relevante. Cuando un usuario forma parte de varias audiencias, se mostrará el paywall de la audiencia con mayor prioridad.
4. <InlineTooltip tooltip="Muestra el paywall asociado a este placement en el código de la app móvil">[iOS](ios-quickstart-paywalls), [Android](android-quickstart-paywalls), [React Native](react-native-quickstart-paywalls), [Flutter](flutter-quickstart-paywalls), y [Unity](unity-quickstart-paywalls)</InlineTooltip>.
