description: "Migra al SDK de Adapty para Android v3.12 para obtener mejor rendimiento y nuevas funciones de monetización."
metadataTitle: "Migración al SDK de Adapty para Android v3.12 | Documentación de Adapty"
---

En Adapty SDK 3.12.0, hemos eliminado el método `logShowOnboarding` del SDK.
Si has estado usando este método, no estará disponible cuando actualices el SDK a la versión 3.12 o posterior.
En su lugar, puedes [crear onboardings en el creador de onboardings sin código de Adapty](onboardings). Los análisis de estos onboardings se registran automáticamente y dispones de muchas opciones de personalización.
