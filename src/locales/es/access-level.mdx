Los niveles de acceso te permiten controlar lo que los usuarios pueden hacer en tu app sin necesidad de codificar IDs de productos específicos. Cada producto define durante cuánto tiempo el usuario obtiene un determinado nivel de acceso. Así, cada vez que un usuario realiza una compra, Adapty concede acceso a la app durante un período concreto (en el caso de suscripciones) o de forma permanente (en el caso de compras de por vida).

Cuando creas una app en el Adapty Dashboard, el nivel de acceso `premium` se genera automáticamente. Este es el nivel de acceso predeterminado y no se puede eliminar.

Puedes tener múltiples niveles de acceso por app. Aquí tienes algunos ejemplos de cuándo pueden resultar útiles:

- En una app de noticias donde vendes suscripciones a diferentes temáticas de forma independiente, puedes crear niveles de acceso como `sports` y `science`.
- En una app de fitness que ofrece entrenamientos en vídeo grabados con una suscripción estándar (usando el nivel de acceso `premium` predeterminado), los usuarios pueden optar por una opción más cara que dé acceso a entrenamientos en directo con un entrenador. En ese caso, puedes crear un nivel `live_coach_access`.
- En una app de aprendizaje de idiomas, puedes crear un nivel de acceso para cada idioma disponible.

Para empezar a trabajar con niveles de acceso en Adapty, ve a **[Products](https://app.adapty.io/access-levels)** desde el menú principal de Adapty y selecciona la pestaña **Access levels**.

La lista **Access levels** muestra todos los niveles de acceso, incluido el `premium` que se añade automáticamente y los que hayas añadido tú en Adapty.
