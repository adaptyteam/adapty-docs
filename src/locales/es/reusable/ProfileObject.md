| app_id            | String     | :heavy_minus_sign: | El ID interno de tu app. Puedes verlo en el Adapty Dashboard: [App Settings -> pestaña General](https://app.adapty.io/settings/general). |
| profile_id        | UUID       | :heavy_minus_sign: | ID de perfil de Adapty. Puedes verlo en el campo **Adapty ID** en el Adapty Dashboard -> [Profiles](https://app.adapty.io/profiles/users) -> página del perfil específico. |
| customer_user_id  | String     | :heavy_plus_sign:  | El ID de tu usuario en tu sistema. Solo funcionará si [identificas a los usuarios](identifying-users) en el código de tu app móvil mediante el SDK. |
| total_revenue_usd | Float      | :heavy_minus_sign: | Un valor flotante que representa los ingresos totales en USD generados en el perfil. |
| segment_hash      | String     | :heavy_minus_sign: | Parámetro interno.                                           |
| timestamp         | Integer    | :heavy_minus_sign: | Tiempo de respuesta en milisegundos; necesario para resolver condiciones de carrera. |
| custom_attributes | Array      | :heavy_minus_sign: | <p>Se permite establecer un máximo de 30 atributos personalizados en el perfil. Si incluyes el array `custom_attributes`, debes proporcionar al menos una clave de atributo.</p><p>**Clave:** La clave debe ser una cadena de no más de 30 caracteres. Solo se permiten letras, números, guiones, puntos y guiones bajos.</p><p>**Valor:** El valor del atributo no puede superar los 30 caracteres. Solo se permiten cadenas y flotantes como valores; los booleanos se convertirán a flotantes. Envía un valor vacío o null para eliminar el atributo.</p> |
| access_levels     | Array      | :heavy_plus_sign:  | Array de objetos [Access level](server-side-api-objects#access-level). Puede ser null si el cliente no tiene niveles de acceso. |
| subscriptions     | Array      | :heavy_plus_sign:  | Array de objetos [Subscription](server-side-api-objects#subscription). Puede ser null si el cliente no tiene suscripciones. |
| non_subscriptions | Array      | :heavy_plus_sign:  | Array de objetos [Non-Subscription](server-side-api-objects#non-subscription). Puede ser null si el cliente no tiene compras. |
