Si ya has implementado el SDK de Adapty en tu app de Capacitor, querrás comprobar que todo está correctamente configurado y que las compras funcionan como se espera en las plataformas iOS y Android. Esto implica probar tanto la integración del SDK como el flujo real de compras con el entorno sandbox de Apple y el entorno de pruebas de Google Play.

## Prueba tu app \{#test-your-app\}

Para realizar pruebas exhaustivas de tus compras in-app, consulta nuestras guías de pruebas específicas por plataforma: [guía de pruebas iOS](test-purchases-in-sandbox) y [guía de pruebas Android](testing-on-android).

## Prepárate para el lanzamiento \{#prepare-for-release\}

Antes de enviar tu app al store, sigue la [lista de verificación de lanzamiento](release-checklist) para confirmar que:

- La conexión con el store y las notificaciones del servidor están configuradas
- Las compras se completan y se reportan a Adapty
- El acceso se desbloquea y se restaura correctamente
- Se cumplen los requisitos de privacidad y revisión
