Возвращает список пейволов вашего приложения.

## Метод и эндпоинт \{#method-and-endpoint\}

```http
GET https://api.adapty.io/api/v2/server-side-api/paywalls/
```

myHeaders.append("Authorization", "Api-Key <YOUR_SECRET_API_KEY>");

const requestOptions = {
  method: "GET",
  headers: myHeaders,
  redirect: "follow"
};

fetch("https://api.adapty.io/api/v2/server-side-api/paywalls/paywalls/?page%5Bnumber%5D=1&page%5Bsize%5D=10", requestOptions)

  .then((result) => console.log(result))

Плейсхолдеры:

- `<YOUR_SECRET_API_KEY>`: Ваш секретный API-ключ для авторизации.

## Параметры \{#parameters\}

### Query-параметры \{#query-parameters\}

| Параметр       | Тип     | Обязательный | Описание                                                     |
| :------------- | :------ | ------------ | :----------------------------------------------------------- |
| `page[number]` | integer | Нет          | Номер страницы для пагинации (по умолчанию: 1)               |
| `page[size]`   | integer | Нет          | Количество элементов на странице (по умолчанию: 20, макс.: 100) |

## Успешный ответ: 200: OK \{#successful-response-200-ok\}

Возвращает постраничный список пейволов с базовой информацией и метаданными списка.

Объект `paywall` содержит следующие поля:

| Название                   | Тип              | Обязательный | Описание                                                                                      |
|----------------------------|------------------|--------------|-----------------------------------------------------------------------------------------------|
| title                      | String           | Да           | Название пейвола, заданное в дашборде Adapty.                                                 |
| paywall_id                 | String(uuid)     | Да           | Уникальный идентификатор пейвола.                                                             |
| use_paywall_builder        | Boolean          | Да           | Использует ли пейвол Paywall Builder.                                                         |
| use_paywall_builder_legacy | Boolean          | Да           | Использует ли пейвол устаревшую версию Paywall Builder.                                       |
| updated_at                 | String           | Да           | Временная метка последнего обновления пейвола.                                                |
| created_at                 | String           | Да           | Временная метка создания пейвола.                                                             |
| state                      | String           | Да           | Текущее состояние пейвола (например, `"live"`).                                               |
| is_deleted                 | Boolean          | Да           | Помечен ли пейвол как удалённый.                                                              |
| web_purchase_url           | String           | Нет          | URL для веб-покупок, если применимо.                                                          |
| products                   | Array of objects | Да           | Массив объектов продуктов с информацией о продуктах пейвола.                                  |

Ответ содержит объект `meta` с объектом `pagination` внутри. Объект `pagination` включает следующие поля:

| Название | Тип     | Обязательный | Описание                      |
|----------|---------|--------------|-------------------------------|
| count    | Integer | Да           | Общее количество пейволов.    |
| page     | Integer | Да           | Номер текущей страницы.       |
| pages    | Integer | Да           | Общее количество страниц.     |
