В Adapty **пейвол** — это удалённо настраиваемый набор продуктов, которые вы продаёте в приложении. Управление продуктами через пейвол позволяет отслеживать эффективность разных наборов продуктов для разных групп пользователей и контролировать отображение каждого продукта.
Вы отображаете пейвол в собственном коде приложения. Adapty предоставляет два инструмента для этого:
- **Remote Config**: [Управляйте элементами пейвола](customize-paywall-with-remote-config) — текстами, медиа и другим контентом — прямо из дашборда, без перевыпуска приложения.
- **Методы покупок SDK**: Вместо прямой интеграции со store API используйте один метод Adapty SDK, который берёт на себя всю логику покупки — см. <InlineTooltip tooltip="Делегируйте обработку покупок Adapty">[iOS](making-purchases), [Android](android-making-purchases), [React Native](react-native-making-purchases), [Flutter](flutter-making-purchases) и [Unity](unity-making-purchases)</InlineTooltip>.
