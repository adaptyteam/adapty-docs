График **Active subscriptions** показывает количество уникальных платных подписок, срок действия которых ещё не истёк на конец каждого выбранного периода. Учитываются активные встроенные подписки, которые уже начались и не истекли; бесплатные пробные периоды и подписки с отменённым автопродлением не включаются. График отражает размер и динамику роста вашей базы подписчиков.

## Расчёт \{#calculation\}

Метрика активных подписок считает платные, не истёкшие подписки на конец каждого периода. Для подписок без льготного периода истечение происходит в момент, когда дата следующего продления наступает без успешного списания.

Например: 500 активных подписок на конец прошлого месяца + 50 новых в этом месяце − 25 истёкших в этом месяце = 525 активных подписок на конец текущего месяца.
:::note
При возврате средств за подписку она вычитается из количества активных подписок. Это гарантирует, что метрика точно отражает подписки, которые в данный момент приносят доход вашему приложению.
:::
