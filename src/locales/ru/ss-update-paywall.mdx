Обновляет [Remote Config](customize-paywall-with-remote-config) конкретного пейвола. Этот эндпоинт позволяет изменять значения Remote Config, которые помогают настраивать внешний вид и поведение пейвола.

:::important
Если вы обновляете Remote Config, он перезапишет все существующие Remote Config!

Если вам нужно сохранить существующие Remote Config, сначала [получите пейвол](api-adapty/operations/getPaywall). Затем скопируйте `remote_configs` оттуда и измените нужные объекты в запросе на обновление.
:::

## Метод и эндпоинт \{#method-and-endpoint\}

```http
PUT https://api.adapty.io/api/v2/server-side-api/paywalls/{paywall_id}/
```

Плейсхолдеры:

- `<YOUR_SECRET_API_KEY>`: Ваш секретный API-ключ для авторизации.
- `{paywall_id}`: Уникальный идентификатор пейвола, который нужно обновить.

## Успешный ответ: 200: OK \{#successful-response-200-ok\}

Возвращает обновлённую информацию о пейволе. Объект `paywall` содержит следующие свойства:

| Название                   | Тип              | Обязательный | Описание                                                                                              |
|----------------------------|------------------|--------------|-------------------------------------------------------------------------------------------------------|
| title                      | String           | Да           | Название пейвола, как задано в дашборде Adapty.                                                      |
| paywall_id                 | String           | Да           | Уникальный идентификатор пейвола.                                                                    |
| use_paywall_builder        | Boolean          | Да           | Использует ли пейвол Paywall Builder.                                                                |
| use_paywall_builder_legacy | Boolean          | Да           | Использует ли пейвол устаревший Paywall Builder.                                                     |
| updated_at                 | String           | Да           | Временная метка последнего обновления пейвола.                                                       |
| created_at                 | String           | Да           | Временная метка создания пейвола.                                                                    |
| state                      | String           | Да           | Текущий статус пейвола: `draft`, `live`, `inactive` или `archived`.                                  |
| is_deleted                 | Boolean          | Да           | Помечен ли пейвол как удалённый.                                                                     |
| web_purchase_url           | String           | Нет          | URL для [веб-покупок](web-paywall), если применимо.                                                  |
| products                   | Array of objects | Да           | Массив объектов [Product](web-api-objects#products-object) с информацией о продуктах.                |
| remote_configs             | Array of objects | Нет          | Массив объектов [RemoteConfig](web-api-objects#remoteconfig-object) с локалью и данными.             |
| main_screenshot            | Object           | Нет          | Объект основного скриншота с полями `image_id` и `url`.                                              |

## Ошибки \{#errors\}

### 400: Bad request \{#400-bad-request\}

Возвращается, если тело запроса некорректно или отсутствуют обязательные поля.

```json showLineNumbers
{
    "errors": [
        {
            "source": "remote_configs",
            "errors": [
                "At least one field must be provided"
            ]
        }
    ],
    "error_code": "validation_error",
    "status_code": 400
}
```

### 401: Unauthorized \{#401-unauthorized\}

<ProfileResponseUnauthorized /> 

### 404: Not found \{#404-not-found\}

Запрос завершился ошибкой, потому что указанный пейвол не найден. Проверьте `paywall_id` на наличие опечаток.

```json showLineNumbers
{
    "errors": [
        {
            "source": null,
            "errors": [
                "Paywall not found"
            ]
        }
    ],
    "error_code": "paywall_does_not_exist",
    "status_code": 404
}
```
