Рады приветствовать вас в Adapty! Наша цель — помочь вам быстро начать работу и добиться наилучших результатов. Это руководство поможет вам начать работу с Adapty, если ваше приложение доступно в Google Play Store.

Интеграция Adapty в мобильное приложение включает установку соединений между вашим приложением и Adapty на уровнях Google Play и SDK. Процесс может показаться объёмным, но встроенный онбординг в дашборде Adapty или инструкции ниже упростят его — обычно это занимает не более 1 часа.

## Чеклист первоначальной интеграции \{#checklist-for-the-initial-integration\}

- [ ] После создания аккаунта в Adapty и указания названия и категории мобильного приложения мы настраиваем его для вас в нашей платформе.
- [ ] [Включите Developer APIs](enabling-of-devepoler-api) в Google Cloud Console
- [ ] [Создайте сервисный аккаунт](create-service-account) в Google Cloud Console
- [ ] [Предоставьте права сервисному аккаунту](grant-permissions-to-service-account) в Google Play Console
- [ ] [Сгенерируйте файл ключа сервисного аккаунта](create-service-account-key-file) в Google Cloud Console
- [ ] [Настройте интеграцию с Google Play](google-play-store-connection-configuration) в дашборде Adapty
- [ ] [Включите Real-time developer notifications (RTDN)](enable-real-time-developer-notifications-rtdn) в Google Play Console
- [ ] Установите и настройте SDK Adapty (можно установить SDK для одного или нескольких фреймворков по мере необходимости)
  - [ ] [Установите SDK Adapty для Android](sdk-installation-android)
  - [ ] [Установите SDK Adapty для React Native](sdk-installation-reactnative)
  - [ ] [Установите SDK Adapty для Flutter](sdk-installation-flutter)
  - [ ] [Установите SDK Adapty для Kotlin Multiplatform](sdk-installation-kotlin-multiplatform)
  - [ ] [Установите SDK Adapty для Unity](sdk-installation-unity)
- [ ] Соберите приложение и запустите его. Достаточно запуска в виде снапшота или в среде песочницы.

:::note
Изменения вступают в силу не менее чем через 24 часа, но есть [лайфхак](https://stackoverflow.com/a/60691844). В [Google Play Console](https://play.google.com/apps/publish/) откройте любое приложение и в разделе **Monetize** перейдите в **Products** -> **Subscriptions**/**In-app products**. Измените описание любого продукта и сохраните изменения. После этого всё должно заработать — изменения в настройках встроенных покупок можно откатить обратно.
:::

После завершения первоначальной интеграции вы [можете начать использовать функции Adapty](product).

Обратите внимание: чтобы пейволы и продукты отображались в вашем мобильном приложении и работала аналитика, необходимо внести изменения в код приложения. В частности, нужно как минимум [отобразить пейволы](android-quickstart-paywalls) и, если вы используете пейволы, созданные не в Paywall Builder, [реализовать процесс покупки](android-making-purchases) в приложении.

:::danger
Перед релизом приложения пройдите чеклист релиза

Перед публикацией приложения обязательно ознакомьтесь с [чеклистом релиза](release-checklist). Он поможет убедиться, что все необходимые шаги выполнены, и содержит критерии для оценки успешности интеграции.
:::
