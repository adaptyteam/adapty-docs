Рады видеть вас в Adapty! Наша цель — помочь вам быстро разобраться и получить максимум от вашего приложения. Этот гайд поможет начать работу с Adapty, если ваше приложение доступно в App Store.

Интеграция Adapty в мобильное приложение предполагает установку связей между вашим приложением и Adapty на уровне App Store и SDK. Хотя на первый взгляд это может показаться сложным, следование онбордингу в дашборде Adapty или этим инструкциям позволит выполнить всё за 30 минут.

## Гайд по начальной интеграции \{#guide-for-the-initial-integration\}

- [ ] После создания аккаунта в Adapty и указания названия и категории приложения мы настраиваем его на стороне платформы Adapty.
- [ ] [Сгенерируйте ключ встроенных покупок](generate-in-app-purchase-key) в App Store Connect
- [ ] [Настройте интеграцию с App Store](app-store-connection-configuration) в дашборде Adapty и App Store Connect
- [ ] Если в вашем приложении есть пробные периоды или другие promotional offer, [настройте promotional offers App Store](app-store-connection-configuration#step-4-for-trials-and-special-offers--set-up-promotional-offers) в дашборде Adapty.
- [ ] [Включите серверные уведомления App Store](enable-app-store-server-notifications) в App Store Connect
- [ ] Установите Adapty SDK для используемых фреймворков:
  - [ ] [Установите Adapty SDK для нативного iOS](sdk-installation-ios)
  - [ ] [Установите Adapty SDK для React Native](sdk-installation-reactnative)
  - [ ] [Установите Adapty SDK для Flutter](sdk-installation-flutter)
  - [ ] [Установите Adapty SDK для Unity](sdk-installation-unity)
  - [ ] [Установите Adapty SDK для Kotlin Multiplatform](sdk-installation-kotlin-multiplatform)
- [ ] Соберите приложение и запустите его в режиме песочницы.

После завершения начальной интеграции вы [можете начать использовать функции Adapty](product).

Помните, что для отображения пейволов и продуктов в мобильном приложении и корректной работы аналитики необходимо внести изменения в код приложения. В частности, нужно как минимум [отобразить пейволы](ios-quickstart-paywalls) и, если вы используете пейволы, созданные не в Paywall Builder, [обработать процесс покупки](making-purchases) внутри приложения.

:::danger
Пройдите чеклист перед релизом приложения

Перед выпуском приложения обязательно ознакомьтесь с [чеклистом для релиза](release-checklist). Это поможет убедиться, что все необходимые шаги выполнены перед публикацией приложения с Adapty SDK.
:::
