На пейволе можно использовать любые изображения или видео, но иногда нужно показать персонализированный контент — например, видео с выбранным пользователем аватаром. Именно для этого и существуют пользовательские медиафайлы.

Пользовательский медиафайл — это изображение или видео, которое ваше приложение запрашивает по ID из кода. Он заменяет стандартный медиафайл, добавленный на пейвол в Paywall Builder.

:::info
Для использования этой функции обновите Adapty SDK до версии 3.7.0 или выше.
:::

## Где можно использовать пользовательские медиафайлы? \{#where-can-i-use-custom-media\}

Везде, где обычно используются стандартные изображения или видео:

- В качестве главного изображения
- В качестве главного видео
- В качестве обычной иконки
- В фоне карточки
- В фоне карусели

## Как использовать пользовательские медиафайлы? \{#how-to-use-custom-media\}
Чтобы настроить пользовательский медиафайл:

1. Установите флажок **Use custom media ID** под областью загрузки.

2. Введите ID медиафайла. Для главных изображений и главных видео ID предопределены.

3. Загрузите резервное изображение или видео в секции загрузки файла.

4. Подробнее см. в <InlineTooltip tooltip="гайдах о том, как получить пейволы Paywall Builder в вашем приложении">[iOS](get-pb-paywalls), [Android](android-get-pb-paywalls), [React Native](react-native-get-pb-paywalls), [Flutter](flutter-get-pb-paywalls) и [Unity](unity-get-pb-paywalls)</InlineTooltip> — затем вызовите медиафайл по его ID в коде.

Если версия Adapty SDK ниже 3.7.0 или пользовательский ID медиафайла не определён в коде, вместо него будет отображаться резервное изображение или видео.
