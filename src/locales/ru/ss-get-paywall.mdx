Возвращает конкретный пейвол вашего приложения.

## Метод и эндпоинт \{#method-and-endpoint\}

```http
GET https://api.adapty.io/api/v2/server-side-api/paywalls/{paywall_id}/
```

## Пример запроса \{#example-request\}

<Tabs groupId="api-lang" queryString> 

<TabItem value="curl" label="cURL" default> 

```bash showLineNumbers
curl --location 'https://api.adapty.io/api/v2/server-side-api/paywalls/{paywall_id}/' \
--header 'Content-Type: application/json' \
--header 'Authorization: Api-Key <YOUR_SECRET_API_KEY>'
```

</TabItem> 

<TabItem value="python" label="Python" default> 

```python showLineNumbers

url = "https://api.adapty.io/api/v2/server-side-api/paywalls/{paywall_id}/"

headers = {
    "Content-Type": "application/json",
    "Authorization": "Api-Key <YOUR_SECRET_API_KEY>"
}

response = requests.get(url, headers=headers)

print(response.text)
```

</TabItem> 

<TabItem value="js" label="JavaScript" default> 

```javascript showLineNumbers
const myHeaders = new Headers();
myHeaders.append("Authorization", "Api-Key <YOUR_SECRET_API_KEY>");
myHeaders.append("Content-Type", "application/json");

const requestOptions = {
  method: "GET",
  headers: myHeaders,
  redirect: "follow"
};

fetch("https://api.adapty.io/api/v2/server-side-api/paywalls/{paywall_id}/", requestOptions)
  .then((response) => response.text())
  .then((result) => console.log(result))
  .catch((error) => console.error(error));
```

</TabItem> 

</Tabs>

Плейсхолдеры:

- `<YOUR_SECRET_API_KEY>`: Ваш секретный API-ключ для авторизации.
- `{paywall_id}`: Уникальный идентификатор пейвола.

## Успешный ответ: 200: OK \{#successful-response-200-ok\}

Объект `paywall` в ответе содержит следующие поля:

| Название                  | Тип              | Обязательное       | Описание                                                                                 |
|---------------------------|------------------|--------------------|---------------------------------------------------------------------------------------------|
| title                     | String           | Да      | Название пейвола, заданное в дашборде Adapty.                               |
| paywall_id                | String           | Да      | Уникальный идентификатор пейвола.                                                       |
| use_paywall_builder       | Boolean          | Да      | Использует ли пейвол Paywall Builder.                                               |
| use_paywall_builder_legacy| Boolean          | Да      | Использует ли пейвол устаревшую версию Paywall Builder.                                        |
| updated_at                | String           | Да      | Временная метка последнего обновления пейвола.                                                |
| created_at                | String           | Да      | Временная метка создания пейвола.                                                     |
| state                     | String           | Да      | Текущее состояние пейвола: `draft`, `live`, `inactive` или `archived`.               |
| is_deleted                | Boolean          | Да      | Помечен ли пейвол как удалённый.                                                   |
| web_purchase_url          | String           | Нет       | URL для [веб-покупок](web-paywall), если применимо.                                        |
| products                  | Array of objects | Да      | Массив объектов [Product](web-api-objects#products-object) с информацией о продуктах. |
| remote_configs            | Array of objects | Нет       | Массив объектов [RemoteConfig](web-api-objects#remoteconfig-object) с локалью и данными.  |
| main_screenshot           | Object           | Нет       | Объект основного скриншота с полями `image_id` и `url`.                                           |

### 404: Not found \{#404-not-found\}

Запрос завершился ошибкой, так как указанный пейвол не найден. Проверьте `paywall_id` на наличие опечаток.

```json showLineNumbers
{
    "errors": [
        {
            "source": null,
            "errors": [
                "Paywall not found"
            ]
        }
    ],
    "error_code": "paywall_does_not_exist",
    "status_code": 404
}
```
