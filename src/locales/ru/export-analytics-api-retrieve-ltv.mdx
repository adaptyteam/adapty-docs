Возвращает данные LTV для оценки долгосрочного потенциала выручки от клиентов за период их взаимодействия с продуктом.

## Эндпоинт и метод \{#endpoint-and-method\}

```http
POST https://api-admin.adapty.io/api/v1/client-api/metrics/ltv/
```

## Параметры \{#parameters\}

| Название     | Тип                                      | Обязательный       | Описание                                                     |
| ------------ | ---------------------------------------- | ------------------ | ------------------------------------------------------------ |
| filters      | [MetricsFilters](#metricsfilters-object) | :heavy_plus_sign:  | Объект с параметрами фильтрации. Подробнее см. ниже.         |
| period_unit  | String                                   | :heavy_minus_sign: | <p>Задаёт временной интервал для агрегации аналитических данных — результаты будут сгруппированы по выбранным периодам: дням, неделям, месяцам и т. д. Возможные значения:</p><ul><li>day</li><li>week</li><li>month</li><li>quarter</li><li>year</li></ul> |
| period_type  | String                                   | :heavy_minus_sign: | <p>Возможные значения:</p><ul><li>renewals</li><li>days</li></ul> |
| segmentation | String                                   | :heavy_minus_sign: | <p>Возможные значения:</p><ul><li>day</li><li>week</li><li>month</li><li>year</li><li>country</li><li>product</li><li>paywall</li><li>paywalls_group</li><li>audience</li><li>placement</li><li>duration</li><li>store</li></ul> |
| format       | String                                   | :heavy_minus_sign: | <p>Формат экспортируемого файла. Доступные варианты:</p><ul><li>json</li><li>csv</li></ul> |
