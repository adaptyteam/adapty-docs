Push-уведомления в [iOS-приложении Adapty](https://apps.apple.com/us/app/adapty/id6739359219) помогают следить за происходящим с вашими приложениями в режиме реального времени. Неважно, отслеживаете ли вы новые триалы, продления подписок или установки — вы будете получать мгновенные обновления, не открывая дашборд.

Вы можете задать настройки уведомлений по умолчанию для всех приложений, а затем скорректировать их для отдельных приложений, если нужен более тонкий контроль. Это особенно удобно при запуске нового приложения, когда хочется пристально следить за первой активностью.

Общее количество уведомлений по всем вашим приложениям ограничено 1000 в день.

## Настройка Push-уведомлений \{#configure-push-notifications\}

Чтобы настроить push-уведомления в iOS-приложении Adapty:

1. Откройте приложение Adapty.
2. Перейдите в **Settings → Notifications**.
3. Нажмите **Enable notifications**, чтобы разрешить приложению отправлять уведомления.
4. Выберите, использовать ли настройки по умолчанию или настроить уведомления для каждого приложения отдельно. Обратите внимание: индивидуальные настройки приложения имеют приоритет над настройками по умолчанию.
5. Выберите, о каких событиях вы хотите получать уведомления.

## Поддерживаемые события \{#supported-events\}

Вы можете получать push-уведомления для следующих типов событий:

* Access level updated
* Auto-renew turned off
* Auto-renew turned on
* Billing issue detected
* Entered grace period
* Non-subscription purchase
* Non-subscription purchase refunded
* Subscription cancelled
* Subscription deferred
* Subscription initial purchase
* Subscription paused
* Subscription refunded
* Subscription renewed
* Trial canceled
* Trial converted
* Trial started
