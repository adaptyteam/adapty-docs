Экспорт аналитических данных в CSV даёт вам возможность глубже изучить метрики производительности приложения, настраивать отчёты и анализировать тренды со временем. С помощью Adapty API вы можете легко выгружать детальную аналитику в формат CSV, что упрощает отслеживание, обмен данными и уточнение выводов по мере необходимости.

## Начало работы с API для экспорта аналитики \{#getting-started-with-the-api-for-analytics-export\}

С помощью API экспорта аналитики вы можете, например:

1. **Анализировать MRR от маркетинговых кампаний**: измерить эффективность прошлогодних маркетинговых кампаний в конкретной стране, чтобы определить, какие из них принесли наибольший доход, с еженедельной детализацией. Используйте метод [Retrieve analytics data](api-export-analytics/operations/retrieveAnalyticsData).

2. **Отслеживать удержание когорт со временем**: следить за удержанием по когортам, чтобы выявлять точки оттока и сравнивать когорты между собой — это помогает обнаружить тренды и ключевые моменты, где стратегии вовлечения могут повысить удержание. С ограничением по конкретному стору, стране и продукту. Используйте метод [Retrieve cohort data](api-export-analytics/operations/retrieveCohortData).

3. **Оценивать конверсию по каналам**: анализировать коэффициенты конверсии по ключевым каналам привлечения, чтобы понять, какие из них наиболее эффективны для первых покупок. Это помогает направлять маркетинговый бюджет на наиболее результативные каналы. Используйте метод [Retrieve conversion data](api-export-analytics/operations/retrieveConversionData).

4. **Контролировать отток**: отслеживать скорость отписок, чтобы выявлять паттерны оттока или оценивать успех усилий по удержанию — с фокусом на конкретную страну и продукт. Используйте метод [Retrieve funnel data](api-export-analytics/operations/retrieveFunnelData).

5. **Оценивать LTV по сегментам пользователей**: определять пожизненную ценность различных сегментов пользователей, чтобы понять, какие группы приносят наибольший доход со временем. Сосредоточьтесь на высокоценных сегментах, например долгосрочных подписчиках, и используйте результаты для уточнения стратегий привлечения. Используйте метод [Retrieve LTV data](api-export-analytics/operations/retrieveLTVData).

6. **Анализировать удержание по странам**: изучать показатели удержания по регионам, чтобы выявлять рынки с высокой вовлечённостью и выстраивать стратегии локализации или региональные стратегии. Используйте метод [Retrieve retention data](api-export-analytics/operations/retrieveRetentionData).
