Добавляйте заголовки, параграфы и списки одним кликом, стилизуйте их под ваш бренд и используйте [динамические переменные](onboarding-variables) для персонализации контента для каждого пользователя.
## Настройте текстовые стили \{#set-up-text-styles\}

Панель **Styles** включает заранее настроенные текстовые стили: H1, H2, H3, Button Label, Body, Caption и Small Label. Нажмите на любой стиль, чтобы изменить гарнитуру шрифта, начертание, размер, выравнивание, оформление и другие свойства.

Когда вы [добавляете текстовый элемент](#add-text), можно выбрать один из настроенных здесь стилей.

:::important
Изменения стиля применяются ко всем текстовым элементам, которые его используют, на всех экранах.
:::

Чтобы создать новый стиль:

1. Нажмите <Inline id="Styles.webp" alt="Palette"/>, чтобы открыть панель **Styles**.
2. На вкладке **Text** нажмите **Create style**.
<ZoomImage id="new-text-style.webp" width="500px" alt="Новый стиль текста" />
3. Введите название и настройте типографику:
- **Font family**: выберите шрифт. Подробнее о работе с пользовательскими шрифтами — в [гайде](using-custom-fonts-in-paywall-builder).
- **Weight** и **size**: задайте начертание и размер шрифта.
- **Color**: задайте цвет текста.
- **Line height**: задайте межстрочный интервал или оставьте значение **Auto**.
- **Alignment**: задайте горизонтальное (left, center, right) и вертикальное (top, middle, bottom) выравнивание.
- **Decoration**: добавьте подчёркивание или зачёркивание.
- **Truncate**: ограничивает количество отображаемых строк. Текст сверх этого лимита обрезается. Полезно, когда длина контента варьируется из-за динамических переменных или локализации.
4. Нажмите **Create**.
:::note
SF Pro недоступен для кросс-платформенного использования. Рекомендуем Inter как близкую альтернативу.
:::

<ZoomImage id="text-style-details.webp" width="500px" alt="Параметры настройки стиля текста" />
## Добавление текста \{#add-text\}

Чтобы добавить текстовый элемент:
1. Нажмите **+** в верхнем левом углу. Выберите **Text**. Выберите один из [стилей текста](#set-up-text-styles), заданных заранее или настроенных вами.

<ZoomImage id="new-text.webp" width="500px" alt="New text" />
2. Нажмите на новый элемент и отредактируйте его содержимое в разделе **Content** панели **Design** справа.
3. При необходимости настройте типографику в панели **Design**. Или выделите текст в превью, чтобы открыть тултип для быстрой настройки стиля.
4. По желанию в панелях **Design** и **Interaction** можно применить любые другие настройки, доступные для компонентов флоу. Подробнее см. в разделе [Стилизация элементов](builder-styling).
:::tip
Если нужно использовать один и тот же текстовый элемент на нескольких экранах, скопируйте и вставьте его: выделите элемент, нажмите Ctrl+C (или ⌘+C на Mac), перейдите на другой экран и нажмите Ctrl+V (или ⌘+V на Mac).
:::

<ZoomImage id="text-settings.webp" width="500px" alt="Настройки текстового элемента" />
### Изменение стиля для части текста \{#change-styling-for-parts-of-text\}

Чтобы изменить стиль только для части текстового элемента:

1. Выделите нужный фрагмент текстового элемента в разделе **Content**.
2. В появившемся тултипе измените цвет текста, примените жирное начертание, подчёркивание, курсив или зачёркивание, либо добавьте URL.

Предварительный просмотр обновится сразу же.

## Добавление списков \{#add-lists\}

:::note
Элементы списка — это контейнеры, состоящие из отдельных компонентов-элементов. Для простых маркированных или нумерованных списков в тексте используйте текстовый элемент и примените нужное форматирование через панель **Design**.
:::

1. Нажмите **+** в верхнем левом углу. Выберите **List** и один из шаблонов списка.

2. Откройте панель **Design** справа, чтобы редактировать элементы списка или загрузить изображение в качестве маркера.
