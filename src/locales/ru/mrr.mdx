График Monthly Recurring Revenue (MRR) отображает нормализованный доход от активных платных подписок за месяц. Он помогает оценить темп роста и масштаб бизнеса вне зависимости от колебаний, связанных с разными сроками подписок.

### Расчёт \{#calculation\}

:::warning
Калькулятор ниже **не учитывает** [комиссию стора и налоги](how-adapty-analytics-works#tax-and-commission-logic). Сравнивайте результат с расчётами **валовой выручки**.
:::

MRR — это предсказуемый ежемесячный доход вашего подписочного бизнеса. Для каждого типа подписки умножьте её цену на количество подписчиков и разделите на количество месяцев в расчётном периоде, затем сложите результаты. Введите значения своих подписок в интерактивный калькулятор, чтобы рассчитать валовый MRR:

<SimpleCalculator client:load
    heading="MRR"
    formuLatex="\sum_{subscriptions}^{}\frac{P_s\times N_s}{D_m}"
    variables={[
        {
            nameInTheFormula: "P_s",
            variableName: "subscriptionPrice",
            variableDescription: "Цена",
            variableValue: 10
        },
        {
            nameInTheFormula: "N_s",
            variableName: "activeSubs",
            variableDescription: "Подписчики",
            variableValue: 1,
            isInteger: true
        },
        {
            nameInTheFormula: "D_m",
            variableName: "duration",
            variableDescription: "Период подписки",
            variableValue: 1,
            options: [
                { label: "Еженедельно", value: 0.23 },
                { label: "Ежемесячно", value: 1 },
                { label: "2 месяца", value: 2 },
                { label: "3 месяца", value: 3 },
                { label: "6 месяцев", value: 6 },
                { label: "Ежегодно", value: 12 }
            ]
        }
    ]}
    formulaCalculation="(subscriptionPrice * activeSubs) / duration"
    isSum={true}
    defaultRows={[
        { subscriptionPrice: 240, activeSubs: 2, duration: 12},
        { subscriptionPrice: 30, activeSubs: 10, duration: 1},
        { subscriptionPrice: 10, activeSubs: 20, duration: 0.23},
    ]}
/>

Где:
Ps — цена подписки
Ns — количество активных платных подписок данного типа. Adapty считает активной любую платную подписку, срок действия которой ещё не истёк.
Dsm — длительность подписки в месяцах (0.23 для еженедельных подписок)

Важно учитывать, что Adapty не включает в расчёт MRR неповторяющиеся подписки, расходуемые покупки и разовые покупки — они не являются предсказуемым и регулярным доходом.

По сути, MRR показывает доход от всех активных подписок, нормализованный к одному месяцу. Например, для годовой подписки вместо учёта полной суммы с момента оформления доход делится на 12 равных частей, равномерно распределяемых на 12 месяцев.

Подписки, по которым был оформлен возврат, в MRR не учитываются. При возврате подписка исключается из расчёта MRR за все периоды, в которые она была активна, — таким образом MRR отражает только регулярный доход от действующих подписок.

Например, если есть 2 активные годовые подписки по $240 и 10 ежемесячных подписок по $30:
MRR = (2 \* $240 / 12) + (10 \* $30 / 1) + (20 \* $10 / 0.23) = $1209.5

### Применение графика MRR \{#mrr-chart-usage\}

MRR — ключевая метрика для бизнеса, основанного на регулярных доходах от подписок. Она отражает не только размер базы подписчиков, но и нормализует разные сроки подписок к единому знаменателю — ежемесячному регулярному доходу. Благодаря этому MRR даёт реальную картину темпа роста бизнеса, упрощая точное отслеживание траектории развития.

Чтобы использовать MRR максимально эффективно, сегментируйте когорты подписчиков по месяцу первой покупки и переключите разрешение на «по месяцам». Это позволит построить график с накоплением, который покажет, как ежемесячные когорты подписчиков менялись со временем. Такой подход помогает выявлять тенденции и закономерности в базе подписчиков, упрощая корректировку бизнес-стратегии и оптимизацию продуктов и маркетинговых усилий.
