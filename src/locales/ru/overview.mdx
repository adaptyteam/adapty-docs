[Overview](https://app.adapty.io/overview) — раздел дашборда Adapty, где все ключевые метрики собраны в одном месте. Вы можете настроить, какие графики отображать, и просматривать данные по всем приложениям сразу (в отличие от раздела [Charts](charts), предназначенного для детального анализа отдельного приложения).

Вы можете настроить набор отображаемых графиков и их порядок. Для этого нажмите **Edit** в правом верхнем углу, а затем удалите ненужные графики, добавьте новые или перетащите существующие в нужном порядке. Настроить содержимое Overview также можно через меню **Add**:

## Элементы управления \{#controls\}

Инструменты для детального анализа данных в Overview очень похожи на те, что есть в разделе [Charts](charts), — большинство из них описано в статье [Элементы управления аналитикой](controls-filters-grouping-compare-proceeds).

Есть одно важное отличие: здесь можно группировать и фильтровать данные по стране, стору и, что особенно полезно, по приложению — ведь по умолчанию Overview показывает данные по всем приложениям сразу. Это помогает понять, какой вклад каждое приложение вносит в ваши бизнес-метрики:

:::note
**Часовой пояс и настройки установок**

Обратите внимание: эти настройки применяются ко всем вашим приложениям и переопределяют то, что задано в [App settings](general).

- **Installs**: по умолчанию установки считаются по `device_id` — новая установка или переустановка на устройстве засчитывается как отдельная установка. Изменить это можно, нажав **Edit**. Подробное описание других вариантов см. в разделе [Определение установок для аналитики](general#4-installs-definition-for-analytics).
- **Timezone**: по умолчанию часовой пояс страницы **Overview** наследуется от одного из ваших приложений. Если у ваших приложений разные часовые пояса для отчётов, настройте часовой пояс Overview, нажав **Edit** и выбрав нужный вариант из выпадающего списка.

:::
