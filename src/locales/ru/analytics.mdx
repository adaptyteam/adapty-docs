Аналитика Adapty — это мощный набор инструментов для визуализации и анализа данных. Созданная для команд разработки приложений, она фокусируется на финансовых показателях вашего приложения и поведении пользователей.

В дополнение к стандартным метрикам, доступным в аналитике стора (например, выручка или установки), Adapty рассчитывает составные метрики: пожизненную ценность пользователя, когортный анализ и другое. Отдельные разделы анализируют поведение пользователей, приводящее к потере дохода: графики удержания, данные по решению проблем с оплатой. Изучайте графики, чтобы лучше понять поведение пользователей, и принимайте решения на основе данных для оптимизации приложения.

Прочитайте гайд [Как работает аналитика Adapty](how-adapty-analytics-works), чтобы узнать подробнее об источниках и обработке данных. Для быстрого ознакомления со всеми доступными метриками и их различиями см. статью [Таблица сравнения метрик](metric-comparison-table).

:::tip
Подключите [Adapty User Acquisition](adapty-user-acquisition), чтобы расширить аналитические возможности Adapty. UA использует маркетинговые данные для анализа связи между рекламными кампаниями и поведением пользователей. Его графики отображают полную картину экономики вашего приложения с расчётом ROI и точными данными об атрибуции.
:::

## Фильтры и элементы группировки \{#filters-and-grouping-controls\}

:::link
Основная статья: [Элементы управления](controls-filters-grouping-compare-proceeds)
:::

Используйте фильтры и элементы группировки для просмотра данных по конкретным сегментам пользователей. Например, можно отфильтровать график удержания по стору и сравнить поведение пользователей на Android и iOS.

<ZoomImage id="analytics/controls.webp" width="700px" alt="Элементы управления аналитикой" />

В любом представлении аналитики Adapty вы можете:

    * Изменять диапазон дат для визуализации данных.
    * Фильтровать данные по стору, атрибуции или другим параметрам.
    * Группировать данные внутри графика по стране, [сегменту пользователей](segments) или другим критериям.

[Графики](charts) также позволяют сравнивать одну и ту же метрику за два разных периода времени.
