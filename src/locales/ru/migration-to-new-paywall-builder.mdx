Рады представить наш [**новый Paywall Builder**](adapty-paywall-builder)! Этот продвинутый no-code инструмент создан для того, чтобы создание пейволов стало максимально простым и мощным — вы сможете делать красивые, привлекательные пейволы без каких-либо технических или дизайнерских навыков.

## Ключевые возможности нового Paywall Builder \{#key-features-of-the-new-paywall-builder\}

- **Расширенный выбор шаблонов**: выбирайте из большого набора профессионально разработанных шаблонов для быстрого старта. Они предлагают разные стили и макеты под любые задачи и предпочтения.
- **Больше гибкости**: используйте слои дизайна и новые элементы — карусели, карточки, список продуктов и футер. Это даёт творческую свободу для реализации любого пейвола, который вы задумали.
- **Улучшенные существующие элементы**: существующие элементы значительно доработаны — теперь у вас больше опций и возможностей для воплощения идей.

## Параллельные версии Paywall Builder \{#parallel-paywall-builder-versions\}

Adapty одновременно поддерживает две версии Paywall Builder:

- **Новый Paywall Builder**: находится на вкладке **Builder & Generator** пейвола в дашборде Adapty — это самая актуальная и функциональная версия. Пейволы, созданные здесь, требуют iOS, Android и React Native SDK версии 3.0 или выше, а также Flutter и Unity SDK версии 3.3.0 или выше.
- **Legacy Paywall Builder**: находится на вкладке **Legacy Builder** — устаревшая версия, которую стоит использовать только для поддержки старых версий приложений с SDK ниже v3.x.x. Для новых пейволов её использовать не рекомендуется, так как скоро она будет упразднена.

## Миграция пейволов в новый Builder \{#migrating-paywalls-to-the-new-builder\}

При миграции пейвола из legacy builder в новый builder создаётся новая версия пейвола на вкладке **Builder & Generator**. Её можно редактировать в новом Paywall Builder — она будет отображаться в приложениях с Adapty SDK v3.0 и выше. Legacy-версия остаётся на вкладке **Legacy Builder** и поддерживает приложения с SDK 2.x и ниже.

Вы будете поддерживать пейволы в обоих форматах независимо друг от друга — изменения в одном формате не влияют на другой.

## Шаги миграции пейвола \{#steps-to-migrate-a-paywall\}

Чтобы мигрировать пейвол в новый Paywall Builder:

1. Откройте пейвол, который хотите мигрировать.
2. Перейдите на вкладку **Builder & Generator**.
3. Нажмите **Migrate paywall**.

5. Нажмите **Save**.
6. Если есть проблемы, они будут выделены красным и вы сразу их увидите. Исправьте их и сохраните пейвол снова.
