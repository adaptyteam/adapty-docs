Возвращает информацию о плейсментах пейволов или онбордингов: пейволы, сегменты, аудитории и A/B-тесты. Это позволяет убедиться, что всё настроено правильно, не открывая каждый плейсмент по отдельности.

:::note
Как альтернатива, вы можете [экспортировать те же данные из дашборда](export-placements).
:::

## Эндпоинт и метод \{#endpoint-and-method\}

```
POST https://api-admin.adapty.io/api/v1/client-api/exports/placements/
```

## Пример запроса \{#example-request\}

<Tabs groupId="api-lang" queryString>
<TabItem value="curl" label="cURL" default> 

```bash showLineNumbers
curl -X POST --location "https://api-admin.adapty.io/api/v1/client-api/exports/placements/" \
    -H "Authorization: Api-Key <YOUR_SECRET_API_KEY>" \
    -H "Accept: application/json" \
    -H "Content-Type: application/json" \
    -d '{
          "filters": {
            "placement_type": "paywall"
          }
        }'
```

</TabItem>

</TabItem>

<TabItem value="js" label="JavaScript" default> 

```javascript showLineNumbers
const myHeaders = new Headers();
myHeaders.append("Authorization", "Api-Key <YOUR_SECRET_API_KEY>");
myHeaders.append("Accept", "application/json");

const raw = JSON.stringify({
  "filters": {
      "placement_type": "paywall"
    }
});

const requestOptions = {
  method: "POST",
  headers: myHeaders,
  body: raw,
  redirect: "follow"
};

fetch("https://api-admin.adapty.io/api/v1/client-api/exports/placements/", requestOptions)
  .then((response) => response.text())
  .then((result) => console.log(result))
  .catch((error) => console.error(error));
```

</TabItem> 

</Tabs>

Плейсхолдеры:

- `<YOUR_SECRET_API_KEY>`: Ваш секретный API-ключ для авторизации.

:::tip
В зависимости от того, как вы планируете использовать ответ, вы можете установить `Accept` в `text/csv`, чтобы получить его в формате CSV, или `application/json`, чтобы получить JSON-файл.
:::

## Параметры \{#parameters\}

| Параметр | Тип    | Обязателен в запросе | Может быть null в запросе | Описание                                                                                                                                                                                                                              |
|----------|--------|----------------------|---------------------------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| filters  | Object | :heavy_plus_sign:    | :heavy_minus_sign:        | Фильтр, определяющий, какие плейсменты будут включены в ответ. <br/> На данный момент можно фильтровать только по `placement_type`: `paywall` или `onboarding`. Таким образом, значение параметра `filters` — объект с одной парой ключ-значение. |

```json showLineNumbers
{
  "filters": {
    "placement_type": "paywall"
  }
}
```

## Успешный ответ: 200: OK \{#successful-response-200-ok\}

| Параметр                     | Тип    | Описание                                                                                  |
|------------------------------|--------|-------------------------------------------------------------------------------------------|
| developer_id                 | String | ID плейсмента.                                                                            |
| placement_name               | String | Название плейсмента.                                                                      |
| audience_name                | String | Название аудитории.                                                                       |
| segment_name                 | String | ID аудитории.                                                                             |
| cross_placement_ab_test_name | String | Название кросс-плейсментного A/B-теста. Может быть `null` в JSON или пустым в CSV.       |
| ab_test_name                 | String | Название A/B-теста. Может быть `null` в JSON или пустым в CSV.                           |
| paywall_name                 | String | Название пейвола (возвращается только если `placement_type` равен `paywall`).             |
| onboarding_name              | String | Название онбординга (возвращается только если `placement_type` равен `onboarding`).       |
