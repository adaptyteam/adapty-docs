title: "Миграция Adapty Capacitor SDK на v. 3.16"
description: "Перейдите на Adapty Capacitor SDK v3.16 для повышения производительности и новых функций монетизации."
metadataTitle: "Миграция на Adapty Capacitor SDK v3.16 | Документация Adapty"
---

Начиная с Adapty SDK v.3.16.0 требуется Capacitor 8. Если вам нужен Capacitor 7, используйте Adapty SDK v.3.15.

Чтобы перейти на Capacitor SDK v.3.16, убедитесь, что ваш проект использует Capacitor 8. Если вы всё ещё используете Capacitor 7, у вас есть два варианта:

1. **Обновиться до Capacitor 8**: следуйте [официальному гайду по миграции Capacitor](https://capacitorjs.com/docs/updating/8-0), чтобы обновить проект, а затем установите Adapty SDK v.3.16.
2. **Остаться на Adapty SDK v.3.15**: если обновление до Capacitor 8 невозможно, продолжайте использовать Adapty SDK v.3.15, который поддерживает Capacitor 7.
