## Авторизация \{#authorization\}

Запросы к API должны быть аутентифицированы с помощью вашего публичного API-ключа, передаваемого в заголовке **Authorization** со значением `Api-Key {your_public_api_key}`, например `Api-Key public_live_...`. Найти этот ключ можно в [дашборде Adapty -> **App Settings** -> вкладка **General** -> раздел **API keys**](https://app.adapty.io/settings/general).

## Формат запросов \{#request-format\}

- **Заголовок Content-Type**:  
  Установите заголовок **Content-Type** в значение `application/json`, чтобы API корректно обработал ваш запрос.
- **Тело запроса**:  
  API ожидает, что тело запроса будет передано в формате JSON.
