Apple Ads Manager имеет двустороннюю интеграцию с Apple Ads: вы получаете данные о результативности практически в реальном времени и можете создавать и редактировать ключевые слова прямо в дашборде Adapty — это гораздо удобнее, чем в нативном интерфейсе.

Если вы создаёте ключевое слово в нативном дашборде Apple Ads, оно автоматически появится в Apple Ads Manager в течение 24 часов.

Помимо [просмотра подробной аналитики](adapty-ads-manager-analytics), вы можете управлять всеми настройками ключевых слов:
- Добавлять ключевые слова в группы объявлений
- Добавлять минус-слова
- Добавлять ключевые слова в формате SKAG (Single Keyword Ad Group)
- Редактировать ключевые слова прямо в таблице
- Выполнять массовые действия над несколькими ключевыми словами
- Запускать и приостанавливать ключевые слова

## Что такое ключевые слова \{#what-are-keywords\}

Ключевые слова — это поисковые запросы, по которым ваши объявления показываются в результатах поиска App Store. Они организованы в [группы объявлений](ads-manager-create-ad-group), которые входят в [кампании](ads-manager-create-campaign). Такая иерархическая структура позволяет эффективно организовывать рекламную стратегию и управлять ею.

:::important
Ключевые слова применимы только для кампаний с типом размещения **Search results**. Для кампаний с другими типами размещения (Search tab или Product pages) ключевые слова не используются.
:::

### Стандартные ключевые слова \{#standard-keywords\}

Стандартные ключевые слова — это основные запросы, по которым вы делаете ставки для показа объявлений. Когда пользователи ищут эти запросы в App Store, ваше объявление может появиться в результатах поиска.

### Минус-слова \{#negative-keywords\}

Минус-слова не позволяют вашему объявлению показываться по нерелевантным для вашего приложения запросам. Добавляя минус-слова, вы сокращаете расходы на нецелевые показы.

Минус-слова можно добавлять на уровне группы объявлений или в виде кросс-групповых минус-слов, применяющихся сразу к нескольким кампаниям.

### Ключевые слова в формате SKAG (Single Keyword Ad Group) \{#keywords-as-skag-single-keyword-ad-group\}

SKAG (Single Keyword Ad Group) — это стратегия, при которой для каждого ключевого слова создаётся отдельная группа объявлений. Такой подход позволяет:
- Точно управлять ставками для высокоценных ключевых слов
- Лучше анализировать результативность на уровне отдельного ключевого слова

SKAG особенно полезен для выявления наиболее эффективных ключевых слов и максимального раскрытия их потенциала через отдельные группы объявлений.

## Добавление ключевых слов \{#add-keywords\}

Чтобы добавить ключевые слова в группу объявлений:

1. Перейдите в **Ads Manager** через боковое меню. На любой вкладке нажмите **+** над таблицей, затем выберите **Add keywords** из выпадающего списка.

<ZoomImage id="add-keywords-action.webp" width="700px" /> 

2. В модальном окне выберите кампании и группы объявлений, в которые хотите добавить ключевые слова. После выбора групп объявлений в одной кампании можно выбрать другую кампанию и добавить ещё группы.

<ZoomImage id="select-ad-groups-keywords.webp" width="700px" />

3. Нажмите **Select**, чтобы продолжить.
4. В диалоге **Add keywords** введите ключевые слова в поле **Keywords list**. Если у вас есть файл с ключевыми словами в формате CSV, можно вставить его содержимое — Apple Ads Manager загрузит все ключевые слова сразу.
5. Для каждого ключевого слова в таблице настройте:
   - **Match type**: выберите **Exact** или **Broad**
   - **CPT bid**: укажите максимальную ставку за клик для этого ключевого слова или оставьте поле пустым, чтобы использовать стандартную максимальную ставку CPT группы объявлений
6. Проверьте ключевые слова и нажмите **Add X keywords** (где X — количество добавляемых ключевых слов).

:::important
После сохранения ключевого слова его тип соответствия изменить нельзя. Если нужно изменить тип соответствия, удалите ключевое слово и добавьте его заново с нужным типом.
:::

## Добавление минус-слов \{#add-negative-keywords\}

Чтобы добавить минус-слова:

1. Перейдите в **Ads Manager** через боковое меню. На любой вкладке нажмите **+** над таблицей, затем выберите **Add negative keywords** из выпадающего списка.
   <ZoomImage id="add-negative-keywords.webp" width="700px" />
2. В модальном окне **Add negative keywords to** выберите уровень, на котором хотите добавить минус-слова:
   - **Selected campaigns**: добавить минус-слова на уровне кампании.
   - **Selected ad groups**: добавить минус-слова на уровне группы объявлений.
   - **All ad groups in selected campaigns**: добавить минус-слова на уровне групп объявлений для всех групп в выбранных кампаниях.
   :::note
   Учтите следующее:
   - Минус-слова на уровне группы объявлений имеют приоритет над минус-словами на уровне кампании.
   - Если вы добавляете минус-слова ко всем группам объявлений в выбранных кампаниях, при создании новых групп в этих кампаниях их нужно будет добавлять вручную.
   :::

  <ZoomImage id="select-campaigns.gif" width="700px" /> 

3. Введите минус-слова в поле **Keywords list**. Если у вас есть файл с ключевыми словами в формате CSV, можно вставить его содержимое — Apple Ads Manager загрузит все ключевые слова сразу.
4. Для каждого ключевого слова в таблице выберите **Match type**:
   - **Exact**: исключает только точное ключевое слово или очень близкие его варианты.
   - **Broad**: исключает ключевое слово и связанные поисковые запросы.
      Или отметьте нужные ключевые слова флажками и измените тип соответствия для них всех сразу.
5. Проверьте минус-слова и нажмите **Add X keywords** (где X — количество добавляемых ключевых слов).

:::note
Кросс-групповые минус-слова особенно полезны, когда нужно исключить определённые запросы сразу из нескольких кампаний — это экономит время и обеспечивает единообразие рекламной стратегии.
:::

## Добавление ключевых слов в формате SKAG \{#add-keywords-as-skag\}

Чтобы добавить ключевые слова в формате SKAG (Single Keyword Ad Group):

1. Перейдите в **Ads Manager** через боковое меню. На любой вкладке нажмите **+** над таблицей, затем выберите **Add keywords as SKAG** из выпадающего списка.
   <ZoomImage id="add-skag-keywords.webp" width="700px" />
2. Выберите кампании, в которых хотите создать группы объявлений SKAG. Можно выбрать несколько кампаний.
3. По умолчанию новые группы объявлений создаются с настройками по умолчанию, таргетированными на всех пользователей. Если хотите изменить это, выберите **Copy settings from ad group** и укажите существующую группу объявлений, настройки которой нужно скопировать.
   <ZoomImage id="copy-settings-from-ad-group.webp" width="700px" />
4. Настройте параметры новых групп объявлений:
   - **Ad group name prefix**: необязательный префикс для имени каждой группы объявлений (например, «SKAG_» создаст «SKAG_keyword1», «SKAG_keyword2» и т. д.). Нажмите **Tag**, чтобы динамически добавить ключевое слово, название кампании и страну в имена групп.
   - **CPT bid** и **CPA cap**: задайте ставку для всех ключевых слов сразу или выберите **Set CPT bid and CPA cap for each word manually** и задайте их для каждого ключевого слова отдельно.
5. Введите ключевые слова в поле **Keywords list**. Если у вас есть файл с ключевыми словами в формате CSV, можно вставить его содержимое — Apple Ads Manager загрузит все ключевые слова сразу.
6. Для каждого ключевого слова в таблице выберите **Match type**:
   - **Exact**: соответствует только точному ключевому слову или очень близким его вариантам
   - **Broad**: соответствует ключевому слову и связанным поисковым запросам
     Или отметьте нужные ключевые слова флажками и измените тип соответствия для них всех сразу.
7. Выберите **Check for duplicates in target campaign**, чтобы убедиться в отсутствии одинаковых ключевых слов в целевых кампаниях.
8. Нажмите **Create**, чтобы создать группы объявлений SKAG.

Каждое ключевое слово будет помещено в собственную группу объявлений в каждой выбранной кампании, что позволяет управлять ими и оптимизировать их независимо.

<ZoomImage id="new-skag.webp" width="700px" />

## Редактирование ключевых слов \{#edit-keywords\}

Чтобы отредактировать существующие ключевые слова:

1. Перейдите в **Ads Manager > Keywords** или **Ads Manager > Negative keywords** и найдите нужное ключевое слово в таблице, либо перейдите на страницу кампании, затем — группы объявлений, и найдите ключевое слово там.
2. Отредактируйте значения прямо в таблице:
   - **CPT bid**: нажмите на значение ставки и введите новую максимальную ставку за клик
   - **Status**: используйте переключатель, чтобы приостановить или активировать ключевое слово

:::note
Правки ключевых слов, сделанные напрямую в Apple Ads, автоматически синхронизируются с Apple Ads Manager, но могут отображаться с задержкой.
:::

## Массовые действия \{#bulk-actions\}

Вы можете выполнять массовые действия над несколькими ключевыми словами одновременно, чтобы сэкономить время и управлять ключевыми словами эффективнее.

Чтобы выполнить массовые действия:

1. Перейдите на вкладку **Ads Manager > Keywords** или **Ads Manager > Negative keywords**.
2. Выберите несколько ключевых слов, отметив флажки рядом с нужными.
3. Нажмите выпадающий список **Actions** и выберите одно из следующих действий:
   - **Add as keywords**: добавить выбранные ключевые слова как стандартные
   - **Add as negative keywords**: добавить выбранные ключевые слова как минус-слова
   - **Add as SKAG**: создать Single Keyword Ad Groups для выбранных ключевых слов
   - **Activate**: активировать выбранные ключевые слова
   - **Pause**: приостановить выбранные ключевые слова
   - **Edit CPT bids**: изменить ставки CPT для выбранных ключевых слов. Доступны разные способы:
     - **Set to**: задать конкретное значение ставки для нескольких ключевых слов.
     - **Increase by/decrease by**: увеличить или уменьшить ставки на заданную сумму в USD или на определённый процент от ставки. При желании можно задать верхний предел ставки, чтобы случайно не потратить слишком много.
     - **Set to average CPT**: использовать метрику CPT (стоимость за клик) для выравнивания ставки. Задайте коэффициент-множитель. Например, установите множитель 0,9, если результативность ниже ожиданий, или 1,1, если она превышает их.
     - **Set to average CPA**: использовать метрику CPA (стоимость за привлечение) для выравнивания ставки. Задайте коэффициент-множитель.

:::tip
Массовые действия особенно полезны для:
- Конвертации ключевых слов между разными типами (стандартные, минус-слова, SKAG)
- Быстрого добавления ключевых слов с другими типами соответствия для нескольких ключевых слов сразу
- Фильтрации наиболее эффективных ключевых слов и корректировки их ставок
- Выявления низкоэффективных ключевых слов и их приостановки
:::

## Экспорт ключевых слов \{#export-keywords\}

Чтобы экспортировать таблицу ключевых слов в формате CSV, нажмите значок загрузки над таблицей и выберите **Export current page** или **Export all pages**.

**Export all pages** скачивает все ключевые слова со всех страниц в один файл. Прогресс загрузки отображается в модальном окне; загрузку можно отменить в любой момент. Доступны два дополнительных фильтра:

- **Enabled only**: включать только активные ключевые слова.
- **With spend ≥**: включать только ключевые слова с расходами выше указанного порога.
- **Group by country**: разбивать каждую строку ключевого слова по странам.

Таблица экспортируется в том виде, в котором она отображается на вашем дашборде, с выбранными вами столбцами.

<ZoomImage id="export-keywords.webp" width="700px" alt="Export keywords" />

## Графики на уровне ключевых слов \{#explore-keyword-level-charts\}

Вы можете открыть график для любого ключевого слова прямо из таблицы **Ads Manager > Keywords**. Это позволяет детально анализировать результативность каждого отдельного ключевого слова по дням.

Чтобы отобразить график, нажмите на значок графика рядом с ключевым словом в таблице.

<ZoomImage id="keyword-chart.webp" width="500px" />

По умолчанию график отображает метрику **Spend** для выбранного ключевого слова.

Вы можете отображать несколько метрик одновременно, чтобы отслеживать корреляции и изменения во времени. Нажмите **+**, чтобы добавить новую метрику.

Нажмите **Reset**, чтобы начать заново, или просто снимите флажки с метрик, чтобы скрыть их.

<ZoomImage id="keyword-chart.gif" width="700px" />

## История ставок \{#bid-history\}

Чтобы просмотреть историю ставок для ключевого слова, нажмите кнопку **Bid History** рядом с ним в таблице. Откроется панель с двумя вкладками.

- Вкладка **Metrics** отображает график с метриками за период. Добавлять и убирать метрики можно так же, как в графиках на уровне ключевых слов: нажмите **+**, чтобы добавить, или снимите флажки, чтобы скрыть. В каждой точке изменения ставки появляется маркер — наведите на него курсор, чтобы увидеть точное значение ставки на эту дату. Это помогает сопоставить изменения ставок с изменениями результативности: если метрика упала или резко выросла после изменения, маркер укажет точный момент.
- Вкладка **Bid History** показывает таблицу всех изменений ставок в обратном хронологическом порядке с предыдущим и новым значением ставки, датой каждого изменения и тем, что его вызвало.
