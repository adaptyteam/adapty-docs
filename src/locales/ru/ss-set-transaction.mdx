Создаёт новую транзакцию для пользователя вашего приложения в Adapty и предоставляет уровень доступа. Транзакция, созданная этим методом, появится в вашей [аналитике](https://app.adapty.io/analytics) и [**Event Feed**](https://app.adapty.io/event-feed), а также будет отправлена во все интеграции.

Этот метод предпочтительнее метода [Grant access level](api-adapty/operations/grantAccessLevel).

:::important
Если продукт не [создан в Adapty](create-product), транзакция всё равно будет записана в базу данных Adapty — она отобразится в аналитике и попадёт в события интеграций.

Однако пользователь получит только стандартный уровень доступа `premium`. Поэтому, если вы хотите гибко управлять доступом и назначать разные уровни, необходимо [создать продукты](create-product) в Adapty.
:::

## Метод и эндпоинт \{#method-and-endpoint\}

```http
POST https://api.adapty.io/api/v2/server-side-api/purchase/set/transaction/
```

Формат запроса зависит от того, является ли покупка **подпиской** или **разовой покупкой**.

```javascript showLineNumbers
const myHeaders = new Headers();
myHeaders.append("adapty-customer-user-id", "<YOUR_CUSTOMER_USER_ID>");
myHeaders.append("Content-Type", "application/json");
myHeaders.append("Authorization", "Api-Key <YOUR_SECRET_API_KEY>");

const raw = JSON.stringify({
  "purchase_type": "subscription",
  "store": "app_store",
  "environment": "Production",
  "store_product_id": "1year.premium",
  "store_transaction_id": "30002109551456",
  "store_original_transaction_id": "30002109551456",
  "is_family_shared": false,
  "price": {
    "country": "US",
    "currency": "USD",
    "value": 9.99
  },
  "purchased_at": "2022-10-12T09:42:50.000000+0000",
  "variation_id": "81109d24-ea95-4806-9ec7-b482bbd1a33d",
  "originally_purchased_at": "2022-10-12T09:42:50.000000+0000",
  "renew_status": true,
  "expires_at": "2026-10-12T09:42:50.000000+0000"
});

const requestOptions = {
  method: "POST",
  headers: myHeaders,
  body: raw,
  redirect: "follow"
};

fetch("https://api.adapty.io/api/v2/server-side-api/purchase/set/transaction/", requestOptions)
  .then((response) => response.text())
  .then((result) => console.log(result))
  .catch((error) => console.error(error));
```
