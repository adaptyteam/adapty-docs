Лента событий позволяет визуально отслеживать [события](events), генерируемые Adapty, и проверять статус их экспорта в сторонние интеграции, включая вебхук.

:::warning

Обратите внимание, что транзакции, созданные через [серверный API (версия 1)](server-side-api-specs-legacy#requests), не отображаются в **Event Feed**. Чтобы они попадали в ленту, используйте [серверный API (версия 2)](api-adapty/operations/setTransaction).

:::

:::note
Статус отправки для AppsFlyer, Facebook Ads и Branch может быть неточным, поскольку эти сервисы не всегда возвращают ошибки при их возникновении.
:::

Чтобы открыть профиль пользователя, инициировавшего транзакцию, нажмите кнопку **View Profile** в деталях события.
