Adapty SDK v3.0 добавляет поддержку нового [Adapty Paywall Builder](adapty-paywall-builder) — обновлённой версии no-code инструмента для создания пейволов. Благодаря максимальной гибкости и широким возможностям дизайна ваши пейволы станут ещё эффективнее и прибыльнее.
## Обновление до версии 3.0.1 \{#upgrade-to-version-301\}

1. Обновитесь до версии 3.0.1 обычным способом.
2. Замените файлы резервного пейвола:
   1. [Скачайте последнюю версию](fallback-paywalls) из дашборда Adapty.
   2. Сохраните их на устройстве пользователя и передайте в метод `.setFallbackPaywalls`, как описано [здесь](react-native-use-fallback-paywalls).
