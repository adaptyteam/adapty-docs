| Название | Обязательный | Описание |
| ---------------- | -------- | ------------------------------------------------------------ |
| store            | true     | Стор приложения |
| locale           | False    | Идентификатор локали пейвола. Ожидается в виде языкового кода, состоящего из одного или нескольких подтегов, разделённых символом «-». Первый подтег обозначает язык, второй — регион (поддержка регионов будет добавлена позже). Пример: `en` означает английский язык, `en-US` — американский английский. Если параметр не указан, пейвол будет создан в локали по умолчанию. |
| placement_id     | true     | Идентификатор [плейсмента](placements). Это значение вы указали при создании плейсмента в дашборде Adapty. |
| customer_user_id | true*    | Идентификатор пользователя в вашей системе. Требуется указать либо `customer_user_id`, либо `profile_id`. |
| profile_id       | true*    | Идентификатор пользователя в Adapty. Требуется указать либо `customer_user_id`, либо `profile_id`. |

**Пример**

```
{
  "store": "adapty",
  "locale": "string",
  "placement_id": "string",
  "customer_user_id": "string"
}
```
