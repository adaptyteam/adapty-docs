Если законодательные требования предписывают использование серверов в ЕС, Adapty предоставляет возможность подключения к кластеру ЕС. Следуйте этим шагам:

1. Настройте Adapty SDK для региона ЕС.
2. Зарегистрируйтесь в Adapty через специальную ссылку для регистрации в ЕС.
3. Войдите в Adapty через специальную ссылку для входа в ЕС.

После настройки вы сможете использовать дашборд Adapty в обычном режиме по адресу `app.adapty.io`.

## Шаг 1. Настройте Adapty SDK для региона ЕС \{#step-1-configure-adapty-sdk-for-eu-region\}

<Tabs groupId="current-os" queryString> 
<TabItem value="swift" label="iOS"> 

Установите Adapty SDK версии 3.0.3 или выше, как описано в разделе [Установка и настройка Adapty SDK](sdk-installation-ios). В процессе настройки добавьте кластер ЕС следующим образом:

Включите параметр `serverCluster` в конфигурацию:

<Tabs groupId="current-os" queryString>

<TabItem value="swift" label="Swift" default>

```swift showLineNumbers
// In your AppDelegate class:

let configurationBuilder =
    AdaptyConfiguration
        .Builder(withAPIKey: "PUBLIC_SDK_KEY")
        .with(observerMode: false)
        .with(customerUserId: "YOUR_USER_ID")
        .with(idfaCollectionDisabled: false)
        .with(ipAddressCollectionDisabled: false)
        // highlight-next-line
        .with(serverCluster: .eu)

Adapty.activate(with: configurationBuilder.build()) { error in
  // handle the error
}
```

Установите Adapty SDK версии 3.0.4 или выше, как описано в разделе [Установка и настройка Adapty SDK](sdk-installation-android). В процессе настройки добавьте кластер ЕС следующим образом:

Используйте метод `.withServerCluster` в конфигурации:

<Tabs groupId="current-os" queryString>
  <TabItem value="kotlin" label="Kotlin" default>

```kotlin showLineNumbers
override fun onCreate() {
    super.onCreate()
    Adapty.activate(
      applicationContext,
      AdaptyConfig.Builder("PUBLIC_SDK_KEY")
          .withObserverMode(false) //default false
          .withCustomerUserId(customerUserId) //default null
          .withIpAddressCollectionDisabled(false) //default false
          // highlight-next-line
          .withServerCluster(AdaptyConfig.ServerCluster.EU)
          .build()
    )  
}
```

## Шаг 2. Зарегистрируйтесь в кластере ЕС \{#step-2-register-with-eu-cluster\}

Для регистрации компании в кластере ЕС используйте https://app.adapty.io/eu-registration. После регистрации вы сможете [добавлять сотрудников](members-settings#add-a-member) в дашборде Adapty в обычном режиме.

## Шаг 3: Войдите через специальную ссылку для ЕС \{#step-3-log-in-with-eu-specific-link\}

Направьте членов команды для входа по ссылке https://app.adapty.io/eu-login. После входа они смогут использовать https://app.adapty.io для обычной работы с дашбордом.

:::warning

Новые пользователи или существующие пользователи, выполняющие вход с нового устройства, должны использовать https://app.adapty.io/eu-login при первом входе.

:::

Это поможет вам соблюдать требования законодательства ЕС по защите данных при использовании Adapty!
