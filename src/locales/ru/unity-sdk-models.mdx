### AdaptyPrice

```typescript
type AdaptyPrice = {
  amount: number;
  currencyCode: string | undefined;
  currencySymbol: string | undefined;
  localizedString: string | undefined;
};
```

| Поле | Тип | Описание |
|---|---|---|
| `amount` | `number` | Числовое значение цены в единицах локальной валюты. |
| `currencyCode` | `string \| undefined` | Трёхбуквенный код валюты по стандарту ISO 4217. Например, `USD`, `EUR`. |
| `currencySymbol` | `string \| undefined` | Символ валюты. Например, `$`, `€`. |
| `localizedString` | `string \| undefined` | Локализованное строковое представление цены. |
| Название | Тип | Описание |
| :---------------- | :--------------- | :------------------------------------------------------------------------------------------------------------------------------- |
| Amount | double | Цена продукта со скидкой в местной валюте |
| CurrencyCode | string (optional) | Код валюты локали, используемый для форматирования цены продукта |
| CurrencySymbol | string (optional) | Символ валюты локали, используемый для форматирования цены продукта |
| LocalizedString | string (optional) | Отформатированная цена скидки для локали пользователя |

### AdaptyProfile.AccessLevel

Информация об [уровне доступа пользователя.](https://swift.adapty.io/documentation/adapty/adaptyprofile/accesslevel)
| Название | Тип | Описание |
|----|----|-----------|
| Id | string | Уникальный идентификатор уровня доступа, заданный вами в дашборде Adapty |
| IsActive | bool | Активен ли уровень доступа. Как правило, достаточно проверить только это свойство, чтобы определить, есть ли у пользователя доступ к премиум-функциям |
| VendorProductId | string | Идентификатор продукта в App Store Connect, который открыл этот уровень доступа |
| Store | string | Стор, в котором была совершена покупка, открывшая этот уровень доступа. Возможные значения: app_store, play_store, adapty |
| ActivatedAt | DateTime | Время активации уровня доступа |
| RenewedAt | DateTime (optional) | Время последнего продления уровня доступа |
| ExpiresAt | DateTime (optional) | Время истечения уровня доступа (может быть в прошлом или равно null для пожизненного доступа) |
| IsLifetime | bool | Активен ли уровень доступа бессрочно (без даты истечения). Если значение равно true, проверять expires_at не нужно — достаточно проверить isActive |
| ActiveIntroductoryOfferType | string (optional) | Тип активного introductory offer. Возможные значения: free_trial, pay_as_you_go, pay_up_front. Если значение не равно null, оффер был применён в текущем расчётном периоде подписки |
| ActivePromotionalOfferType | string (optional) | Тип активного promotional offer. Возможные значения: free_trial, pay_as_you_go, pay_up_front. Если значение не равно null, оффер был применён в текущем расчётном периоде подписки |
| ActivePromotionalOfferId | string (optional) | Идентификатор активного promotional offer |
| OfferId | string (optional) | Идентификатор оффера |
| WillRenew | bool | Настроено ли автоматическое продление подписки |
| IsInGracePeriod | bool | Находится ли автопродляемая подписка в льготном периоде |
| UnsubscribedAt | DateTime (optional) | Время отмены автопродляемой подписки. Подписка при этом может оставаться активной — это лишь означает, что автопродление отключено. Сбрасывается в null, если пользователь возобновляет подписку |
| BillingIssueDetectedAt | DateTime (optional) | Время обнаружения проблемы с оплатой. Подписка при этом может оставаться активной. Сбрасывается в null после успешного списания |
| StartsAt | DateTime (optional) | Время начала действия уровня доступа (может быть в будущем) |
| CancellationReason | string (optional) | Причина отмены подписки. Возможные значения: voluntarily_cancelled, billing_error, refund, price_increase, product_was_not_available, unknown |
| IsRefund | bool | Был ли возврат средств за покупку |

### Перечисления \{#enums\}

#### AdaptySubscriptionPeriodUnit \{#adaptysubscriptionperiodunit\}
- `Day` — период в днях
- `Week` — период в неделях
- `Month` — период в месяцах
- `Year` — период в годах

#### AdaptySubscriptionRenewalType \{#adaptysubscriptionrenewaltype\}
- `Autorenewable` — автоматически возобновляемая подписка
- `Prepaid` — предоплаченная подписка

#### AdaptySubscriptionOfferType \{#adaptysubscriptionoffertype\}
- `Introductory` — introductory offer
- `Promotional` — promotional offer
- `WinBack` — win-back offer

#### AdaptyPaymentMode \{#adaptypaymentmode\}
- `FreeTrial` — бесплатный пробный период
- `PayAsYouGo` — оплата по мере использования
- `PayUpFront` — единовременная оплата
#### AdaptySubscriptionUpdateReplacementMode
- `ImmediateWithTimeProration` - Немедленно с пересчётом времени
- `ImmediateAndChargeProratedPrice` - Немедленно с пропорциональным списанием
- `ImmediateWithoutProration` - Немедленно без пересчёта
- `Deferred` - Отложенно
- `ImmediateAndChargeFullPrice` - Немедленно с полной оплатой

#### AdaptyPurchaseResultType
- `Pending` - Покупка ожидает подтверждения
- `UserCancelled` - Пользователь отменил покупку
- `Success` - Покупка выполнена успешно
#### AdaptyLogLevel
- `Error` - Логирование ошибок
- `Warn` - Логирование предупреждений
- `Info` - Логирование информации
- `Verbose` - Подробное логирование
- `Debug` - Логирование отладки

#### AdaptyProfileGender
- `Female` - Женский пол
- `Male` - Мужской пол
- `Other` - Другой пол

#### AdaptyRefundPreference
- `NoPreference` - Без предпочтений по возврату
- `Grant` - Одобрить возврат
- `Decline` - Отклонить возврат

#### AppTrackingTransparencyStatus
- `NotDetermined` - Статус не определён
- `Restricted` - Статус ограничен
- `Denied` - Статус отклонён
- `Authorized` - Статус авторизован
