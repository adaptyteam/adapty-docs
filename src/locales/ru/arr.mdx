График годового регулярного дохода отображает выручку от всех активных автообновляемых подписок, приведённую к годовому периоду. Активной считается любая оплаченная и не истёкшая подписка. ARR — ключевая метрика для отслеживания роста подписочного бизнеса и прогнозирования будущей выручки.

### Расчёт \{#calculation\}

:::warning
Калькулятор ниже **не учитывает** [комиссию стора и налоги](how-adapty-analytics-works#tax-and-commission-logic). Сравнивайте результат с расчётами **валовой выручки**.
:::

ARR отражает предсказуемый регулярный годовой доход подписочного бизнеса. Эта метрика особенно полезна, когда годовые подписки составляют основную долю продаж.

Для каждого типа подписки разделите произведение цены и числа подписчиков на количество лет в расчётном периоде, затем сложите результаты. Введите значения своих подписок в интерактивный калькулятор, чтобы рассчитать валовой ARR:

<SimpleCalculator client:load
    heading="ARR"
    formuLatex="\sum \frac{P_s \times U_s}{D_y}"
    variables={[
        {
            nameInTheFormula: "P_s",
            variableName: "price",
            variableDescription: "Цена подписки",
            variableValue: 240
        },
        {
            nameInTheFormula: "U_s",
            variableName: "subs",
            variableDescription: "Активные платные подписки",
            variableValue: 2,
            isInteger: true
        },
        {
            nameInTheFormula: "D_y",
            variableName: "periods",
            variableDescription: "Период подписки",
            variableValue: 1,
            options: [
                { label: "Еженедельно", value: "1/52" },
                { label: "Ежемесячно", value: "1/12" },
                { label: "2 месяца", value: "2/12" },
                { label: "3 месяца", value: "3/12" },
                { label: "6 месяцев", value: "6/12" },
                { label: "Годовая", value: 1 }
            ]
        }
    ]}
    formulaCalculation="(price * subs ) / periods"
    isSum={true}
    defaultRows={[
        { price: 240, subs: 2, periods: "1" },
        { price: 30, subs: 10, periods: "1/12" },
        { price: 10, subs: 20, periods: "1/52" }
    ]}
/>

Например, если есть 2 активные годовые подписки по $240, 10 ежемесячных по $30 и 20 еженедельных по $10:
ARR = (2 _ $240 / 1) + (10 _ $30 / (1/12)) + (20\*$10 / (1/52)) = $14 480

:::note
ARR не учитывает подписки, по которым был оформлен возврат. При возврате подписка исключается из расчёта MRR за все периоды, в которые она была активна, — это гарантирует, что MRR отражает только регулярную выручку от действующих подписок.
:::

### Использование графика ARR \{#arr-chart-usage\}

График годового регулярного дохода (ARR) — удобный инструмент для оценки роста и масштаба подписочного бизнеса. Эта широко используемая метрика даёт нормализованное представление о регулярной выручке за 12-месячный период. Чтобы лучше понять, что движет вашим ARR, можно сегментировать данные по ключевым группам подписчиков — например, по стору или длительности продукта. Так вы определите, какие сегменты приносят наибольший доход, и скорректируете бизнес-стратегию.
