Возвращает данные воронки для отслеживания прохождения пользователей через определённые этапы конверсии.

## Эндпоинт и метод \{#endpoint-and-method\}

```http
POST https://api-admin.adapty.io/api/v1/client-api/metrics/funnel/
```
## Пример запроса \{#request-example\}

```bash
curl --location 'https://api-admin.adapty.io/api/v1/client-api/metrics/funnel/' \
--header 'Authorization: Api-Key <YOUR_SECRET_API_KEY>' \
--header 'Content-Type: application/json' \

}'
```

## Параметры \{#parameters\}

| Название      | Тип                                      | Обязательный       | Описание                                                     |
| ------------- | ---------------------------------------- | ------------------ | ------------------------------------------------------------ |
| filters       | [MetricsFilters](#metricsfilters-object) | :heavy_plus_sign:  | Объект с параметрами фильтрации. Подробнее см. ниже.         |
| period_unit   | String                                   | :heavy_minus_sign: | <p>Задаёт временной интервал для агрегации аналитических данных, чтобы просматривать результаты, сгруппированные по выбранным периодам: дням, неделям, месяцам и т. д. Возможные значения:</p><ul><li>day</li><li>week</li><li>month</li><li>quarter</li><li>year</li></ul> |
| show_value_as | String                                   | :heavy_minus_sign: | <p>Определяет способ отображения значений. Возможные значения:</p><ul><li>absolute: в процентах от общего числа</li><li>relative: в процентах от начального значения, где для периодов обновления отсчёт начинается со 100%</li><li>both</li></ul> |
| segmentation  | String                                   | :heavy_minus_sign: | <p>Задаёт основу для сегментации. Возможные значения:</p><ul><li>app_id</li><li>period</li><li>renewal_status</li><li>cancellation_reason</li><li>store_product_id</li><li>country</li><li>store</li><li>purchase_container_id</li><li>paywall_id</li><li>audience_id</li><li>placement_id</li><li>attribution_source</li><li>attribution_status</li><li>attribution_channel</li><li>attribution_campaign</li><li>attribution_adgroup</li><li>attribution_adset</li><li>attribution_creative</li><li>duration</li><li>default</li></ul> |
| format        | String                                   | :heavy_minus_sign: | <p>Задаёт формат экспортируемого файла. Доступные варианты:</p><ul><li>json</li><li>csv</li></ul> |
