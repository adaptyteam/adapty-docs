Добавляет идентификаторы интеграции к профилю.

## Эндпоинт и метод \{#endpoint-and-method\}

```
POST https://api.adapty.io/api/v2/server-side-api/integration/profile/set/integration-identifiers/
```

Плейсхолдеры:

- `<YOUR_CUSTOMER_USER_ID>`: уникальный ID пользователя в вашей системе.
- `<YOUR_SECRET_API_KEY>`: секретный API-ключ для авторизации.

#### Параметры \{#parameters\}

| Параметр                         | Тип    | Обязательный | Nullable | Описание                                                     |
| -------------------------------- | ------ | ------------ | -------- | ------------------------------------------------------------ |
| adjust_device_id                 | String | Нет          | Нет      | ID пользователя в сети в [интеграции с Adjust](adjust).      |
| airbridge_device_id              | String | Нет          | Нет      | ID устройства пользователя в [интеграции с Airbridge](airbridge). |
| amplitude_device_id              | String | Нет          | Нет      | ID устройства пользователя в [интеграции с Amplitude](amplitude). |
| amplitude_user_id                | String | Нет          | Нет      | ID пользователя в [интеграции с Amplitude](amplitude).       |
| appmetrica_device_id             | String | Нет          | Нет      | ID устройства пользователя в [интеграции с AppMetrica](appmetrica). |
| appmetrica_profile_id            | String | Нет          | Нет      | ID пользователя в [интеграции с AppMetrica](appmetrica).     |
| appsflyer_id                     | String | Нет          | Нет      | ID пользователя в сети в [интеграции с AppsFlyer](appsflyer). |
| branch_id                        | String | Нет          | Нет      | Branch Key приложения пользователя в интеграции с Branch.    |
| facebook_anonymous_id            | String | Нет          | Нет      | ID пользователя в [интеграции с Facebook Ads](facebook-ads). |
| firebase_app_instance_id         | String | Нет          | Нет      | ID пользователя в [интеграции с Firebase](firebase-and-google-analytics). |
| mixpanel_user_id                 | String | Нет          | Нет      | ID пользователя в [интеграции с Mixpanel](mixpanel).         |
| one_signal_player_id             | String | Нет          | Нет      | ID пользователя в [интеграции с OneSignal](onesignal). Устаревший идентификатор. |
| one_signal_subscription_id       | String | Нет          | Нет      | ID пользователя в [интеграции с OneSignal](onesignal). Рекомендуемый идентификатор. |
| posthog_distinct_user_id         | String | Нет          | Нет      | ID пользователя в [интеграции с PostHog](posthog).           |
| pushwoosh_hwid                   | String | Нет          | Нет      | ID устройства пользователя в [интеграции с Pushwoosh](pushwoosh). |
| tenjin_analytics_installation_id | String | Нет          | Нет      | ID устройства пользователя в [интеграции с Tenjin](tenjin).  |

## Успешный ответ: 200: OK \{#successful-response-200-ok\}

Запрос выполнен успешно. Тело ответа пустое.
