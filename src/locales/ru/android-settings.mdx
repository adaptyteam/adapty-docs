| Поле                            | Описание                                                                                                                                                                                      |
| :------------------------------ | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Package name**                | Имя пакета — уникальный идентификатор вашего приложения в Google Play Store. Он необходим для базовой работы Adapty, в том числе для обработки подписок.                                      |
| **Service account key file**    | [Ключи](create-service-account) для безопасной аутентификации и проверки покупок.                                                                                               |
| **Google Play RTDN topic name** | URL для получения [серверных уведомлений](enable-real-time-developer-notifications-rtdn) от Play Store, чтобы отслеживать изменения статуса подписок пользователей и реагировать на них. |
