Вы можете создавать пользовательские [сегменты](segments) прямо из [Apple Ads Manager](adapty-ads-manager), выбирая кампании, группы объявлений или ключевые слова и превращая их в сегменты буквально в несколько кликов. Это упрощает персонализацию пейволов и офферов по источнику привлечения — без ручной настройки условий сегмента.

После создания сегмента его можно использовать для назначения разных продуктов и цен, запуска A/B-тестов и кастомизации внешнего вида пейволов.

## Сценарии использования \{#use-cases\}

Вот несколько примеров того, как сегменты, созданные на основе данных Apple Ads, можно применять на практике:

- **Пейволы на основе ключевых слов**. Показывайте пейвол с акцентом на функции пользователям по высокочастотным целевым ключевым словам, а общий пейвол — пользователям по широким словам для охвата.

- **Офферы на уровне кампаний**. Предлагайте более длинные пробные периоды или специальные цены пользователям из выбранных кампаний Apple Ads, сохраняя стандартный оффер для остальных.

- **Согласованность объявления и пейвола**. Направляйте пользователей из групп объявлений, продвигающих конкретные функции, на пейволы, которые в первую очередь акцентируют именно эти функции.

- **Оптимизация высокодоходных кампаний**. Показывайте пейвол с полной ценой и упором на премиум-план пользователям из кампаний, которые стабильно обеспечивают более высокий lifetime value.

## Создание сегментов \{#create-segments\}

Чтобы создать сегмент из Apple Ads Manager:

1. Перейдите в **Ads Manager** и откройте вкладку **Campaigns**, **Ad groups** или **Keywords**. Установите флажки рядом с нужными объектами. Обратите внимание: если вы выберете несколько объектов, для них будет создан один общий сегмент, а не отдельный сегмент для каждого.
2. Нажмите **Actions > Create segment from campaigns/ad groups/keywords**.

<ZoomImage id="create-segment-from.webp" width="700px" /> 

3. При необходимости обновите детали сегмента в окне **Create segment**:
    - **Adapty project**: приложение в Adapty, в котором нужно создать этот сегмент.
    - **Build segment from campaign/ad group**: при создании сегмента из кампаний или групп объявлений на этом шаге можно скорректировать выбранные кампании или группы.
    - **Segment name** — название сегмента.
    - **Segment description** — описание сегмента.

<ZoomImage id="asa-segment.png" width="700px" />

4. Нажмите **Create**.
5. После создания сегмента можно подготовить его к использованию:

- Добавьте его в [плейсмент](placements), чтобы использовать с существующим пейволом или онбордингом.
- Создайте новый [пейвол](adapty-paywall-builder) или [онбординг](onboardings), который будет отображаться пользователям из этого сегмента.
- Запустите [A/B-тест](ab-tests).
