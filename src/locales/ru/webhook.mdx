Вебхук — это эффективный способ получать уведомления о [событиях](webhook-event-types-and-fields#webhook-event-types) в реальном времени, особенно для отслеживания изменений подписок и покупок. Это позволяет мониторить статус подписчиков и реагировать на него.

В отличие от API-запросов, требующих постоянного опроса, вебхук настраивается один раз и автоматически отправляет данные через HTTP при возникновении события.

:::tip
Настраиваете это с помощью AI-агента? Смотрите [Обработка событий подписки Adapty с помощью вебхуков](handle-webhooks-with-ai) — полное руководство на одной странице.
:::
Интегрировав вебхуки, вы сможете:

- Отслеживать подписки и покупки в своей серверной системе.
- Автоматизировать процессы и рабочие флоу на основе жизненного цикла подписок.
- Взаимодействовать с подписчиками: напоминать им о преимуществах приложения, обрабатывать решения об отписке и решать проблемы с оплатой.
- Проводить детальный анализ поведения пользователей.

**Характеристики интеграции**
| Integration characteristic | Description                                                 |
| :------------------------- | :---------------------------------------------------------- |
| Расписание                   | Обновления в реальном времени                                           |
| Направление данных             | Односторонняя передача данных: от Adapty к вашему серверу       |
| Флоу интеграции Adapty    | События отправляются сервером Adapty сразу после их получения |
## События, отправляемые на webhook \{#events-sent-to-webhook\}

Все типы событий, которые можно отправить на webhook, описаны на странице [Типы и поля событий webhook](webhook-event-types-and-fields).

Вы можете отправлять все события или выбрать только нужные. Обратитесь к странице [Флоу событий](event-flows), чтобы определить, какие события вам необходимы. Ненужные типы событий можно отключить при [настройке интеграции с Webhook](set-up-webhook-integration#configure-webhook-integration-in-the-adapty-dashboard). Там же при необходимости можно заменить стандартные идентификаторы событий Adapty на собственные.
