Возвращает аналитические данные о поведении пользователей и ключевых метриках для использования в графиках.

## Эндпоинт и метод \{#endpoint-and-method\}

```http
POST https://api-admin.adapty.io/api/v1/client-api/metrics/analytics/
```
## Пример запроса \{#request-example\}

const myHeaders = new Headers();
myHeaders.append("Authorization", "Api-Key <YOUR_SECRET_API_KEY>");
myHeaders.append("Content-Type", "application/json");

const raw = JSON.stringify({
  "chart_id": "revenue",
  "filters": {
    "date": [
      "2024-01-01",
      "2024-12-31"

  "segmentation": "attribution_campaign"
});

const requestOptions = {
  method: "POST",
  headers: myHeaders,
  body: raw,
  redirect: "follow"
};

fetch("https://api-admin.adapty.io/api/v1/client-api/metrics/analytics/", requestOptions)
  .then((response) => response.text())
  .then((result) => console.log(result))
  .catch((error) => console.error(error));

| period_unit  | String                                          | Нет      | Задаёт временной интервал для агрегации аналитических данных: по дням, неделям, месяцам и т.д. | date_type    | String                                          | Нет      | Возможные значения: <ul><li>purchase_date (по умолчанию)</li><li>profile_install_date</li></ul>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |

| format       | String                                          | Нет      | Задаёт формат экспортируемого файла. Возможные значения: <ul><li>json (по умолчанию)</li><li>csv</li></ul>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |

### Объект MetricsFilters \{#metricsfilters-object\}

Критерии фильтрации отличаются в зависимости от графика. Варианты:

### Для ARPU и installs \{#for-arpu-and-installs\}

| Название             | Тип                          | Обязательный       | Описание                                                     |
| -------------------- | ---------------------------- | ------------------ | ------------------------------------------------------------ |
| date                 | массив строк (String)        | :heavy_plus_sign:  | Укажите дату или период, за который нужно получить данные графика. |
| store                | массив строк (String)        | :heavy_minus_sign: | Фильтр по магазину приложений, в котором была совершена покупка. Возможные значения: **app_store**, **play_store**, **stripe** и любой пользовательский идентификатор стора. Если вы используете кастомный стор, укажите его ID, заданный в дашборде Adapty. |
| country              | массив строк (String)        | :heavy_minus_sign: | Фильтр по двухбуквенному коду страны, в которой была совершена покупка, в формате ISO 3166-1. |
| attribution_source   | массив строк (String)        | :heavy_minus_sign: | Источник интеграции атрибуции. Возможные значения: <ul><li>adjust</li><li>airbridge</li><li>apple_search_ads</li><li>appsflyer</li><li>branch</li><li>custom</li></ul> |
| attribution_status   | массив строк (String)        | :heavy_minus_sign: | Указывает, является ли атрибуция органической или неорганической. Возможные значения: <ul><li>organic</li><li>non-organic</li><li>unknown</li></ul> |
| attribution_channel  | массив строк (String)        | :heavy_minus_sign: | Маркетинговый канал, приведший к транзакции.                 |
| attribution_campaign | массив строк (String)        | :heavy_minus_sign: | Маркетинговая кампания, приведшая к транзакции.              |
| attribution_adgroup  | массив строк (String)        | :heavy_minus_sign: | Группа объявлений атрибуции, приведшая к транзакции.         |
| attribution_adset    | массив строк (String)        | :heavy_minus_sign: | Набор объявлений атрибуции, приведший к транзакции.          |
| attribution_creative | массив строк (String)        | :heavy_minus_sign: | Конкретные визуальные или текстовые элементы объявления или кампании, отслеживаемые для оценки эффективности (например, клики, конверсии). |
