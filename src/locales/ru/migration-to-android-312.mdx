В Adapty SDK 3.12.0 мы удалили метод `logShowOnboarding` из SDK.
Если вы использовали этот метод, он будет недоступен после обновления SDK до версии 3.12 и выше.
Вместо этого вы можете [создавать онбординги в конструкторе онбордингов Adapty без кода](onboardings). Аналитика по этим онбордингам отслеживается автоматически, и у вас есть широкие возможности для кастомизации.
