В Adapty SDK 3.12.0 метод `logShowOnboarding` был удалён из SDK.
Если вы использовали этот метод, он станет недоступен после обновления SDK до версии 3.12 или выше.

Вместо него вы можете [создавать онбординги в конструкторе онбордингов Adapty без кода](onboardings). Аналитика для этих онбордингов отслеживается автоматически, а возможностей для кастомизации у вас будет предостаточно.
