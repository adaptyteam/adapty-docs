| Name          | Type             | Description                                                                                                                      |

### AdaptySubscriptionOffer

`AdaptySubscriptionOffer` — это объект, содержащий информацию о предложении для подписки. Его структура зависит от платформы.

Если продукт является подпиской, то поле `subscriptionOffer` объекта `AdaptyPaywallProduct` будет содержать это значение.

### Перечисления

#### AdaptyPeriodUnit
- `day` - День
- `week` - Неделя
- `month` - Месяц
- `year` - Год

#### AdaptySubscriptionOfferPaymentMode
- `free_trial` - Бесплатный пробный период
- `pay_as_you_go` - Оплата по мере использования
- `pay_up_front` - Единовременная оплата

#### AdaptyRenewalType
- `prepaid` - Предоплаченная подписка
- `autorenewable` - Автопродлеваемая подписка

#### AdaptyProfile.Gender
- `f` - Женский
- `m` - Мужской
