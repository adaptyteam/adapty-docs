## Объект Offer \{#offer-object\}

Этот объект содержит сведения о применённом предложении.

```json showLineNumbers
{
  "category": "no_offer",
  "type": "free_trial",
  "id": "Offer Name"
}

```

### Свойства \{#properties\}

| Название | Тип    | Обязательное | Описание                                                  |
| -------- | ------ | ------------ | --------------------------------------------------------- |
| category | String | Да           | Категория применённого предложения. Возможные значения: `introductory`, `promotional`, `winback`. |
| type     | String | Да           | Тип применённого предложения. Возможные значения: `free_trial`, `pay_as_you_go`, `pay_up_front`. |
| Id       | String | Да           | **Название предложения** применённого оффера, указанное в разделе [**Products**](https://app.adapty.io/products) дашборда Adapty. |

## Объект Products \{#products-object\}

Этот объект содержит сведения о продукте в Adapty.

### Свойства \{#properties-1\}

| Название                       | Тип     | Обязательное | Описание                                                  |
| ------------------------------ | ------- | ------------ | --------------------------------------------------------- |
| title                          | String  | Нет          | **Название продукта** из раздела [**Products**](https://app.adapty.io/products) дашборда Adapty. |
| is_consumable                  | Boolean | Да           | Указывает, является ли продукт расходуемой покупкой.      |
| adapty_product_id              | UUID    | Нет          | Внутренний идентификатор продукта в Adapty.               |
| vendor_product_id              | String  | Да           | Идентификатор продукта в сторах.                          |
| introductory_offer_eligibility | Boolean | Нет          | Указывает, имеет ли пользователь право на iOS introductory offer. |
| promotional_offer_eligibility  | Boolean | Нет          | Указывает, имеет ли пользователь право на promotional offer. |
| base_plan_id                   | String  | Нет          | [Идентификатор базового плана](https://support.google.com/googleplay/android-developer/answer/12154973) для Google Play или [идентификатор цены](https://docs.stripe.com/products-prices/how-products-and-prices-work#use-products-and-prices) для Stripe. |
| offer                          | JSON    | Нет          | Объект [Offer](web-api-objects#offer-object) в формате JSON. |

### Пример \{#example\}

```json showLineNumbers
{
  "title": "Monthly Subscription w/o Trial",
  "is_consumable": true,
  "adapty_product_id": "InternalProductId",
  "vendor_product_id": "onemonth_no_trial",
  "introductory_offer_eligibility": false,
  "promotional_offer_eligibility": true,
  "base_plan_id": "B1",
  "offer": {
    "category": "promotional",
    "type": "pay_up_front",
    "id": "StoreOfferId"
  }
}

```

### Свойства \{#properties-2\}

| Название | Тип    | Обязательное      | Описание                                                  |
| -------- | ------ | ----------------- | --------------------------------------------------------- |
| lang     | String | :heavy_plus_sign: | <p>Код локали для [локализации пейвола](localizations-and-locale-codes). Использует теги языка и региона, разделённые дефисом (**-**).</p><p>Примеры: `en` для английского, `pt-br` для бразильского португальского.</p><p>Подробнее см. в разделе [Локализации и коды локалей](localizations-and-locale-codes).</p> |
| data     | String | :heavy_plus_sign: | Сериализованная JSON-строка, представляющая Remote Config вашего пейвола. Найти её можно на вкладке **Remote Config** конкретного пейвола в дашборде Adapty. |
