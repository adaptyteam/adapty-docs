Adapty позволяет измерять конверсию ваших пейволов. Однако для этого необходимо фиксировать момент, когда пейвол был показан пользователю — без этого мы узнаём только о тех, кто совершил покупку, и упускаем тех, кто этого не сделал. Используйте этот запрос, чтобы зафиксировать просмотр пейвола.

## Эндпоинт и метод \{#endpoint-and-method\}

```text showLineNumbers
POST https://api.adapty.io/api/v2/web-api/paywall/visit/
```

## Параметры \{#parameters\}

| Название         | Тип           | Обязательный       | Описание                                                                                                                                                                             |
| ---------------- | ------------- | ------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| customer_user_id | String        | :heavy_plus_sign:* | <p>Идентификатор пользователя в вашей системе.</p><p>* Требуется либо `customer_user_id`, либо `profile_id`.</p>                                                                     |
| profile_id       | String        | :heavy_plus_sign:* | <p>Идентификатор пользователя в Adapty.</p><p>* Требуется либо `customer_user_id`, либо `profile_id`.</p>                                                                            |
| visited at       | ISO 8601 date | :heavy_minus_sign: | Дата и время, когда пользователь открыл пейвол.                                                                                                                                      |
| store            | String        | :heavy_plus_sign:  | Стор, в котором был куплен продукт. Возможные значения: **app_store**, **play_store**, **stripe** или **Store ID** вашего [кастомного стора](custom-store). |
| variation_id     | String        | :heavy_plus_sign:  | ID варианта, используемый для отслеживания покупок до конкретного пейвола, с которого они были совершены.                                                                             |
