В мире со множеством культур важно адаптировать продукт под каждую страну. Для этого можно использовать локализации пейволов. Для каждого пейвола можно создавать версии на разных языках под нужды конкретных локальных рынков.

В зависимости от того, чем вы пользуетесь для создания флоу, добавление локали различается:
1. [Добавление локали в Adapty Flow Builder](add-paywall-locale-in-adapty-paywall-builder): Если вы создаёте флоу в Adapty Flow Builder, вы можете локализовать все элементы прямо в конструкторе, включая текст и медиафайлы. Чтобы управлять тем, какая локализация отображается, передайте локаль в метод `getFlow`.
2. [Добавление локали в Remote Config](add-remote-config-locale): Кроме того, Adapty позволяет управлять локализациями Remote Config без повторного развёртывания приложения.
