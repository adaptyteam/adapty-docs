Пользовательский HTML позволяет создавать уникальные взаимодействия, встраивать сторонние виджеты или быстро тестировать экспериментальные элементы без обновления приложения.

:::note
Элементы пользовательского HTML не предзагружаются и не кэшируются, поэтому рекомендуем использовать Raw HTML только для небольших и лёгких элементов.
:::

Чтобы вставить пользовательский HTML-код:

1. Нажмите **Add** в верхнем левом углу.
2. Перейдите в **Media & Display** и выберите **Raw HTML**.
3. Введите или отредактируйте HTML-код в правой панели.
