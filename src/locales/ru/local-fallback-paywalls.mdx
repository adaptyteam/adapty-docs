Adapty автоматически генерирует JSON-файлы конфигурации для ваших [резервных пейволов](/fallback-paywalls) — по одному на каждую платформу. Эти файлы также содержат резервные данные для ваших [онбордингов](local-fallback-onboarding).

Если в одном плейсменте есть несколько пейволов или онбордингов, резервная версия будет включать вариант с наибольшим весом или наиболее широкой аудиторией. Adapty обновляет эти файлы при каждом изменении пейволов или онбордингов.

Чтобы скачать резервные конфигурации, выполните следующие шаги:

1. Откройте страницу **[Placements](https://app.adapty.io/placements)**.
2. Нажмите кнопку **Fallbacks**.
3. Выберите целевую платформу (*iOS* или *Android*) из выпадающего списка.
4. Выберите версию SDK, чтобы начать загрузку.
