Добавляет маркетинговые данные пользовательской атрибуции к профилю.

## Эндпоинт и метод \{#endpoint-and-method\}

```text showLineNumbers
POST https://api.adapty.io/api/v2/web-api/attribution/
```

## Пример запроса \{#request-example\}

## Параметры \{#parameters\}

| Название            | Тип    | Обязательный       | Описание                                                     |
| ------------------- | ------ | ------------------ | ------------------------------------------------------------ |
| status              | String | :heavy_plus_sign:  | <p>Указывает, является ли атрибуция органической или неорганической.</p><p>Возможные значения:</p><ul><li> organic</li><li> non_organic</li><li> unknown</li></ul> |
| attribution_user_id | String | :heavy_minus_sign: | ID пользователя, присвоенный источником атрибуции.           |
| channel             | String | :heavy_minus_sign: | Название маркетингового канала.                              |
| campaign            | String | :heavy_minus_sign: | Название маркетинговой кампании.                             |
| ad_group            | String | :heavy_minus_sign: | Группа объявлений атрибуции.                                 |
| ad_set              | String | :heavy_minus_sign: | Набор объявлений атрибуции.                                  |
| creative            | String | :heavy_minus_sign: | Ключевое слово креатива атрибуции.                           |
| customer_user_id    | String | :heavy_plus_sign:* | <p>ID пользователя, который вы используете в своём приложении для его идентификации. Например, это может быть UUID пользователя, email или любой другой идентификатор. Null, если вы его не задали. Найти его можно в поле **Customer User ID** профиля в [дашборде Adapty](https://app.adapty.io/profiles/users).</p><p>* Требуется указать либо `customer_user_id`, либо `profile_id`.</p> |
| profile_id          | String | :heavy_plus_sign:* | <p>Идентификатор пользователя в Adapty. Найти его можно в поле **Adapty ID** профиля в [дашборде Adapty](https://app.adapty.io/profiles/users).</p><p>* Требуется указать либо `customer_user_id`, либо `profile_id`.</p> |
