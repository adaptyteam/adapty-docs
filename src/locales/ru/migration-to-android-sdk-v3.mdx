Adapty SDK v.3.0 добавляет поддержку нового [Adapty Paywall Builder](adapty-paywall-builder) — обновлённого инструмента для создания пейволов без написания кода. Благодаря максимальной гибкости и богатым возможностям дизайна ваши пейволы станут ещё эффективнее и прибыльнее.

Adapty SDK поставляется как BoM (Bill of Materials), что гарантирует согласованность версий Adapty SDK и AdaptyUI SDK в вашем приложении.

Чтобы перейти на v3.0, обновите код следующим образом:

<Tabs groupId="current-os" queryString>
  <TabItem value="module-level build.gradle" label="module-level build.gradle" default>

```diff showLineNumbers
dependencies {
    ...
-   implementation 'io.adapty:android-sdk:2.11.5'
-   implementation 'io.adapty:android-ui:2.11.3'
+   implementation platform('io.adapty:adapty-bom:3.0.4')
+   implementation 'io.adapty:android-sdk'
+   implementation 'io.adapty:android-ui'
}
```

</TabItem>
