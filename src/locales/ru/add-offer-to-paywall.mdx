Офферы в App Store и Google Play — это специальные предложения или скидки, которые эти платформы предоставляют для встроенных покупок.

Чтобы оффер отображался и был доступен для выбора на [пейволе](paywalls) в вашем приложении, выполните следующие шаги:

При [настройке продуктов на пейволе](create-paywall) выберите оффер, [созданный ранее](create-offer) для этого продукта, из списка **Offer**. Список доступен только для продуктов, у которых есть офферы.

:::info

Пейволы, созданные с помощью Adapty Paywall Builder, отображают только первую фазу [многофазного оффера подписки Google](https://support.google.com/googleplay/android-developer/answer/12154973). При этом, когда пользователь совершает покупку, все фазы оффера применяются в соответствии с настройками в Google Play.

:::
