Bir flow veya paywall oluştturduysanız, bunu kullanıcıya göstermek için mobil uygulama kodunuzda ayrıca render etmenize gerek yok. Söz konusu flow veya paywall, içinde neyin gösterileceğini ve nasıl gösterileceğini zaten içeriyor.

Aşağıda kullanılan `AdaptyUI.FlowConfiguration` nesnesini edinmek için [Flow ve paywallları getir](get-pb-paywalls) sayfasına bakın.

## SwiftUI'da flow ve paywallları sunma \{#present-flows-and-paywalls-in-swiftui\}

### Ekranda modal görünüm olarak göster \{#present-as-a-modal-view\}

Bir flow veya paywall'ı cihaz ekranında modal görünüm olarak göstermek için SwiftUI'da `.flow` modifier'ını kullanın. En temel çağrı; `isPresented`, `flowConfiguration` ve beş zorunlu callback'i gerektirir:
```swift showLineNumbers title="SwiftUI"
.flow(
    isPresented: $flowPresented,
    flowConfiguration: <AdaptyUI.FlowConfiguration>,
    didFinishPurchase: { _, _ in /* dismiss, or do nothing to let the flow continue */ },
    didFailPurchase: { _, _ in /* handle the error */ },
    didFinishRestore: { _ in /* check access level and dismiss */ },
    didFailRestore: { _ in /* handle the error */ },
    didReceiveError: { _ in flowPresented = false }
)
```

Daha fazla kontrol için, buton tıklamalarını yönetmek amacıyla `didPerformAction` gibi isteğe bağlı callback'ler ekleyin:
```swift showLineNumbers title="SwiftUI"
@State var flowPresented = false // ensure that you manage this variable state and set it to `true` at the moment you want to show the flow or paywall

var body: some View {
  Text("Hello, AdaptyUI!")
      .flow(
          isPresented: $flowPresented,
          flowConfiguration: <AdaptyUI.FlowConfiguration>,
          didPerformAction: { action in
              switch action {
                  case .close:
                      flowPresented = false
                  default:
                      // Handle other actions
                      break
              }
          },
          didFinishPurchase: { product, purchaseResult in /* dismiss, or do nothing to let the flow continue */ },
          didFailPurchase: { product, error in /* handle the error */ },
          didFinishRestore: { profile in /* check access level and dismiss */ },
          didFailRestore: { error in /* handle the error */ },
          didReceiveError: { error in flowPresented = false }
      )
}
```

### Modal olmayan görünüm olarak sunma \{#present-as-a-non-modal-view\}

Flow'ları ve paywall'ları uygulamanızın navigasyon akışında navigasyon hedefleri veya satır içi görünümler olarak da sunabilirsiniz. SwiftUI görünümlerinizde doğrudan `AdaptyFlowView` kullanın:
```swift showLineNumbers title="SwiftUI"
AdaptyFlowView(
    flowConfiguration: <AdaptyUI.FlowConfiguration>,
    didFinishPurchase: { product, purchaseResult in
        // Dismiss the view, or do nothing to let the flow continue
    },
    didFailPurchase: { product, error in
        // Handle purchase failure
    },
    didFinishRestore: { profile in
        // Handle successful restore
    },
    didFailRestore: { error in
        // Handle restore failure
    },
    didReceiveError: { error in
        // Handle the error (rendering or JS exception from the flow script).
    }
)
```

## Flow'ları ve paywallları UIKit'te sunma \{#present-flows-and-paywalls-in-uikit\}

Flow'u veya paywall'ı cihaz ekranında görüntülemek için şu adımları izleyin:

1. Görüntülemek istediğiniz görsel flow'u `AdaptyUI.flowController(with:delegate:)` metodunu kullanarak başlatın:

     ```swift showLineNumbers title="Swift"
     import AdaptyUI

     let visualFlow = try AdaptyUI.flowController(
         with: <AdaptyUI.FlowConfiguration>,
         delegate: <AdaptyFlowControllerDelegate>
     )
     ```

    İstek parametreleri:
| Parametre | Zorunluluk | Açıklama |
   | :----------------------- | :------- | :---------- |
   | **flowConfiguration**    | zorunlu | Akış veya paywallın görsel detaylarını içeren bir `AdaptyUI.FlowConfiguration` nesnesi. `AdaptyUI.getFlowConfiguration(forFlow:)` metodunu kullanın. Daha fazla bilgi için [Flow ve paywallları alma](get-pb-paywalls) konusuna bakın. |
   | **delegate**             | zorunlu | Flow ve paywall olaylarını dinlemek için kullanılan `AdaptyFlowControllerDelegate`. Daha fazla bilgi için [Flow & paywall olaylarını yönetme](ios-handling-events) konusuna bakın. |

    Döndürür:
| Nesne | Açıklama |
   | :---------------------- | :------------------------------------------------------- |
   | **AdaptyFlowController** | İstenen flow veya paywall ekranını temsil eden bir nesne. |

2. Nesne başarıyla oluşturulduktan sonra cihazın ekranında görüntüleyebilirsiniz:

   ```swift showLineNumbers title="Swift"
   present(visualFlow, animated: true)
   ```

## SwiftUI'da paywallları sunma \{#present-paywalls-in-swiftui\}

### Cihaz ekranında modal görünüm olarak sunma \{#present-as-a-modal-view\}

Görsel paywallı cihaz ekranında modal görünüm olarak göstermek için SwiftUI'da `.paywall` modifier'ını kullanın:
```swift showLineNumbers title="SwiftUI"
@State var paywallPresented = false // ensure that you manage this variable state and set it to `true` at the moment you want to show the paywall

var body: some View {
  Text("Hello, AdaptyUI!")
      .paywall(
          isPresented: $paywallPresented,
          paywallConfiguration: <AdaptyUI.PaywallConfiguration>,
          didPerformAction: { action in
              switch action {
                  case .close:
                      paywallPresented = false
                  default:
                      // Handle other actions
                      break
              }
          },
          didFinishPurchase: { product, profile in paywallPresented = false },
          didFailPurchase: { product, error in /* handle the error */ },
          didFinishRestore: { profile in /* check access level and dismiss */  },
          didFailRestore: { error in /* handle the error */ },
          didFailRendering: { error in paywallPresented = false }
      )
}
```

## UIKit'te paywall'ları sunma \{#present-paywalls-in-uikit\}

Paywall'ı cihaz ekranında görüntülemek için şunları yapın:

1. Görüntülemek istediğiniz paywall'ı `.paywallController(for:products:viewConfiguration:delegate:)` metodunu kullanarak başlatın:

     ```swift showLineNumbers title="Swift"
     import AdaptyUI

     let visualPaywall = AdaptyUI.paywallController(
         with: <paywall configuration object>,
         delegate: <AdaptyPaywallControllerDelegate>
     )
     ```

    İstek parametreleri:
| Parametre                | Zorunluluk | Açıklama |
   | :----------------------- | :------- | :---------- |
   | **paywall configuration**         | zorunlu | Paywallın görsel ayrıntılarını içeren bir `AdaptyUI.PaywallConfiguration` nesnesi. `AdaptyUI.getPaywallConfiguration(forPaywall:locale:)` metodunu kullanın. Daha fazla bilgi için [Paywall Builder paywalllarını ve yapılandırmalarını getirme](get-pb-paywalls) konusuna bakın. |
   | **delegate**            | zorunlu | Paywall olaylarını dinlemek için bir `AdaptyPaywallControllerDelegate`. Daha fazla bilgi için [Paywall olaylarını yönetme](ios-handling-events) konusuna bakın. |
Döner:

   | Nesne                   | Açıklama                                             |
   | :---------------------- | :--------------------------------------------------- |
   | **AdaptyPaywallController** | İstenen paywall ekranını temsil eden bir nesne |

2. Nesne başarıyla oluşturulduktan sonra cihaz ekranında gösterebilirsiniz:

   ```swift showLineNumbers title="Swift"
   present(visualPaywall, animated: true)
   ```
