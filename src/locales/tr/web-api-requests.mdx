Adapty'nin sunucu taraflı API'si, abonelik verilerinize programatik olarak erişip yönetmenizi sağlar; böylece mevcut servisleriniz ve altyapınızla sorunsuz entegrasyon kurabilirsiniz. Platformlar arası veri senkronizasyonu, access level tanımlama veya Stripe'ta satın alma doğrulama gibi işlemlerde bu API, sistemlerinizi senkronize tutmak ve kullanıcılarınızı aktif kılmak için ihtiyacınız olan araçları sunar.

## Postman koleksiyonu ve ortamı \{#postman-collection-and-environment\}

Web API'yi daha kolay kullanabilmeniz için Postman'a indirip aktarabileceğiniz bir koleksiyon ve ortam dosyası hazırladık.

- **İstek Koleksiyonu**: Adapty web API'sinde mevcut tüm istekleri içerir. Ortamda tanımlayabileceğiniz değişkenler kullandığını unutmayın.
- **Ortam**: Değerleri bir kez tanımlayabileceğiniz değişkenlerin listesini içerir. Sunucu taraflı API, web API ve analytics export API için birleşik bir ortam hazırladık. Bu ortamı aktif hale getirdikten sonra Postman, tanımlanan değişken değerlerini isteklerinizde otomatik olarak kullanır.

:::tip

[Koleksiyonu ve ortamı indirin](https://raw.githubusercontent.com/adaptyteam/adapty-docs/refs/heads/main/Downloads/Adapty_Web_API_postman_collection.zip)

:::

Bir koleksiyonu ve ortamı Postman'a nasıl aktaracağınız hakkında bilgi almak için lütfen [Postman belgelerine](https://learning.postman.com/docs/getting-started/importing-and-exporting/importing-data/) başvurun.

## Kullanılan değişkenler \{#variables-used\}

İş akışınızı kolaylaştırmak için sunucu taraflı API, web API ve analytics export API için birleşik bir ortam oluşturduk. Aşağıda web API'sine özgü değişkenler yer almaktadır:

| Değişken                | Açıklama                                                     | Örnek Değer                                             |
| ----------------------- | ------------------------------------------------------------ | ------------------------------------------------------- |
| public_api_key          | [**App settings**](https://app.adapty.io/settings/general) sayfasındaki **Public SDK key** alanında bulabilirsiniz. | `public_live_Pj1P1xzM.2CvSvE1IalQRFjsWy6csBVNpH33atnod` |
| adapty-customer-user-id | Sisteminizde kullandığınız kullanıcı ID'si. Adapty Kontrol Paneli'nde Profil'in **Customer user ID** alanında bulabilirsiniz. | `john.doe@example.com`                                  |
| adapty-profile-id       | Adapty tarafından atanan kullanıcı ID'si. Adapty Kontrol Paneli'nde Profil'in **Adapty ID** alanında bulabilirsiniz. | `3286abd3-48b0-4e9c-a5f6-ac0a006333a6`                  |
