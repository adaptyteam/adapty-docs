---
title: "Kotlin Multiplatform SDK'da ATT ile Başa Çıkma"
description: "Abonelik kurulumunu ve yönetimini kolaylaştırmak için Kotlin Multiplatform üzerinde Adapty'ye başlayın."
---

Uygulamanız AppTrackingTransparency framework'ünü kullanıyor ve kullanıcıya uygulama izleme yetkilendirme isteği gösteriyorsa, [yetkilendirme durumunu](https://developer.apple.com/documentation/apptrackingtransparency/attrackingmanager/authorizationstatus/) Adapty'ye göndermeniz gerekir.

```kotlin showLineNumbers
import com.adapty.kmp.Adapty
import com.adapty.kmp.models.AdaptyProfileParameters

val profileParameters = AdaptyProfileParameters.Builder()
    .withAttStatus(3) // 3 = ATTrackingManagerAuthorizationStatusAuthorized
    .build()

Adapty.updateProfile(profileParameters)
    .onSuccess {
        // ATT status updated successfully
    }
    .onError { error ->
        // handle AdaptyError
    }
```

:::warning
Bu değeri değiştiği anda mümkün olduğunca erken göndermenizi önemle tavsiye ederiz; ancak bu şekilde veriler yapılandırdığınız entegrasyonlara zamanında iletilir.
:::
