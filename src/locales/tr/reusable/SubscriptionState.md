| Kullanıcı **durumu** | Açıklama |
| :--------------------- | :----------------------------------------------------------- |
| Subscribed | Kullanıcının otomatik yenileme etkin şekilde aktif bir aboneliği var. |
| Auto-renew off | Kullanıcı otomatik yenilemeyi kapattı ancak abonelik süresi dolana kadar premium özelliklere erişimi devam ediyor. |
| Subscription cancelled | Kullanıcı aboneliğini iptal etti ve abonelik tamamen sona erdi. |
| Billing issue | Abonelik veya deneme süresi dolduktan sonra ödeme alınamadı. |
| Grace period | Abonelik veya deneme süresi dolduğunda ödeme alınamadığı için kullanıcı şu anda ek süre içinde. |
| Active trial | Kullanıcının şu anda deneme sürecinde olan aktif bir aboneliği var. |
| Trial cancelled | Kullanıcı deneme süresini iptal etti ve aktif bir aboneliği yok. |
| Never subscribed | Kullanıcı hiç abonelik başlatmadı veya deneme süresine girmedi; ücretsiz kullanıcı olarak kalmaya devam ediyor. |
