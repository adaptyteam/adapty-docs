[Asapty](https://asapty.com/) entegrasyonunu kullanarak Search Ads kampanyalarınızı optimize edebilirsiniz. Adapty, abonelik olaylarını Asapty'ye gönderir; böylece Apple Search Ads attribution verilerine dayalı özel kontrol panelleri oluşturabilirsiniz.

Bu entegrasyon, Adapty'ye herhangi bir attribution verisi eklemez; zira ihtiyacımız olan her şeyi doğrudan [ASA](apple-search-ads) üzerinden alıyoruz.

## Entegrasyonu kurma \{#set-up-integration\}

### Adapty'yi Asapty'ye bağlama \{#connect-adapty-to-asapty\}

Asapty entegrasyonunu kurmak için Adapty kontrol panelinde [Integrations > Asapty](https://app.adapty.io/integrations/asapty) bölümüne gidin ve Asapty ID alanını doldurun.

### Olayları ve etiketleri yapılandırma \{#configure-events-and-tags\}

Kimlik bilgilerinin altında, Adapty'den Asapty'ye gönderebileceğiniz üç grup olay yer almaktadır. İhtiyacınız olanları etkinleştirmeniz yeterli. Adapty'nin sunduğu olayların tam listesine [buradan](events) ulaşabilirsiniz.

### Uygulamanızı Asapty'ye bağlama \{#connect-your-app-to-asapty\}

Yukarıdaki adımları tamamladıktan sonra Adapty, attribution verilerini Asapty'den otomatik olarak alır. Uygulama kodunuzda attribution verilerini açıkça talep etmenize gerek yoktur. Daha doğru attribution verileri için Asapty'yi her olayın verisiyle birlikte `customerUserId` paylaşacak şekilde yapılandırın.

## Asapty olay yapısı \{#asapty-event-structure\}

Adapty, olayları sorgu parametreleri aracılığıyla GET isteğiyle Asapty'ye gönderir. Her olay URL'si şu şekilde görünür:

```
https://asapty.com/_api/mmpEvents/?source=adapty&asaptyid=a1b2c3d4&keywordid=12345&adgroupid=67890&campaignid=11223&conversiondate=1709294400000&event_name=subscription_renewed&install_time=1709100000&app_name=MyApp&json=%7B%22af_revenue%22%3A%229.99%22%2C%22af_currency%22%3A%22USD%22...%7D
```

Sorgu parametreleri:

| Parametre        | Tür    | Açıklama                                                      |
|:-----------------|:-------|:--------------------------------------------------------------|
| `source`         | String | Her zaman "adapty" değerini alır.                             |
| `asaptyid`       | String | Kimlik bilgilerinizdeki Asapty ID.                            |
| `keywordid`      | String | Apple Search Ads Anahtar Kelime ID'si (varsa).                |
| `adgroupid`      | String | Apple Search Ads Reklam Grubu ID'si (varsa).                  |
| `campaignid`     | String | Apple Search Ads Kampanya ID'si (varsa).                      |
| `conversiondate` | Long   | Olayın **milisaniye** cinsinden zaman damgası.                |
| `event_name`     | String | Olay adı (Adapty olayından eşlenen).                          |
| `install_time`   | Long   | Kurulumun saniye cinsinden zaman damgası.                     |
| `app_name`       | String | Adapty'deki uygulama başlığı (varsa).                         |
| `json`           | String | Olay ayrıntılarını içeren URL kodlu JSON dizesi (aşağıya bakın). |

`json` parametresi, aşağıdaki alanları içeren URL kodlu bir JSON dizesidir:

| Parametre                 | Tür    | Açıklama                                            |
|:--------------------------|:-------|:----------------------------------------------------|
| `af_revenue`              | String | Dize olarak gelir miktarı.                          |
| `af_currency`             | String | Para birimi kodu (ör. "USD").                       |
| `transaction_id`          | String | Mağaza İşlem ID'si.                                 |
| `original_transaction_id` | String | Orijinal Mağaza İşlem ID'si.                        |
| `purchase_date`           | Long   | Milisaniye cinsinden satın alma zaman damgası.      |
| `original_purchase_date`  | Long   | Milisaniye cinsinden orijinal satın alma zaman damgası. |
| `environment`             | String | `Production` veya `Sandbox`.                        |
| `vendor_product_id`       | String | Mağazadaki Ürün ID'si.                              |
| `profile_country`         | String | Kullanıcının IP'sine göre ülke kodu.                |
| `store_country`           | String | Kullanıcının mağaza ülke kodu.                      |

## Sorun giderme \{#troubleshooting\}

- Adapty'de [Apple Search Ads](apple-search-ads) yapılandırdığınızdan ve [kimlik bilgilerini yüklediğinizden](https://app.adapty.io/settings/apple-search-ads) emin olun; bunlar olmadan Asapty çalışmaz.
- Yalnızca ayrıntılı, organik olmayan ASA attribution verisine sahip profiller olaylarını Asapty'ye iletebilir. Attribution yeterli değilse "The user profile is missing the required integration data." mesajını görürsünüz.
- Entegrasyon yapılandırılmadan önce oluşturulan profiller olaylarını Asapty'ye iletamaz.
- Doğru kuruluma rağmen Adapty entegrasyonu çalışmıyorsa [**App Settings** -> **Apple Search Ads** sekmesinde](https://app.adapty.io/settings/apple-search-ads) **Receive Apple Search Ads attribution in Adapty** geçişinin etkin olduğundan emin olun.
