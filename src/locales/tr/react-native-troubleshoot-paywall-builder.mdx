Bu kılavuz, React Native SDK'da Adapty Paywall Builder ile tasarlanan paywallları kullanırken karşılaşılan yaygın sorunları çözmenize yardımcı olur.

## Bir paywall yapılandırmasını getirme başarısız oluyor \{#getting-a-paywall-configuration-fails\}

**Sorun**: Bir flow veya paywall için view yapılandırmasını getirme başarısız oluyor.

**Neden**: Paywall, Paywall Builder'da cihazda gösterim için etkinleştirilmemiş.

**Çözüm**: Paywall Builder'da **Show on device** geçiş düğmesini etkinleştirin.

## Paywall görüntüleme sayısı çok yüksek \{#the-paywall-view-number-is-too-big\}

**Sorun**: Paywall görüntüleme sayısı beklenenin iki katını gösteriyor.

**Neden**: Kodunuzda `logShowFlow` (React Native SDK v4+) / `logShowPaywall` çağırıyor olabilirsiniz; bu, Paywall Builder veya Flow Builder kullanıyorsanız görüntüleme sayısını ikiye katlar. Bu araçlarla oluşturulmuş flow'lar ve paywall'lar için analizler otomatik olarak izlenir, bu yüzden bu metodu kullanmanıza gerek yoktur.

**Çözüm**: Paywall Builder veya Flow Builder kullanıyorsanız, kodunuzda `logShowFlow` (React Native SDK v4+) / `logShowPaywall` çağırmadığınızdan emin olun.

## Diğer sorunlar \{#other-issues\}

**Sorun**: Yukarıda ele alınmayan başka Paywall Builder sorunları yaşıyorsunuz.

**Çözüm**: Gerekirse [migration kılavuzlarını](react-native-sdk-migration-guides) kullanarak SDK'yı en son sürüme güncelleyin. Pek çok sorun yeni SDK sürümlerinde çözülmüştür.
