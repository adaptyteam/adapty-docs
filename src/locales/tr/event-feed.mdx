Etkinlik akışı, Adapty tarafından oluşturulan [Etkinlikleri](events) görsel olarak takip etmenizi ve webhook dahil üçüncü taraf entegrasyonlara aktarım durumlarını kontrol etmenizi sağlar.

:::warning

[Sunucu tarafı API (sürüm 1)](server-side-api-specs-legacy#requests) kullanılarak oluşturulan işlemlerin **Event Feed**'de görünmediğini unutmayın. Bu işlemlerin dahil edilmesini sağlamak için [sunucu tarafı API (sürüm 2)](api-adapty/operations/setTransaction)'yi kullanın.

:::
