Etkinlik akışı, Adapty tarafından oluşturulan [Etkinlikleri](events) görsel olarak takip etmenizi ve webhook dahil üçüncü taraf entegrasyonlara aktarım durumlarını kontrol etmenizi sağlar.
:::warning
Event Feed şunları göstermez:
- **Sunucu tarafı API v1 işlemleri**: [Sunucu tarafı API (sürüm 1)](server-side-api-specs-legacy#requests) kullanılarak oluşturulanlardır. Bunların görünmesi için bunun yerine [sunucu tarafı API (sürüm 2)](api-adapty/operations/setTransaction) kullanın.
- **Profili olmayan olaylar**: SDK bir kullanıcıyı tanımlamadan önce gelen işlemler — örneğin, mağaza sunucu bildirimleri. Bunları dışa aktarmalara dahil etmek için [S3](s3-exports) veya [Google Cloud Storage](google-cloud-storage) entegrasyonunda **Include events without profile** seçeneğini etkinleştirin.
:::
