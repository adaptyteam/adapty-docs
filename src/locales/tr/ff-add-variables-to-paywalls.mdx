[Gerekli tüm ürün verilerini aldıktan](ff-action-flow) sonra, bu verileri FlutterFlow'da tasarladığınız güzel paywalla eşleştirme zamanı. Bu örnekte ürün başlığını ve fiyatını eşleştireceğiz.

## Adım 2.1. Paywall sayfasına ürün başlığı ekleme \{#step-21-add-product-title-to-paywall-page\}

1. Paywall sayfanızdaki ürün metnine çift tıklayın. **Set from Variable** penceresinde `getPaywallProductResult` değişkenini arayın ve seçin.

  <Zoom>
    <img src="/assets/shared/img/ff-paywall-text.webp"
    style={{
      border: '1px solid #727272', /* border width and color */
      width: '700px', /* image width */
      display: 'block', /* for alignment */
      margin: '0 auto' /* center alignment */
    }}
  />
  </Zoom>

2. Alanları aşağıdaki gibi doldurun:
  - **Available Options**: Data Structured Field
  - **Select Field**: value
  - **Available Options**: Item at Index
  - **List Index Options**: First
  - **Available Options**: Data Structured Field
  - **Select Field**: localizedTitle
  - **Default Variable Value**: null
  - **UI Builder Display Value**: Herhangi bir şey; örnekte `product.title` kullanılmıştır.

  <Zoom>
    <img src="/assets/shared/img/ff-product.webp"
    style={{
      border: '1px solid #727272', /* border width and color */
      width: '700px', /* image width */
      display: 'block', /* for alignment */
      margin: '0 auto' /* center alignment */
    }}
  />
  </Zoom>

3. Değişiklikleri kaydetmek için **Confirm**'e tıklayın.

## Adım 2.2. Paywall sayfasına fiyat metni ekleme \{#step-22-add-price-text-to-paywall-page\}

Fiyat metni için Adım 2.1'deki adımları aşağıda gösterildiği gibi tekrarlayın:

1. Paywall sayfanızdaki fiyat metnine çift tıklayın. **Set from Variable** penceresinde `getPaywallProductResult` değişkenini arayın ve seçin.

<Zoom>
  <img src="/assets/shared/img/ff-price.webp"
  style={{
    border: '1px solid #727272', /* border width and color */
    width: '700px', /* image width */
    display: 'block', /* for alignment */
    margin: '0 auto' /* center alignment */
  }}
/>
</Zoom>

2. Alanları aşağıdaki gibi doldurun:

- **Available Options**: Data Structured Field
- **Select Field**: value
- **Available Options**: Item at Index
- **List Index Options**: First
- **Available Options**: Data Structured Field
- **Select Field**: price
- **Default Variable Value**: null
- **UI Builder Display Value**: Herhangi bir şey; örnekte `product.price` kullanılmıştır.

3. Değişiklikleri kaydetmek için **Confirm** düğmesine tıklayın.

### Paywall sayfasına yerel para biriminde fiyat ekleme \{#add-price-in-local-currency-to-paywall-page\}

1. Paywall sayfanızdaki fiyata çift tıklayın. **Set from Variable** penceresinde `getPaywallProductResult` değişkenini arayın ve seçin.

2. Alanları aşağıdaki gibi doldurun:

- **Available Options**: Data Structured Field
- **Select Field**: value
- **Available Options**: Item at Index
- **List Index Options**: First
- **Available Options**: Data Structured Field
- **Select Field**: price
- **Available Options**: Data Structured Field
- **Select Field**: amount
- **Available Options**: Decimal
- **Decimal Type**: Automatic
- **Default Variable Value**: null
- **UI Builder Display Value**: Herhangi bir şey; örnekte `price.amount` kullanılmıştır.

3. Değişiklikleri kaydetmek için **Confirm**'e tıklayın.

İşte bu kadar! Artık uygulamanızı başlattığınızda, Adapty paywallındaki ürün verileri doğrudan paywall sayfanızda görünecek!
