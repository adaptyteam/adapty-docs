Uygulamanız AppTrackingTransparency çerçevesini kullanıyor ve kullanıcıya uygulama takip yetkilendirme isteği sunuyorsa, [yetkilendirme durumunu](https://developer.apple.com/documentation/apptrackingtransparency/attrackingmanager/authorizationstatus/) Adapty'ye göndermeniz gerekir.

```dart showLineNumbers
final builder = AdaptyProfileParametersBuilder()
  ..setAppTrackingTransparencyStatus(AdaptyIOSAppTrackingTransparencyStatus.authorized);

try {
  await Adapty().updateProfile(builder.build());
} on AdaptyError catch (adaptyError) {
  // handle the error
} catch (e) {
  // handle unknown error
}
```

:::warning
Bu değeri, değiştiği anda mümkün olan en kısa sürede göndermenizi kesinlikle öneririz. Ancak bu şekilde veriler, yapılandırdığınız entegrasyonlara zamanında iletilir.
:::
