---
title: "Adapty Capacitor SDK'yı v. 3.16'ya Taşıma"
description: "Daha iyi performans ve yeni monetizasyon özellikleri için Adapty Capacitor SDK v3.16'ya geçiş yapın."
metadataTitle: "Adapty Capacitor SDK v3.16'ya Geçiş | Adapty Dokümanları"
---

Adapty SDK v.3.16.0 itibarıyla Capacitor 8 zorunlu hale gelmiştir. Capacitor 7'ye ihtiyaç duyuyorsanız Adapty SDK v.3.15'i kullanın.

Capacitor SDK v.3.16'ya yükseltmek için projenizin Capacitor 8 kullandığından emin olun. Hâlâ Capacitor 7 kullanıyorsanız iki seçeneğiniz var:

1. **Capacitor 8'e Yükseltme**: Projenizi güncellemek için [resmi Capacitor taşıma kılavuzunu](https://capacitorjs.com/docs/updating/8-0) takip edin, ardından Adapty SDK v.3.16'yı yükleyin.
2. **Adapty SDK v.3.15'te Kalma**: Capacitor 8'e yükseltmek mümkün değilse Capacitor 7'yi destekleyen Adapty SDK v.3.15'i kullanmaya devam edin.
