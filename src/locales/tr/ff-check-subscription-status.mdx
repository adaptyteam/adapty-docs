Bir kullanıcının belirli ücretli içeriğe erişimi olup olmadığını belirlerken, onun access level'ını doğrulamanız gerekir. Bu, kullanıcının en az bir access level'a sahip olup olmadığını ve bu level'ın gerekli olan mı olduğunu kontrol etmek anlamına gelir.

Bunu, mevcut tüm access level'ları içeren kullanıcı profilini kontrol ederek yapabilirsiniz.

Şimdi, kullanıcıların ürününüzü satın almasına izin verelim:

1. Ücretli içeriği göstermesi gereken butona çift tıklayın ve henüz açık değilse sağ paneldeki **Actions** bölümünü açın.

2. **Action Flow Editor**'ı açın.

3. **Select Action Trigger** penceresinde **On Tap** seçeneğini seçin.

4. **No Actions Created** penceresinde **Add Conditional Action** butonuna tıklayın.

5. Eylem argümanlarını ayarlamak için **UNSET**'e tıklayın ve `currentProfile` değişkenini seçin. Bu, geçerli kullanıcının profil verilerini tutan Adapty değişkenidir.

6. Alanları aşağıdaki gibi doldurun:
- **Available Options**: Data Structure Field
- **Select Field**: accessLevels
- **Available Options**: Filter List Items
- **Filter Conditions**: 
  1. **Conditions -> Single Condition** seçeneğini seçin ve **UNSET**'e tıklayın.
  2. **First value** alanında **Source** olarak **Item in list** seçin ve alanları şu şekilde doldurun:
     - **Available Options**: Data Structure Field
     - **Select Field**: accessLevelIdentifier
  3. Filtre operatörünü **Equal to** olarak ayarlayın.
  4. **Second value** yanındaki **UNSET**'e tıklayın ve **Value** alanına access level'ınızın ID'sini girin; örneğimizde `premium` kullanıyoruz.
     <Zoom>
     <img src="/assets/shared/img/ff-filter.webp"

     width: '500px', /* image width */
     display: 'block', /* for alignment */
     margin: '0 auto' /* center alignment */
     }}
     />
     </Zoom>
  5. **Confirm**'e tıklayın ve aşağıdaki diğer alanları doldurmaya devam edin.
- **Available Options**: Item at Index
- **List Index Options**: First
- **Available Options**: Data Structure Field
- **Select Field**: accessLevel
- **Available Options**: Data Structure Field
- **Select Field**: isActive

7. **Confirm**'e tıklayın.

Şimdi, sonraki adım için eylemleri ekleyin — kullanıcının doğru aboneliğe sahip olup olmadığına göre. Ya onları premium abonelere açık olan sayfaya yönlendirin ya da erişim satın alabilmeleri için paywall sayfasını açın.
