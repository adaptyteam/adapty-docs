```javascript showLineNumbers
    try {
      final result = await adapty.makePurchase(product: product);
    } on AdaptyError catch (adaptyError) {

      }

    }
```

## Diğer sorunlar \{#other-issues\}

Henüz bir çözüm bulamadıysanız, deneyebileceğiniz diğer adımlar şunlardır:
- **SDK'yı en son sürüme yükseltme**: En kararlı olduklarından ve bilinen sorunlara yönelik düzeltmeler içerdiklerinden, her zaman en son SDK sürümlerine yükseltmenizi öneririz.
- **Destek ekibiyle iletişime geçin veya diğer geliştiricilerden yardım alın** [destek forumunda](https://adapty.featurebase.app/).
- **Destek ekibiyle [support@adapty.io](mailto:support@adapty.io) adresinden veya sohbet aracılığıyla iletişime geçin**: SDK'yı yükseltmeye hazır değilseniz veya yükseltme sorunu çözmediyse destek ekibimizle iletişime geçin. [Ayrıntılı günlük kaydını etkinleştirip](sdk-installation-flutter#logging) günlükleri ekiple paylaşırsanız sorununuz daha hızlı çözülecektir. İlgili kod parçacıklarını da ekleyebilirsiniz.
