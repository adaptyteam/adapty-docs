SDK tarafından döndürülen her hata `AdaptyErrorCode` türündedir. İşte bir örnek:
:::tip
**Hata ayıklamadan önce ayrıntılı günlükleri etkinleştirin.** Çoğu `AdaptyError`, altta yatan bir StoreKit, Play Billing, ağ veya backend hatasını sarar. Ayrıntılı günlükler açıkken (`await Adapty().setLogLevel(AdaptyLogLevel.verbose)` — bkz. [Günlük Kaydı](sdk-installation-flutter#logging)), sarılmış hata konsola yazdırılır ve genellikle asıl nedeni size söyler.
:::

:::important
Bu çözümler sorununuzu gidermezse, destek ekibiyle iletişime geçmeden önce atacağınız adımlar için [Diğer sorunlar](#other-issues) bölümüne bakın; bu sayede size daha verimli yardımcı olabiliriz.
:::
```javascript showLineNumbers
    try {
      final result = await adapty.makePurchase(product: product);
    } on AdaptyError catch (adaptyError) {
      if (adaptyError.code == AdaptyErrorCode.paymentCancelled) {
        // Cancelled
      }
    } catch (e) {
    }
```

## Diğer sorunlar \{#other-issues\}

Henüz bir çözüm bulamadıysanız, sonraki adımlar şunlar olabilir:
- **SDK'yı en son sürüme yükseltin**: Daha kararlı olan ve bilinen sorunlara yönelik düzeltmeler içeren en son SDK sürümlerine yükseltmenizi her zaman tavsiye ederiz.
- **Destek ekibiyle iletişime geçin veya diğer geliştiricilerden yardım alın** [destek forumunda](https://adapty.featurebase.app/).
- **Destek ekibiyle [support@adapty.io](mailto:support@adapty.io) adresinden veya sohbet üzerinden iletişime geçin**: SDK'yı yükseltmeye hazır değilseniz ya da yükseltme sorunu çözmediyse destek ekibimizle iletişime geçin. [Ayrıntılı günlük kaydını etkinleştirip](sdk-installation-flutter#logging) günlükleri ekiple paylaşırsanız sorununuz daha hızlı çözülecektir. İlgili kod parçacıklarını da ekleyebilirsiniz.
