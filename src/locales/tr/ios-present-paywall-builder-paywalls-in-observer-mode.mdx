Paywall Builder kullanarak bir paywall özelleştirdiyseniz, bunu kullanıcıya göstermek için mobil uygulama kodunuzda ayrıca render etmeniz gerekmez. Böyle bir paywall, hem neyin gösterileceğini hem de nasıl gösterileceğini içerir.

:::warning
Bu bölüm yalnızca [Observer mode](observer-vs-full-mode) için geçerlidir. Observer mode'da çalışmıyorsanız [iOS - Present Paywall Builder paywalls](ios-present-paywalls) sayfasına bakın.
:::

<SDKv4>

<details>
   <summary>Flow sunmaya başlamadan önce (Genişletmek için tıklayın)</summary>
1. Adapty'nin [App Store ile](initial_ios) ilk entegrasyonunu ayarlayın.
   2. Adapty SDK'yı kurun ve yapılandırın. `observerMode` parametresini `true` olarak ayarladığınızdan emin olun. [iOS SDK kurulum kılavuzuna](sdk-installation-ios#activate-adapty-module-of-adapty-sdk) başvurun.
   3. Adapty Kontrol Paneli'nde [ürünler oluşturun](create-product).
   4. [Builder'larda flow'ları veya paywall'ları yapılandırın](create-paywall) ve ürünleri bunlara atayın.
   5. [Placement'lar oluşturun ve flow'larınızı veya paywall'larınızı bunlara atayın](create-placement).
   6. Mobil uygulama kodunuzda [flow'ları ve yapılandırmalarını alın](get-pb-paywalls).
</details>

<TabItem value="swift" label="Swift" default>

1. `AdaptyObserverModeResolver` nesnesini uygulayın. Protokol, SDK v3 ile aynıdır — observer mode, flow ve paywall rendering arasında değişmez:
   ```swift showLineNumbers title="Swift"
   func observerMode(didInitiatePurchase product: AdaptyPaywallProduct,
                     onStartPurchase: @escaping () -> Void,
                     onFinishPurchase: @escaping () -> Void) {
          // use the product object to handle the purchase
          // call onStartPurchase / onFinishPurchase to notify AdaptyUI about the purchase progress
   }

   func observerModeDidInitiateRestorePurchases(onStartRestore: @escaping () -> Void,
                                                onFinishRestore: @escaping () -> Void) {
          // call onStartRestore / onFinishRestore to notify AdaptyUI about the restore progress
   }
   ```

2. Çözümleyicinizi `observerModeResolver:` parametresi olarak geçirerek bir flow yapılandırma nesnesi oluşturun:

   ```swift showLineNumbers title="Swift"
   do {
       let flowConfiguration = try await AdaptyUI.getFlowConfiguration(
           forFlow: flow,
           observerModeResolver: <AdaptyObserverModeResolver>
       )
   } catch {
       // handle the error
   }
   ```

   İstek parametreleri:
| Parametre                | Zorunluluk | Açıklama                                                                                                              |
   | :----------------------- | :------- | :----------------------------------------------------------------------------------------------------------------------- |
   | **forFlow**              | zorunlu | `Adapty.getFlow(placementId:)` aracılığıyla elde edilen bir `AdaptyFlow` nesnesi. Bkz. [Flow ve paywall'ları alma](get-pb-paywalls). |
   | **observerModeResolver** | zorunlu | Yukarıda uyguladığınız `AdaptyObserverModeResolver`.                                                                  |
3. `AdaptyUI.flowController(with:delegate:)` kullanarak flow controller'ı başlatın:

   ```swift showLineNumbers title="Swift"
   import AdaptyUI

   let visualFlow = try AdaptyUI.flowController(
       with: flowConfiguration,
       delegate: <AdaptyFlowControllerDelegate>
   )
   ```

   İstek parametreleri:
| Parametre | Varlık | Açıklama |
   | :------------------------- | :------- | :----------------------------------------------------------------------------------------------------------------------------------------------- |
   | **flowConfiguration**      | gerekli | Flow'un görsel ayrıntılarını içeren bir `AdaptyUI.FlowConfiguration` nesnesi. Bkz. [Flow ve paywallları alma](get-pb-paywalls).                     |
   | **delegate**               | gerekli | Flow olaylarını dinlemek için bir `AdaptyFlowControllerDelegate`. Bkz. [Flow ve paywall olaylarını yönetme](ios-handling-events).                             |
Döndürür:

   | Nesne                | Açıklama                                               |
   | :------------------- | :----------------------------------------------------- |
   | AdaptyFlowController | İstenen flow ekranını temsil eden bir nesne.           |

4. Controller'ı gösterin:

   ```swift showLineNumbers title="Swift"
   present(visualFlow, animated: true)
   ```

:::warning
[Paywallları satın alma işlemleriyle ilişkilendirmeyi](report-transactions-observer-mode) unutmayın. Aksi takdirde Adapty, satın almanın hangi paywall'dan geldiğini belirleyemez.
:::

<TabItem value="swiftui" label="SwiftUI" default>

SwiftUI'da, resolver ile flow yapılandırmasını çekin ve `.flow` modifier'ına geçirin:
```swift showLineNumbers title="SwiftUI"
@State var flowPresented = false
@State var flowConfiguration: AdaptyUI.FlowConfiguration?

var body: some View {
    Text("Hello, AdaptyUI!")
        .flow(
            isPresented: $flowPresented,
            flowConfiguration: flowConfiguration,
            didPerformAction: { action in
                switch action {
                    case .close:
                        flowPresented = false
                    default:
                        break
                }
            },
            didFinishPurchase: { product, purchaseResult in /* dismiss, or do nothing to let the flow continue */ },
            didFailPurchase: { product, error in /* handle the error */ },
            didFinishRestore: { profile in /* check access level and dismiss */ },
            didFailRestore: { error in /* handle the error */ },
            didReceiveError: { error in flowPresented = false }
        )
        .task {
            flowConfiguration = try? await AdaptyUI.getFlowConfiguration(
                forFlow: flow,
                observerModeResolver: <AdaptyObserverModeResolver>
            )
        }
}
```

`getFlowConfiguration` üzerindeki `observerModeResolver:` parametresi, render edilen flow'un özel satın alma mantığınıza uymasını sağlar — modifier'ın kendisi de tam mod ile aynı callback'leri kullanır.

:::warning
[Satın alma işlemlerini paywall'larla ilişkilendirmeyi](report-transactions-observer-mode) unutmayın. Aksi takdirde Adapty, satın almanın kaynak paywall'ını belirleyemez.
:::

<TabItem value="swift" label="Swift" default> 

1. `AdaptyObserverModeResolver` nesnesini uygulayın:
   ```swift showLineNumbers title="Swift"
   func observerMode(didInitiatePurchase product: AdaptyPaywallProduct,
                     onStartPurchase: @escaping () -> Void,
                     onFinishPurchase: @escaping () -> Void) {
          // use the product object to handle the purchase
          // use the onStartPurchase and onFinishPurchase callbacks to notify AdaptyUI about the process of the purchase
   }

   func observerModeDidInitiateRestorePurchases(onStartRestore: @escaping () -> Void,
                                                onFinishRestore: @escaping () -> Void) {
          // use the onStartRestore and onFinishRestore callbacks to notify AdaptyUI about the process of the restore
   }
   ```

`observerMode(didInitiatePurchase:onStartPurchase:onFinishPurchase:)` olayı, kullanıcının bir satın alma işlemi başlattığını size bildirir. Bu geri çağırıma yanıt olarak kendi özel satın alma akışınızı tetikleyebilirsiniz.

`observerModeDidInitiateRestorePurchases(onStartRestore:onFinishRestore:)` olayı, kullanıcının bir geri yükleme işlemi başlattığını size bildirir. Bu geri çağırıma yanıt olarak kendi özel geri yükleme akışınızı tetikleyebilirsiniz.
Ayrıca, satın alma veya geri yükleme sürecini AdaptyUI'ya bildirmek için aşağıdaki callback'leri çağırmayı unutmayın. Bu, loader gösterimi gibi paywall'un düzgün çalışması için gereklidir:
| Callback           | Açıklama                                                                                    |
   | :----------------- | :------------------------------------------------------------------------------------------- |
   | onStartPurchase()  | Satın alma işleminin başladığını AdaptyUI'ya bildirmek için bu callback çağrılmalıdır.       |
   | onFinishPurchase() | Satın alma işleminin tamamlandığını AdaptyUI'ya bildirmek için bu callback çağrılmalıdır.    |
   | onStartRestore()   | Geri yükleme işleminin başladığını AdaptyUI'ya bildirmek için bu callback çağrılmalıdır.     |
   | onFinishRestore()  | Geri yükleme işleminin tamamlandığını AdaptyUI'ya bildirmek için bu callback çağrılmalıdır. |
2. Bir paywall yapılandırma nesnesi oluşturun:

   ```swift showLineNumbers title="Swift"
   do {
       let paywallConfiguration = try AdaptyUI.getPaywallConfiguration(
        forPaywall: <paywall object>, 
        observerModeResolver: <AdaptyObserverModeResolver>
        )
   } catch {
       // handle the error
   }
   ```

   İstek parametreleri:
| Parametre                | Varlık   | Açıklama                                                                                                                                                                                                                                                                                                              |
   | :----------------------- | :------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   | **Paywall**              | zorunlu  | İstenen paywall için bir controller elde etmek amacıyla kullanılan `AdaptyPaywall` nesnesi.                                                                                                                                                                                                                           |
   | **ObserverModeResolver** | zorunlu  | Önceki adımda uyguladığınız `AdaptyObserverModeResolver` nesnesi.                                                                                                                                                                                                                                                     |
3. Görüntülemek istediğiniz görsel paywallı başlatmak için `.paywallController(for:products:viewConfiguration:delegate:)` metodunu kullanın:

   ```swift showLineNumbers title="Swift"
   import AdaptyUI
   
   let visualPaywall = AdaptyUI.paywallController(
       with: <paywall configuration object>,
       delegate: <AdaptyPaywallControllerDelegate>
   )
   ```

İstek parametreleri:
| Parametre                | Varlık | Açıklama                                                                                                                                                                                                                                                                                                           |
| :----------------------- | :------- | :-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Paywall Configuration**              | zorunlu | Paywallın görsel detaylarını içeren bir `AdaptyUI.PaywallConfiguration` nesnesi. `AdaptyUI.getPaywallConfiguration(forPaywall:locale:)` metodunu kullanın. Daha fazla bilgi için [Paywall Builder paywalllarını ve yapılandırmalarını getirme](get-pb-paywalls) konusuna bakın.                                                                                                                                                                                                                                             |
| **Delegate**             | zorunlu | Paywall olaylarını dinlemek için bir `AdaptyPaywallControllerDelegate`. Daha fazla bilgi için [Paywall olaylarını yönetme](ios-handling-events) konusuna bakın.                                                                                                                                                                                 |
Döndürür:

| Nesne                   | Açıklama                                             |
| :---------------------- | :--------------------------------------------------- |
| AdaptyPaywallController | İstenen paywall ekranını temsil eden bir nesne        |

Nesne başarıyla oluşturulduktan sonra şu şekilde görüntüleyebilirsiniz:

```swift showLineNumbers title="Swift"
present(visualPaywall, animated: true)
```
:::warning
[Satın alma işlemlerine paywallları bağlamayı](report-transactions-observer-mode) unutmayın. Aksi takdirde Adapty, satın almanın hangi paywalldan geldiğini belirleyemez.
