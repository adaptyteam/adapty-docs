### AdaptySubscriptionPeriod

`AdaptySubscriptionPeriod`, bir abonelik dönemini tanımlar ve iki özellik içerir:

| Parametre | Tip | Açıklama |
|-----------|-----|----------|
| `unit` | `AdaptyPeriodUnit` | Dönem birimini belirtir: `day`, `week`, `month` veya `year`. |
| `numberOfUnits` | `Int` | Belirtilen birimdeki dönem sayısını belirtir. |

| Name          | Type             | Description                                                                                                                      |
| :------------ | :--------------- | :------------------------------------------------------------------------------------------------------------------------------- |
| numberOfUnits | number           | Periyot birimlerinin sayısı                                                                                                      |
| unit          | [AdaptyPeriodUnit](#adaptyperiodunit) | Abonelik periyodunun belirtildiği zaman birimi. Olası değerler: `day`, `week`, `month`, `year` |

### AdaptyProfile.AccessLevel

[Kullanıcının access level'ı](https://swift.adapty.io/documentation/adapty/adaptyprofile/accesslevel) hakkında bilgi.

| Ad | Tür | Açıklama |
|----|----|-----------|
| id | string | Adapty Kontrol Paneli'nde sizin tarafınızdan yapılandırılan access level'ın benzersiz tanımlayıcısı |
| isActive | boolean | Access level'ın aktif olup olmadığı. Kullanıcının premium özelliklere erişip erişemeyeceğini belirlemek için genellikle yalnızca bu özelliği kontrol etmeniz yeterlidir |
| vendorProductId | string | Bu access level'ı kilit açan App Store Connect'teki ürünün tanımlayıcısı |
| store | string | Bu access level'ı kilit açan satın almanın yapıldığı mağaza. Olası değerler: app_store, play_store, adapty |
| activatedAt | string | Access level'ın etkinleştirildiği zaman (ISO 8601 formatı) |
| renewedAt | string (isteğe bağlı) | Access level'ın yenilendiği zaman (ISO 8601 formatı) |
| expiresAt | string (isteğe bağlı) | Access level'ın sona ereceği zaman (ISO 8601 formatı; geçmişte bir tarih olabilir ve süresiz erişim için null olabilir) |
| isLifetime | boolean | Access level'ın ömür boyu (son kullanma tarihi olmaksızın) aktif olup olmadığı. true olarak ayarlanmışsa expires_at değerini kontrol etmenize gerek yoktur; sadece isActive'i kontrol edebilirsiniz |
| activeIntroductoryOfferType | string (isteğe bağlı) | Aktif başlangıç teklifinin türü. Olası değerler: free_trial, pay_as_you_go, pay_up_front. Değer null değilse, teklif mevcut abonelik dönemi boyunca uygulanmış demektir |
| activePromotionalOfferType | string (isteğe bağlı) | Aktif promosyon teklifinin türü. Olası değerler: free_trial, pay_as_you_go, pay_up_front. Değer null değilse, teklif mevcut abonelik dönemi boyunca uygulanmış demektir |
| activePromotionalOfferId | string (isteğe bağlı) | Aktif promosyon teklifinin tanımlayıcısı |
| offerId | string (isteğe bağlı) | Teklif tanımlayıcısı |
| willRenew | boolean | Otomatik yenilenen aboneliğin yenilenecek şekilde ayarlanıp ayarlanmadığı |
| isInGracePeriod | boolean | Otomatik yenilenen aboneliğin ek sürede olup olmadığı |
| unsubscribedAt | string (isteğe bağlı) | Otomatik yenilenen aboneliğin iptal edildiği zaman. Abonelik hâlâ aktif olabilir; bu yalnızca otomatik yenilemenin kapatıldığı anlamına gelir. Kullanıcı aboneliği yeniden etkinleştirirse null olarak ayarlanacaktır (ISO 8601 formatı) |
| billingIssueDetectedAt | string (isteğe bağlı) | Ödeme sorununun tespit edildiği zaman. Abonelik hâlâ aktif olabilir. Ödeme gerçekleştirilirse null olarak ayarlanacaktır (ISO 8601 formatı) |
| startsAt | string (isteğe bağlı) | Bu access level'ın başladığı zaman (ISO 8601 formatı; gelecekte bir tarih olabilir) |
| cancellationReason | string (isteğe bağlı) | Aboneliğin iptal edilme nedeni. Olası değerler: voluntarily_cancelled, billing_error, refund, price_increase, product_was_not_available, unknown |
| isRefund | boolean | Satın almanın iade edilip edilmediği |

### Numaralandırmalar

#### AdaptyPeriodUnit
- `day` - Gün dönemi
- `week` - Hafta dönemi
- `month` - Ay dönemi
- `year` - Yıl dönemi

#### AdaptySubscriptionOfferPaymentMode
- `free_trial` - Ücretsiz deneme
- `pay_as_you_go` - Kullandıkça öde
- `pay_up_front` - Peşin öde

#### AdaptyRenewalType
- `prepaid` - Ön ödemeli abonelik
- `autorenewable` - Otomatik yenilenen abonelik

#### AdaptyProfile.Gender
- `f` - Kadın
- `m` - Erkek
- `o` - Diğer

#### AdaptyAndroidSubscriptionUpdateReplacementMode
- `immediate_with_time_proration` - Zaman eşit dağılımıyla anında
- `immediate_and_charge_prorated_price` - Anında ve orantılı fiyat tahsil et
- `immediate_without_proration` - Eşit dağılım olmadan anında
- `deferred` - Ertelenmiş
- `immediate_and_charge_full_price` - Anında ve tam fiyat tahsil et
