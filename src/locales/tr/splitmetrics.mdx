[SplitMetrics Acquire](https://splitmetrics.com/acquire/) entegrasyonu sayesinde Apple Search Ads reklamlarınızın aboneliklerden ne kadar gelir getirdiğini tam olarak görebilirsiniz. Ayrıca kullanıcılarınızı aylarca takip ederek reklamlarınızın zaman içinde ne kadar kazandırdığını öğrenebilirsiniz.
Ayrıca Adapty, Apple Search Ads attribution verilerine dayanarak özel panolar ve otomasyon oluşturabilmeniz için SplitMetrics Acquire'a [abonelik etkinlikleri](events) gönderir.
Attribution verisi Adapty'ye eklenmez; zira ASA'dan doğrudan ihtiyacımız olan her şeye zaten sahibiz.
## SplitMetrics Acquire entegrasyonu nasıl kurulur \{#how-to-set-up-splitmetrics-acquire-integration\}

Adapty'nin sağladığı varsayılan etkinlik adlarını kullanmanızı öneririz. Ancak etkinlik adlarını ihtiyaçlarınıza göre değiştirebilirsiniz. Adapty, sunucudan sunucuya entegrasyon aracılığıyla SplitMetrics Acquire'a abonelik etkinlikleri gönderir; böylece tüm abonelik etkinliklerini SplitMetrics kontrol panelinizde görüntüleyebilirsiniz.

## SDK yapılandırması \{#sdk-configuration\}

SDK tarafında herhangi bir şey yapılandırmanıza gerek yok; ancak daha iyi doğruluk için Adapty'ye `customerUserId` göndermenizi öneririz.

:::warning
[Apple Search Ads](apple-search-ads)'i Adapty'de yapılandırdığınızdan ve [kimlik bilgilerini yüklediğinizden](https://app.adapty.io/settings/apple-search-ads) emin olun; bunlar olmadan SplitMetrics Acquire çalışmaz.
:::

## Sorun giderme \{#troubleshooting\}

Doğru kuruluma rağmen SplitMetrics Acquire entegrasyonu çalışmıyorsa:

- [App Settings -> Apple Search Ads sekmesinde](https://app.adapty.io/settings/apple-search-ads) **Receive Apple Search Ads attribution in Adapty** geçişinin etkin olduğundan, Adapty'de [Apple Search Ads](apple-search-ads)'in yapılandırıldığından ve [kimlik bilgilerinin yüklendiğinden](https://app.adapty.io/settings/apple-search-ads) emin olun; bunlar olmadan SplitMetrics çalışmaz.
- Profillerin organik olmayan ASA attribution'ına sahip olduğundan emin olun. Yalnızca ayrıntılı, organik olmayan ASA attribution'ına sahip profiller etkinliklerini Adapty'ye iletir.

## SplitMetrics Acquire etkinlik yapısı \{#splitmetrics-acquire-event-structure\}

Adapty, etkinlikleri sorgu parametreleri kullanarak GET isteğiyle SplitMetrics Acquire'a gönderir. Her etkinlik şu şekilde yapılandırılmıştır:

```json
{
  "source": "Apple Search Ads",
  "app_id": "123456789",
  "name": "subscription_renewed",
  "type": "subscription_renewed",
  "revenue": 9.99,
  "currency": "USD",
  "tap_time": "2024-03-01 12:00:00",
  "open_time": "2024-03-01 12:05:00",
  "event_time": "2024-03-02 12:00:00",
  "adaccount_id": "123456",
  "campaign_id": "123456789",
  "adgroup_id": "123456789",
  "keyword_id": "123456789",
  "creative_set_id": "123456789",
  "Ad_id": "123456789",
  "country_or_region": "US",
  "conversion_type": "Download",
  "user_id": "user_12345",
  "att_status": "3",
  "device_type": "iphone",
  "app_version": "1.2.3",
  "sdk_version": "2.10.0",
  "ios_version": "17.2",
  "event_value": "{\"vendor_product_id\":\"yearly.premium.6999\",\"original_transaction_id\":\"GPA.3383...\"}",
  "event_id": "123e4567-e89b-12d3-a456-426614174000"
}
```

Parametreler:

| Parametre           | Tür    | Açıklama                                                                                                              |
|:--------------------|:-------|:----------------------------------------------------------------------------------------------------------------------|
| `source`            | String | Her zaman "Apple Search Ads".                                                                                         |
| `app_id`            | String | Apple App ID.                                                                                                         |
| `name`              | String | Etkinlik adı (Adapty etkinliğinden eşlenir).                                                                          |
| `type`              | String | Etkinlik türü (`name` ile aynı).                                                                                      |
| `revenue`           | Float  | Gelir miktarı.                                                                                                        |
| `currency`          | String | Para birimi kodu.                                                                                                     |
| `tap_time`          | String | Reklama tıklanma tarihi ve saati.                                                                                     |
| `open_time`         | String | Uygulamanın açıldığı (kurulduğu) tarih ve saat.                                                                      |
| `event_time`        | String | Etkinliğin gerçekleştiği tarih ve saat.                                                                               |
| `adaccount_id`      | String | ASA Organizasyon ID'si.                                                                                               |
| `campaign_id`       | String | ASA Kampanya ID'si.                                                                                                   |
| `adgroup_id`        | String | ASA Reklam Grubu ID'si.                                                                                               |
| `keyword_id`        | String | ASA Anahtar Kelime ID'si.                                                                                             |
| `creative_set_id`   | String | ASA Kreatif Set ID'si.                                                                                                |
| `Ad_id`             | String | ASA Reklam ID'si.                                                                                                     |
| `country_or_region` | String | Mağaza ülkesi veya bölgesi.                                                                                           |
| `conversion_type`   | String | Dönüşüm türü (örn. "Download").                                                                                       |
| `user_id`           | String | Müşteri Kullanıcı ID'si veya Adapty Profil ID'si.                                                                     |
| `att_status`        | String | İzleme kullanım durumu (0-3).                                                                                         |
| `device_type`       | String | Cihaz türü (örn. "iphone", "ipad").                                                                                   |
| `app_version`       | String | Uygulama sürümü.                                                                                                      |
| `sdk_version`       | String | Adapty SDK sürümü.                                                                                                    |
| `ios_version`       | String | iOS sürümü.                                                                                                           |
| `event_value`       | String | Mevcut tüm [etkinlik ayrıntılarını](webhook-event-types-and-fields#for-most-event-types) içeren JSON dizesi.         |
| `event_id`          | String | Benzersiz etkinlik ID'si (UUID).                                                                                      |
