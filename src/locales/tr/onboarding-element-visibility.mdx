Belirli elementlere koşullu görünürlük de ekleyebilirsiniz. Koşullu elementler yalnızca belirli quiz sorularına belirli yanıtlar veren kullanıcılara görünür.

Örneğin, "Deneyim seviyeniz nedir?" sorusuna "Başlangıç" yanıtını veren kullanıcılar bir sonraki adımda ek ayrıntılar görecektir.

Bir elementi koşullu hale getirmek için:
1. Sağ taraftaki **Element** sekmesine gidin.
2. **Visible** bölümünde **Conditional** seçeneğini seçin.
3. Aşağıdakileri seçerek koşulu ayarlayın:
- quiz ID
- operatör
- quiz yanıtı
4. (İsteğe bağlı) Birden fazla koşul eklemek için **Advanced condition** seçeneğine tıklayın.

Örneğin, quiz ID olarak 'goal', operatör olarak 'Has' ve quiz yanıtı olarak 'Education' seçtiyseniz, element yalnızca o quiz için verilen yanıtlar arasında 'Education' seçeneğini içeren kullanıcılara görünür olacaktır.
