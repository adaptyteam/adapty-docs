Adapty SDK'yı React Native uygulamanıza entegre ettiyseniz, her şeyin doğru kurulduğunu ve satın almaların hem iOS hem de Android platformlarında beklendiği gibi çalıştığını test etmek isteyeceksinizdir. Bu süreç; SDK entegrasyonunun ve Apple'ın sandbox ortamı ile Google Play'in test ortamında gerçek satın alma akışının test edilmesini kapsar.

## Uygulamanızı test edin \{#test-your-app\}

Uygulama içi satın almalarınızı kapsamlı şekilde test etmek için platforma özel test kılavuzlarımıza bakın: [iOS test kılavuzu](test-purchases-in-sandbox) ve [Android test kılavuzu](testing-on-android).

## Yayına almaya hazırlık \{#prepare-for-release\}

Uygulamanızı mağazaya göndermeden önce aşağıdakileri doğrulamak için [Yayın kontrol listesi](release-checklist)'ni takip edin:

- Mağaza bağlantısı ve sunucu bildirimleri yapılandırılmış olmalı
- Satın almalar tamamlanıp Adapty'ye raporlanmalı
- Erişim doğru şekilde açılıp geri yüklenebilmeli
- Gizlilik ve inceleme gereksinimleri karşılanmış olmalı
