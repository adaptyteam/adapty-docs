<SDKv4>

Adapty Flow Builder veya Paywall Builder kullanarak flow'lar veya paywall'lar oluşturuyorsanız, butonları doğru şekilde ayarlamak kritik öneme sahiptir:
1. [Builder'da bir düğme ekleyin](paywall-buttons) ve ona önceden tanımlanmış bir eylem atayın ya da özel bir eylem kimliği oluşturun.
2. Atadığınız her eylemi işlemek için uygulamanıza kod yazın.

Bu kılavuz, özel ve önceden tanımlanmış eylemleri kodunuzda nasıl yöneteceğinizi gösterir.

:::warning
**Yalnızca satın almalar, yenileme işlemleri, flow/paywall kapatma ve URL açma otomatik olarak işlenir.** Diğer tüm düğme eylemleri, uygulama kodunda uygun bir yanıt uygulaması gerektirir.
:::
## Flow ve paywallları kapatma \{#close-flows-and-paywalls\}

Flow veya paywallınıza bir kapatma butonu eklemek için:

1. Builder'da bir buton ekleyin ve **Close** eylemini atayın.
2. Uygulama kodunuzda `close` eylemi için bir handler uygulayın.

:::info
Android SDK'da `close` eylemi, varsayılan olarak flow veya paywallı kapatmayı tetikler. Ancak gerekirse bu davranışı kodunuzda değiştirebilirsiniz. Örneğin, bir flow'un kapatılması başka bir flow'un açılmasını tetikleyebilir.
:::
```kotlin
override fun onActionPerformed(action: AdaptyUI.Action, context: Context) {
    when (action) {
        AdaptyUI.Action.Close -> (context as? Activity)?.onBackPressed() // default behavior
    }
}
```

## Flow ve paywalllardan URL açma \{#open-urls-from-flows-and-paywalls\}

:::tip
Bir grup bağlantı eklemek istiyorsanız (örneğin, kullanım koşulları ve satın alma geri yükleme), builder'da bir **Link** öğesi ekleyin ve bunu **Open URL** eylemiyle düğmelerle aynı şekilde yönetin.
:::

Flow veya paywallınızdan bir bağlantı açan bir düğme eklemek için (örneğin, **Terms of use** veya **Privacy policy**):

1. Builder'da bir düğme ekleyin, **Open URL** eylemini atayın ve açmak istediğiniz URL'yi girin.
2. Uygulama kodunuzda, alınan URL'yi tarayıcıda açan `openUrl` eylemi için bir işleyici uygulayın.
:::info
Android SDK'da `openUrl` eylemi varsayılan olarak URL'yi açar. Ancak gerekirse bu davranışı kodunuzda değiştirebilirsiniz.
:::

```kotlin
override fun onActionPerformed(action: AdaptyUI.Action, context: Context) {
   when (action) {    
       is AdaptyUI.Action.OpenUrl -> {
           val intent = Intent(Intent.ACTION_VIEW, Uri.parse(action.url)) // default behavior
           context.startActivity(intent)
       }
   }
}
```
## Özel eylemleri yönetme \{#handle-custom-actions\}

Başka eylemleri yöneten bir buton eklemek için:

1. Builder'da bir buton ekleyin, ona **Custom** eylemini atayın ve bir ID verin.
2. Uygulama kodunuzda, oluşturduğunuz eylem ID'si için bir işleyici uygulayın.

Örneğin, başka bir abonelik teklifi setiiniz veya tek seferlik satın almalarınız varsa, başka bir flow ya da paywall görüntüleyecek bir buton ekleyebilirsiniz:
```kotlin
override fun onActionPerformed(action: AdaptyUI.Action, context: Context) {
   when (action) {
       is AdaptyUI.Action.Custom -> {
           if (action.customId == "openNewPaywall") {
               // Display another flow or paywall
           }
       }
   }
}
```

:::warning

## Paywallları kapatma \{#close-paywalls\}

Paywallınızı kapatacak bir düğme eklemek için:

1. Paywall builder'da bir düğme ekleyin ve **Close** eylemini atayın.
2. Uygulama kodunuzda, paywallı kapatan `close` eylemi için bir işleyici uygulayın.

:::info
Android SDK'da `close` eylemi, paywall'ı varsayılan olarak kapatır. Ancak gerekirse bu davranışı kodunuzda geçersiz kılabilirsiniz. Örneğin, bir paywallı kapatmak başka bir paywallın açılmasını tetikleyebilir.
:::
```kotlin
override fun onActionPerformed(action: AdaptyUI.Action, context: Context) {
    when (action) {
        AdaptyUI.Action.Close -> (context as? Activity)?.onBackPressed() // default behavior
    }
}
```

## Uygulamaya giriş yapma \{#log-into-the-app\}

Kullanıcıların uygulamanıza giriş yapmasını sağlayan bir buton eklemek için:

1. Paywall builder'da bir buton ekleyin ve **Login** eylemini atayın.
2. Uygulama kodunuzda, kullanıcınızı tanımlayan `login` eylemi için bir işleyici uygulayın.

```kotlin
override fun onActionPerformed(action: AdaptyUI.Action, context: Context) {
   when (action) {
       AdaptyUI.Action.Login -> {
           val intent = Intent(context, LoginActivity::class.java)
           context.startActivity(intent)
       }
   }
}
```
