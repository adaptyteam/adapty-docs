<SDKv4>

iOS uygulamanız çocuklara yönelikse [Apple](https://developer.apple.com/kids/) politikalarına uymanız gerekir. Adapty SDK kullanıyorsanız, birkaç basit adımla SDK'yı bu politikalara uygun şekilde yapılandırabilir ve uygulama mağazası incelemelerinden geçebilirsiniz.
## Neler gerekli? \{#whats-required\}

Aşağıdaki verilerin toplanmasını devre dışı bırakmak için Adapty SDK'yı yapılandırmanız gerekir:

- [IDFA (Reklamverenler için Tanımlayıcı)](https://en.wikipedia.org/wiki/Identifier_for_Advertisers)
- [IP adresi](https://www.ftc.gov/system/files/ftc_gov/pdf/p235402_coppa_application.pdf)
Ayrıca müşteri kullanıcı kimliğini dikkatli kullanmanızı öneririz. `<FirstName.LastName>` formatındaki kullanıcı kimliği, e-posta kullanımı gibi kişisel veri toplama olarak değerlendirilecektir. Çocuk Modu için en iyi uygulama, uyumluluğu sağlamak amacıyla rastgele oluşturulmuş veya anonimleştirilmiş tanımlayıcılar (ör. hash'lenmiş kimlikler veya cihaz tarafından oluşturulan UUID'ler) kullanmaktır.
## Kids Modunu Etkinleştirme \{#enabling-kids-mode\}
### Adapty Kontrol Paneli'ndeki Güncellemeler \{#updates-in-the-adapty-dashboard\}

Adapty Kontrol Paneli'nde IP adresi toplamasını devre dışı bırakmanız gerekiyor. Bunun için [App settings](https://app.adapty.io/settings/general) sayfasına gidin ve **Collect users' IP address** altındaki **Disable IP address collection** seçeneğine tıklayın.
### Mobil uygulama kodunuzdaki güncellemeler \{#updates-in-your-mobile-app-code\}

SDK 4.0 ile birlikte Kids Mode, `KidsMode` adlı bir Swift package trait'idir. Bu trait'i etkinleştirmek, IDFA ve AdSupport'u tüm SDK genelinde derleme aşamasında devre dışı bırakır — normal **Adapty** ve **AdaptyUI** modüllerini ve normal `import Adapty` / `import AdaptyUI` ifadelerini kullanmaya devam edersiniz.

:::note
`KidsMode` trait'i, SDK `4.0.0-beta.2` sürümünden itibaren kullanılabilir. SDK 4.0 ile birlikte SDK yalnızca Swift Package Manager aracılığıyla kurulmaktadır — CocoaPods artık desteklenmemektedir.
:::
<Tabs>
<TabItem value="xcode" label="Xcode" default>
1. [Adapty SDK'yı](sdk-installation-ios) her zamanki gibi yükleyin; standart **Adapty** ve **AdaptyUI** modüllerini seçin.
2. Xcode 26.4 veya sonraki sürümlerde proje ayarlarınızı açın, **Package Dependencies** görünümüne gidin ve AdaptySDK-iOS bağımlılığı için **KidsMode** trait'ini etkinleştirin.
:::note
Xcode 26.4'ten önceki sürümler, bir Xcode projesinde kullanıcı arayüzü üzerinden trait'leri etkinleştiremez. Bu durumda, `KidsMode` trait'i etkinleştirilmiş şekilde Adapty'ye bağımlı küçük bir yerel Swift paketi oluşturun (**Package.swift** sekmesine bakın) ve uygulama hedefinizi bu pakete bağımlı yapın.
:::
</TabItem>
<TabItem value="spm" label="Package.swift">
Adapty'yi `Package.swift` dosyasında bağımlılık olarak ekliyorsanız, trait'i paket bildiriminde etkinleştirin. Trait'ler için `swift-tools-version` 6.1 veya üstü gereklidir.
```swift showLineNumbers title="Package.swift"
.package(
    url: "https://github.com/adaptyteam/AdaptySDK-iOS.git",
    exact: "4.0.0-beta.2",
    traits: ["KidsMode"]
)
```
</TabItem>

<TabItem value="cocoapods" label="CocoaPods">
1. Podfile dosyanızı güncelleyin:

   - Eğer `post_install` bölümünüz **yoksa**, aşağıdaki kod bloğunun tamamını ekleyin.
   - Eğer `post_install` bölümünüz **varsa**, vurgulanan satırları mevcut bölümle birleştirin.
   ```ruby showLineNumbers title="Podfile"
   def adapty_enable_kids_mode(installer)
     installer.pods_project.targets.each do |target|
       next unless target.name == 'Adapty'
       target.build_configurations.each do |config|
         flags = config.build_settings['OTHER_SWIFT_FLAGS'] || '$(inherited)'
         flags = flags.join(' ') if flags.is_a?(Array)
         config.build_settings['OTHER_SWIFT_FLAGS'] = "#{flags} -DADAPTY_KIDS_MODE"
       end
       target.frameworks_build_phase.files.dup.each do |bf|
         target.frameworks_build_phase.remove_build_file(bf) if bf.display_name.to_s.include?('AdSupport')
       end
     end
     installer.pods_project.save
     Dir.glob(File.join(installer.sandbox.root, 'Target Support Files', '**', '*.xcconfig')).each do |xc|
       File.write(xc, File.read(xc).gsub(/\s*-framework\s+"?AdSupport"?/, ''))
     end
   end

   post_install do |installer|
     # ... keep your existing post_install body (Flutter adds one automatically) ...

     adapty_enable_kids_mode(installer)   # <-- enable Adapty Kids Mode
   end
   ```

2. Değişiklikleri uygulamak için aşağıdaki komutu çalıştırın:

   ```sh showLineNumbers title="Shell"
   pod install
   ```
