App Store sunucu bildirimlerini ayarlamak, veri doğruluğunu sağlamak açısından kritik öneme sahiptir; bu sayede iade işlemleri ve diğer etkinlikler dahil olmak üzere App Store'dan anlık güncellemeler alabilirsiniz.
:::important
App Store Server Notifications V2 desteği için Adapty iOS SDK 2.10.0 veya üzeri gereklidir.
:::

1. Adapty Kontrol Paneli'nde **URL for App Store server notification** bağlantısını kopyalayın.

<Zoom>
  <img src="/assets/shared/img/2901185-app_server_notifications.webp"
  style={{
    border: '1px solid #727272', /* border width and color */
    width: '700px', /* image width */
    display: 'block', /* for alignment */
    margin: '0 auto' /* center alignment */
  }}
/>
</Zoom>
2. [App Store Connect](https://appstoreconnect.apple.com/apps) sayfasını açın. Uygulamanızı seçin ve **General** → **App Information** bölümüne, ardından **App Store Server Notifications** alt bölümüne gidin.
3. Kopyaladığınız **URL for App Store server notification** değerini **Production Server URL** ve **Sandbox Server URL** alanlarına yapıştırın.

## Ham etkinlikleri iletme \{#raw-events-forwarding\}

Zaman zaman Apple'dan gelen ham S2S etkinliklerini doğrudan almak isteyebilirsiniz. Adapty kullanırken bunları almaya devam etmek için endpoint'inizi **URL for forwarding raw Apple events** alanına ekleyin; Apple'dan gelen ham etkinlikleri olduğu gibi size ileteceğiz.
