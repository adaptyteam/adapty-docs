[Resmi AppsFlyer What's New](https://support.appsflyer.com/hc/en-us/articles/20509378973457-Bulletin-Upgrading-the-AppsFlyer-S2S-API) sayfasına göre AppsFlyer, API kullanımını daha güvenli hale getirmek ve dolandırıcılığı azaltmak amacıyla uygulama içi etkinlikler için sunucudan sunucuya (S2S) API'sini yükseltti. Mevcut endpoint ileride kullanımdan kaldırılacak; bu nedenle geçişi şimdiden planlamaya başlamanızı öneririz.

Adapty, AppsFlyer S2S API 3'ü destekler ve API 2'den sorunsuz bir geçiş sunar. Bu geçişin tek yönlü olduğunu unutmayın; değişikliği yaptıktan sonra API 2'ye geri dönemezsiniz.

AppsFlyer S2S API 2'den 3'e geçmek için:

1. [AppsFlyer sitesini](https://www.appsflyer.com/home) açın ve giriş yapın.

2. Kontrol panelinin sol üst köşesinde **Hesap adınıza** tıklayın ve **Security Center**'ı seçin.

3. **Manage your account security** penceresinde **Manage your AppsFlyer API and S2S tokens** düğmesine tıklayın.

4. S2S tokenınız yoksa **New token** düğmesine tıklayın. Zaten varsa 8. adıma geçin.

<Zoom>
  <img src="/assets/shared/img/7934920-appsflyer_new_token.webp"
  style={{
    border: '1px solid #727272', /* border width and color */
    width: '700px', /* image width */
    display: 'block', /* for alignment */
    margin: '0 auto' /* center alignment */
  }}
/>
</Zoom>

5. **New token** penceresinde token için bir ad girin. Bu ad yalnızca sizin referansınız içindir.

6. **Choose type** listesinden **S2S** seçeneğini seçin.

7. Yeni tokeni kaydetmek için **Create new token** düğmesine tıklamayı unutmayın.

8. **Tokens** penceresinde S2S tokenı kopyalayın.

<Zoom>
  <img src="/assets/shared/img/d014c25-appsflyer_tokens.webp"
  style={{
    border: '1px solid #727272', /* border width and color */
    width: '700px', /* image width */

    margin: '0 auto' /* center alignment */
  }}
/>
</Zoom>

9. Adapty Kontrol Paneli'nde [**Integrations** -> **AppsFlyer**](https://app.adapty.io/integrations/appsflyer) bölümünü açın.

10. **AppsFlyer S2S API** alanında **API 3**'ü seçin.

<Zoom>
  <img src="/assets/shared/img/c0b3e72-appsflyer_switch_API.webp"
  style={{
    border: '1px solid #727272', /* border width and color */

    display: 'block', /* for alignment */
    margin: '0 auto' /* center alignment */

/>

11. Kopyaladığınız S2S anahtarını **Dev key for iOS** ve **Dev key for Android** alanlarına yapıştırın.

12. Geçişi onaylamak için **Save** düğmesine tıklayın.

Bu işlemin ardından entegrasyonunuz anında AppsFlyer S2S API 3'e geçer ve yeni etkinlikleriniz şu URL'ye gönderilmeye başlar: `https://api3.appsflyer.com/inappevent`.
