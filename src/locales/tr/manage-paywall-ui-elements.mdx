Bir şablon seçtikten sonra, şablonun öğeleri sol panelde görüntülenir. Paywall'unuzdaki öğeleri düzenlemek için bu paneli kullanın. Öğeler, paywall'da sol paneldeki sırayla görünür.

:::warning

Bu bölüm, iOS, Android ve React Native SDK'larının 3.0 veya daha yüksek sürümleriyle, Flutter ve Unity SDK'larının ise 3.3.0 veya daha yüksek sürümleriyle çalışan yeni Paywall Builder'ı açıklamaktadır. Adapty SDK v2.x veya öncesiyle uyumlu eski Paywall Builder hakkında bilgi için bkz. [Eski Paywall Builder](adapty-paywall-builder-legacy).

:::

## Öğe ekleme \{#add-element\}

Düzenin üstüne bir öğe eklemek için:

1. Sol paneldeki **Add Element** düğmesine tıklayın.
2. Eklemek istediğiniz öğeyi seçin. Yeni öğe, **Footer**'ın üstünde listede görünecektir.

## Paywall öğesini yeniden adlandırma \{#rename-paywall-element\}

Bir öğeyi yeniden adlandırmak için:

1. Sol paneldeki öğeye tıklayarak ayrıntılarını açın.

<Zoom>
  <img src="/assets/shared/img/a5b26ba-edit_element.webp"
  style={{
    border: '1px solid #727272', /* border width and color */
    width: '700px', /* image width */
    display: 'block', /* for alignment */
    margin: '0 auto' /* center alignment */
  }}
/>
</Zoom>

2. Sağ paneldeki üç nokta düğmesine tıklayın ve **Edit** seçeneğini belirleyin.

3. Öğenin yeni adını yazın ve **Enter** tuşuna basın.

## Öğeyi gizleme / gösterme \{#hide--show-element\}

Bir öğeyi oluşturup yapılandırmış olsanız bile, onu paywall'dan geçici olarak gizleyebilirsiniz. Bu, yaptığınız tüm yapılandırmaları kaybetmeden öğeyi daha sonra eklemek istediğinizde oldukça kullanışlıdır. Bir öğe gizlendikten sonra, paywall öğe hiç eklenmemiş gibi görünür; tüm hizalamalar ve boşluklar yeniden hesaplanıp çizilir.

Bir öğeyi gizlemek için:

1. Sol paneldeki öğeye tıklayarak ayrıntılarını açın.

2. Sağ paneldeki üç nokta düğmesine tıklayın ve **Hide** seçeneğini belirleyin.

<Zoom>
  <img src="/assets/shared/img/pb-hide-element.webp"
  style={{
    border: '1px solid #727272', /* border width and color */
    width: '700px', /* image width */
    display: 'block', /* for alignment */
    margin: '0 auto' /* center alignment */
  }}
/>
</Zoom>

Gizlenen öğe, ana panelde not olarak ve seçildiğinde sol panelde işaretlenir.

<Zoom>
  <img src="/assets/shared/img/pb-hidden-element.webp"

    width: '700px', /* image width */

    margin: '0 auto' /* center alignment */

Öğeyi tekrar göstermek için sağ paneldeki üç nokta düğmesine tıklayın ve **Show** seçeneğini belirleyin.
