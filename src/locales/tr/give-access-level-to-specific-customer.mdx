Adapty Kontrol Paneli'nden belirli bir müşterinin access level'ını manuel olarak ayarlayabilirsiniz. Bu özellik özellikle destek senaryolarında işe yarar. Örneğin, harika bir yorum bırakan bir kullanıcıya teşekkür olarak premium kullanımını bir hafta uzatmak isteyebilirsiniz.

## Adapty Kontrol Paneli'nde belirli bir müşteriye access level verme \{#give-access-level-to-a-specific-customer-in-the-adapty-dashboard\}

1. Adapty ana menüsünden **[Profiles and Segments](https://app.adapty.io/placements)** bölümüne gidin.

<Zoom>
  <img src="/assets/shared/img/profiles-list.png"
  style={{
    border: '1px solid #727272', /* border width and color */
    width: '700px', /* image width */
    display: 'block', /* for alignment */
    margin: '0 auto' /* center alignment */
  }}
/>
</Zoom>

2. Erişim vermek istediğiniz müşteriye tıklayın.

3. **Add access level** düğmesine tıklayın.

<Zoom>
  <img src="/assets/shared/img/add-access-level.png"
  style={{
    border: '1px solid #727272', /* border width and color */
    width: '700px', /* image width */
    display: 'block', /* for alignment */
    margin: '0 auto' /* center alignment */
  }}
/>
</Zoom>

4. Vermek istediğiniz access level'ı ve bu müşteri için ne zaman sona ereceğini seçin.

5. **Apply** düğmesine tıklayın.

## API aracılığıyla belirli bir müşteriye access level verme \{#give-access-level-to-a-specific-customer-via-api\}

Adapty API'sini kullanarak sunucunuzdan bir müşteriye access level verme seçeneğiniz de bulunmaktadır. Bu özellik, referans bonusları veya ürünlerinizle ilgili diğer etkinlikler için oldukça kullanışlıdır. Ayrıntılı bilgi için [Sunucu taraflı API ile access level verme](api-adapty/operations/grantAccessLevel) sayfasına bakın.
