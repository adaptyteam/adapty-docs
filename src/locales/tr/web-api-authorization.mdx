## Yetkilendirme \{#authorization\}

API isteklerinin, **Authorization** başlığında `Api-Key {your_public_api_key}` değeriyle (örneğin `Api-Key public_live_...`) public API anahtarınızla doğrulanması gerekir. Bu anahtarı [Adapty Kontrol Paneli -> **App Settings** -> **General** sekmesi -> **API keys** bölümünde](https://app.adapty.io/settings/general) bulabilirsiniz.

## İstek formatı \{#request-format\}

- **Content-Type başlığı**:
  API'nin isteğinizi işleyebilmesi için **Content-Type** başlığını `application/json` olarak ayarlayın.
- **Gövde**:
  API, istek gövdesinin JSON formatında olmasını bekler.
