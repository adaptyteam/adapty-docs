## Yetkilendirme \{#authorization\}

API istekleri, **Authorization** başlığına `Api-Key {your_public_api_key}` değeri verilerek kimlik doğrulaması yapılmalıdır; örneğin `Api-Key public_live_...`. Bu anahtarı [Adapty Kontrol Paneli -> **App Settings** -> **General** sekmesi -> **API keys** bölümü](https://app.adapty.io/settings/general) içinde bulabilirsiniz.

:::important
API anahtarları uygulamaya özgüdür. Birden fazla uygulamanız varsa her biri için farklı anahtar kullandığınızdan emin olun.
:::
## İstek formatı \{#request-format\}

- **Content-Type başlığı**:
  API'nin isteğinizi işleyebilmesi için **Content-Type** başlığını `application/json` olarak ayarlayın.
- **Gövde**:
  API, isteğin gövdesinin JSON formatında olmasını bekler.
