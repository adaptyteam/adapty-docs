Adapty'nin veri erişimini otomatikleştirebilmesi için Google Play Console'da bir servis hesabı gereklidir.

1. Google Cloud Console'un [**IAM & Admin** -> **Service accounts**](https://console.cloud.google.com/iam-admin/serviceaccounts) bölümünü açın. Doğru projeyi kullandığınızdan emin olun.

<Zoom>
  <img src="/assets/shared/img/17bbf45-google_cloud_create_service_account.webp"
  style={{
    border: '1px solid #727272', /* border width and color */
    width: '700px', /* image width */
    display: 'block', /* for alignment */
    margin: '0 auto' /* center alignment */
  }}
/>
</Zoom>

2. **Service accounts** penceresinde **Create service account** düğmesine tıklayın.

<Zoom>
  <img src="/assets/shared/img/b93eec1-service_account_details.webp"
  style={{
    border: '1px solid #727272', /* border width and color */
    width: '700px', /* image width */
    display: 'block', /* for alignment */
    margin: '0 auto' /* center alignment */
  }}
/>
</Zoom>

3. **Create service account** penceresinin **Service account details** alt bölümünde istediğiniz **Service Account Name**'i girin. Bu hesabın amacını belirtmek için adın içine "Adapty" eklemenizi öneririz. **Service account ID** otomatik olarak oluşturulacaktır.

4. Servis hesabı e-posta adresini kopyalayın ve ileride kullanmak üzere kaydedin.

5. **Create and continue** düğmesine tıklayın.

<Zoom>
  <img src="/assets/shared/img/e69d713-grant_access_to_project.webp"
  style={{
    border: '1px solid #727272', /* border width and color */
    width: '700px', /* image width */
    display: 'block', /* for alignment */
    margin: '0 auto' /* center alignment */
  }}
/>
</Zoom>

6. **Grant this service account access to project** alt bölümündeki **Select a role** açılır listesinden **Pub/Sub -> Pub/Sub Admin** seçeneğini seçin. Bu rol, gerçek zamanlı geliştirici bildirimlerini etkinleştirmek için gereklidir.

<Zoom>
  <img src="/assets/shared/img/976299c-service_account_role.webp"
  style={{
    border: '1px solid #727272', /* border width and color */
    width: '700px', /* image width */
    display: 'block', /* for alignment */
    margin: '0 auto' /* center alignment */
  }}
/>
</Zoom>

7. **Add another role** düğmesine tıklayın.

8. Yeni açılan **Role** açılır listesinden **Monitoring -> Monitoring Viewer** seçeneğini seçin. Bu rol, bildirim kuyruğunun izlenmesine olanak tanımak için gereklidir.

9. **Continue** düğmesine tıklayın.

<Zoom>
  <img src="/assets/shared/img/ffe8d82-grant_user_access.webp"
  style={{
    border: '1px solid #727272', /* border width and color */
    width: '700px', /* image width */
    display: 'block', /* for alignment */
    margin: '0 auto' /* center alignment */
  }}
/>
</Zoom>

10. Herhangi bir değişiklik yapmadan **Done** düğmesine tıklayın. **Service accounts** penceresi açılacaktır.
