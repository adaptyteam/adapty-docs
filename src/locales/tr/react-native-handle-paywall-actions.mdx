<SDKv4>

Adapty Flow Builder veya Paywall Builder kullanarak flow'lar veya paywall'lar oluşturuyorsanız, butonları doğru şekilde ayarlamanız çok önemlidir:

1. [Flow/paywall builder'da bir buton](paywall-buttons) ekleyin ve ona ya mevcut bir eylem atayın ya da özel bir eylem ID'si oluşturun.
2. Atadığınız her eylemi yönetmek için uygulamanızda kod yazın.

Bu rehber, özel ve mevcut eylemleri kodunuzda nasıl yöneteceğinizi gösterir.

:::warning
**Satın almalar, geri yüklemeler, flow/paywall kapatma işlemleri ve URL açma otomatik olarak yönetilir.** Bunların varsayılan davranışını yapılandırabilir veya özel eylemler için yanıtlar uygulayabilirsiniz.
:::

:::note
SDK, push bildirimleri veya kamera erişimi gibi sistem izni istekleri için bir `onRequestPermission` flow işleyicisi sunar. Flow'lar bu istekleri henüz tetiklemez, bu yüzden şimdilik uygulamanıza gerek yoktur.
:::

## Flow'ları ve paywall'ları kapatma \{#close-flows-and-paywalls\}

Flow'unuzu veya paywall'ınızı kapatacak bir buton eklemek için:

1. Builder'da bir buton ekleyin ve ona **Close** eylemini atayın.
2. Uygulama kodunuzda, flow veya paywall'ı kapatan `close` eylemi için bir işleyici uygulayın.

:::info
React Native SDK'da `close` eylemi varsayılan olarak flow veya paywall'ı kapatmayı tetikler. Ancak gerekirse bu davranışı kodunuzda geçersiz kılabilirsiniz. Örneğin, bir flow'u kapatmak başka birini açmayı tetikleyebilir.
:::

<Tabs groupId="presentation-method" queryString>
<TabItem value="platform" label="React bileşeni" default>

React bileşeni için, kapatma eylemini tekil olay işleyici prop'ları aracılığıyla yönetin:

```javascript

function MyPaywall({ flow }) {
  const onCloseButtonPress = useCallback<FlowEventHandlers['onCloseButtonPress']>(() => {
    // Handle close button press - navigate away or hide component
    navigation.goBack();
  }, [navigation]);

  return (
    <AdaptyFlowView
      flow={flow}
      style={styles.container}
      onCloseButtonPress={onCloseButtonPress}
    />
  );
}
```

</TabItem>
<TabItem value="standalone" label="Modal sunum">

Modal sunum için, kapatma işleyicisini uygulayın:

```javascript

const view = await createFlowView(flow);

const unsubscribe = view.setEventHandlers({
    onCloseButtonPress() {
        return true; // allow flow or paywall closing
    }
});
```

</TabItem>
</Tabs>

## Flow'lardan ve paywall'lardan URL açma \{#open-urls-from-flows-and-paywalls\}

:::tip
Bir grup bağlantı eklemek istiyorsanız (örneğin kullanım koşulları ve satın alma geri yükleme), builder'da bir **Link** öğesi ekleyin ve onu **Open URL** eylemine sahip butonlarla aynı şekilde yönetin.
:::

Flow'unuzdan veya paywall'ınızdan bir bağlantı açan bir buton eklemek için (örneğin **Kullanım koşulları** veya **Gizlilik politikası**):

1. Builder'da bir buton ekleyin, ona **Open URL** eylemini atayın ve açmak istediğiniz URL'yi girin.
2. Uygulama kodunuzda, alınan URL'yi bir tarayıcıda açan `openUrl` eylemi için bir işleyici uygulayın.

:::info
React Native SDK'da `openUrl` eylemi varsayılan olarak URL'yi açmayı tetikler. Ancak gerekirse bu davranışı kodunuzda geçersiz kılabilirsiniz.
:::

<Tabs groupId="presentation-method" queryString>
<TabItem value="platform" label="React bileşeni" default>

React bileşeni için, URL açmayı olay işleyici prop'u aracılığıyla yönetin:

```javascript

function MyPaywall({ flow }) {
  const onUrlPress = useCallback<FlowEventHandlers['onUrlPress']>((url) => {
    Linking.openURL(url);
  }, []);

  return (
    <AdaptyFlowView
      flow={flow}
      style={styles.container}
      onUrlPress={onUrlPress}
    />
  );
}
```

</TabItem>
<TabItem value="standalone" label="Modal sunum">

Modal sunum için, URL işleyicisini uygulayın:

```javascript

const view = await createFlowView(flow);

const unsubscribe = view.setEventHandlers({
    onUrlPress(url) {
        Linking.openURL(url);
        return false; // Keep flow or paywall open
    },
});
```

</TabItem>
</Tabs>

## Özel eylemleri yönetme \{#handle-custom-actions\}

Diğer herhangi bir eylemi yöneten bir buton eklemek için:

1. Builder'da bir buton ekleyin, ona **Custom** eylemini atayın ve bir ID atayın.
2. Uygulama kodunuzda, oluşturduğunuz eylem ID'si için bir işleyici uygulayın.

Örneğin, başka bir abonelik teklifleri veya tek seferlik satın alma grubunuz varsa, başka bir flow veya paywall görüntüleyecek bir buton ekleyebilirsiniz:

<Tabs groupId="presentation-method" queryString>
<TabItem value="platform" label="React bileşeni" default>

React bileşeni için, özel eylemleri olay işleyici prop'u aracılığıyla yönetin:

```javascript

function MyPaywall({ flow }) {
  const onCustomAction = useCallback<FlowEventHandlers['onCustomAction']>((actionId) => {
    if (actionId === 'openNewPaywall') {
      // Display another flow or paywall
    }
  }, []);

  return (
    <AdaptyFlowView
      flow={flow}
      style={styles.container}
      onCustomAction={onCustomAction}
    />
  );
}
```

```javascript
const unsubscribe = view.setEventHandlers({

});
```

## Uygulamaya giriş yapma \{#log-into-the-app\}

Kullanıcıları uygulamanıza giriş yaptıran bir buton eklemek için:

1. Paywall builder'da bir buton ekleyin ve ona **Login** eylemini atayın.
2. Uygulama kodunuzda, kullanıcınızı tanımlayan `login` eylemi için bir işleyici uygulayın.

<Tabs groupId="presentation-method" queryString>
<TabItem value="platform" label="React bileşeni" default>

React bileşeni için, girişi olay işleyici prop'u aracılığıyla yönetin:

```javascript

function MyPaywall({ paywall }) {
  const onCustomAction = useCallback<EventHandlers['onCustomAction']>((actionId) => {
    if (actionId === 'login') {
      navigation.navigate('Login');
    }
  }, [navigation]);

  return (
    <AdaptyPaywallView
      paywall={paywall}
      style={styles.container}
      onCustomAction={onCustomAction}
    />
  );
}
```

```javascript

const view = await createPaywallView(paywall);

});
```
