metadataTitle: "Adapty Android SDK v3.0'a Geçiş | Adapty Dokümanları"
---

Adapty SDK v3.0, yeni ve heyecan verici [Adapty Paywall Builder](adapty-paywall-builder)'ı destekler; bu, paywall oluşturmak için kullanılan kodsuz, kullanıcı dostu aracın yeni sürümüdür. Maksimum esneklik ve zengin tasarım özellikleriyle paywallleriniz çok daha etkili ve kârlı hale gelecek.

Adapty SDK'ları bir BoM (Bill of Materials) olarak sunulur; bu sayede uygulamanızdaki Adapty SDK ve AdaptyUI SDK sürümleri tutarlı kalır.

v3.0'a geçiş yapmak için kodunuzu aşağıdaki gibi güncelleyin:
<Tabs groupId="current-os" queryString>
  <TabItem value="module-level build.gradle" label="module-level build.gradle" default>

```diff showLineNumbers
dependencies {
    ...
-   implementation 'io.adapty:android-sdk:2.11.5'
-   implementation 'io.adapty:android-ui:2.11.3'
+   implementation platform('io.adapty:adapty-bom:3.0.4')
+   implementation 'io.adapty:android-sdk'
+   implementation 'io.adapty:android-ui'
}
```
