Kendi satın alma altyapınız varsa ve henüz Adapty'ye tamamen geçmeye hazır değilseniz, [Observer mode](observer-vs-full-mode)'u inceleyebilirsiniz. Temel haliyle Observer Mode, gelişmiş analitik ve attribution ile analitik sistemlerle sorunsuz entegrasyon sunar.
Eğer bu ihtiyaçlarınızı karşılıyorsa, tek yapmanız gereken:
1. Adapty SDK'yı yapılandırırken `observerMode` parametresini `true` olarak ayarlayarak bunu etkinleştirmek. Kurulum talimatları için [Capacitor](sdk-installation-capacitor#activate-adapty-module-of-adapty-sdk) belgelerine bakın.
2. Mevcut satın alma altyapınızdan Adapty'ye [işlemleri bildirmek](report-transactions-observer-mode-capacitor).
:::tip
SDK v4'te, Adapty tarafından render edilen flow'ları ve paywall'ları Observer modunda da sunabilirsiniz: kullanıcı satın alma veya geri yükleme düğmesine dokunduğunda, SDK eylemi kodunuza devreder, böylece satın alma veya geri yüklemeyi kendiniz gerçekleştirebilirsiniz. Bkz. [Flow'ları Observer modunda sunma](capacitor-present-flows-in-observer-mode).
:::
### Observer mode kurulumu \{#observer-mode-setup\}

Satın alma işlemlerini ve abonelik durumunu kendiniz yönetip Adapty'yi yalnızca abonelik olaylarını ve analitiği göndermek için kullanıyorsanız Observer mode'u etkinleştirin.

:::important
Observer mode'da çalışırken Adapty SDK hiçbir işlemi kapatmaz; bu nedenle işlemleri kendiniz yönettiğinizden emin olun.
:::

```typescript showLineNumbers

try {
  await adapty.activate({
    apiKey: 'YOUR_PUBLIC_SDK_KEY',
    params: {
      observerMode: true // Enable observer mode
    }
  });
} catch (error) {
  console.error('Failed to activate Adapty:', error);
}
```

Parametreler:

| Parametre                   | Açıklama                                                     |
| --------------------------- | ------------------------------------------------------------ |
| **observerMode**            | [Observer mode](observer-vs-full-mode)'u kontrol eden boolean bir değer. Varsayılan değer `false`'tur. |

## Observer Mode'da Adapty paywall'larını kullanma \{#using-adapty-paywalls-in-observer-mode\}

Adapty'nin paywall'larını ve A/B testi özelliklerini de kullanmak istiyorsanız bunu yapabilirsiniz; ancak Observer mode'da birkaç ek adım gereklidir. Yukarıdaki adımlara ek olarak şunları yapmanız gerekir:

1. [Remote config paywall'lar](present-remote-config-paywalls-capacitor) için paywall'ları normalde olduğu gibi gösterin.
2. Paywall'ları satın alma işlemleriyle [ilişkilendirin](report-transactions-observer-mode-capacitor).
