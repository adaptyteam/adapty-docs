metadataTitle: "Uygulama İçi Satın Alma Anahtarı Oluşturma | Adapty Dokümanları"
---

**In-App Purchase Key**, satın almaların gerçekliğini doğrulamak amacıyla App Store Connect içinde oluşturulan özel bir API anahtarıdır.

:::note
App Store Server API için API anahtarları oluşturabilmek üzere App Store Connect'te Admin veya Account Holder rolüne sahip olmanız gerekir. API Anahtarlarının nasıl oluşturulacağını [Apple Developer Dokümantasyonu](https://developer.apple.com/documentation/appstoreserverapi/creating-api-keys-to-authorize-api-requests)'nda da okuyabilirsiniz.
:::

1. **App Store Connect**'i açın. [**Users and Access** → **Integrations** → **In-App Purchase**](https://appstoreconnect.apple.com/access/integrations/api/subs) bölümüne gidin.

2. Ardından **Active** başlığının yanındaki ekle düğmesine **(+)** tıklayın.

3. Açılan **Generate In-App Purchase Key** penceresinde, ileride başvurmak üzere anahtara bir ad girin. Bu ad Adapty'de kullanılmayacaktır.

4. **Generate** düğmesine tıklayın. **Generate in-App Purchase Key** penceresi kapandıktan sonra oluşturulan anahtarı **Active** listesinde göreceksiniz.

   

<Zoom>

    margin: '0 auto' /* center alignment */

/>
</Zoom>

5. API anahtarınızı oluşturduktan sonra, anahtarı dosya olarak indirmek için **Download In-App Purchase Key** düğmesine tıklayın.

   

<Zoom>
  <img src="/assets/shared/img/d59faff-download_in-app_purchase_key.webp"
  style={{
    border: '1px solid #727272', /* border width and color */
    width: '700px', /* image width */
    display: 'block', /* for alignment */
    margin: '0 auto' /* center alignment */
  }}
/>
</Zoom>

6. **Download in-App Purchase Key** penceresinde **Download** düğmesine tıklayın. Dosya bilgisayarınıza kaydedilir.  
   Bu dosyayı güvende tutmak ve ileride Adapty Kontrol Paneli'ne yüklemek için saklamak çok önemlidir. Oluşturulan dosya yalnızca bir kez indirilebileceğinden, yükleyene kadar güvenli bir yerde sakladığınızdan emin olun. **In-App Purchase** bölümünden oluşturulan .p8 anahtarı, [Adapty'nin App Store ile ilk entegrasyonu yapılandırılırken](app-store-connection-configuration#step-3-upload-in-app-purchase-key-file) kullanılacaktır.
