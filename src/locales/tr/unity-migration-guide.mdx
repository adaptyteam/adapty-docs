## Migrasyon Rehberleri \{#migration-guides\}

### [Unity Adapty SDK 3.x'e migrasyon rehberi](unity-sdk-migration-guides)

Eski sürümlerden Unity Adapty SDK 3.x'e nasıl geçileceğini öğrenin.

## Yenilikler \{#whats-new\}

### Sürüm 3.x \{#version-3x\}

- Geliştirilmiş paywall sunumu
- İyileştirilmiş hata yönetimi
- Daha iyi C# desteği
- Performans optimizasyonları

### Sürüm 2.x \{#version-2x\}

- Yeni onboarding özellikleri
- Geliştirilmiş analitik
- İyileştirilmiş satın alma akışı
- Hata düzeltmeleri ve kararlılık iyileştirmeleri

## Kırıcı Değişiklikler \{#breaking-changes\}

### Sürüm 3.x \{#version-3x-breaking\}

- Güncellenmiş observer API'ı
- Değiştirilen paywall sunum metodları
- Düzenlenmiş hata yönetimi yapısı

### Sürüm 2.x \{#version-2x-breaking\}

- Güncellenmiş onboarding API'ı
- Değiştirilen profil yapısı
- Düzenlenmiş satın alma akışı

## Migrasyon Kontrol Listesi \{#migration-checklist\}

Yeni bir sürüme geçerken:

- [ ] Kırıcı değişiklikleri inceleyin
- [ ] API çağrılarını güncelleyin
- [ ] Tüm işlevselliği test edin
- [ ] Hata yönetimini güncelleyin
- [ ] Analitik takibini doğrulayın
- [ ] Tüm platformlarda test edin
