タイトル、段落、またはリストをワンクリックで追加し、ブランドに合わせてスタイルを設定し、[動的変数](onboarding-variables)を使って各ユーザー向けにコンテンツをパーソナライズできます。

## テキストスタイルを設定する \{#set-up-text-styles\}

**Styles** パネルには、H1、H2、H3、Button Label、Body、Caption、Small Label といった事前設定済みのテキストスタイルが含まれています。スタイルをクリックすると、フォントファミリー、ウェイト、サイズ、配置、装飾などのプロパティを編集できます。

[テキスト要素を追加する](#add-text)際には、ここで設定したスタイルから選択できます。

:::important
スタイルへの変更は、そのスタイルを使用しているすべての画面のすべてのテキスト要素に適用されます。
:::

新しいスタイルを作成するには：

1. <Inline id="Styles.webp" alt="Palette"/> をクリックして **Styles** パネルを開きます。
2. **Text** タブで **Create style** をクリックします。

<ZoomImage id="new-text-style.webp" width="500px" alt="新しいテキストスタイル" />

3. 名前を入力し、タイポグラフィを設定します：
- **Font family**：書体を選択します。カスタムフォントの使い方については、[ガイド](using-custom-fonts-in-paywall-builder)を参照してください。
- **Weight** と **size**：フォントのウェイトとサイズを設定します。
- **Color**：テキストの色を設定します。
- **Line height**：行間を設定します。**Auto** のままにしておくこともできます。
- **Alignment**：水平方向（左、中央、右）および垂直方向（上、中央、下）の配置を設定します。
- **Decoration**：アンダーラインまたは取り消し線を適用します。
- **Truncate**：表示する行数を制限します。その制限を超えたテキストは切り捨てられます。動的変数やローカライズによってコンテンツの長さが変わる場合に便利です。
4. **Create** をクリックします。

:::note
SF Pro はクロスプラットフォームでの使用には対応していません。近い代替フォントとして Inter をお勧めします。
:::

<ZoomImage id="text-style-details.webp" width="500px" alt="テキストスタイルの設定オプション" />

## テキストを追加する \{#add-text\}

テキスト要素を追加するには：
1. 左上の **+** をクリックします。**Text** を選択し、事前設定済みまたは自分で設定した[テキストスタイル](#set-up-text-styles)から選択します。

<ZoomImage id="new-text.webp" width="500px" alt="新しいテキスト" />

2. 新しい要素をクリックし、右側の **Design** パネルの **Content** セクションでコンテンツを編集します。
3. 必要に応じて、**Design** パネルでタイポグラフィを調整します。または、プレビュー内のテキストを選択してツールチップを開き、スタイルをすばやくカスタマイズできます。
4. オプションとして、**Design** および **Interaction** パネルで、フロー内のコンポーネントに利用可能な他の設定も行えます。詳細については、[要素のスタイル設定](builder-styling)を参照してください。

:::tip
同じテキスト要素を複数の画面で使用したい場合は、コピー＆ペーストを使ってください：要素を選択し、Ctrl+C（Macでは ⌘+C）を押し、別の画面に移動して Ctrl+V（Macでは ⌘+V）を押して貼り付けます。
:::

<ZoomImage id="text-settings.webp" width="500px" alt="テキスト要素の設定" />

### テキストの一部だけスタイルを変更する \{#change-styling-for-parts-of-text\}

テキスト要素の一部だけスタイルを変更するには：

1. **Content** セクションでテキスト要素の一部を選択します。
2. 表示されるツールチップで、テキストの色を変更したり、太字、アンダーライン、イタリック、取り消し線の書式を適用したり、URLを追加したりできます。

プレビューはすぐに更新されます。

<ZoomImage id="edit-text-part.webp" width="500px" alt="テキスト部分の編集" />

## リンクを追加する \{#add-links\}

フローでは、テキストをクリック可能なリンクに変換する方法が2つあります。テキストの役割に応じて選択してください：

- **インラインリンク** — 段落内に埋め込まれた「詳細はこちら」のような、流れるコピーの中にURLを含める場合。常にアプリ内ブラウザで開きます。
- **Open URL アクション** — 利用規約ボタンのような、独立したタップターゲットの場合。アプリ内ブラウザまたは外部ブラウザのどちらでも開けます。

### インラインリンク \{#inline-link\}

テキスト要素の一部をリンクにするには：

1. **Content** セクションでテキストを選択します。
2. 書式設定ツールチップでリンクアイコンをクリックします。
3. ポップアップに移動先の URL を貼り付けます。

<ZoomImage id="inline-link-demo.webp" width="400px" alt="リンクボタンがハイライトされた書式設定ツールチップ" />

### Open URL アクション \{#open-url-action\}

:::link
メイン記事：[アクション](onboarding-actions#open-url)
:::

ボタン全体をリンクにするには：

1. [ボタン](builder-elements#buttons)を追加するか、**Links** プリセット（Restore / Terms of Service / Privacy Policy ボタンが並んだ既製の行）を使用します。

    <ZoomImage id="links-button-template.webp" width="500px" alt="Terms of use、Restore、Privacy policy ボタンが表示されたキャンバス上の Links プリセット" />

2. **Layers** パネルでボタンを選択し、右パネルの **Interactions** タブを開きます。

    <ZoomImage id="button-selected.webp" width="500px" alt="Layers パネルで選択された Terms of Use ボタン（ネストされたテキスト要素は表示されているが未選択の状態）" />

3. [**Open URL**](onboarding-actions#open-url) アクションの移動先を設定します。

    :::important
    空の Open URL アクションは[プレビューと公開をブロックします](builder-save-publish#troubleshooting)。
    :::

    <ZoomImage id="open-url.webp" width="500px" alt="Interactions パネルでの Open URL アクションの設定" />

## 条件付きテキストを追加する \{#add-conditional-text\}

条件付きテキストは、条件に基づいてテキスト要素に表示する内容を変えます。たとえば、ユーザーが年間プランを選択したときと月間プランを選択したときで、見出しに異なるメッセージを表示できます。条件付きテキストは[条件付き表示](onboarding-element-visibility)と同じように機能しますが、要素の表示・非表示を切り替えるのではなく、コンテンツ自体を入れ替えます。

条件付きテキストを設定するには：

1. キャンバスでテキスト要素を選択します。
2. **Design** パネルの **Content** セクションで **Conditional** を選択します。

<ZoomImage id="flow-conditional-text.webp" width="500px" alt="Design パネルで Content が Conditional に設定されている状態" />

3. **if** 条件を作成します。**Custom**、**Products**、または **Elements** タブからプロパティを選択し、演算子を設定して、マッチさせる値を入力します。プロパティの種類の詳細については、[条件付き表示](onboarding-element-visibility)を参照してください。
4. **then** の下に、条件が true の場合に表示するテキストを入力します。リッチテキスト書式は[通常のテキスト](#change-styling-for-parts-of-text)と同じように機能します。[変数](onboarding-variables)を挿入するには、**&#123; &#125; Add variable** をクリックします。
5. **else** の下に、どの条件にも一致しない場合に表示するフォールバックテキストを入力します。
6. （オプション）**+ Add else/if** をクリックして、それぞれ独自のテキストを持つ条件をさらに追加します。

:::tip
別の言語の条件付きテキストを編集するには、エディタ下部でアクティブなロケールを切り替えます。まず **Localizations** パネルで言語を追加してください。詳しくは[フロービルダーでロケールを追加する](add-paywall-locale-in-adapty-paywall-builder)を参照してください。
:::

<ZoomImage id="flow-conditional-text-1.gif" width="500px" alt="if、then、else のブランチで条件付きテキストを設定している様子" />

## リストを追加する \{#add-lists\}

:::note
リスト要素は、個々のアイテムコンポーネントで構成されたコンテナです。流れるテキスト内にシンプルな箇条書きや番号付きリストを作るには、テキスト要素を使って **Design** パネルから好みの書式を適用してください。
:::

1. 左上の **+** をクリックします。**List** を選択し、リストテンプレートの中から1つ選びます。

2. 右側の **Design** パネルでリストアイテムを編集したり、アイテムマーカーとして画像をアップロードしたりできます。

<ZoomImage id="add-list.webp" width="500px" alt="リスト要素を追加する" />
