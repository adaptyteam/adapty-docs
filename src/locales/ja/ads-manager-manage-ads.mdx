Adapty Ads ManagerはApple Adsと双方向に連携しており、ほぼリアルタイムのパフォーマンスデータを取得できるほか、ネイティブUIよりはるかに使いやすいAdaptyダッシュボードから直接広告を作成・編集できます。
Apple のネイティブ広告ダッシュボードで広告を作成すると、24 時間以内に Adapty Ads Manager に自動的に表示されます。
## 広告とは \{#what-are-ads\}

広告とは、[キャンペーン](ads-manager-create-campaign)内の[広告グループ](ads-manager-create-ad-group)に割り当てられた広告クリエイティブです。1つの広告グループにつき、アクティブな広告を1つ割り当てることができます。

## 広告を作成する \{#create-ads\}

始める前に、以下を作成しておく必要があります：

- **Ad group**（広告グループ）。[Adapty Ads Manager ダッシュボードで直接作成](ads-manager-create-ad-group)できます。
- **Custom product page**（カスタムプロダクトページ）。[Apple Ads で直接設定](https://developer.apple.com/help/app-store-connect/create-custom-product-pages/configure-multiple-product-page-versions/)する必要があります。広告で使用する前に、App Store の審査に通過している必要があります。

:::note
選択した広告グループ内にすでにアクティブな広告がある場合、新しい広告を実行するために一時停止されます。
:::

新しい Apple Ads の広告を作成するには：
1. サイドバーメニューから **Ads Manager** に移動します。任意のタブで、テーブル上の **+** をクリックし、**Create ad** を選択します。

<ZoomImage id="create-ad.webp" width="900px" />

2. 広告を出稿するアプリを選択します。
3. 広告のグループを選択します。
4. 広告名を入力します。
5. 広告のステータスを選択します。後から広告を開始したい場合は、**Status** チェックボックスをオフにします。

<ZoomImage id="create-ad-meta.webp" />

6. **Select CPP** をクリックします。App Store で承認済みのカスタムプロダクトページがすべて表示されます。選択できるカスタムプロダクトページは1つのみです。

<ZoomImage id="select-cpp.gif" />
7. **Create ad** をクリックします。
## 広告を編集する \{#edit-ads\}

:::note
広告を作成した後は、名前のみ編集できます。CPP の変更や別の広告グループへの移動はできません。
:::

広告名を編集するには、以下のいずれかの方法を使用してください：

- **Ads Manager > Ads** で広告名をクリックします。広告名を編集し、その横にあるチェックマークをクリックします。
  <ZoomImage id="edit-ad-name.gif" width="500px" />

- 広告名の横にあるチェックボックスを選択し、**Actions > Edit ad** をクリックします。広告名を変更して **Save changes** をクリックします。
   <ZoomImage id="edit-ad-name.webp" width="500px"/>
:::note
Apple Ads で直接行った広告の編集内容は、Adapty Ads Manager に自動的に同期されますが、Adapty Ads Manager に反映されるまでに時間がかかる場合があります。
:::
## 広告のエクスポート \{#export-ads\}

広告テーブルをCSVとしてエクスポートするには、テーブル上部のダウンロードアイコンをクリックし、**Export current page** または **Export all pages** を選択してください。

**Export all pages** を選択すると、すべてのページにわたる広告データが1つのファイルにダウンロードされます。ダウンロードの進捗はモーダルで確認でき、いつでもキャンセルできます。また、以下の任意フィルターも利用できます：

- **Enabled only**: アクティブな広告のみを含めます。
- **With spend ≥**: 指定した閾値以上の支出がある広告のみを含めます。
- **Group by country**: 各広告の行を国別に分類します。
テーブルは、ダッシュボードに表示されている状態（選択した列）でそのままエクスポートされます。

<ZoomImage id="export-ads.webp" width="700px" alt="Export ads" />
## 広告を開始・一時停止する \{#launch--pause-ads\}

Apple Ads Managerから任意の広告を開始または一時停止するには、以下のいずれかの方法を使用します。

- **Ads Manager > Ads** の **Status** 列のチェックボックスをオンまたはオフにします。
  <ZoomImage id="ad-status.webp" width="500px"/>

- 広告名の横のチェックボックスを選択し、**Actions > Edit ad** をクリックします。広告名を変更し、**Save changes** をクリックします。
  <ZoomImage id="edit-ad-name.webp" width="500px"/>
