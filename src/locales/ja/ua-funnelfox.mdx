[FunnelFox](https://funnelfox.com) は、App Store の手数料やその他の制限を回避しつつ、ストア外でユーザーを獲得・課金できる web2app ファネル構築プラットフォームです。

連携が完了すると、FunnelFox はトランザクションイベントを Adapty のアトリビューションに送信し、ウェブのタッチポイントから有料サブスクリプションまでの完全なアトリビューションパスを確認できるようになります。

インテグレーションを設定するには、**Project ID** を使って 1 つ以上の FunnelFox プロジェクトをお使いの Adapty アプリにリンクしてください。
## 仕組み \{#how-it-works\}

ユーザーが FunnelFox のファネルで購入を完了すると、FunnelFox はトランザクションイベントを Adapty Attribution に送信します。Adapty は **Project ID** を使用して、そのトランザクションがどのアプリに属するかを識別します。イベントは保存され、Adapty Attribution のアナリティクスに表示されます。

各トランザクションには以下が含まれます：
- **サブスクリプションライフサイクルイベント**: 開始、更新、キャンセル、トライアル変換、返金など
- **アトリビューションデータ**: キャンペーン、広告セット、広告識別子、UTMパラメーター、プラットフォームのクリックID（fbclid、ttclid、gclid）
- **ファネルおよび実験データ**: FunnelFoxのファネル名と実験名（A/B テストのバリアントを比較するために使用）

Adaptyはトランザクション内のクリックIDから**チャンネル**（Facebook、TikTok、Google、またはオーガニック）を自動的に判別します。手動での設定は不要です。
:::note
FunnelFox のトランザクションは、ウェブ購入にはアプリのインストールイベントがないため、インストール日ではなく**初回課金日**をコホート日として使用します。
:::
## 連携を設定する \{#configure-integration\}

### ステップ 1. FunnelFox で Project ID を取得する \{#step-1-get-your-project-id-in-funnelfox\}

1. FunnelFox ダッシュボードの左サイドバーで **Settings** をクリックします。
2. **Project info** セクションで **ID** の値をコピーします。

<ZoomImage id="ff-project-id.webp" width="500px" />

### ステップ 2. Adapty UA でプロジェクトを追加する \{#step-2-add-the-project-in-adapty-ua\}

1. Adapty UA で [**Integrations > FunnelFox**](https://app.adapty.io/ua/integrations/funnelfox) に移動します。
2. FunnelFox からコピーした Project ID を貼り付けます。
3. **Save** をクリックします。

追加の FunnelFox プロジェクトを接続するには、**Add project** をクリックし、プロジェクトごとに両方のステップを繰り返してください。
