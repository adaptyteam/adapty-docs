## 設定 \{#configuration\}

### Android

1. フォールバック設定ファイルをアプリケーションに追加します。以下のいずれかのディレクトリを選択してください：
   * **android/app/src/main/assets/**
   * **android/app/src/main/res/raw/**
:::note
`res/raw` フォルダには特別なファイル命名規則があります（先頭は英字、大文字不可、アンダースコア以外の特殊文字不可、スペース不可）。
:::

2. `FileLocation` 定数の `android` プロパティを更新してください：
   * ファイルが `assets` ディレクトリにある場合は、そのディレクトリからの相対パスを渡してください。
   * ファイルが `res/raw` ディレクトリにある場合は、拡張子なしのファイル名を渡してください。
### iOS

1. フォールバックJSONファイルをプロジェクトバンドルに追加します：XCodeの **File** メニューを開き、**Add Files to "YourProjectName"** を選択します。
2. 設定ファイルの名前を `FileLocation` 定数の `ios` プロパティに指定します。

## 例 \{#example\}

```typescript showLineNumbers

const fileLocation = {
  ios: {
    fileName: 'ios_fallback.json'
  },
  android: {
    //if the file is located in 'android/app/src/main/assets/'
    relativeAssetPath: 'android_fallback.json'
  }
};

await adapty.setFallback({ fileLocation });
```

:::important
`setFallback` は、SDK がターゲットのフロー、ペイウォール、またはオンボーディングを取得する前に実行する必要があります。
:::

パラメーター：
| パラメーター            | 説明                                              |
| :------------------- | :------------------------------------------------------- |
| **fileLocation** | フォールバック設定ファイルの場所を表すオブジェクト。 |
