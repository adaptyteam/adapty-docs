Adapty Ads Manager は Apple Ads と双方向に連携しています。ほぼリアルタイムのパフォーマンスデータを取得できるほか、ネイティブ UI よりもはるかに便利な方法で、Adapty ダッシュボードから直接キーワードを作成・編集できます。
Apple Adsのネイティブダッシュボードでキーワードを作成すると、24時間以内に自動的にAdapty Ads Managerに表示されます。

[包括的なアナリティクスの確認](adapty-ads-manager-analytics)に加えて、キーワードのすべての設定を管理できます：
- 広告グループへのキーワードの追加
- 除外キーワードの追加
- SKAG（Single Keyword Ad Group）としてのキーワードの追加
- テーブル内でのキーワードの直接編集
- 複数のキーワードに対する一括操作
- キーワードの開始・一時停止
## キーワードとは \{#what-are-keywords\}

キーワードとは、App Storeの検索結果に広告を表示させるための検索語句です。[広告グループ](ads-manager-create-ad-group)内に整理され、広告グループは[キャンペーン](ads-manager-create-campaign)に属します。この階層構造により、広告戦略を体系的に管理・整理できます。

:::important
キーワードは**Search results**プレースメントタイプのキャンペーンにのみ適用されます。他のプレースメントタイプ（Search TabやProduct Pages）のキャンペーンにはキーワードは使用されません。
:::

### 標準キーワード \{#standard-keywords\}

標準キーワードは、広告を表示させるために入札する主要な語句です。ユーザーがApp Storeでこれらの語句を検索すると、広告が検索結果に表示される場合があります。

### 除外キーワード \{#negative-keywords\}

除外キーワードは、アプリに関連しない検索で広告が表示されるのを防ぎます。除外キーワードを追加することで、関連性の低い検索への無駄な費用を削減できます。

除外キーワードは広告グループレベルで追加するか、複数のキャンペーンに同時に適用するクロスグループ除外キーワードとして追加できます。

### SKAGとしてのキーワード（Single Keyword Ad Group） \{#keywords-as-skag-single-keyword-ad-group\}

SKAG（Single Keyword Ad Group）は、1つの広告グループに1つのキーワードのみを含む形で個別の広告グループを作成する戦略です。このアプローチにより、以下のメリットが得られます。
- 価値の高いキーワードに対する入札の精密なコントロール
- キーワードレベルでのパフォーマンス分析の向上

SKAGは、パフォーマンスの高いキーワードを特定し、専用の広告グループを通じてその可能性を最大化するのに特に有効です。

## キーワードの追加 \{#add-keywords\}

広告グループにキーワードを追加するには：

1. サイドバーメニューから **Ads Manager** に移動します。任意のタブでテーブル上の **+** をクリックし、ドロップダウンから **Add keywords** を選択します。

<ZoomImage id="add-keywords-action.webp" width="700px" /> 

2. モーダルで、キーワードを追加したいキャンペーンと広告グループを選択します。1つのキャンペーンで広告グループを選択した後、別のキャンペーンを選択してさらに広告グループをリストに追加することもできます。

<ZoomImage id="select-ad-groups-keywords.webp" width="700px" />
3. **Select** をクリックして次に進みます。
4. **Add keywords** ダイアログで、**Keywords list** フィールドにキーワードを入力します。カンマ区切りのファイルにキーワードが含まれている場合は、その内容を貼り付けると Adapty Ads Manager が一括でキーワードをアップロードします。
5. テーブルの各キーワードに対して次の項目を設定します：
   - **Match type**：**Exact**（完全一致）または **Broad**（部分一致）を選択します
   - **CPT bid**：このキーワードのタップあたりの最大入札額を設定するか、空のままにしてアドグループのデフォルト最大 CPT 入札額を使用します
6. キーワードを確認し、**Add X keywords**（X は追加するキーワードの数）をクリックします。
:::important
キーワードを保存した後は、マッチタイプを変更することはできません。マッチタイプを変更する場合は、そのキーワードを削除し、希望のマッチタイプで再度追加してください。
:::

<ZoomImage id="add-keywords-dialog.gif" width="700px" />
## ネガティブキーワードを追加する \{#add-negative-keywords\}

ネガティブキーワードを追加するには：
1. サイドバーメニューから **Ads Manager** に移動します。任意のタブで、テーブル上部の **+** をクリックし、ドロップダウンから **Add negative keywords** を選択します。
   <ZoomImage id="add-negative-keywords.webp" width="700px" />
2. **Add negative keywords to** モーダルで、ネガティブキーワードを追加するレベルを選択します。
   - **Selected campaigns**: キャンペーンレベルのネガティブキーワードを追加します。
   - **Selected ad groups**: 広告グループレベルのネガティブキーワードを追加します。
   - **All ad groups in selected campaigns**: 選択したキャンペーンのすべての広告グループに、広告グループレベルのネガティブキーワードを追加します。
   :::note
   以下の点に注意してください。
   - 広告グループレベルのネガティブキーワードは、キャンペーンレベルのネガティブキーワードよりも優先されます。
   - 選択したキャンペーンのすべての広告グループにネガティブキーワードを追加した場合、後でキャンペーンに新しい広告グループを追加する際は、手動でネガティブキーワードを追加する必要があります。
   :::
<ZoomImage id="select-campaigns.gif" width="700px" />
3. **Keywords list** フィールドにネガティブキーワードを入力します。カンマ区切りのキーワードファイルがある場合は、その内容をそのまま貼り付けると、Adapty Ads Manager がキーワードを一括でアップロードします。
4. テーブル内の各キーワードに対して、**Match type** を選択します：
   - **Exact**：完全一致のキーワード、またはごく近い表記ゆれのみを除外します。
   - **Broad**：キーワードおよび関連する検索ワードを除外します。
      または、チェックボックスを使って複数のキーワードを選択し、マッチタイプをまとめて変更することもできます。
5. ネガティブキーワードを確認し、**Add X keywords**（X は追加するキーワード数）をクリックします。
:::note
クロスグループのネガティブキーワードは、複数のキャンペーンにわたって特定の検索語句を一括除外したい場合に特に便利で、時間を節約しながら広告戦略全体の一貫性を確保できます。
:::
   <ZoomImage id="negative-keywords.gif" width="700px" />
## キーワードをSKAGとして追加する \{#add-keywords-as-skag\}

キーワードをSKAG（Single Keyword Ad Group）として追加するには：
1. サイドバーメニューから **Ads Manager** に移動します。任意のタブで、テーブル上部の **+** をクリックし、ドロップダウンから **Add keywords as SKAG** を選択します。
   <ZoomImage id="add-skag-keywords.webp" width="700px" />
2. SKAG 広告グループを作成するキャンペーンを選択します。複数のキャンペーンを選択できます。
3. デフォルトでは、すべてのユーザーをターゲットにしたデフォルト設定で新しい広告グループが作成されます。変更する場合は、**Copy settings from ad group** を選択し、設定のコピー元となる既存の広告グループを選択してください。
   <ZoomImage id="copy-settings-from-ad-group.webp" width="700px" />
4. 新しい広告グループの設定を構成します：
   - **Ad group name prefix**：各広告グループ名に追加するオプションのプレフィックスです（例：「SKAG_」と設定すると「SKAG_keyword1」「SKAG_keyword2」などが作成されます）。**Tag** をクリックすると、キーワード、キャンペーン名、国をグループ名に動的に追加できます。
   - **CPT bid** と **CPA cap**：すべてのキーワードに一括で入札額を設定するか、**Set CPT bid and CPA cap for each word manually** を選択してキーワードごとに個別設定します。
5. **Keywords list** フィールドにキーワードを入力します。カンマ区切りのキーワードファイルがある場合は、その内容を貼り付けることで Adapty Ads Manager がキーワードを一括でアップロードします。
6. テーブル内の各キーワードについて、**Match type** を選択します：
   - **Exact**：キーワードまたは非常に近いバリエーションのみにマッチします
   - **Broad**：キーワードおよび関連する検索語句にマッチします
     または、チェックボックスを選択して一括でマッチタイプを変更することもできます。
7. **Check for duplicates in target campaign** を選択して、ターゲットキャンペーンに同一のキーワードがないことを確認します。
8. **Create** をクリックして SKAG 広告グループを作成します。
各キーワードは、選択したキャンペーンごとに個別の広告グループに配置されるため、独立して管理・最適化できます。

<ZoomImage id="new-skag.webp" width="700px" />
## キーワードを編集する \{#edit-keywords\}

既存のキーワードを編集するには：

1. **Ads Manager > Keywords** または **Ads Manager > Negative keywords** に移動し、テーブルで編集したいキーワードを見つけます。または、キャンペーンページに移動してから広告グループページに進み、キーワードを見つけます。
2. テーブル内で値を直接編集します：
   - **CPT bid**：入札額をクリックして、新しいタップあたりの最大コスト入札額を入力します
   - **Status**：トグルスイッチを使ってキーワードを一時停止または有効化します

:::note
Apple Ads で直接行ったキーワードの編集は Adapty Ads Manager に自動的に同期されますが、Adapty Ads Manager に反映されるまでに時間がかかる場合があります。
:::
<ZoomImage id="edit-keywords.webp" width="500px" />
## 一括操作 \{#bulk-actions\}

複数のキーワードに対して一括操作を行うことで、時間を節約しながらキーワードをより効率的に管理できます。

一括操作を行うには：
1. **Ads Manager > Keywords** または **Ads Manager > Negative keywords** タブに移動します。
2. 管理したいキーワードの横にあるチェックボックスをオンにして、複数のキーワードを選択します。
3. **Actions** ドロップダウンをクリックし、以下のいずれかのオプションを選択します：
   - **Add as keywords**: 選択したキーワードを標準キーワードとして追加します
   - **Add as negative keywords**: 選択したキーワードをネガティブキーワードとして追加します
   - **Add as SKAG**: 選択したキーワードに対してシングルキーワード広告グループを作成します
   - **Activate**: 選択したキーワードを有効化します
   - **Pause**: 選択したキーワードを一時停止します
   - **Edit CPT bids**: 選択したキーワードのCPT入札額を編集します。編集方法は以下のとおりです：
     - **Set to**: 複数の入札額を特定の金額に設定します。
     - **Increase by/decrease by**: 入札額をUSD単位またはパーセント単位で増減します。誤って使いすぎないよう、任意で入札額の上限を設定することもできます。
     - **Set to average CPT**: CPT（タップ単価）指標を使用して、入札額をその値に合わせます。乗数係数を設定します。たとえば、パフォーマンスが期待を下回っている場合は0.9、上回っている場合は1.1に設定します。
     - **Set to average CPA**: CPA（獲得単価）指標を使用して、入札額をその値に合わせます。乗数係数を設定します。
:::tip
一括操作が特に役立つ場面：
- キーワードを異なるタイプ（スタンダード、ネガティブ、SKAG）に変換する
- 複数のキーワードに対して他のマッチタイプのキーワードを素早く追加する
- 上位のパフォーマンスキーワードを絞り込んで入札額を調整する
- パフォーマンスが低いキーワードを特定して一時停止する
:::

## キーワードのエクスポート \{#export-keywords\}

キーワードテーブルを CSV としてエクスポートするには、テーブル上部のダウンロードアイコンをクリックし、**Export current page** または **Export all pages** を選択してください。

**Export all pages** を選択すると、すべてのページのキーワードが 1 つのファイルにまとめてダウンロードされます。ダウンロードの進捗はモーダルで確認でき、いつでもキャンセルできます。以下の 2 つのオプションフィルターを利用できます：

- **Enabled only**：有効なキーワードのみを含めます。
- **With spend ≥**：指定した閾値以上の支出があるキーワードのみを含めます。
- **Group by country**：各キーワードの行を国別に分解します。
テーブルは、ダッシュボードで表示するように選択した列と同じ状態でエクスポートされます。

## キーワードレベルのチャートを確認する \{#explore-keyword-level-charts\}

**Ads Manager > Keywords** テーブルから、任意のキーワードのチャートを直接開けます。これにより、各キーワードの日別パフォーマンスを詳細に分析できます。

チャートを表示するには、テーブル内のキーワードの隣にあるチャートアイコンをクリックしてください。

<ZoomImage id="keyword-chart.webp" width="500px" />

デフォルトでは、選択したキーワードの **Spend** 指標が表示されます。

複数の指標を同時に表示して、相関関係や時系列の変化を把握できます。**+** をクリックして新しい指標を追加してください。
**リセット**をクリックして最初からやり直すか、指標のチェックボックスをオフにして非表示にします。

<ZoomImage id="keyword-chart.gif" width="700px" />
## 入札履歴 \{#bid-history\}

キーワードの入札履歴を表示するには、テーブル内のキーワード横にある **Bid History** ボタンをクリックします。パネルが開き、2つのタブが表示されます。
- **Metrics** タブには、時系列の指標チャートが表示されます。指標の追加・削除はキーワードレベルのチャートと同じ方法で行えます — **+** をクリックして追加するか、チェックボックスをオフにして非表示にしてください。入札額が変更された各時点にマーカーが表示され、マーカーにカーソルを合わせると該当日の正確な入札額を確認できます。これを活用することで、入札変更とパフォーマンスの変化を関連付けられます — 変更後に指標が下落またはスパイクした場合、マーカーがその発生タイミングを正確に示します。
- **Bid History** タブには、すべての入札変更が新しい順に一覧表示され、変更前後の入札額、変更日、変更のトリガーが記録されています。
