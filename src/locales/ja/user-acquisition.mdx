Adapty Attributionは、ウェブからアプリへのキャンペーンにおける広告費用とサブスクリプション収益を紐付け、アプリの経済全体を一か所で把握できるようにします。

Adapty Attributionで収益データを確認するには、まずAdaptyダッシュボードでインテグレーションを有効にする必要があります。APIキー、トークン、識別子などを渡す必要はありません。Adapty SDKを更新・設定するだけです。
:::important
Adapty Attribution は以下のバージョンから利用できます：
- iOS、Android、Flutter SDK バージョン 3.9.1 以上
- React Native および Capacitor SDK バージョン 3.10.0 以上
- Unity SDK バージョン 3.12.0 以上
- Kotlin Multiplatform SDK バージョン 3.15.0 以上
:::
## 始める前に \{#before-you-start\}

収益データとキャンペーンのパフォーマンスを連携させるために、Adapty で購入を追跡できるよう設定してください：
- すでに **Adapty を使ってアプリ内課金を実装済みの場合**、この段階では他に何もする必要はありません。
- **アプリ内課金をまだ実装していなく、Adapty を使いたい場合**は、[クイックスタートガイド](quickstart) の手順を完了して、購入処理を Adapty に委任してください。
- **Adapty を使わずにアプリ内課金をすでに実装済みで、Adapty への移行を予定していない場合**は、[オブザーバーモードでお使いのプラットフォーム向けに Adapty SDK をインストール](implement-observer-mode)してください。この段階では、プロジェクトに SDK を追加し、オブザーバーモードを有効にして起動し、トランザクションを報告するだけで構いません。
このセットアップにより、ウェブからアプリへのアトリビューションが有効になります：
- ユーザーがアプリをインストールすると、Adapty SDKがリンクパラメータからインストールの詳細を取得し、Adapty アトリビューションがキャンペーンの詳細を取得できるようになります
- Adapty SDKはアプリ内のすべての収益関連イベントを把握し、それらをウェブキャンペーンにアトリビュートできます。
## ステップ 1. Adapty Attribution を開く \{#step-1-open-adapty-attribution\}

:::important
Adapty ロゴをクリックしても **Attribution** が表示されない場合は、ブラウザの設定で adapty.io の Cookie とサイトデータを削除してから、ページをリロードしてください。
:::

ヘッダーの Adapty ロゴをクリックし、**Attribution** を選択します。

<ZoomImage id="product-switcher.webp" width="500px" alt="Adapty product switcher" />

サブスクリプションイベントは自動的に Adapty Attribution に流れ込み始めます。キャンペーンデータは、ステップ 2 でデータソースを接続した後に表示されます。
イベント配信を一時停止するには、Adapty ダッシュボードで **Integrations > Adapty** を開き、トグルをオフにしてください。
### サポートされているイベント \{#supported-events\}

デフォルトでは、Adaptyはユーザーアクイジションへ3つのグループのイベントを送信します：
- トライアル
- サブスクリプション
- 問題

サポートされているイベントの全一覧は[こちら](events)で確認できます。

## ステップ 2. 広告プラットフォームを連携してトラッキングリンクを追加する \{#step-2-connect-your-ad-platform-and-add-tracking-links\}

Adapty はトラッキングリンクを使って、アプリのインストールとキャンペーンデータを紐付けます。
Adapty Attribution で計測したい広告キャンペーンには、必ずトラッキングリンクをリンク先 URL として設定してください。

複数の広告プラットフォームで広告を配信している場合は、プラットフォームごとに個別にトラッキングリンクを設定してください。

Adapty が広告プラットフォームと連携する方法は 2 つあります。
- **ネイティブ連携（Meta Ads、TikTok Ads）。** Adapty が広告プラットフォームに直接接続します。トラッキングリンクは自動的に生成され、キャンペーンパラメータはリンクの使用箇所に応じて動的に入力されます。異なるキャンペーン、広告セット、クリエイティブに同じリンクを使い回しても、Adapty が正しいキャンペーンデータと広告費を自動的に受け取ります。
- **トラッキングリンクのみ（その他すべての広告プラットフォーム）。** Adapty は広告プラットフォームに接続しません。トラッキングリンクは手動で作成し、リンク作成時にすべてのキャンペーンパラメーターを明示的に定義する必要があります。これらのプラットフォームでは広告費用データは利用できません。

1. Adapty Attribution ダッシュボードの [Integrations > Meta](https://app.adapty.io/ua/integrations/facebook/accounts) に移動し、**Continue with Facebook** をクリックします。

3. リクエストされた権限を確認し、**Save** をクリックします。
4. **Web campaigns** タブに切り替えて、**Create campaign** をクリックします。アプリを選択し、**Save** をクリックします。

<ZoomImage id="ua-new-campaign.webp" width="500px" />

5. **General** タブで、**iOS** および/または **Android** セクションを展開し、App StoreおよびGoogle PlayのアプリケーションURLを貼り付けます。その後、**Save** をクリックします。

<ZoomImage id="ua-url.gif" width="900px" />
6. **Click link** フィールドの値を**1つのリンク**またはプラットフォーム別リンクからコピーします。次に、Meta Ads Manager で広告を開き、このリンクを宛先 URL として貼り付けます。

:::important
**Website URL** フィールドには `https://api-ua.adapty.io/api/v1/attribution/click` を貼り付けてください。リンクの残りの部分は、**Tracking** セクションの **URL parameters** フィールドに貼り付けてください。これにより、Meta 広告の審査が通りやすくなります。詳しくは、[Meta Ads Manager での広告設定に関する推奨事項](meta-create-campaign)をご覧ください。

7. Meta Ads で広告を開始すると、そのデータが Adapty Attribution ダッシュボードで分析できるようになります。

5. **General** タブで **iOS** または **Android**（もしくは両方）のセクションを展開し、App Store または Google Play のアプリURLを貼り付けます。その後、**Save** をクリックします。

その他の広告プラットフォーム向けのトラッキングリンクを作成するには：

1. Adapty Attribution ダッシュボードで、サイドバーメニューから **Tracking links** に移動します。そこで **Create link** をクリックします。

    <ZoomImage id="new-tracking-link.webp" width="700px" />

2. リストからアプリを選択し、**Next** をクリックします。

    <ZoomImage id="ua-choose-app.webp" width="700px" />
3. リンクのパラメーターを入力して、トラッキングしたいキャンペーンと広告に紐づけます。
4. デフォルトでは One Link が作成されます。これはユーザーのプラットフォームを自動的に検出し、クリックをトラッキングした後に App Store または Google Play へリダイレクトします。

    プラットフォームごとに別々のリダイレクト URL を使いたい場合は、**One Link** チェックボックスをオフにして、プラットフォーム別のストアリンクを手動で入力してください。

    - **iOS link** または **Android link** — ストアごとに個別のリンクを用意したい場合に使うオプションのリンクです。
7. 広告プラットフォームに移動し、広告の遷移先 URL としてリンクを貼り付けます。

## ステップ3. ウェブからアプリへのキャンペーンを開始して結果を確認する \{#step-3-launch-your-web-to-app-campaign-and-view-results\}

キャンペーンが稼働し、ユーザーがアプリをインストールし始めると、Adaptyはインストールと収益をキャンペーンにアトリビュートし始めます。

[Adapty UAアナリティクスダッシュボード](ua-analytics)では、以下のようなキャンペーンレベルの指標を確認できます：
- インストール数とコンバージョン数
- サブスクリプションと購入による収益
- 広告プラットフォーム、キャンペーン、広告セット、クリエイティブ別のパフォーマンス内訳

指標は、アプリからインストールおよび収益イベントを受信した時点で表示されます。広告費用データはネイティブインテグレーションを持つプラットフォームでのみ利用できます。
