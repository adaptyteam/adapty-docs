[Adapty Ads Manager](adapty-ads-manager)でキャンペーン、広告グループ、またはキーワードを選択し、数クリックでセグメントに変換することで、ユーザーの[セグメント](segments)を直接作成できます。これにより、セグメント条件を手動で設定することなく、獲得元に基づいてペイウォールやオファーを簡単にパーソナライズできます。

セグメントを作成したら、それを使って異なるプロダクトや価格を設定したり、A/B テストを実施したり、ペイウォールの外観をカスタマイズしたりできます。
## ユースケース \{#use-cases\}

以下は、Apple Ads データから作成したセグメントを実際にどのように活用できるかの例です。

- **キーワードベースのペイウォール**。購買意欲の高いキーワードからのユーザーには機能重視のペイウォールを表示し、より広い探索系キーワードからのユーザーには汎用的なペイウォールを表示する。

- **キャンペーンレベルのオファー**。特定の Apple Ads キャンペーンからのユーザーには長めのトライアルや特別価格を提供し、それ以外のユーザーには標準的なオファーを維持する。

- **広告クリエイティブとペイウォールの一貫性**。特定の機能を訴求している広告グループからのユーザーを、その機能を前面に出したペイウォールへ誘導する。
- **高ROIキャンペーンの最適化**。生涯価値の高いユーザーを継続的に獲得しているキャンペーンからのユーザーに対して、プレミアム優先・定価のペイウォールを表示します。
## セグメントの作成 \{#create-segments\}

Adapty Ads Manager からセグメントを作成するには：

1. **Ads Manager** に移動し、**Campaigns**、**Ad groups**、または **Keywords** タブに切り替えます。使用したいエンティティの横にあるチェックボックスを選択します。複数のエンティティを選択した場合、それぞれに個別のセグメントが作成されるのではなく、すべてを対象とした1つのセグメントが作成されます。
2. **Actions > Create segment from campaigns/ad groups/keywords** をクリックします。

<ZoomImage id="create-segment-from.webp" width="700px" />
3. 必要に応じて、**Create segment** ウィンドウでセグメントの詳細を更新します：
    - **Adapty project**: このセグメントを作成したい Adapty のアプリ。
    - **Build segment from campaign/ad group**: キャンペーンまたは広告グループからセグメントを作成する場合、このステップで選択したキャンペーンや広告グループを調整できます。
    - **Segment name**
    - **Segment description**

<ZoomImage id="asa-segment.png" width="700px" />

4. **Create** をクリックします。
5. セグメントが作成されたら、使用する準備を行います：
- [プレースメント](placements)に追加して、既存のペイウォールまたはオンボーディングと組み合わせて使用する
- セグメントのユーザーに表示する新しい[ペイウォール](adapty-paywall-builder)または[オンボーディング](onboardings)をデザインする
- [A/B テスト](ab-tests)を実施する
