**Adapty Developer CLI** は、ダッシュボードを開かずに Adapty アカウントを管理するためのコマンドラインツールです。ターミナルや自動化環境から、主要な設定操作を行うことができます。

**CLI でできること:**

- Adapty アカウントへの iOS・Android アプリの作成と設定
- アクセスレベルの定義（アプリが実行時に確認するサブスクリプションのティア）
- プロダクトの設定と App Store・Google Play のストア ID へのマッピング
- ペイウォールの作成とプロダクトの割り当て
- SDK 経由でペイウォールを取得するためのプレースメントの設定

:::link
AI アシスタントや MCP クライアントを使用していますか？LLM が CLI を利用しやすくする [Adapty CLI スキル](https://github.com/adaptyteam/adapty-cli/tree/main/skills/adapty-cli) が利用可能です。
:::
