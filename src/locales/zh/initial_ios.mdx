欢迎加入 Adapty！我们的首要目标是帮助您快速上手，并为您的应用取得最佳成果。本指南旨在帮助您在应用已上架 App Store 的情况下开始使用 Adapty。

将 Adapty 集成到您的移动应用中，需要在 App Store 和 SDK 两个层面建立应用与 Adapty 之间的连接。虽然表面上看起来较为复杂，但按照 Adapty 看板中的用户引导或本指南的说明操作，您可以在 30 分钟内完成集成。

## 初始集成指南 \{#guide-for-the-initial-integration\}

- [ ] 在 Adapty 创建账户并填写您的移动应用名称和类别后，我们将在 Adapty 平台内为您完成应用的配置。
- [ ] 在 App Store Connect 中[生成应用内购买密钥](generate-in-app-purchase-key)
- [ ] 在 Adapty 看板和 App Store Connect 中[配置 App Store 集成](app-store-connection-configuration)
- [ ] 如果您的应用包含试用期或其他促销活动，请在 Adapty 看板中[配置 App Store 促销活动](app-store-connection-configuration#step-4-for-trials-and-special-offers--set-up-promotional-offers)。
- [ ] 在 App Store Connect 中[启用 App Store 服务器通知](enable-app-store-server-notifications)
- [ ] 为您使用的框架安装 Adapty SDK：
  - [ ] [为原生 iOS 安装 Adapty SDK](sdk-installation-ios)
  - [ ] [为 React Native 安装 Adapty SDK](sdk-installation-reactnative)
  - [ ] [为 Flutter 安装 Adapty SDK](sdk-installation-flutter)
  - [ ] [为 Unity 安装 Adapty SDK](sdk-installation-unity)
  - [ ] [为 Kotlin Multiplatform 安装 Adapty SDK](sdk-installation-kotlin-multiplatform)
- [ ] 构建您的应用并在沙盒模式下运行。

完成初始集成后，您即可[开始使用 Adapty 的功能](product)。

请注意，要在您的移动应用中显示付费墙和产品，以及让数据分析正常工作，您需要对应用代码进行相应修改。具体来说，您至少需要[展示付费墙](ios-quickstart-paywalls)，如果您使用的是非付费墙编辑工具创建的付费墙，还需要在应用中[处理购买流程](making-purchases)。

:::danger
在发布应用前请仔细检查发布清单

在发布应用之前，请务必仔细查阅[发布清单](release-checklist)。这将确保您在集成 Adapty SDK 的应用上线前已完成所有必要步骤。
:::
