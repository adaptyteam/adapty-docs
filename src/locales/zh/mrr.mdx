月度经常性收入（MRR）数据图表展示来自活跃付费订阅的收入，并将其换算为月度数值。无论订阅时长如何，它都能反映您的订阅业务所产生的稳定收入。

如需查看各订阅同期群随时间推移对经常性收入的贡献，可按首次购买月份对数据图表进行分组，并切换至月度视图。堆叠面积图能直观呈现每个同期群逐月的收入贡献情况。

## 计算方式 \{#calculation\}

:::warning
以下计算器**未考虑**[平台抽成和税费](how-adapty-analytics-works#commissions-and-taxes)。请将结果与您的**gross revenue**数据进行对比。
:::

MRR 将每笔订阅的收入换算为月度等值金额——例如一笔 $240 的年度订阅，每月贡献 $20，而非一次性计入 $240。这样无论订阅的计费周期如何分布，MRR 都能保持稳定。
MRR 是所有订阅类型的（价格 × 活跃订阅者数 ÷ 以月为单位的计费周期）之和。周订阅的计费周期约为 0.23 个月。
<SimpleCalculator client:load

                { label: "每月", value: 1 },
                { label: "2个月", value: 2 },

        { subscriptionPrice: 240, activeSubs: 2, duration: 12},
        { subscriptionPrice: 30, activeSubs: 10, duration: 1},
        { subscriptionPrice: 10, activeSubs: 20, duration: 0.23},
    ]}
/>
MRR 不包含：

- 一次性购买、消耗型商品和不可续订的订阅（这些不产生可预测的周期性收入）。
- 已退款的订阅——从其有效的所有周期中移除。
