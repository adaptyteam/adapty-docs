深色模式已成为改善移动应用用户体验的重要功能，尤其对于长时间与内容交互的用户而言。大多数支持深色模式的应用都会选择为付费墙提供两个不同版本（浅色和深色）以保持一致性，Adapty 正好支持这一功能。

启用深色模式后，付费墙会自动匹配设备当前的模式：如果设备处于浅色模式，则显示浅色版本；如果设备处于深色模式，则显示深色版本。

:::warning

深色模式支持以下版本：

- iOS：从 v3.1.0 开始
- Android：从 v3.1.1 开始
- React Native：从 v3.1.0 开始
- Flutter：从 v3.2.0 开始
- Unity：从 v3.3.0 开始

备用付费墙也支持深色模式。

:::

要为付费墙设置深色模式，请按以下步骤操作：

1. 首先，在付费墙的 **Layout settings** 中启用深色模式：

<Zoom>
  <img src="/assets/shared/img/dark-mode.webp"
  style={{
    border: 'none', /* border width and color */
    width: '700px', /* image width */
    display: 'block', /* for alignment */
    margin: '0 auto' /* center alignment */
  }}
/>
</Zoom>

2. 现在，您可以分别配置浅色和深色模式。要切换到深色模式，请打开付费墙左侧菜单中的 **Dark Mode** 开关：

<Zoom>
     <img src="/assets/shared/img/dark-mode-switch.webp"
     style={{
       border: 'none', /* border width and color */
       width: '700px', /* image width */
       display: 'block', /* for alignment */
       margin: '0 auto' /* center alignment */
     }}
     />
     </Zoom>

3. 切换到深色模式后，您可以根据需要调整各元素。深色模式允许您使用不同的图片或视频，以及独立的颜色和背景选项。

    width: '700px', /* image width */
    display: 'block', /* for alignment */
    margin: '0 auto' /* center alignment */
  }}
  />
  </Zoom>
