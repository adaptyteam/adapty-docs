Adapty SDK v3.0 带来了全新的 [Adapty 付费墙编辑工具](adapty-paywall-builder)支持——这是一款全新的无代码、易上手的付费墙创建工具。凭借极高的灵活性和丰富的设计能力，你的付费墙将变得更加高效、更具盈利潜力。
## 升级流程 \{#upgrade-process\}

Unity 的升级流程与其他平台相同：

1. 升级至 Adapty SDK v3.x
2. 将现有付费墙迁移至新版付费墙编辑工具

有关 Unity 专属的详细迁移说明，请参阅 [Unity SDK 安装指南](sdk-installation-unity)，并遵循主迁移指南中概述的通用迁移步骤。
