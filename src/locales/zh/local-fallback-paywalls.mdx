Adapty 会自动为你的[备用付费墙](/fallback-paywalls)生成 JSON 配置文件，每个平台对应一个文件。这些文件同时包含用户引导的备用数据。

如果某个版位下有多个付费墙或用户引导，备用版本将包含权重最高或受众范围最广的那个变体。每当你修改付费墙或用户引导时，Adapty 都会更新这些文件。

请按照以下步骤下载备用配置：

1. 打开 **[Placements](https://app.adapty.io/placements)** 页面。
2. 点击 **Fallbacks** 按钮。
3. 从下拉菜单中选择目标平台（*iOS* 或 *Android*）。
4. 选择你的 SDK 版本以开始下载。
