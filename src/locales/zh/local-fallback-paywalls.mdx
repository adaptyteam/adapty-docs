Adapty 会自动为您的[备用付费墙](/fallback-paywalls)生成 JSON 配置文件，每个平台各一份。这些文件同时包含您的[用户引导](local-fallback-onboarding)的备用数据。

如果某个版位包含多个付费墙或用户引导，备用版本将使用权重最高或目标受众最广的实验变体。每当您修改付费墙或用户引导时，Adapty 都会更新这些文件。

请按照以下步骤下载您的备用配置文件：

1. 打开 **[Placements](https://app.adapty.io/placements)** 页面。
2. 点击 **Fallbacks** 按钮。
3. 从下拉菜单中选择目标平台（*iOS* 或 *Android*）。
4. 选择您的 SDK 版本以开始下载。
