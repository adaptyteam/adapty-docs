Apple Ads Manager 与 Apple Ads 实现了双向集成：您可以获得近实时的效果数据，还可以直接在 Adapty 看板中创建和编辑广告，比在原生界面操作更加便捷。

如果您在原生 Apple Ads 看板中创建了广告，它将在 24 小时内自动出现在 Apple Ads Manager 中。

## 什么是广告 \{#what-are-ads\}

广告是分配到[广告组](ads-manager-create-ad-group)中的广告素材，广告组隶属于某个[广告系列](ads-manager-create-campaign)。每个广告组可以分配一个有效广告。

## 创建广告 \{#create-ads\}

开始之前，请确保您已创建以下内容：

- **广告组**。您可以[直接在 Apple Ads Manager 看板中创建](ads-manager-create-ad-group)。
- **自定义产品页面**。您需要[直接在 Apple Ads 中进行设置](https://developer.apple.com/help/app-store-connect/create-custom-product-pages/configure-multiple-product-page-versions/)。该页面必须经过 App Store 审核通过后，才能在广告中使用。

:::note
如果所选广告组中已有一个有效广告，该广告将被暂停，以运行新广告。
:::

要创建新的 Apple Ads 广告：

1. 从侧边栏菜单进入 **Ads Manager**。在任意选项卡中，点击表格上方的 **+**，然后选择 **Create ad**。

<ZoomImage id="create-ad.webp" width="900px" />

2. 选择您要投放广告的应用。
3. 为广告选择广告组。
4. 输入广告名称。
5. 选择广告状态。如果您希望稍后再开始投放广告，请清除 **Status** 复选框。

<ZoomImage id="create-ad-meta.webp" />

6. 点击 **Select CPP**。您将看到应用中所有经 App Store 审核通过的自定义产品页面。您只能选择一个自定义产品页面。

<ZoomImage id="select-cpp.gif" />

7. 点击 **Create ad**。

## 编辑广告 \{#edit-ads\}

:::note
广告创建后，您只能编辑其名称，无法更改其自定义产品页面或将其移动到其他广告组。
:::

要编辑广告名称，请使用以下任意方式：

- 在 **Ads Manager > Ads** 中点击广告名称，编辑广告名称后点击旁边的对勾图标。
  <ZoomImage id="edit-ad-name.gif" width="500px" />

- 选中广告名称旁边的复选框，然后点击 **Actions > Edit ad**。修改广告名称后点击 **Save changes**。
   <ZoomImage id="edit-ad-name.webp" width="500px"/>

:::note
直接在 Apple Ads 中对广告进行的编辑会自动同步到 Apple Ads Manager，但可能需要一些时间才能在 Apple Ads Manager 中显示。
:::

## 启动与暂停广告 \{#launch--pause-ads\}

要在 Apple Ads Manager 中启动或暂停任意广告，请使用以下任意方式：

- 在 **Ads Manager > Ads** 的 **Status** 列中选中或清除复选框。
  <ZoomImage id="ad-status.webp" width="500px"/>

- 选中广告名称旁边的复选框，然后点击 **Actions > Edit ad**。修改广告名称后点击 **Save changes**。
  <ZoomImage id="edit-ad-name.webp" width="500px"/>
