Apple Ads Manager 与 Apple Ads 实现了双向集成：您可以获得近实时的效果数据，并且可以直接在 Adapty 看板中创建和编辑广告，比原生界面更加便捷。
如果您在原生 Apple Ads 看板中创建广告，该广告将在 24 小时内自动显示在 Apple Ads Manager 中。
## 什么是广告 \{#what-are-ads\}

广告是分配给[广告组](ads-manager-create-ad-group)（位于[广告系列](ads-manager-create-campaign)内）的广告素材。每个广告组可分配一个有效广告。
## 创建广告 \{#create-ads\}

在开始之前，请确保您已创建以下内容：

- **Ad group**。您可以[直接在 Apple Ads Manager 看板中创建](ads-manager-create-ad-group)。
- **Custom product page**。您需要[直接在 Apple Ads 中进行设置](https://developer.apple.com/help/app-store-connect/create-custom-product-pages/configure-multiple-product-page-versions/)。该页面必须经过 App Store 审核通过后，才能在广告中使用。

:::note
如果所选广告组中已有正在运行的广告，该广告将被暂停，以便运行新广告。
:::

创建新的 Apple Ads 广告：
1. 从侧边栏菜单进入 **Ads Manager**。在任意标签页中，点击表格上方的 **+** 并选择 **Create ad**。

<ZoomImage id="create-ad.webp" width="900px" />

2. 选择您要投放广告的应用。
3. 为广告选择一个广告组。
4. 输入广告名称。
5. 选择广告状态。如果您希望稍后再开始投放广告，请取消勾选 **Status** 复选框。

<ZoomImage id="create-ad-meta.webp" />

6. 点击 **Select CPP**。您将看到应用在 App Store 审核通过的所有自定义产品页面。您只能选择一个自定义产品页面。

<ZoomImage id="select-cpp.gif" />
7. 点击 **Create ad**。
## 编辑广告 \{#edit-ads\}

:::note
创建广告后，您只能编辑其名称，无法更改其 CPP 或将其移动到其他广告组。
:::

要编辑广告名称，请使用以下任一方式：

- 在 **Ads Manager > Ads** 中点击广告名称，编辑后点击旁边的对勾图标。
  <ZoomImage id="edit-ad-name.gif" width="500px" />

- 勾选广告名称旁边的复选框，然后点击 **Actions > Edit ad**。修改广告名称后点击 **Save changes**。
   <ZoomImage id="edit-ad-name.webp" width="500px"/>
:::note
直接在 Apple Ads 中对广告所做的编辑会自动同步到 Apple Ads Manager，但可能需要一些时间才能在 Apple Ads Manager 中显示。
:::
## 导出广告 \{#export-ads\}

如需将广告表格导出为 CSV 文件，请点击表格上方的下载图标，然后选择 **Export current page** 或 **Export all pages**。

**Export all pages** 会将所有分页中的广告数据合并到一个文件中下载。下载进度弹窗会实时显示进度，您可以随时取消。此外，还提供两个可选筛选条件：

- **Enabled only**：仅包含处于启用状态的广告。
- **With spend ≥**：仅包含花费超过指定阈值的广告。
- **Group by country**：将每条广告数据按国家/地区拆分展示。
表格将按照您在看板上的显示效果导出，包含您选择展示的列。

<ZoomImage id="export-ads.webp" width="700px" alt="Export ads" />
## 启动与暂停广告 \{#launch--pause-ads\}

要从 Apple Ads Manager 启动或暂停任何广告，请使用以下任一方式：

- 在 **Ads Manager > Ads** 的 **Status** 列中选中或取消选中复选框。
  <ZoomImage id="ad-status.webp" width="500px"/>

- 选中广告名称旁边的复选框，然后点击 **Actions > Edit ad**。修改广告名称后点击 **Save changes**。
  <ZoomImage id="edit-ad-name.webp" width="500px"/>
