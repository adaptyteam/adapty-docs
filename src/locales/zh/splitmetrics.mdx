通过 [SplitMetrics Acquire](https://splitmetrics.com/acquire/) 集成，您可以精确了解 Apple Search Ads 从订阅中获得的收益，并追踪用户数月，以了解广告随时间产生的收益。

此外，Adapty 会将[订阅事件](events)发送至 SplitMetrics Acquire，使您可以在那里基于 Apple Search Ads 归因构建自定义看板和自动化流程。
该集成不会向 Adapty 添加任何归因数据，因为我们已经直接从 ASA 获取了所需的全部信息。

## 如何设置 SplitMetrics Acquire 集成 \{#how-to-set-up-splitmetrics-acquire-integration\}

要集成 SplitMetrics Acquire，请前往 [Integrations > SplitMetrics Acquire](https://app.adapty.io/integrations/splitmetrics) 并填写凭据。

我们建议使用 Adapty 提供的默认事件名称。当然，您也可以根据需要修改事件名称。Adapty 将通过服务端对服务端集成将订阅事件发送至 SplitMetrics Acquire，让您可以在 SplitMetrics 看板中查看所有订阅事件。

## SDK 配置 \{#sdk-configuration\}

您无需在 SDK 端进行任何配置，但我们建议向 Adapty 发送 `customerUserId` 以提高准确性。

:::warning
请确保您已在 Adapty 中配置了 [Apple Search Ads](apple-search-ads) 并[上传了凭据](https://app.adapty.io/settings/apple-search-ads)，否则 SplitMetrics Acquire 将无法正常工作。
:::

## 故障排查 \{#troubleshooting\}

如果在正确设置后 SplitMetrics Acquire 集成仍无法正常工作，请检查以下几点：

- 确保已在 [App Settings -> Apple Search Ads tab](https://app.adapty.io/settings/apple-search-ads) 中启用了 **Receive Apple Search Ads attribution in Adapty** 开关，并在 Adapty 中配置了 [Apple Search Ads](apple-search-ads) 且[上传了凭据](https://app.adapty.io/settings/apple-search-ads)，否则 SplitMetric 将无法工作。
- 确认用户画像具有非自然流量的 ASA 归因。只有拥有详细非自然流量 ASA 归因的用户画像才会将其事件发送至 Adapty。

## SplitMetrics Acquire 事件结构 \{#splitmetrics-acquire-event-structure\}

Adapty 通过 GET 请求以查询参数的形式将事件发送至 SplitMetrics Acquire。每个事件的结构如下：

```json
{
  "source": "Apple Search Ads",
  "app_id": "123456789",
  "name": "subscription_renewed",
  "type": "subscription_renewed",
  "revenue": 9.99,
  "currency": "USD",
  "tap_time": "2024-03-01 12:00:00",
  "open_time": "2024-03-01 12:05:00",
  "event_time": "2024-03-02 12:00:00",
  "adaccount_id": "123456",
  "campaign_id": "123456789",
  "adgroup_id": "123456789",
  "keyword_id": "123456789",
  "creative_set_id": "123456789",
  "Ad_id": "123456789",
  "country_or_region": "US",
  "conversion_type": "Download",
  "user_id": "user_12345",
  "att_status": "3",
  "device_type": "iphone",
  "app_version": "1.2.3",
  "sdk_version": "2.10.0",
  "ios_version": "17.2",
  "event_value": "{\"vendor_product_id\":\"yearly.premium.6999\",\"original_transaction_id\":\"GPA.3383...\"}",
  "event_id": "123e4567-e89b-12d3-a456-426614174000"
}
```

参数说明：

| 参数                  | 类型     | 描述                                                                                                            |
|:--------------------|:-------|:--------------------------------------------------------------------------------------------------------------|
| `source`            | String | 始终为 "Apple Search Ads"。                                                                                       |
| `app_id`            | String | Apple App ID。                                                                                                  |
| `name`              | String | 事件名称（从 Adapty 事件映射而来）。                                                                                        |
| `type`              | String | 事件类型（与 `name` 相同）。                                                                                            |
| `revenue`           | Float  | 收益金额。                                                                                                         |
| `currency`          | String | 货币代码。                                                                                                         |
| `tap_time`          | String | 广告点击的日期和时间。                                                                                                   |
| `open_time`         | String | 应用打开（安装）的日期和时间。                                                                                               |
| `event_time`        | String | 事件发生的日期和时间。                                                                                                   |
| `adaccount_id`      | String | ASA 组织 ID。                                                                                                    |
| `campaign_id`       | String | ASA 广告系列 ID。                                                                                                  |
| `adgroup_id`        | String | ASA 广告组 ID。                                                                                                   |
| `keyword_id`        | String | ASA 关键词 ID。                                                                                                   |
| `creative_set_id`   | String | ASA 创意素材组 ID。                                                                                                 |
| `Ad_id`             | String | ASA 广告 ID。                                                                                                    |
| `country_or_region` | String | 商店所在国家或地区。                                                                                                    |
| `conversion_type`   | String | 转化类型（例如 "Download"）。                                                                                          |
| `user_id`           | String | 客户用户 ID 或 Adapty 用户画像 ID。                                                                                     |
| `att_status`        | String | 追踪使用状态（0-3）。                                                                                                  |
| `device_type`       | String | 设备类型（例如 "iphone"、"ipad"）。                                                                                     |
| `app_version`       | String | 应用程序版本。                                                                                                       |
| `sdk_version`       | String | Adapty SDK 版本。                                                                                                |
| `ios_version`       | String | iOS 版本。                                                                                                       |
| `event_value`       | String | 包含所有可用[事件详情](webhook-event-types-and-fields#for-most-event-types)的 JSON 字符串。 |
| `event_id`          | String | 唯一事件 ID（UUID）。                                                                                                |
