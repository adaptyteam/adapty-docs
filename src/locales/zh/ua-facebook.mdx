Adapty UA 的 Meta 集成让您可以跨 Facebook、Instagram、Messenger 和 Audience Network 追踪和优化广告系列效果。

:::tip
请参阅我们的 [Meta Ads Manager 广告投放设置指南](meta-create-campaign)。
:::

## 第一步：连接您的 Facebook 账户 \{#step-1-connect-your-facebook-account\}

要将 Meta Ads 连接到 Adapty UA，请从左侧边栏进入 **Integrations > Meta**。您有两种连接方式：

- **Continue with Facebook**：通过 OAuth 连接。如果您使用个人或企业 Facebook 账户登录 Meta Ads Manager，请选择此方式。
- **Add system token**：使用永久系统用户令牌进行连接。如果您的组织通过 Meta Business 系统用户管理广告账户，请选择此方式。

<Tabs>
<TabItem value="oauth" label="Continue with Facebook">
:::important
确保您的 Facebook 账户有权访问您需要的广告活动和像素。
:::

1. 点击 **Continue with Facebook**。

<ZoomImage id="ua-connect-meta.webp" width="700px" />

2. 使用您的 Facebook 账户登录，然后点击 **Continue**。

<ZoomImage id="ua-fb-sign.webp" width="500px" />

3. 查看所请求的权限，然后点击 **Save**。

</TabItem>
<TabItem value="system" label="Add system token">
1. 在 Meta Business Manager 中[生成系统用户访问令牌](https://developers.facebook.com/documentation/ads-commerce/marketing-api/collaborative-ads/managed-partner-ads/api-guide/prerequisites/generate-access-token-system-user)。

2. 点击 **Add system token**。

<ZoomImage id="ua-add-meta-system-token.webp" width="700px" alt="Add system token button on the Meta integration page in Adapty UA" />

3. 粘贴令牌并点击 **Connect**。

<ZoomImage id="ua-connect-meta-system-token.webp" width="700px" alt="System token input field on the Meta integration page in Adapty UA" />
</TabItem>
</Tabs>

完成后，您的所有广告账户将被添加到 Adapty UA。您可以继续添加广告系列。
## 第二步：添加广告系列 \{#step-2-add-campaigns\}

要将 Meta 广告系列添加到 Adapty 用户获取并追踪 Meta 广告在 Adapty 中的效果：

1. 切换到 **Web campaigns** 选项卡，点击 **Create campaign**。选择应用后点击 **Save**。

<ZoomImage id="ua-new-campaign.webp" width="500px" />

2. 在 **General** 选项卡中，展开 **iOS** 和/或 **Android** 部分，粘贴 App Store 和/或 Google Play 的应用 URL。

<ZoomImage id="ua-url.gif" width="900px" />

3. 复制 **Click link** 字段的值。然后在 Meta Ads Manager 中打开你的广告，将此链接粘贴进去。这样 Adapty 就能将安装和购买行为关联到 Meta 上的广告。

<ZoomImage id="ua-lnk.webp" width="900px" />

4. 如需将转化事件回传给 Meta，你还可以在 Adapty UA 中将 Meta 的像素与广告系列关联。操作方法：在 **Pixel** 下拉菜单中选择一个已有的像素。
## 第三步：映射事件 \{#step-3-map-events\}

要将转化事件回传至 Meta 以优化广告系列，您需要在 **Events names** 部分配置事件映射。这将使 Adapty 在用户在应用内执行操作时，自动将订阅事件发送到您的 Meta 像素。

在 **Events names** 部分，开启您想在 Meta Ads Manager 中追踪的事件。对于每个已启用的事件，从下拉菜单中选择对应的 Meta 事件，或设置自定义事件。默认情况下，Adapty 会将 Adapty 事件映射到 Meta 的标准事件。

点击 **Save** 应用您的事件映射配置。

## 其他配置 \{#additional-configuration\}

### 附加参数 \{#additional-parameters\}

**Additional parameter** 字段允许您添加自定义数据点，用于在 Adapty 外部进行分析。当您需要将特定的广告系列或用户数据传递给外部分析工具或归因合作伙伴时，此功能非常有用。

在 **Additional parameter** 字段中，输入您希望包含在归因追踪中的任何自定义数据。附加参数将包含在发送至 Meta 的所有归因数据中，可用于高级广告系列分析和优化。

例如，如果您正在运行同一广告系列的多个变体，可以添加 `variant=A` 或 `variant=B` 来区分不同的创意方案。

:::important
附加参数会更改您在 Meta Ads Manager 中粘贴的**点击链接**。如果您已经复制了该链接并在之后添加了自定义参数，请确保复制并粘贴包含自定义参数的最新点击链接。
:::

### 设置 \{#settings\}

**Settings** 标签页用于控制 Adapty 如何将用户行为与你的 Meta Ads 广告活动进行匹配。这些设置决定了确定性归因和概率性归因匹配的时间窗口。

如需配置，请前往广告活动配置中的 **Settings** 标签页。你将看到两个主要设置项：
- **确定性匹配窗口**：使用精确的设备标识符（如 iOS 上的 IDFA 或 Android 上的 Advertising ID）将用户与广告活动进行高精度匹配。建议将其设置为 168 小时（7 天）以获得最佳归因准确性——这也是默认推荐值。当用户点击 Meta 广告并在此窗口内安装应用时，Adapty 可以通过设备标识符明确地将该安装归因到具体的广告点击。
- **Probabilistic matching window**：当确定性匹配不可用时，系统会使用统计建模和设备指纹技术来匹配用户。大多数广告活动建议将此项设置为 6 小时——这是默认值，适用于大多数场景。对于点击量较高的广告活动，可以将其缩短至 1-2 小时。对于因隐私设置或其他原因无法进行确定性匹配的用户，Adapty 会在此较短的时间窗口内使用概率匹配。

点击 **Save** 以应用您的设置。

<ZoomImage id="ua-meta-attribution-settings.webp" width="900px" />
### 收入覆盖 \{#revenue-override\}

如果您追踪试用事件并希望 Meta 将收入归因于这些事件，请使用 **Revenue override** 部分。当 **Trial started** 事件启用时，该部分会显示出来。

对于每个试用事件目标，请设置订阅价格的百分比作为上报收入。例如，设置为 30% 时，Adapty 会将订阅价格的 30% 作为试用事件的转化值发送给 Meta。

要添加覆盖规则：
1. 在 **Events names** 部分启用 **Trial started**。
2. 在 **Revenue override** 中，点击 **Add override**。
3. 选择目标事件并输入收入百分比（0–100）。
4. 点击 **Save**。

<ZoomImage id="revenue-override.webp" width="500px" />
### 发送所有事件 \{#send-all-events\}

默认情况下，Adapty 仅将归因于 Meta 广告系列的用户事件发送到你的 Pixel。启用 **Send all events** 后，来自自然流量和未归因用户的事件也会转发到 Pixel。

启用后，无论广告系列归因如何，每次安装和交易事件都会发送到 Pixel。这样可以为 Meta 提供更广泛的转化数据，用于受众建模和广告系列优化。

要启用此选项，请在广告系列设置中选择 **Send all events (forward organic/non-attributed events to this pixel)**，然后点击 **Save**。
