Adapty 根据 HTTP 状态码判断是否成功送达，将 `200-399` 范围之外的所有响应视为失败。

您可以在 Adapty 看板的 **Event List** 中跟踪集成事件的状态。无论特定集成是否启用了某种事件类型，系统都会显示所有已启用集成的状态。

- 黑色：事件已成功发送。
- <span style={{ color: 'grey' }}>灰色：</span>该事件类型在此集成中已禁用。
- <span style={{ color: 'red' }}>红色：</span>集成存在需要关注的问题。

如需查看失败事件的详细信息，请将鼠标悬停在集成名称上，即可看到包含具体错误信息的提示框。
