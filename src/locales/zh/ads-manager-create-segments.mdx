您可以通过选择广告系列、广告组或关键词，直接在 [Apple Ads Manager](adapty-ads-manager) 中创建用户[市场细分](segments)，并通过几次点击将其转化为细分。这样便可轻松根据获客来源为付费墙和优惠进行个性化设置，无需手动配置细分条件。

创建市场细分后，您可以使用它来分配不同的产品和价格、运行 A/B 测试以及自定义付费墙外观。

## 使用场景 \{#use-cases\}

以下是基于 Apple Ads 数据创建的市场细分在实践中的一些应用示例：

- **基于关键词的付费墙**。向来自高意向关键词的用户展示以功能为重点的付费墙，向来自更广泛发现类关键词的用户展示通用付费墙。

- **广告系列级别的优惠**。为来自特定 Apple Ads 广告系列的用户提供更长的试用期或特别定价，同时为其他用户保持标准优惠。

- **广告创意与付费墙的一致性**。将来自推广特定功能的广告组的用户引导至优先展示这些功能的付费墙。

- **高 ROI 广告系列优化**。向来自持续带来更高生命周期价值的广告系列的用户，展示以高级版为主的全价付费墙。

## 创建市场细分 \{#create-segments\}

要从 Apple Ads Manager 创建市场细分：

1. 前往 **Ads Manager**，切换至 **Campaigns**、**Ad groups** 或 **Keywords** 标签页。勾选您想要使用的实体旁边的复选框。请注意，如果您选择多个实体，它们将被用于为所有实体创建一个市场细分，而不是为每个实体分别创建一个细分。
2. 点击 **Actions > Create segment from campaigns/ad groups/keywords**。

<ZoomImage id="create-segment-from.webp" width="700px" />

3. 如有需要，在 **Create segment** 窗口中更新细分详情：
    - **Adapty project**：您希望在 Adapty 中创建此细分的应用。
    - **Build segment from campaign/ad group**：从广告系列或广告组创建细分时，您可以在此步骤调整所选的广告系列或广告组。
    - **Segment name**
    - **Segment description**

<ZoomImage id="asa-segment.png" width="700px" />

4. 点击 **Create**。
5. 现在，当您的市场细分创建完成后，您可以准备使用它：

- 将其添加到[版位](placements)，以便与现有的付费墙或用户引导配合使用
- 设计一个新的[付费墙](adapty-paywall-builder)或[用户引导](onboardings)，该内容将展示给细分中的用户
- 运行 [A/B 测试](ab-tests)
