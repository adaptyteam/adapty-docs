Adapty 的付费墙编辑工具允许你自定义产品及其优惠的所有文字内容。如果你需要支持多语言，我们强烈建议使用变量。

### 工作原理 \{#how-it-works\}

当你在产品文字中添加我们列表中的标签变量时，SDK 会从应用商店拉取预先获取的本地化数据来替换这些标签。这样可以确保付费墙上的文字始终精准匹配当前语言环境。

**示例**：假设你有一个"高级订阅"产品，同时面向美国和西班牙市场。在美国，它可能显示为"Premium Subscription for $4.99/month"，而在西班牙则会显示为"Suscripción Premium por €4.99/mes"。

标签变量可以根据直接来自商店的数据自动本地化这些字符串，确保标题和价格始终准确无误。

### 如何使用标签变量 \{#how-to-use-tag-variables\}

:::note
标签变量只能用于付费墙编辑工具的"产品"组件中，用来描述产品及其优惠内容。
:::

1. 在付费墙编辑工具的左侧面板中，选择你想要自定义的产品。
2. 在任意文本字段中使用[下方表格](paywall-builder-tag-variables#full-list-of-variables)中的变量来描述产品及其优惠。

### 完整变量列表 \{#full-list-of-variables\}

| 标签变量                  | 说明                                                                                                                                                                                                        | 示例                 |
| :------------------------ | :---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | :------------------- |
| `<PROD_TITLE/>`           | 产品的本地化名称                                                                                                                                                                                            | Premium Subscription |
| `<PROD_PRICE/>`           | 产品的本地化价格。对于订阅，此为单个计费周期的价格。                                                                                                                                                        | $9.99                |
| `<PROD_PRICE_PER_DAY/>`   | 订阅价格除以计费周期天数所得的每日价格。**非订阅类产品返回空字符串。**                                                                                                                                      | $0.33                |
| `<PROD_PRICE_PER_WEEK/>`  | 订阅价格除以计费周期周数所得的每周价格。**非订阅类产品返回空字符串。**                                                                                                                                      | $2.33                |
| `<PROD_PRICE_PER_MONTH/>` | 订阅价格除以计费周期月数所得的每月价格。若计费周期不足一个月，则调整为代表完整一个月的价格。**非订阅类产品返回空字符串。**                                                                                  | $9.99                |
| `<PROD_PRICE_PER_YEAR/>`  | 调整为代表完整一年使用费用的订阅价格。例如，月度订阅价格将乘以 12，而年度价格保持不变。**非订阅类产品返回空字符串。**                                                                                       | $119.88              |
| `<OFFER_PRICE/>`          | 优惠（新用户优惠或促销活动）的本地化价格。**仅适用于自动续费订阅，若用户不符合任何优惠资格则返回空字符串。**                                                                                               | $0.99                |
| `<OFFER_BILLING_PERIOD/>` | 优惠（新用户优惠或促销活动）的本地化计费周期。对于免费试用和预付优惠，与 `<OFFER_FULL_DURATION/>` 相同。**仅适用于自动续费订阅，若用户不符合任何优惠资格则返回空字符串。**                                  | 1 week               |
| `<OFFER_FULL_DURATION/>`  | 优惠（新用户优惠或促销活动）的本地化完整持续时长。**仅适用于自动续费订阅，若用户不符合任何优惠资格则返回空字符串。**                                                                                       | 1 month              |

### 不同优惠类型的优惠标签 \{#offer-tags-for-different-offer-types\}

:::note
你可以在[这里](offers)了解更多关于优惠及其在 Adapty 中的配置方式。
:::

优惠标签理解起来可能有点复杂，我们通过一个例子来说明。假设你有一个每周订阅产品"高级订阅"，价格为 $5，包含三种可能的优惠：

- **按用量付费（Pay As You Go）**：前 3 周每周 $3，之后每周 $5。
- **预付（Pay Up Front）**：前 3 周一次性付 $8，之后每周 $5。
- **免费试用（Free Trial）**：第一周免费，之后每周 $5。

对于此产品，`<PROD_TITLE/>` 为"高级订阅"，`<PROD_PRICE/>` 为 $5。而优惠标签的值会根据用户符合资格的优惠类型有所不同：

| 标签变量                  | 按用量付费    | 预付        | 免费试用   |
| :------------------------ | :------------ | :---------- | :--------- |
| `<OFFER_PRICE/>`          | $3            | $8          | $0         |
| `<OFFER_BILLING_PERIOD/>` | 1 week        | 3 weeks     | 1 week     |
| `<OFFER_FULL_DURATION/>`  | 3 weeks       | 3 weeks     | 1 week     |

因此，除"按用量付费"类型之外的其他优惠，`<OFFER_BILLING_PERIOD/>` 和 `<OFFER_FULL_DURATION/>` 的值将保持一致。
