[Apple Ads Manager](adapty-ads-manager) 是您用于 Apple Ads 的优化与分析平台。

本指南将帮助您通过以下两步开始使用 Apple Ads Manager：

1. 安装 Adapty SDK，让其追踪您的购买数据。
2. 将 Apple Ads Manager 连接到您的 Apple Ads 账户，以导入历史数据并开始追踪实时更新。

:::note
Apple Ads Manager 不使用 **App settings** 中的 [Apple Ads 集成](apple-search-ads)。

要使用 Apple Ads Manager，您只需完成本指南中描述的设置即可。
:::

## 1. 安装 Adapty SDK \{#1-install-the-adapty-sdk\}

:::important
Adapty Ads Manager 是一款**独立产品**。即使您的付费墙、订阅或分析并非由 Adapty 处理，您也可以使用它——无需将整个技术栈迁移到 Adapty。

要获取准确的收入数据，最低配置是以观察者模式安装 Adapty SDK，并在 Adapty 中启用 App Store 服务器通知。
:::

要将您的收入数据与广告系列效果关联起来，请让 Adapty 追踪您的购买行为：

1. 第一步取决于您是否已经实现了应用内购买：
   - 如果您**已经使用 Adapty 实现了应用内购买**，此阶段无需进行任何其他操作。
   - 如果您**已经在不使用 Adapty 的情况下实现了应用内购买**，且不打算迁移到 Adapty，请以观察者模式为您的平台安装 Adapty SDK。此阶段您只需将 SDK 添加到项目中，以观察者模式激活它，并上报交易记录：
     - [iOS](implement-observer-mode)
     - [Android](implement-observer-mode-android)
     - [React Native](implement-observer-mode-react-native)
     - [Flutter](implement-observer-mode-flutter)
     - [Unity](implement-observer-mode-unity)
     - [Kotlin Multiplatform](implement-observer-mode-kmp)
     - [Capacitor](implement-observer-mode-capacitor)
   - 如果您**尚未实现应用内购买且希望使用 Adapty**，请完成[快速入门指南](quickstart)中的步骤，将购买处理委托给 Adapty。
2. 要直接从 App Store 接收与收入相关的更新，请[在 Adapty 中启用 App Store 服务器通知](enable-app-store-server-notifications)。

## 2. 连接 Apple Ads \{#2-connect-apple-ads\}

:::important
您需要在 Apple Ads 中拥有 **Account Admin** 角色才能将 Apple Ads 连接到 Adapty。
:::

现在，您需要将 Apple Ads Manager 账户连接到您的 Apple Ads 账户：

1. 在 Adapty 看板上，点击左上角切换到 Apple Ads Manager。

<ZoomImage id="apple-ads.webp" width="500px" />

2. 点击 **Continue with Apple**。

<ZoomImage id="continue-with-apple.webp" width="700px" />

3. 登录您的 Apple 账户。
4. 选择您希望授予 Apple Ads Manager 的访问权限：
- **Read and Write**：提供对所有广告系列组的访问权限。
- **Limited access**：选择特定广告系列组，并分配 **Read & Write** 角色，以仅授予对这些组的访问权限。

<ZoomImage id="apple-ads-access.webp" width="700px" />

5. 点击 **Grant access**。

<ZoomImage id="grant-access.webp" width="300px" />

完成后，Adapty 将开始从 Apple Ads 同步您的历史数据。您可以立即开始探索 Adapty Ads Manager，但所有历史数据的导入需要一些时间。
