查阅 Adapty 的分步指南，了解如何将 Adapty SDK 与 Webhook 及 ETL 选项（如 Amazon S3 和 Google Cloud Storage）进行集成。

通过 Webhook 集成，您可以实时接收有关用户行为和事件的通知。这些通知可以自定义，并推送至您指定的端点，让您能够轻松监控和分析应用数据。请参阅以下文档，深入了解 Adapty 的 Webhook 集成及其在应用中的实现方式：

- [Webhook](webhook)

Adapty 提供了一项便捷功能，可自动定期推送与您应用相关的所有交易数据。借助该功能，您可以轻松地将交易数据每日导出至多种云存储服务提供商。数据以 gzip 压缩的 .csv 文件格式上传，便于高效存储和分析。请参阅以下易于使用的指南，高效管理您的数据并优化数据管理流程：

- [Amazon S3](s3-exports)
- [Google Cloud Storage](google-cloud-storage)
