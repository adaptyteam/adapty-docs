将分析数据导出为 CSV 格式，可以让您更灵活地深入了解应用的性能指标、自定义报告并分析长期趋势。借助 Adapty API，您可以轻松将详细的分析数据提取为 CSV 格式，方便您随时跟踪、共享和优化数据洞察。

## 分析导出 API 入门 \{#getting-started-with-the-api-for-analytics-export\}

通过分析导出 API，您可以实现以下操作：

1. **分析营销活动带来的 MRR**：衡量去年营销活动在特定国家/地区的影响，按周跟踪以了解哪些活动带来了最高收益。可使用 [获取分析数据](api-export-analytics/operations/retrieveAnalyticsData) 方法。

2. **跟踪同期群随时间变化的留存率**：按同期群跟踪留存情况，找出流失节点并比较不同同期群的表现，从而发现趋势和关键时机，以便通过互动策略提升留存率。支持限定特定应用商店、特定国家/地区和特定产品。可使用 [获取同期群数据](api-export-analytics/operations/retrieveCohortData) 方法。

3. **评估各渠道的转化率**：分析关键获客渠道的转化率，了解哪些渠道在推动首次购买方面最为有效，从而优先将营销预算投入高效渠道。可使用 [获取转化数据](api-export-analytics/operations/retrieveConversionData) 方法。

4. **查看流失率**：监控用户取消订阅的速度，以发现流失规律或评估留存工作的成效，支持聚焦特定国家/地区和特定产品。可使用 [获取漏斗数据](api-export-analytics/operations/retrieveFunnelData) 方法。

5. **按用户市场细分评估 LTV**：识别不同用户市场细分的生命周期价值，了解哪些群体随时间推移带来了最高收益。重点关注高价值市场细分（如长期订阅用户），并利用结果优化获客策略。可使用 [获取 LTV 数据](api-export-analytics/operations/retrieveLTVData) 方法。

6. **按国家/地区查看留存率**：分析各地区的留存率，发现高互动市场，为本地化或区域策略提供指导。可使用 [获取留存数据](api-export-analytics/operations/retrieveRetentionData) 方法。
