只需单击即可添加标题、段落或列表，将其样式化以匹配你的品牌，并使用[动态变量](onboarding-variables)为每位用户个性化内容。
## 设置文字样式 \{#set-up-text-styles\}

**Styles** 面板包含预配置的文字样式：H1、H2、H3、Button Label、Body、Caption 和 Small Label。点击任意样式可编辑其字体族、字重、字号、对齐方式、装饰效果及其他属性。

当你[添加文字元素](#add-text)时，可以从这里设置好的样式中进行选择。

:::important
对某个样式的修改会应用于所有使用该样式的文字元素，且对所有屏幕均生效。
:::

要创建新样式：

1. 点击 <Inline id="Styles.webp" alt="Palette"/> 打开 **Styles** 面板。
2. 在 **Text** 标签页中，点击 **Create style**。
<ZoomImage id="new-text-style.webp" width="500px" alt="新文字样式" />
3. 输入名称并配置排版样式：
- **Font family**：选择字体。关于自定义字体的使用方法，请参阅[指南](using-custom-fonts-in-paywall-builder)。
- **Weight** 和 **size**：设置字体粗细和大小。
- **Color**：设置文字颜色。
- **Line height**：设置行间距，或保持 **Auto** 不变。
- **Alignment**：设置水平对齐（左、中、右）和垂直对齐（上、中、下）。
- **Decoration**：添加下划线或删除线效果。
- **Truncate**：限制显示的行数，超出部分将被截断。当内容因动态变量或本地化而长度不一时，此功能尤为实用。
4. 点击 **Create**。
:::note
SF Pro 不支持跨平台使用。我们推荐使用 Inter 作为近似替代字体。
:::

<ZoomImage id="text-style-details.webp" width="500px" alt="文字样式配置选项" />
## 添加文本 \{#add-text\}

添加文本元素的步骤如下：
1. 点击左上角的 **+**，选择 **Text**，然后从预置或自定义的[文本样式](#set-up-text-styles)中选择一种。

<ZoomImage id="new-text.webp" width="500px" alt="New text" />
2. 点击新元素，在右侧 **Design** 面板的 **Content** 部分编辑其内容。
3. 如需调整排版，可在 **Design** 面板中进行设置。或者，在预览区选中文本，通过弹出的工具提示快速自定义样式。
4. 可选操作：在 **Design** 和 **Interaction** 面板中，你还可以对流程中的组件应用其他可用配置。详情请参阅[元素样式设置](builder-styling)。
:::tip
如果需要在多个屏幕上使用相同的文本元素，可以直接复制粘贴：选中该元素，按 Ctrl+C（Mac 上按 ⌘+C），切换到另一个屏幕，再按 Ctrl+V（Mac 上按 ⌘+V）粘贴即可。
:::

<ZoomImage id="text-settings.webp" width="500px" alt="文本元素设置" />
### 更改部分文本的样式 \{#change-styling-for-parts-of-text\}

如需仅更改文本元素中某一部分的样式：

1. 在 **Content** 部分选中文本元素的某一段文字。
2. 在弹出的工具提示中，更改文字颜色、应用粗体、下划线、斜体或删除线格式，或添加 URL。

预览会立即更新。

## 添加条件文本 \{#add-conditional-text\}

条件文本会根据条件改变文本元素显示的内容。例如，当用户选择年度方案时显示一条信息，选择月度方案时显示另一条信息。条件文本的工作原理与[条件可见性](onboarding-element-visibility)类似，区别在于它是替换内容，而不是显示或隐藏元素。

设置条件文本的步骤如下：

1. 在画布上选择一个文本元素。
2. 在 **Design** 面板的 **Content** 部分，选择 **Conditional**。
<ZoomImage id="flow-conditional-text.webp" width="500px" alt="在设计面板中将内容设置为条件式" />
3. 构建 **if** 条件。从 **Custom**、**Products** 或 **Elements** 标签页中选择一个属性，设置运算符，并输入要匹配的值。有关属性类型的详细信息，请参阅[条件可见性](onboarding-element-visibility)。
4. 在 **then** 下，输入条件为真时要显示的文本。富文本格式设置方式与[普通文本](#change-styling-for-parts-of-text)相同。要插入[变量](onboarding-variables)，请点击 **&#123; &#125; Add variable**。
5. 在 **else** 下，输入当没有条件匹配时显示的备用文本。
6. （可选）点击 **+ Add else/if** 添加更多条件，每个条件都有各自对应的文本。
:::tip
如需编辑其他语言的条件文本，请在编辑器底部切换当前语言环境。请先在 **Localizations** 面板中添加语言——详见[在 Flow Builder 中添加语言环境](add-paywall-locale-in-adapty-paywall-builder)。
:::

<ZoomImage id="flow-conditional-text-1.gif" width="500px" alt="使用 if、then 和 else 分支设置条件文本" />
## 添加列表 \{#add-lists\}

:::note
列表元素是由单独条目组件组成的容器。如需在流动文本中使用简单的项目符号或编号列表，请使用文本元素，并通过 **Design** 面板应用所需格式。
:::

1. 点击左上角的 **+**，选择 **List**，然后选择一个列表模板。

2. 在右侧的 **Design** 面板中编辑列表条目，或上传图片作为条目标记。
