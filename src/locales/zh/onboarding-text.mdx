只需单击即可添加标题、段落或列表，按品牌风格自定义样式，并使用[动态变量](onboarding-variables)为每位用户个性化内容。
## 设置文字样式 \{#set-up-text-styles\}

**Styles** 面板提供了预配置的文字样式：H1、H2、H3、Button Label、Body、Caption 和 Small Label。点击任意样式即可编辑其字体系列、字重、大小、对齐方式、装饰效果等属性。

当你[添加文字元素](#add-text)时，可以从这里设置好的样式中选择。

:::important
对某个样式的修改会应用到所有使用该样式的文字元素，包括所有屏幕。
:::

创建新样式：

1. 点击 <Inline id="Styles.webp" alt="Palette"/> 打开 **Styles** 面板。
2. 在 **Text** 选项卡中，点击 **Create style**。
<ZoomImage id="new-text-style.webp" width="500px" alt="New text style" />

3. 输入名称并配置排版样式——字体族、字重、对齐方式以及[其他属性](#typography-properties)。

    :::warning
   [自定义字体](using-custom-fonts-in-flow-builder)与内置字体的行为不同。请阅读相关指南——部分控件不适用，且每个字体变体需要单独的文件。
   :::

   <ZoomImage id="text-style-details.webp" width="500px" alt="Text style configuration options" />


4. 点击 **Create**。
## 添加文本 \{#add-text\}

添加文本元素的步骤：
1. 点击左上角的 **+**，选择 **Text**，然后从预配置或自定义的[文本样式](#set-up-text-styles)中选择一种。

<ZoomImage id="new-text.webp" width="500px" alt="New text" />
2. 点击新元素，然后在右侧 **Design** 面板的 **Content** 部分编辑其内容。
3. 如有需要，在 **Design** 面板中调整[排版属性](#typography-properties)。或者，在预览中选中文本，弹出提示框以快速自定义样式。
4. 可选步骤：在 **Design** 和 **Interaction** 面板中，还可以为流程中的组件应用其他任意配置。详情请参阅[元素样式设置](builder-styling)。
:::tip
如果需要在多个屏幕上使用同一文本元素，可以复制粘贴：选中该元素，按 Ctrl+C（Mac 上为 ⌘+C），切换到另一个屏幕，再按 Ctrl+V（Mac 上为 ⌘+V）粘贴即可。
:::

<ZoomImage id="text-settings.webp" width="500px" alt="Text element settings" />
### 更改部分文本的样式 \{#change-styling-for-parts-of-text\}

:::warning
**粗体**和**斜体**格式对使用[自定义字体](using-custom-fonts-in-flow-builder)的文本不起作用。如需使用自定义字体的不同字重或样式，请将每种变体作为单独的字体文件上传，然后从 **Font family** 下拉菜单中选择。
:::

如需仅更改文本元素中某些部分的样式：

1. 在 **Content** 部分中选中文本元素的某段内容。
2. 在弹出的工具提示中，更改文本颜色，或应用粗体、下划线、斜体、删除线格式，也可以添加 URL。

预览会立即更新。
<ZoomImage id="edit-text-part.webp" width="500px" alt="Edit text part" />
## 添加原价 \{#add-an-old-price\}

原价元素会在产品实际价格旁边显示一个虚高的"原始"价格，让用户一眼就能看出折扣力度。它会将产品价格乘以你设定的系数，并自动为其添加删除线。

你无需手动编辑该元素的内容。其值会在运行时根据产品在应用商店中的价格计算得出，因此货币单位始终与实际价格保持一致。

添加原价的步骤：

1. 点击左上角的 **+**，选择 **Text**，然后选择 **Old Price**。
<ZoomImage id="old-price-menu.webp" width="500px" alt="展开了 Text 类别的元素菜单，显示 Old Price 项目" />

2. 在 **Layers** 面板中，将元素拖入产品卡片内。该元素同样支持放置在已保存为可复用组件的产品卡片中。

   :::important
   Old Price 元素必须位于产品卡片内部——价格取自该卡片对应的产品。若置于产品卡片外部，编辑工具将显示"Old price"占位符，而非实际价格。
   :::
3. 在 **Design** 面板的 **Price** 区域中，设置 **Multiplier**。Adapty 会将产品价格乘以该值。例如，默认倍数为 2 时，售价 $12.99 的产品会显示划掉的 $25.98。

   <ZoomImage id="old-price-multiplier.webp" width="500px" alt="Design panel with the Price section and the Multiplier field for an Old Price element" />

4. 如有需要，调整[文字排版属性](#typography-properties)。该元素支持与普通文本相同的样式设置。
<ZoomImage id="old-price-result.webp" width="400px" alt="画布上的产品卡片，显示划掉的原价与实际价格并排" />

:::tip
如果你想划掉另一个产品的实际价格——例如，在年度卡片上显示月度套餐的价格——请改用文本元素和变量来构建价格行。请参阅[显示带折扣角标的删除线价格](strikethrough-price)。
:::
## 排版属性 \{#typography-properties\}

每个文本元素和文本样式共享同一套排版控件：

- **Font family**：选择字体——内置字体或[自定义字体](using-custom-fonts-in-flow-builder)。

    :::warning
    不同设备内置的字体各不相同，同一字体在不同设备上的渲染效果也可能有所差异。如果设备上未安装所选字体，系统将使用默认字体（SF Pro / Roboto）。若希望在所有设备上统一显示相同字体，请上传[自定义字体](using-custom-fonts-in-flow-builder)。
    :::

- **Weight**：设置字体粗细。
:::warning
**Weight**、**Bold** 和 **Italic** 控件不适用于[自定义字体](using-custom-fonts-in-flow-builder)，无论是在 **Styles** 面板、**Design** 面板排版区域、内联工具栏还是文本部分格式提示中进行设置。若要应用自定义字体的变体，请将每个变体作为单独的字体文件上传，并从 **Font** 下拉列表中选择合适的字体。
:::
- **Size**：设置字体大小（单位：像素）。
- **Color**：设置文本颜色。
- **Line height**：设置行间距，或保持 **Auto**。
- **Alignment**：设置水平对齐（左对齐、居中、右对齐）和垂直对齐（顶部、居中、底部）。
- **Decoration**：添加下划线或删除线。
- **Truncate**：限制显示的行数，超出部分将被截断。当内容因动态变量或本地化而长度不一时，此选项非常实用。
## 添加链接 \{#add-links\}

流程支持两种将文本转为可点击链接的方式，根据文本的用途选择即可：

- **内联链接** — 适用于正文中嵌入的 URL，例如段落中的"了解更多"引用。始终在应用内浏览器中打开。
- **Open URL 操作** — 适用于独立的点击目标，例如服务条款按钮。可在应用内浏览器或外部浏览器中打开。
### 内联链接 \{#inline-link\}

将文本元素的部分内容转换为链接：

1. 在 **Content** 区域中选中文本。
2. 在格式化工具提示中，点击链接图标。
3. 将目标 URL 粘贴到弹出框中。

<ZoomImage id="inline-link-demo.webp" width="400px" alt="高亮显示链接按钮的格式化工具提示" />
### 打开 URL 操作 \{#open-url-action\}

:::link
主要文章：[操作](onboarding-actions#open-url)
:::

要将整个按钮变成一个链接：

1. 添加一个 [Button](builder-elements#buttons)，或使用 **Links** 预设——这是一个包含 Restore / Terms of Service / Privacy Policy 按钮的现成行。

    <ZoomImage id="links-button-template.webp" width="500px" alt="The Links preset on canvas with Terms of use, Restore, and Privacy policy buttons" />

2. 在 **Layers** 面板中选择该按钮，然后在右侧面板中打开 **Interactions** 选项卡。
<ZoomImage id="button-selected.webp" width="500px" alt="Terms of Use button selected in the Layers panel, with the nested text element visible but unselected" />

3. 为 [**Open URL**](onboarding-actions#open-url) 操作设置目标地址。

    :::important
    空的 Open URL 操作会[阻止预览和发布](builder-save-publish#troubleshooting)。
    :::

    <ZoomImage id="open-url.webp" width="500px" alt="Open URL action configuration in the Interactions panel" />
## 添加条件文本 \{#add-conditional-text\}

条件文本会根据条件改变文本元素显示的内容。例如，当用户选择年付方案时，标题可以显示一条消息；选择月付方案时，则显示另一条消息。条件文本的工作原理类似于[条件可见性](onboarding-element-visibility)，但它是替换内容，而不是显示或隐藏元素。

设置条件文本的步骤：

1. 在画布上选择一个文本元素。
2. 在 **Design** 面板的 **Content** 部分，选择 **Conditional**。
<ZoomImage id="flow-conditional-text.webp" width="500px" alt="在设计面板中将内容设置为条件显示" />
3. 构建 **if** 条件。从 **Custom**、**Products** 或 **Elements** 标签页中选择一个属性，设置运算符，并输入要匹配的值。有关属性类型的详细信息，请参阅[条件可见性](onboarding-element-visibility)。
4. 在 **then** 下，输入条件为真时显示的文本。富文本格式设置方式与[普通文本](#change-styling-for-parts-of-text)相同。要插入[变量](onboarding-variables)，点击 **&#123; &#125; Add variable**。
5. 在 **else** 下，输入当所有条件均不匹配时显示的备用文本。
6. （可选）点击 **+ Add else/if** 添加更多条件，每个条件对应各自的文本。
:::tip
若要为其他语言编辑条件文本，请在编辑器底部切换当前语言环境。请先在 **Localizations** 面板中添加语言——详见[在 Flow Builder 中添加语言环境](add-paywall-locale-in-adapty-paywall-builder)。
:::

<ZoomImage id="flow-conditional-text-1.gif" width="500px" alt="Setting up conditional text with if, then, and else branches" />
## 添加列表 \{#add-lists\}

:::note
列表元素是由各个条目组件组成的容器。如果只需要在正文中添加简单的项目符号列表或编号列表，请使用文本元素，并通过 **Design** 面板设置所需格式。
:::

1. 点击左上角的 **+**，选择 **List**，然后选择一个列表模板。

2. 在右侧的 **Design** 面板中编辑列表条目，或上传图片作为条目标记。
