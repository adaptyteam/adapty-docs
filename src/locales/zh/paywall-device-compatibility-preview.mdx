本节介绍新版付费墙编辑工具，适用于 iOS、Android 和 React Native SDK 3.0 及更高版本，以及 Flutter 和 Unity SDK 3.3.0 及更高版本。有关与 Adapty SDK v2.x 或更早版本兼容的旧版付费墙编辑工具的信息，请参阅[旧版付费墙编辑工具设备兼容性预览](paywall-layout-and-products-legacy#device-compatibility-preview)。

您有两种方式在不同屏幕类型上预览您的付费墙：
- **在设备上预览**：在开发的任何阶段，确保一切在真实设备上按预期显示。
- **在 Adapty 看板中预览**：在设计付费墙时进行预览。

## 在设备上预览 \{#preview-on-devices\}

要在真实设备上预览您的付费墙：

1. [从 App Store 下载 Adapty 应用](https://apps.apple.com/us/app/adapty/id6739359219)。
2. 在付费墙编辑工具中，点击 **Test on device**。

3. 选择付费墙语言区域。
4. 用设备摄像头扫描二维码或打开链接。这将在 Adapty 移动应用中打开您的付费墙。

:::note
在测试模式下，Adapty 无法访问您在商店中的产品，因此付费墙中显示的价格不是真实价格。
:::

## 在 Adapty 看板中预览 \{#preview-in-the-adapty-dashboard\}

您可以使用付费墙编辑工具右侧的面板在不同屏幕类型上预览您的付费墙。这有助于确保您的付费墙在各种设备和屏幕尺寸上都呈现出色。

在这里，您可以：
- 选择要预览付费墙的设备。
- 在水平和垂直预览模式之间切换——对于专为 iPad 设计的付费墙尤为实用。
- 放大或缩小预览。
- 预览[产品信息的标签变量](paywall-builder-tag-variables#how-to-use-tag-variables)。

:::tip
设置元素的[最大宽度](paywall-layout-and-products#content-layout)以优化 iPad 上的布局。
:::
