自定义 HTML 让您可以创建独特的交互效果、嵌入第三方小部件，或在无需更新应用的情况下快速测试实验性元素。

:::note
自定义 HTML 元素不会被预加载或缓存，因此我们建议仅将原始 HTML 用于小型轻量级元素。
:::

要插入自定义 HTML 代码：

1. 点击左上角的 **Add**。
2. 进入 **Media & Display**，选择 **Raw HTML**。
3. 在右侧插入或编辑您的 HTML 代码。
