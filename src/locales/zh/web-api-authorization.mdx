## 授权 \{#authorization\}

API 请求必须通过您的公共 API 密钥进行身份验证，将其作为 **Authorization** 请求头，值为 `Api-Key {your_public_api_key}`，例如 `Api-Key public_live_...`。您可以在 [Adapty 看板 -> **App Settings** -> **General** 标签页 -> **API keys** 部分](https://app.adapty.io/settings/general) 中找到此密钥。

## 请求格式 \{#request-format\}

- **Content-Type 请求头**：
  将 **Content-Type** 请求头设置为 `application/json`，以便 API 处理您的请求。
- **Body**：
  API 要求请求以 JSON 格式作为请求体。
