## 授权 \{#authorization\}

API 请求必须通过你的公开 API 密钥进行身份验证，将其作为 **Authorization** 请求头，值的格式为 `Api-Key {your_public_api_key}`，例如 `Api-Key public_live_...`。可在 [Adapty 看板 -> **App Settings** -> **General** 标签页 -> **API keys** 部分](https://app.adapty.io/settings/general) 找到该密钥。

:::important
API 密钥与应用绑定。如果你有多个应用，请确保为每个应用使用不同的密钥。
:::
## 请求格式 \{#request-format\}

- **Content-Type 请求头**：
  将 **Content-Type** 请求头设置为 `application/json`，API 才能正常处理你的请求。
- **请求体**：
  API 期望请求体使用 JSON 格式。
