将 Adapty SDK 集成到您的应用中，可以实现：

- 开箱即用地处理购买、收据验证和订阅管理
- 无需更新应用即可创建和测试付费墙
- 零配置获取详细的购买分析数据——包含同期群、LTV、流失率和漏斗分析
- 跨应用会话和设备始终保持用户订阅状态最新
- 只需一行代码即可将应用与营销归因和分析服务集成

## 工作原理 \{#how-does-it-work\}

Adapty SDK 的基本实现只需关注三件事：

1. 安装并初始化 SDK。
2. 将应用内购买的处理委托给 Adapty。
3. 在用户画像中监控订阅状态。Adapty 负责判断订阅状态、类型和到期时间——SDK 只需消费这些信息即可。

具体顺序和细节因应用而异，但基本上就是这些。
