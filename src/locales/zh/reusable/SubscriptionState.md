| 用户**状态** | 描述 |
| :--------------------- | :----------------------------------------------------------- |
| Subscribed | 用户拥有已启用自动续订的有效订阅。 |
| Auto-renew off | 用户已关闭自动续订，但在订阅期结束之前仍可访问高级功能。 |
| Subscription cancelled | 用户已取消其订阅，且订阅已完全终止。 |
| Billing issue | 用户因账单问题无法被扣款，发生在订阅或试用到期后。 |
| Grace period | 用户目前处于宽限期，原因是在订阅或试用到期后尝试扣款时出现了账单问题。 |
| Active trial | 用户拥有当前处于试用期的有效订阅。 |
| Trial cancelled | 用户已取消试用，且没有有效订阅。 |
| Never subscribed | 用户从未订阅或开始试用，仍为免费增值用户。 |
