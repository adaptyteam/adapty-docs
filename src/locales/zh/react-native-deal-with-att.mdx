如果您的应用程序使用了 AppTrackingTransparency 框架并向用户呈现应用追踪授权请求，则您应将[授权状态](https://developer.apple.com/documentation/apptrackingtransparency/attrackingmanager/authorizationstatus/)发送给 Adapty。

```typescript showLineNumbers
import { AppTrackingTransparencyStatus } from 'react-native-adapty';

try {
  await adapty.updateProfile({
    // you can also pass a string value (validated via tsc) if you prefer
    appTrackingTransparencyStatus: AppTrackingTransparencyStatus.Authorized,
  });
} catch (error) {
  // handle `AdaptyError`
}
```

:::warning
我们强烈建议您在该值发生变化时尽早发送此值，只有这样，数据才能及时发送到您已配置的集成渠道。
:::
