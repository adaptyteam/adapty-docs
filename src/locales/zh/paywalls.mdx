在 Adapty 中，**付费墙是在应用中展示和销售产品的唯一方式**。通过这种方式，您可以轻松追踪不同产品组合在不同用户群体中的表现，并控制每个产品的展示方式。

Adapty 为您提供两种主要的付费墙设计方式：
- **使用 Adapty 付费墙编辑工具**：您可以[在无代码编辑工具中创建完整的、可直接购买的付费墙](quickstart-paywalls)。Adapty 会自动渲染付费墙，并在后台处理所有复杂的购买流程、收据验证和订阅管理。
- **手动设计付费墙**：手动设计付费墙时，您有以下选项：
   - [使用远程配置](customize-paywall-with-remote-config)：无需在应用代码中硬编码数值，您可以直接在看板中动态管理付费墙元素（如文本和媒体），无需重新部署应用。
   - <InlineTooltip tooltip="Delegate handling purchases to Adapty">[iOS](making-purchases)、[Android](android-making-purchases)、[React Native](react-native-making-purchases)、[Flutter](flutter-making-purchases) 和 [Unity](unity-making-purchases)</InlineTooltip>：自行与应用商店 API 集成非常耗时。您只需使用 Adapty SDK 中的一个方法，即可处理所有复杂的购买逻辑。
