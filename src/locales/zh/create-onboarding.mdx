[用户引导](onboardings)向新用户介绍您的移动应用的价值、功能和使用技巧。

## 第一步：创建用户引导 \{#step-1-create-an-onboarding\}

在 Adapty 看板中创建新的用户引导：

1. 从 Adapty 主菜单进入 **Onboardings**。该页面提供您已设置的所有用户引导的概览及其数据图表。点击 **Create onboarding**。

2. 为您的用户引导创建一个描述性名称，然后点击 **Proceed to build onboarding**。

3. 您将被重定向到用户引导编辑器。

   它包含一个默认的演示模板，您可以通过学习该模板了解用户引导如何收集数据，以及如何[使用变量和问卷对其进行个性化设置](onboarding-user-engagement)。您可以随意删除不需要的屏幕，并在此处[设计您自己的用户引导体验](design-onboarding)。

用户引导发布后，点击 **Back to Adapty**。您的用户引导已创建完成，您可以将其添加到版位以开始使用。

## 第二步：为用户引导创建版位 \{#step-2-create-a-placement-for-your-onboarding\}

1. 从主菜单进入 **Placements**，切换到 **Onboardings** 选项卡，点击 **Create placement**。

   <Zoom>
   <img src="/assets/shared/img/create-onboarding6.png"
   style={{
   border: '1px solid #727272', /* border width and color */
   width: '400px', /* image width */
   display: 'block', /* for alignment */
   margin: '0 auto' /* center alignment */
   }}
   />
   </Zoom>

2. 输入版位名称和 ID。然后点击 **Run onboarding**，选择一个向所有用户展示的用户引导。
3. 如果您为特定用户群体准备了单独的用户引导，请[添加更多目标受众](audience)并为其选择不同的用户引导。

## 第三步：将用户引导集成到您的应用中 \{#step-3-integrate-the-onboarding-into-your-app\}

:::important
用户引导适用于使用 Adapty SDK v3.8.0+（iOS、Android、React Native、Flutter）、v3.14.0+（Unity）或 v3.15.0+（Kotlin Multiplatform、Capacitor）的应用。
:::

要在您的应用中开始显示用户引导，请使用 Adapty SDK 进行集成：
- [iOS](ios-onboardings)
- [Android](android-onboardings)
- [React Native](react-native-onboardings)
- [Flutter](flutter-onboardings)
- [Unity](unity-onboardings)
- [Kotlin Multiplatform](kmp-onboardings)
- [Capacitor](capacitor-onboardings)

要了解哪种用户引导效果更好，您还可以运行 [A/B 测试](ab-tests)。
