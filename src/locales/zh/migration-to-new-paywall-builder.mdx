我们很高兴推出全新的 [**付费墙编辑工具**](adapty-paywall-builder)！这款先进的无代码工具旨在让创建自定义付费墙比以往更加直观、强大，让您能够轻松打造精美、引人入胜的付费墙，无需任何技术或设计经验！

## 新付费墙编辑工具的主要功能 \{#key-features-of-the-new-paywall-builder\}

- **更丰富的模板选择**：从大量专业设计的模板中进行选择，快速启动您的付费墙创建。这些模板提供多种风格和布局，满足不同需求和偏好。
- **更强的灵活性**：借助设计图层和轮播图、卡片、产品列表及页脚等新元素，享受更高的灵活性。这些增强功能赋予您充分的创作自由，可以构建任何您设想的付费墙类型。
- **全面改进的现有元素**：现有元素得到了显著改进，提供更多选项和功能，让您的付费墙创意得以实现。

## 并行运行的付费墙编辑工具版本 \{#parallel-paywall-builder-versions\}

Adapty 同时提供两个版本的付费墙编辑工具：

- **新付费墙编辑工具**：位于 Adapty 看板中付费墙的 **Builder & Generator** 标签下，是最新、最全面的版本。在此创建的付费墙需要 iOS、Android 和 React Native SDK 3.0 或更高版本，以及 Flutter 和 Unity SDK 3.3.0 或更高版本。
- **旧版付费墙编辑工具**：位于 **Legacy Builder** 标签下，此过时版本仅用于支持使用低于 v3.x.x SDK 的旧版应用。我们建议不要将其用于新付费墙，因为它即将被弃用。

## 将付费墙迁移到新编辑工具 \{#migrating-paywalls-to-the-new-builder\}

将付费墙从旧版编辑工具迁移到新编辑工具，将在 **Builder & Generator** 标签中创建付费墙的新版本。此版本可使用新付费墙编辑工具进行编辑，并将在使用 Adapty SDK v3.0 或更高版本的应用中显示。旧版本保留在 **Legacy Builder** 标签中，支持使用 SDK 2.x 或更早版本的应用。

您需要分别维护两种格式的付费墙，一种格式的更改不会影响另一种格式。

## 迁移付费墙的步骤 \{#steps-to-migrate-a-paywall\}

将付费墙迁移到新付费墙编辑工具：

1. 打开您要迁移的付费墙。
2. 打开 **Builder & Generator** 标签。
3. 点击 **Migrate paywall**。
