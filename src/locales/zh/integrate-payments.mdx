要开始使用 Adapty，首先需要与用户购买产品的商店进行集成。Adapty 可连接各种应用商店和网络支付提供商，将您所有的应用内购买和分析数据汇集到一处。

## 与商店和网络支付集成 \{#integrate-with-stores-and-web-payments\}

请在下方选择您的商店以获取详细的集成步骤：

- <a href="https://adapty.io/docs/zh/initial_ios" target="_blank" rel="noopener noreferrer">App Store</a>
- <a href="https://adapty.io/docs/zh/initial-android" target="_blank" rel="noopener noreferrer">Google Play</a>
- 网络支付：
  - <a href="https://adapty.io/docs/zh/stripe" target="_blank" rel="noopener noreferrer">Stripe</a>
  - <a href="https://adapty.io/docs/zh/paddle" target="_blank" rel="noopener noreferrer">Paddle</a>
- <a href="https://adapty.io/docs/zh/custom-store" target="_blank" rel="noopener noreferrer">其他商店</a>

## 后续步骤 \{#next-steps\}

连接好商店或支付平台后，您可以继续[添加产品](quickstart-products.md)。
