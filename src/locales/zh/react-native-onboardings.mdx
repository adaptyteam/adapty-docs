:::warning
**用户引导功能已在 SDK v4 中弃用，将在未来版本中移除。** 该功能不再接受修复或改进。请改用 [流程](react-native-get-pb-paywalls)：与在 WebView 中运行的用户引导不同，流程直接在设备上原生渲染，带来更流畅的动画效果、一致的原生外观体验、更快的加载速度，且无需依赖 WebView 运行时。请参阅[获取流程与付费墙](react-native-get-pb-paywalls)和[展示流程与付费墙](react-native-present-paywalls)开始使用。
:::
