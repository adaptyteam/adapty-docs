了解如何从旧版本迁移到 Unity Adapty SDK 3.x。

## 新特性 \{#whats-new\}

### 版本 3.x \{#version-3x\}

- 增强的付费墙展示
- 改进的错误处理
- 更好的 C# 支持
- 性能优化

### 版本 2.x \{#version-2x\}

- 新增用户引导功能
- 增强的分析能力
- 改进的购买流程
- 缺陷修复与稳定性提升

## 重大变更 \{#breaking-changes\}

### 版本 3.x \{#version-3x-breaking\}

- 更新了观察者 API
- 更改了付费墙展示方法
- 修改了错误处理结构

### 版本 2.x \{#version-2x-breaking\}

- 更新了用户引导 API
- 更改了用户画像结构
- 修改了购买流程

## 迁移检查清单 \{#migration-checklist\}

迁移到新版本时：

- [ ] 审查重大变更
- [ ] 更新 API 调用
- [ ] 测试所有功能
- [ ] 更新错误处理
- [ ] 验证分析跟踪
- [ ] 在所有平台上进行测试
