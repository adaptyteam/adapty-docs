## 2026 年 3 月 \{#march-2026\}

- **开发者 CLI**：无需打开看板，直接从终端管理您的 Adapty 账户。CLI 支持创建应用、定义访问等级、设置产品、创建付费墙以及配置版位——所有操作均可脚本化，适用于自动化环境。此外，还提供 [Adapty CLI 技能](https://github.com/adaptyteam/adapty-cli/tree/main/skills/adapty-cli)，可帮助 AI 编程助手与 CLI 协同工作。[了解更多](developer-cli)
- **Autopilot 中的视觉轮次**：付费墙设计建议现已作为一等轮次纳入您的增长计划——与货币化轮次并列显示在侧边栏中。每个视觉轮次包含一个设计样图、该模式最适用场景的描述，以及其所针对的关键数据图表。[了解更多](autopilot#visual-rounds)

- **向 Autopilot 添加自定义假设**：使用自定义轮次扩展您的增长计划。添加标题、描述、轮次类型（货币化或视觉），以及目标数据图表——对于货币化轮次，还需填写涉及的产品。[了解更多](autopilot#expand-the-plan)
- **重新排序 Autopilot 轮次**：拖拽并重新排序增长计划的各个阶段，以最符合您策略的顺序运行实验。[了解更多](autopilot)

- **Autopilot 中的地理定价轮次**：在增长计划中，将针对特定国家/地区的价格调整作为一种新的轮次类型进行测试。基于市场洞察数据，Autopilot 会针对每个国家/地区推荐提价、降价或维持现价。将推荐方案添加为地理定价轮次，作为 A/B 测试运行——最多可同时运行 5 个。[了解更多](autopilot#geo-pricing-rounds)
- **Apple Ads Manager 中的搜索词自动化**：自动将表现优异的搜索词提升为精确匹配关键词，并在源头将其排除——无需手动下载报告。规则可以从模板创建，也可以从头构建，支持自定义条件和计划。[了解更多](ads-manager-automations-search-terms)
- **Apple Ads Manager 中的最大化转化竞价**：创建广告活动时，您现在可以选择最大化转化作为竞价策略。Apple 的算法会在您的预算范围内最大化下载量，并可设置可选的目标 CPA 进行引导。[了解更多](ads-manager-create-campaign)

- **Adapty UA 中的 Google Ads 和 FunnelFox 集成**：Adapty UA 现已支持与 Google Ads 和 FunnelFox 的新集成。

- **中文文档**：Adapty 文档现已提供中文版本。请使用顶部导航栏中的语言选择器切换语言。
## 2026年2月 \{#february-2026\}

- **按国家设置产品定价**：直接在 Adapty 看板中为各国设置不同价格 —— Adapty 会自动将更改同步至 App Store Connect 和 Google Play。每次价格更新都会记录在审计日志中，确保所有变更有据可查。[了解更多](edit-product)

- **Autopilot 中的国家级竞品定价**：在您的主要市场中，将您的订阅价格与竞品进行比较。[了解更多](autopilot#competitor-analysis)
- **用户引导版本控制**：通过完整的版本历史记录追踪和管理用户引导的版本。查看更改并在需要时回滚。[了解更多](onboarding-version-control)

- **分析中的付费墙转化数据图表**：两个新的转化数据图表——付费墙浏览 → 试用和付费墙浏览 → 付费——展示您的付费墙将浏览者转化为订阅者的效果。[了解更多](analytics-conversion)
- **重复市场细分**：复制现有市场细分及其所有筛选条件，无需从头重建类似的细分。在运行多个营销活动或目标受众存在重叠的 A/B 测试时非常实用。[了解更多](segments#duplicate-segments)

- **Adapty 移动应用中的推送通知**：直接在 Adapty iOS 应用中为 14 种事件类型配置推送通知，无需打开看板即可掌握订阅动态。[了解更多](push-notifications)
- **Kotlin Multiplatform SDK 3.15**：新增用户引导支持、网页付费墙及 API 改进。[了解更多](migration-to-kmp-315)

- **Capacitor SDK 3.16**：新增 Capacitor 8 支持。使用 Capacitor 7 的项目应继续使用 SDK v3.15。[了解更多](migration-to-capacitor-316)
- **LLM 辅助 SDK 集成指南**：借助 AI 编程助手集成 Adapty 的分步指南。每份指南将引导您的 LLM 完成从看板配置到购买的完整实现流程。[iOS](adapty-cursor) | [Android](adapty-cursor-android) | [React Native](adapty-cursor-react-native) | [Flutter](adapty-cursor-flutter) | [Unity](adapty-cursor-unity) | [Kotlin Multiplatform](adapty-cursor-kmp) | [Capacitor](adapty-cursor-capacitor)
## 2026年1月 \{#january-2026\}

- **Capacitor SDK 正式发布**：Capacitor SDK 经过大量测试，现已可用于生产环境。使用 Capacitor 为 iOS 和 Android 构建订阅应用，完整支持 Adapty 集成。[了解更多](capacitor-sdk-overview)

- **新应用的 Autopilot 功能**：即使您的应用尚无大量交易历史记录，Autopilot 分析功能现在也可使用。从第一天起即可获取数据驱动的价格优化建议，并制定您的增长计划。[了解更多](autopilot)
- **Autopilot 中的全球定价机会**：通过针对特定国家/地区的定价建议，识别您表现最佳的市场中的收入潜力。Autopilot 会分析您排名前 5 的国家/地区的转化率和购买力，基于 Adapty 定价指数提供数据驱动的洞察，帮助您决定是提价、降价还是维持现价。[了解更多](autopilot)
- **账单恢复转化数据图表**：全新分析数据图表可追踪从账单问题和宽限期中恢复的收入。监控"Billing issue converted"、"Billing issue converted revenue"、"Grace period converted"和"Grace period converted revenue"，以衡量您的留存恢复效果。

- **在 Apple Ads Manager 中直接管理广告**：无需在平台之间切换，直接在 Adapty 内创建和管理您的 Apple Ads 广告活动。[了解更多](ads-manager-manage-ads)
- **Apple Ads Manager 分析**：在 Adapty 中访问详细的广告级别效果数据图表和归因数据。在统一的看板中查看广告活动效果、广告组分析和归因洞察。[了解更多](adapty-ads-manager-analytics)

- **Apple Ads 归因数据图表**：将多个归因数据图表组合到可自定义的数据图表中，结合订阅数据分析您的 Apple Ads 效果。[了解更多](adapty-ads-manager-analytics#charts)
- **Apple Ads 归因市场细分**：通过简化的两步操作流程，基于 Apple Ads 归因数据创建用户市场细分。按广告活动、广告组或关键词定向用户，实现更精准的分析和实验。[了解更多](ads-manager-create-segments)

- **全新文档平台**：文档网站已迁移至全新平台，支持更快速的功能更新，并通过增强的搜索、导航和内容组织提升用户体验。
## 2025 年 12 月 \{#december-2025\}

- **Apple Ads Manager 文档**：在单一分析看板中将 Apple Search Ads 广告活动数据与收益数据图表整合。新文档涵盖广告活动创建、广告组管理，以及跟踪广告支出 ROI 与订阅表现的方法。[了解更多](ads-manager)
- **应用内 Web 付费墙**：通过应用内浏览器在应用中展示基于 Web 的付费墙，无需外部跳转，提供无缝体验。[iOS](ios-web-paywall#open-web-paywalls-in-an-in-app-browser) | [Android](android-web-paywall#open-web-paywalls-in-an-in-app-browser) | [React Native](react-native-web-paywall#open-web-paywalls-in-an-in-app-browser) | [Flutter](flutter-web-paywall#open-web-paywalls-in-an-in-app-browser)
- **付费墙编辑工具中的逐行 AI 翻译**：选择付费墙中的特定文本元素，借助 AI 辅助进行翻译，使本地化更加精准灵活。[了解更多](add-paywall-locale-in-adapty-paywall-builder#translating-paywalls-with-ai)

- **滚动市场细分**：创建动态目标受众市场细分，根据移动时间窗口自动更新。例如，创建一个"最近 7 天内安装应用的用户"市场细分，持续刷新以始终显示您的最新客户。[了解更多](segments#available-attributes)
- **Meta 和 TikTok 广告活动设置指南**：关于在 Meta（Facebook 和 Instagram）和 TikTok 上创建和追踪广告活动的分步文档，包含转化追踪和数据分析集成。[Meta](meta-create-campaign) | [TikTok](tiktok-create-campaign)
- **手动付费墙实现快速入门指南**：通过分步快速入门指南更快地实现应用内购买，了解如何将 Adapty SDK 集成到您的自定义付费墙 UI 中。[iOS](ios-implement-paywalls-manually) | [Android](android-implement-paywalls-manually) | [React Native](react-native-implement-paywalls-manually) | [Flutter](flutter-implement-paywalls-manually) | [Unity](unity-implement-paywalls-manually) | [Kotlin Multiplatform](kmp-quickstart-manual) | [Capacitor](capacitor-quickstart-manual)
- **用户引导链接的应用内浏览器**：用户引导中的外部链接现在默认在应用内浏览器中打开，使用户留在您的应用中。如有需要，您可以自定义此行为以使用外部浏览器。[iOS](ios-present-onboardings#customize-how-links-open-in-onboardings) | [Android](android-present-onboardings#customize-how-links-open-in-onboardings) | [React Native](react-native-present-onboardings#customize-how-links-open-in-onboardings)
- **改进的 Autopilot 建议**：Autopilot 现在可以根据对您订阅数据的深度分析，提供更优质的价格优化建议。[立即体验 Autopilot](autopilot)

- **文档深色模式**：文档现已支持深色模式，可自动检测系统偏好设置，也可通过右上角的手动开关切换。
