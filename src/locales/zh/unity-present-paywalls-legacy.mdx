如果您已使用付费墙编辑工具自定义了付费墙，则无需在移动应用代码中手动渲染即可将其展示给用户。此类付费墙同时包含展示内容和展示方式。

:::warning

本指南适用于**旧版付费墙编辑工具付费墙**，需要使用 Adapty SDK 2.x 及以下版本。[新版付费墙编辑工具](adapty-paywall-builder)需要 Adapty SDK 3.0 或更高版本，目前尚不支持 Unity。

如需展示远程配置付费墙，请参阅[渲染由远程配置设计的付费墙](present-remote-config-paywalls)。
:::

要展示付费墙，请在通过 `createPaywallView` 方法创建的 `view` 上调用 `view.present()` 方法。每个 `view` 只能使用一次。如果需要再次展示付费墙，请重新调用 `createPaywallView` 以创建新的 `view` 实例。

:::warning

在未重新创建的情况下复用同一个 `view` 可能会导致 `AdaptyUIError.viewAlreadyPresented` 错误。
:::

```csharp showLineNumbers title="Unity"
view.Present((error) => {
  // handle the error
});
```
