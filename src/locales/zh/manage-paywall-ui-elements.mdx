选择模板后，其元素将显示在左侧面板中。使用此面板来整理付费墙上的元素。元素在付费墙上的显示顺序与左侧面板中的顺序一致。

:::warning

本节介绍新版付费墙编辑工具，该工具适用于 iOS、Android 和 React Native SDK 3.0 及以上版本，以及 Flutter 和 Unity SDK 3.3.0 及以上版本。如需了解与 Adapty SDK v2.x 或更早版本兼容的旧版付费墙编辑工具，请参阅[旧版付费墙编辑工具](adapty-paywall-builder-legacy)。

:::

## 添加元素 \{#add-element\}

要在布局上方向付费墙添加元素：

1. 点击左侧面板中的 **Add Element** 按钮。
2. 选择要添加的元素。新元素将出现在 **Footer** 上方的列表中。

## 重命名付费墙元素 \{#rename-paywall-element\}

要重命名元素：

1. 点击左侧面板中的元素以打开其详细信息。

  <img src="/assets/shared/img/a5b26ba-edit_element.webp"

    display: 'block', /* for alignment */
    margin: '0 auto' /* center alignment */

2. 点击右侧面板中的省略号按钮，选择 **Edit** 选项。

3. 输入元素的新名称并按 **Enter**。

## 隐藏/显示元素 \{#hide--show-element\}

即使您已经创建并配置了某个元素，也可以将其从付费墙中临时隐藏。如果您计划稍后添加该元素而不想丢失已完成的配置，这非常方便。隐藏元素后，付费墙看起来就像该元素从未被添加过一样，所有对齐方式和间距都会重新计算并重新绘制。

要隐藏元素：

1. 点击左侧面板中的元素以打开其详细信息。

2. 点击右侧面板中的省略号按钮，选择 **Hide** 选项。

  <img src="/assets/shared/img/pb-hide-element.webp"
  style={{

    width: '700px', /* image width */
    display: 'block', /* for alignment */
    margin: '0 auto' /* center alignment */
  }}
/>
</Zoom>

隐藏的元素会在主面板中以注释形式标记，如果您在左侧面板中选中它，也会显示标记。

<Zoom>
  <img src="/assets/shared/img/pb-hidden-element.webp"

要再次显示元素，点击右侧面板中的省略号按钮，选择 **Show** 选项。
