要打开元素库并向屏幕添加新元素，请点击左侧面板或设备预览上方的加号按钮 <Inline id="plus.svg" alt="Plus" />。
元素分为以下几类：
- [基础元素](#basics)（布局容器、文本、媒体、列表、徽章、勾选标记）
- [表单与问卷](#forms--quiz)（按钮、输入框、问卷、选项卡）
- [付费墙与商务](#paywall--commerce)（产品、试用开关、用户互动、倒计时）
- [进度指示器](#progress)与加载器

每个元素都提供多种预设——包含占位内容或预定义交互的模板。
## 基础知识 \{#basics\}
### 布局 \{#layout\}

:::link
主要文章：[布局元素](builder-containers)
:::

布局元素是用于组织内部子元素的容器。

- **垂直堆叠（Vertical Stack）**：将子元素从上到下排列
- **水平堆叠（Horizontal Stack）**：将子元素从左到右排列
- **分割线（Dividers，水平和垂直）**：用于在视觉上分隔内容区域的线条
- **轮播（Carousel）**：可滑动的容器
- **底部面板（Bottom Sheet）**：从屏幕底部滑出的悬浮面板

<ZoomImage id="layout-element-templates.webp" width="700px" alt="布局元素模板" />
### 文本

:::link
主文章：[文本内容](onboarding-text)
:::

文本可以是静态内容，也可以包含[变量](onboarding-variables)（例如用户名）和[本地化内容](paywall-localization)。


<ZoomImage id="text-templates.webp" width="400px" alt="Text element templates" float="right" />

以下文本预设为标准配置。如需修改此列表，请添加或删除[已保存的文本样式](builder-styling#reusable-styles)：

- H1
- H2
- H3
- Button Label
- Body
- Caption
- Small Label
### 媒体 \{#media\}

:::link
主要文章：[图片、视频与图标](custom-media)
:::

:::note
本节介绍前景媒体元素。如需用图片或视频填满整个屏幕，请更改[屏幕背景](paywall-layout-and-products#screen-background)。
:::

<ZoomImage id="media-templates.webp" width="200px" alt="Media element templates" float="right"/>

- **Icon**：内置图标库中的矢量图标，可自定义大小和颜色
- **Image**：图片——上传本地文件或提供 URL
- **Video**：嵌入式视频播放器，支持最大 50MB 的文件，支持循环播放。
### 列表 \{#list\}

列表元素将内容组织成行和列，用于展示格式统一的数据。在底层实现上，列表是一种[堆叠布局](manage-paywall-ui-elements#layout)。

- **图标列表**：每行包含前置图标和文字标签
- **时间轴**：带有连接步骤指示器的垂直序列
- **图片列表**：每行包含前置图片和文字
- **图标卡片**：以网格形式排列、图标居中的卡片组
- **图片卡片**：带有图片的网格卡片组
- **对比表格**：多列表格，用于跨方案（如免费版与专业版）的功能对比
<ZoomImage id="list-templates.webp" width="700px" alt="列表元素模板"/>
### 徽标 \{#badge\}

<ZoomImage id="badge-template.webp" width="200px" alt="Badge templates" float="right"/>

一种小型叠加标签，用于突出显示某个元素——通常用于推广折扣或特定方案（例如"节省 5%"）。使用[绝对定位](manage-paywall-ui-elements#absolute)将徽标叠加到其他元素上方。
### 勾选标记 \{#checkmarks\}

用于[可选择元素](flow-selectable-elements)内部的选中状态指示图标。每种勾选标记预设均包含选中和未选中两种状态，会根据用户操作自动切换。

<ZoomImage id="checkmark-templates.webp" width="700px" alt="Checkmark templates" float="right"/>


- Checkbox
- Circle
- Radiobutton
- Toggle
## 表单与测验 \{#forms--quiz\}
### 按钮 \{#buttons\}

:::link
主要文章：[按钮](paywall-buttons)
:::

按钮在点击时触发操作——跳转到另一个屏幕、打开 URL 或关闭流程。

<ZoomImage id="button-templates.webp" width="400px" alt="Layout element templates" float="right" />

每个预设都是一个起点——可以重新设计样式并指定任意操作。在 [Interactions](builder-ui#interactions-properties) 标签页中配置按钮行为。
- **Base**：带居中文本的常规按钮
- **Right Icon**：右侧带图标的按钮
- **With Subtitle**：包含两行文字
- **Pulse Animation**：包含动画脉冲效果
- **Purchase**：触发购买操作
- **Secondary Outline**：用于次要操作的描边按钮
- **Back**：返回上一屏幕
- **Close flow**：退出流程
- **View more plans**：展示更多产品选项
- **Links**：一组页脚按钮（恢复购买、服务条款、隐私政策）
### 输入框 \{#inputs\}

:::link
主要文章：[输入框与表单](builder-inputs-and-forms)
:::

输入框允许用户输入数据。每种类型都会强制使用相应的输入方式并进行格式校验。

<ZoomImage id="input-templates.webp" width="400px" alt="Input templates" float="right" />

- 文本
- 电子邮件
- 密码
- 数字
- 手机号码
- 日期
- 时间
- 日期与时间

日期、时间以及日期与时间输入框在点击时会调出设备原生选择器（滚轮或日历）。

使用[变量](onboarding-variables)来处理用户输入并影响条件逻辑。
### 测验 \{#quizzes\}

:::link
主要文章：[调查与测验](onboarding-quizzes)
:::

测验元素提供多选项选择界面，可用于问卷调查、偏好收集和用户细分。设置[交互](onboarding-navigation-branching)以根据用户的回答对流程进行分支跳转。

<ZoomImage id="quiz-templates.webp" width="400px" alt="测验模板" float="right" />
- **Icon Options**：带图标的单列列表
- **Emoji Options**：带表情符号的单列列表
- **Image Options**：带图片的单列列表
- **Icon Grid**：带图标的多列网格
- **Emoji Grid**：带表情符号的多列网格
- **Image Grid**：带照片的多列网格
- **Rating**：数字评分（例如 1–5）
### 选项卡 \{#tabs\}

:::link
主要文章：[选项卡](builder-tabs)
:::

选项卡将页面某一区域拆分为可切换的内容面板。用户选择某个选项卡后，下方内容随之更新。常用于对产品套餐进行分组，或在月付与年付方案之间切换。

<ZoomImage id="tab-templates.webp" width="500px" alt="Tab templates" float="right" />

- **分段控件**：带圆角边框的胶囊形切换器，选中项有包裹样式
- **按钮选项卡**：独立的按钮样式选项卡
- **下划线**：文字标签，选中项下方显示下划线
## 付费墙与商务 \{#paywall--commerce\}
### 产品 \{#products\}

:::link
主要文章：[产品](paywall-product-block)
:::

产品元素用于展示应用内购项目的详情并处理产品选择。每种预设都以不同的布局排列产品数据。关联你的产品，即可将真实数据填充到元素中。

<ZoomImage id="product-templates.webp" width="400px" alt="Product templates" float="right" />
- **垂直列表（Vertical List）**：堆叠排列的产品卡片
- **水平列表（Horizontal List）**：并排展示的产品卡片
- **功能轮播（Feature Carousel）**：可滑动的带功能清单卡片
- **功能卡片（Feature Cards）**：带功能清单的静态卡片
- **横幅列表（Banner List）**：带内联标签的紧凑行布局

### 用户互动 \{#user-engagement\}

预设样式的用户评分模块，有助于建立信任感并提升转化率。这些模板均为基础容器，内含占位内容。

<ZoomImage id="user-engagement-templates.webp" width="400px" alt="User engagement templates" float="right" />

- **Review**：显示星级评分、评论内容及评论者姓名
- **Rating**：以星形图标展示数字评分
- **App Rating**：大号评分数字，配有星级进度条和评论数量
- **Social Proof**：头像叠加展示，并显示用户总数
### 倒计时 \{#countdown\}

:::link
主要文章：[倒计时计时器](paywall-timer)
:::

<ZoomImage id="countdown-templates.webp" width="300px" alt="Countdown timer templates" float="right" />

显示倒计时的小时、分钟和秒数。可用于为限时优惠制造紧迫感。计时器归零时可触发相应操作，例如跳转到其他页面或隐藏折扣标签。
- **Inline**：紧凑型数字显示
- **Inline with units**：带单位标签的数字显示
- **Badge**：小型高亮计时器叠层
- **Blocks**：分别显示天、时、分、秒的独立卡片
## 进度 \{#progress\}

:::link
主要文章：[加载器与进度条](builder-loaders-and-progress-bars)
:::
### 进度指示器 \{#progress-indicators\}

<ZoomImage id="flow-indicators.webp" width="300px" alt="Progress indicator variants" float="right" />

基于步骤的进度条，用于显示用户在多屏流程中所处的位置。适合用于用户引导序列，让用户了解还剩多少步骤。

- **线性（Linear）**：单条连续进度条，随用户操作逐步填满
- **分段（Segmented）**：进度条被分为若干离散段，每步对应一段
- **连接器（Connectors）**：带编号的步骤标记，由短连接线相连
### 加载动画 \{#loaders\}

<ZoomImage id="flow-loaders.webp" width="300px" alt="Loader variants" float="right" />

用于过渡场景的动态加载指示器。当应用处理用户数据时（例如提交问卷后），可使用加载动画。

- **Spinner**：旋转圆形指示器
- **Spinner with label**：带文字标签的旋转指示器（如"Loading..."）
- **Loader**：水平进度条
