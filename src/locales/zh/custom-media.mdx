您可以在付费墙上设置任意图片或视频，但有时您可能希望为用户展示个性化的自定义图片或视频——例如，包含用户所选头像的视频。这就是自定义媒体的用武之地。

自定义媒体是您的应用通过代码中的 ID 调用的图片或视频，它会替换您在付费墙编辑工具中添加的标准媒体文件。

:::info
要使用此功能，请将 Adapty SDK 更新至 3.7.0 或更高版本。
:::

## 自定义媒体可以用在哪里？\{#where-can-i-use-custom-media\}

任何您通常使用常规图片或视频的地方：

- 作为主图
- 作为主视频
- 作为常规图标
- 用于卡片背景
- 用于轮播背景

## 如何使用自定义媒体？\{#how-to-use-custom-media\}
设置自定义媒体的步骤：

1. 勾选上传区域下方的 **Use custom media ID** 复选框。

2. 输入媒体 ID。对于主图和主视频，ID 已预定义。

3. 在文件上传区域上传备用图片或视频。

4. 更多信息请参阅<InlineTooltip tooltip="the guides on how to fetch Paywall Builder paywalls in your app">[iOS](get-pb-paywalls)、[Android](android-get-pb-paywalls)、[React Native](react-native-get-pb-paywalls)、[Flutter](flutter-get-pb-paywalls) 和 [Unity](unity-get-pb-paywalls)</InlineTooltip>，并在代码中通过 ID 调用媒体。

如果 Adapty SDK 版本低于 3.7.0，或者自定义媒体 ID 未在代码中定义，则会显示备用图片或视频。
