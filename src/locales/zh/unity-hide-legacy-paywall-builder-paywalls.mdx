虽然付费墙编辑工具可以在点击"购买"按钮时无缝处理购买流程，但您仍需在 Unity 应用中自行管理付费墙界面的关闭操作。

:::warning
本指南仅涵盖隐藏**旧版付费墙编辑工具付费墙**，该版本支持 Adapty SDK v2.x 及更早版本。
:::

您可以通过调用 `view.Dismiss` 方法来隐藏付费墙视图。

```csharp showLineNumbers
view.Dismiss((error) => {
  // handle the error
});
```
