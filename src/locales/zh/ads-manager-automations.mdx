Adapty Ads Manager 中的自动化规则让你无需手动干预即可响应广告系列效果。你可以根据效果指标定义条件、设置操作并安排规则运行时间——系统会自动完成后续工作。

自动化规则分为两种类型：
| | [关键词规则](ads-manager-automations-keyword-rules) | [搜索词自动化](ads-manager-automations-search-terms) |
|---|---|---|
| **适用对象** | 关键词 | 搜索词（用户实际查询内容） |
| **操作** | 调整出价、启用、暂停、添加为关键词、添加为否定词 | 将词语提升为关键词、添加为否定词 |
| **使用场景** | 根据效果管理关键词完整生命周期 | 从发现流量中构建关键词漏斗 |

## 查看日志 \{#explore-logs\}

**Logs** 标签页展示了所有规则执行的完整历史记录，方便你跟踪性能并排查问题。

要查看日志，请在左侧边栏进入 **Automations**，切换到 **Logs** 标签页。表格中显示以下内容：
- **Rule name**：自动化规则的名称。
- **Created at**：规则执行的日期和时间。
- **Description**：已执行操作、应用条件及目标的摘要（例如："Decrease bid by 25%, IF Spend (previous 3 days) > $20 AND Installs (previous 3 days) is 0, Applied to: Keywords in 3 campaigns"）。
- **Status**：执行状态——Success、Ran with errors 或 Failed。

如需将详细执行数据导出为 CSV，请点击日志条目旁边的下载图标。文件格式因规则类型而异：
- **关键词规则**：列出所有受影响的关键词及其调整前后的出价。
- **搜索词自动化**：列出每个已评估的搜索词，包括来源广告系列和广告组、目标广告组、操作结果、否定结果、出价和匹配类型。

如需按规则名称筛选表格，请使用表格上方的筛选器。

如果看不到最新更新，请点击 **Refresh table** 拉取最新数据。

## 启动和暂停规则 \{#launch-and-pause-rules\}

如需启动或暂停规则，请点击规则旁 **Status** 列中的开关。

暂停功能可让您在特定广告活动期间或进行调整时临时停用规则，而不会丢失规则配置。

:::note
启用规则后，若下次运行时条件满足，该规则可能会覆盖您手动修改的出价。若需要临时进行手动管理，请暂停规则而非删除。
:::

## 立即运行规则 \{#run-a-rule-immediately\}

规则会按照预定计划运行，但您也可以随时立即触发任意规则：

1. 点击规则旁的三个点。
2. 选择 **Run rule right now**。

规则将立即运行，并将配置的操作应用于所有匹配的关键词或搜索词。在添加关键词、广告组或其他结构性变更后，若希望在下次计划时间前提前执行规则，可使用此功能。

<ZoomImage id="asa-run-rule-right-now.webp" width="700px" alt="立即运行规则" />

## 复制规则 \{#duplicate-a-rule\}

在创建相似规则时，可通过复制节省时间：

1. 点击规则旁的三个点。
2. 选择 **Duplicate**。
3. 按需编辑复制的规则，然后点击 **Save**。

<ZoomImage id="asa-duplicate.webp" width="700px" alt="复制规则" />

## 删除规则 \{#delete-a-rule\}

如需移除不再需要的规则：

1. 点击规则旁的三个点。
2. 选择 **Delete** 并确认。

删除规则是永久性操作。如果以后可能还会用到该规则，建议改为暂停。

<ZoomImage id="asa-delete.webp" width="700px" alt="删除规则" />
