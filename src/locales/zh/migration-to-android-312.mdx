---
title: "将 Adapty Android SDK 迁移至 v. 3.12"
description: "迁移至 Adapty Android SDK v3.12，获得更好的性能和新的货币化功能。"
metadataTitle: "迁移至 Adapty Android SDK v3.12 | Adapty 文档"
---

在 Adapty SDK 3.12.0 中，我们已从 SDK 中删除了 `logShowOnboarding` 方法。
如果您一直在使用此方法，当您将 SDK 升级至 3.12 或更高版本时，该方法将不再可用。

作为替代，您可以[在 Adapty 无代码用户引导编辑工具中创建用户引导](onboardings)。这些用户引导的分析数据会自动追踪，并且您拥有丰富的自定义选项。
