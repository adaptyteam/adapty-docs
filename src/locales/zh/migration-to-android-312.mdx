---
title: "将 Adapty Android SDK 迁移至 v3.12"
description: "迁移至 Adapty Android SDK v3.12，获得更好的性能与全新的变现功能。"
metadataTitle: "迁移至 Adapty Android SDK v3.12 | Adapty 文档"
---

在 Adapty SDK 3.12.0 中，我们已从 SDK 中移除了 `logShowOnboarding` 方法。
如果你之前使用过该方法，升级至 SDK 3.12 或更高版本后将无法继续使用它。
作为替代，您可以[在 Adapty 无代码用户引导编辑工具中创建用户引导](onboardings)。这些用户引导的分析数据会自动追踪，并且您拥有丰富的自定义选项。
