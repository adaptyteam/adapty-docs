每当您 **Save**、**Preview** 或 **Publish** 用户引导时，Adapty 都会将当前版本添加到您的**版本历史记录**中。您可以浏览版本历史记录并切换到任意已保存的版本。

:::warning
Adapty 不会在[编辑过程](design-onboarding)中自动保存您的用户引导。如果您希望将其添加到版本历史记录中，请 **Publish** 您的用户引导。
:::

点击页面右上角的"versions"菜单，查看您的版本历史记录。每个版本均包含时间戳，以便您了解保存时间。点击某条记录上的箭头按钮，即可恢复到该状态。

:::important
恢复到早期版本后，请记得 **publish** 所做的更改。
:::
