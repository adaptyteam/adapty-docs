产品列表是付费墙的核心元素，它以有条理、吸引人的方式展示您的产品。该列表对于引导用户完成购买至关重要。

**Content** 标签页包含将在付费墙中展示的产品。这些产品与您在创建付费墙时添加的产品相同。

您可以调整产品列表，这将影响付费墙 **General** 标签页中的产品列表。

查看产品列表后：

1. 在 **Selected product** 字段中选择默认预选的产品。
2. 在 **Products** 部分的 **Style** 标签页中，定义产品在选中或未选中状态下的显示效果。
3. 在 **Layout** 标签页中配置模块的整体视图，或[添加产品分组](#add-products-group)以组合布局。

:::warning

本节介绍新版付费墙编辑工具，适用于 iOS、Android 和 React Native SDK 3.0 及更高版本，以及 Flutter 和 Unity SDK 3.3.0 及更高版本。有关兼容 Adapty SDK v2.x 及更早版本的旧版付费墙编辑工具的信息，请参阅[旧版付费墙编辑工具的付费墙产品](paywall-layout-and-products-legacy#products)。

:::

## 通过试用开关在两个产品集之间切换 \{#switch-between-2-product-sets-by-trial-toggle\}

为了创建多样化的用户体验，您可以允许用户使用开关在两组产品之间切换。这对于区分标准产品和试用产品尤为有用。

添加开关的步骤：

1. 在 **Products** 元素中，将 **Products grouping** 选项更改为 **Toggle (for free trials and other offers)**。这将添加两个子部分：**Toggle on** 和 **Toggle off**。

## 通过标签页在产品集之间切换 \{#switch-between-product-sets-by-tab\}

- 您的应用提供多种按周、按月或按年的订阅方案
- 您有不同等级，如 Plus、Gold 或 Premium

您还可以在标签页中添加功能列表等元素，帮助用户了解不同等级之间的差异。

这样可以让用户首先专注于您的主要选项，同时仍然允许他们在感兴趣时探索其他方案。

## 添加产品分组 \{#add-products-group\}

如果您想对不同产品同时应用垂直和水平布局，或在产品之间添加文本，可以添加另一个产品分组。

:::note
添加产品分组将禁用[产品分组](#switch-between-2-product-sets-by-trial-toggle)选项。请在添加新产品分组或在同一模块内对产品进行分组之间做出选择。
:::

添加产品分组的步骤：

1. 点击 **Footer** 上的 **Add element** 或 **+**。
2. 选择 **Products**。
3. 添加产品。由于同一产品不能出现在不同分组中，您需要先从另一个分组中删除该产品。
