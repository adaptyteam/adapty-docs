Adapty User Acquisition 帮助你将广告支出与订阅收入关联起来，适用于 web-to-app 推广活动，让你在一个地方全面掌握应用的商业数据。

要在 Adapty User Acquisition 中查看收入数据，你需要先在 Adapty 看板中启用该集成。无需传入任何 API 密钥、令牌或标识符，只需更新并配置 Adapty SDK 即可。
:::important
用户获取功能需要以下版本支持：
- iOS、Android 和 Flutter SDK 3.9.0 或更高版本。
- React Native 和 Capacitor SDK 3.10.0 或更高版本。
- Unity SDK 3.12.0 或更高版本。
- Kotlin Multiplatform SDK 3.15.0 或更高版本。
:::
## 开始之前 \{#before-you-start\}

要将收益数据与广告活动效果关联起来，请让 Adapty 跟踪您的购买记录：
- 如果您**已经通过 Adapty 实现了应用内购买**，此阶段无需进行任何额外操作。
- 如果您**尚未实现应用内购买且希望使用 Adapty**，请按照[快速入门指南](quickstart)完成相关步骤，将购买处理委托给 Adapty。
- 如果您**已经在不使用 Adapty 的情况下实现了应用内购买**，且不打算迁移到 Adapty，请[以观察者模式为您的平台安装 Adapty SDK](implement-observer-mode)。此阶段只需将 SDK 添加到项目中，以观察者模式激活它，并上报交易记录：
此设置可实现网页到应用的归因：
- 当用户安装您的应用时，Adapty SDK 会从链接参数中获取安装详情，Adapty UA 即可获取广告系列详情
- Adapty SDK 能感知应用内所有与收入相关的事件，并将其归因到网页广告系列。
## 步骤 1. 启用用户获取集成 \{#step-1-enable-the-user-acquisition-integration\}

要开始向 Adapty UA 发送收入事件：
1. 在 Adapty 看板中前往 [Integrations > Adapty](https://app.adapty.io/integrations/user-acquisition)。
2. 打开开关。

事件开始触发后，你将看到每个事件的以下详情：
- 事件名称
- 状态
- 环境
- 日期时间

## 步骤 2. 连接广告平台并添加追踪链接 \{#step-2-connect-your-ad-platform-and-add-tracking-links\}

Adapty 通过追踪链接将应用安装与推广活动数据关联起来。
对于所有需要在 Adapty UA 中进行效果衡量的广告推广活动，你必须将追踪链接设为目标 URL。

如果你在多个平台上投放广告，需要为每个平台分别设置追踪链接。

Adapty 与广告平台的对接方式有两种：
- **原生集成（Meta Ads、TikTok Ads）。** Adapty 直接与广告平台对接。追踪链接自动生成，广告系列参数根据链接的使用位置动态填充。同一条链接可跨不同广告系列、广告组或素材复用，Adapty 会自动获取正确的广告系列数据和广告支出。
- **仅限追踪链接（所有其他广告平台）。** Adapty 不会连接广告平台。追踪链接需手动创建，所有推广活动参数必须在创建链接时明确指定。这些平台不支持广告支出数据。

<Tabs>
<TabItem value="meta" label="Meta Ads" default>
要为 Meta Ads 创建追踪链接：
1. 在 Adapty UA 看板中前往 [Integrations > Meta](https://app.adapty.io/ua/integrations/facebook/accounts)，然后点击 **Continue with Facebook**。
<ZoomImage id="ua-connect-meta.webp" width="700px" />
2. 使用您的 Facebook 账号登录，然后点击 **Continue**。

<ZoomImage id="ua-fb-sign.webp" width="500px" />

3. 查看所请求的权限，然后点击 **Save**。
4. 切换到 **Web campaigns** 标签页，点击 **Create campaign**。选择应用后点击 **Save**。

<ZoomImage id="ua-new-campaign.webp" width="500px" />

5. 在 **General** 标签页中，展开 **iOS** 和/或 **Android** 部分，粘贴 App Store 和/或 Google Play 的应用 URL，然后点击 **Save**。

<ZoomImage id="ua-url.gif" width="900px" />
6. 复制 **Click link** 字段中**某一链接**或特定平台链接的值。然后在 Meta Ads Manager 中打开你的广告，将该链接粘贴为目标 URL。

:::important
在 **Website URL** 字段中，粘贴 `https://api-ua.adapty.io/api/v1/attribution/click`。将链接的其余部分粘贴到 **Tracking** 部分的 **URL parameters** 字段中。这有助于你的 Meta 广告通过审核。查看更多[在 Meta Ads Manager 中设置广告的建议](meta-create-campaign)。
:::
7. 现在，当您在 Meta Ads 中投放广告后，其数据将可在 Adapty UA 看板中进行分析。
</TabItem>
<TabItem value="tiktok" label="TikTok for Business">
要为 TikTok for Business 创建追踪链接：
1. 在 Adapty UA 看板中前往 [Integrations > TikTok Ads](https://app.adapty.io/ua/integrations/tiktok/accounts)，然后点击 **Continue with TikTok**。
   <ZoomImage id="ua-connect-tiktok.webp" width="700px" />
2. 使用您的 TikTok 账户登录，然后点击 **Continue**。

3. 查看所请求的权限，然后点击 **Save**。
<ZoomImage id="ua-tiktok-sign.webp" width="500px" />
4. 切换到 **Web campaigns** 标签页，点击 **Create campaign**。选择应用后点击 **Save**。

    <ZoomImage id="ua-new-campaign-tiktok.webp" width="500px" />

5. 在 **General** 标签页中，展开 **iOS** 和/或 **Android** 部分，粘贴 App Store 和/或 Google Play 的应用链接，然后点击 **Save**。

    <ZoomImage id="ua-url.gif" width="900px" />
6. 复制 **Click link** 字段中**某一个链接**或特定平台链接的值。然后在 TikTok Ads Manager 中创建广告时，将该值粘贴到 **Advanced Settings** 部分的 **Tracking URL** 字段中。这样 Adapty 就能将安装和购买行为与 TikTok 广告关联起来。请参阅[在 TikTok Ads 中设置广告系列的指南](tiktok-create-campaign)。

    <ZoomImage id="ua-lnk.webp" width="900px" />
7. 现在，当你在 TikTok for Business 中投放广告后，其数据将可在 Adapty UA 看板中进行分析。
</TabItem>
<TabItem value="others" label="其他广告平台">
要为其他广告平台创建跟踪链接：

1. 在 Adapty UA 看板中，从侧边栏菜单进入 **Tracking links**，然后点击 **Create link**。

    <ZoomImage id="new-tracking-link.webp" width="700px" />

2. 从列表中选择你的应用，然后点击 **Next**。

    <ZoomImage id="ua-choose-app.webp" width="700px" />
3. 填写链接参数，将其与您要跟踪的广告活动和广告相匹配。
4. 默认情况下，您创建的是 One Link。它会自动检测用户的平台，并在记录点击后将其重定向到 App Store 或 Google Play。

    如果您希望为每个平台单独设置重定向 URL，请取消勾选 **One Link** 复选框，并手动提供各平台的应用商店链接。

    <ZoomImage id="one-link.webp" width="700px" />
5. 点击 **Create**。
6. 打开你的跟踪链接页面，从以下某个部分复制 **Click link**：
    - **One link** – 使用此链接跟踪点击，并自动将用户重定向到正确的商店。
    - **iOS link** 或 **Android link** — 如果你希望为每个商店使用单独的链接，可选择对应平台的版本。
7. 前往你的广告平台，将链接作为广告目标 URL 粘贴到广告中。

</TabItem>
</Tabs>
## 第三步：启动网页到应用活动并查看结果 \{#step-3-launch-your-web-to-app-campaign-and-view-results\}

活动上线且用户开始安装您的应用后，Adapty 将开始将安装和收入归因到您的活动。

在 [Adapty UA 分析看板](ua-analytics)中，您将看到活动级别的数据图表，例如：
- 安装量和转化率
- 订阅和购买收入
- 按广告平台、活动、广告组和创意细分的效果

数据图表在从您的应用接收到安装和收入事件后即会显示。广告支出数据仅适用于具有原生集成的平台。
