Adapty 用户获取（User Acquisition）帮助您在网页转应用（web-to-app）广告活动中将广告支出与订阅收入关联起来，让您在一处全面了解应用的经济状况。

要在 Adapty 用户获取中查看收入数据，您必须先在 Adapty 看板中启用该集成。无需传递任何 API 密钥、令牌或标识符，只需更新并配置 Adapty SDK 即可。

:::warning
用户获取功能仅适用于以下版本：
- iOS、Android 和 Flutter Adapty SDK 3.9.0 或更高版本。
- React Native Adapty SDK 3.10.0 或更高版本。
- Unity SDK 3.12.0 或更高版本。
:::

## 开始之前 \{#before-you-start\}

要将您的收入数据与广告活动效果相关联，请让 Adapty 追踪您的购买记录：

- 如果您**已经通过 Adapty 实现了应用内购买**，在此阶段无需做任何其他操作。
- 如果您**尚未实现应用内购买且希望使用 Adapty**，请按照[快速入门指南](quickstart)完成相关步骤，将购买处理委托给 Adapty。
- 如果您**已经在不使用 Adapty 的情况下实现了应用内购买**，且不打算迁移到 Adapty，请[以观察者模式为您的平台安装 Adapty SDK](implement-observer-mode)。在此阶段，您只需将 SDK 添加到项目中，以观察者模式激活它，并上报交易信息。

此配置可启用网页转应用归因：
- 当用户安装您的应用时，Adapty SDK 会从链接参数中获取安装详情，以便 Adapty UA 获取广告活动详情。
- Adapty SDK 了解应用内所有与收入相关的事件，并可将其归因到网页广告活动。

## 第一步：启用用户获取集成 \{#step-1-enable-the-user-acquisition-integration\}

要开始向 Adapty UA 发送收入事件：
1. 在 Adapty 看板中，前往 [Integrations > Adapty](https://app.adapty.io/integrations/user-acquisition)。
2. 打开开关。

一旦您的事件开始触发，您将看到每个事件的以下详情：
- 事件名称
- 状态
- 环境
- 日期时间

## 第二步：连接您的广告平台并添加追踪链接 \{#step-2-connect-your-ad-platform-and-add-tracking-links\}

Adapty 使用追踪链接将应用安装与广告活动数据关联起来。
您必须在每个希望在 Adapty UA 中衡量的广告活动中，将追踪链接用作目标 URL。

如果您在多个平台上投放广告，请分别为每个平台设置追踪链接。

Adapty 与广告平台的协作有两种方式：

- **原生集成（Meta Ads、TikTok Ads）。** Adapty 直接连接广告平台。追踪链接自动生成，广告活动参数根据链接的使用位置动态填充。您可以在不同广告活动、广告组或创意素材中使用同一链接，Adapty 将自动接收正确的广告活动数据和广告支出。

- **仅追踪链接（所有其他广告平台）。** Adapty 不连接广告平台。追踪链接需手动创建，所有广告活动参数必须在创建链接时明确定义。这些平台无法获取广告支出数据。

<Tabs>
<TabItem value="meta" label="Meta Ads" default>
要为 Meta Ads 创建追踪链接：
1. 在 Adapty UA 看板中，前往 [Integrations > Meta](https://app.adapty.io/ua/integrations/facebook/accounts)，然后点击 **Continue with Facebook**。

<ZoomImage id="ua-connect-meta.webp" width="700px" />
2. 使用您的 Facebook 账号登录，然后点击 **Continue**。

<ZoomImage id="ua-fb-sign.webp" width="500px" />

3. 查看所请求的权限，然后点击 **Save**。
4. 切换到 **Web campaigns** 标签页，点击 **Create campaign**。选择应用并点击 **Save**。

<ZoomImage id="ua-new-campaign.webp" width="500px" />

5. 在 **General** 标签页中，展开 **iOS** 和/或 **Android** 部分，粘贴 App Store 和/或 Google Play 应用 URL，然后点击 **Save**。

<ZoomImage id="ua-url.gif" width="900px" />

6. 复制**单个链接**或平台专属链接的 **Click link** 字段值。然后，在 Meta Ads Manager 中，打开您的广告，将此链接粘贴为目标 URL。

:::important
在 **Website URL** 字段中，粘贴 `https://api-ua.adapty.io/api/v1/attribution/click`。将链接的其余部分粘贴到 **Tracking** 部分的 **URL parameters** 字段中。这有助于您的 Meta 广告通过审核。请参阅[在 Meta Ads Manager 中设置广告的更多建议](meta-create-campaign)。
:::

7. 现在，当您在 Meta Ads 中启动广告后，其数据将可在 Adapty UA 看板中进行分析。
</TabItem>
<TabItem value="tiktok" label="TikTok for Business">
要为 TikTok for Business 创建追踪链接：
1. 在 Adapty UA 看板中，前往 [Integrations > TikTok Ads](https://app.adapty.io/ua/integrations/tiktok/accounts)，然后点击 **Continue with TikTok**。
   <ZoomImage id="ua-connect-tiktok.webp" width="700px" />
2. 使用您的 TikTok 账号登录，然后点击 **Continue**。

3. 查看所请求的权限，然后点击 **Save**。

    <ZoomImage id="ua-tiktok-sign.webp" width="500px" />
4. 切换到 **Web campaigns** 标签页，点击 **Create campaign**。选择应用并点击 **Save**。

    <ZoomImage id="ua-new-campaign-tiktok.webp" width="500px" />

5. 在 **General** 标签页中，展开 **iOS** 和/或 **Android** 部分，粘贴 App Store 和/或 Google Play 应用 URL，然后点击 **Save**。

    <ZoomImage id="ua-url.gif" width="900px" />

6. 复制**单个链接**或平台专属链接的 **Click link** 字段值。然后，在 TikTok Ads Manager 中创建广告时，将此值粘贴到 **Advanced Settings** 部分的 **Tracking URL** 字段中。这将允许 Adapty 将安装和购买与 TikTok 中的广告关联起来。请参阅[在 TikTok Ads 中设置广告活动的指南](tiktok-create-campaign)。

    <ZoomImage id="ua-tiktok-lnk.webp" width="900px" />
7. 现在，当您在 TikTok for Business 中启动广告后，其数据将可在 Adapty UA 看板中进行分析。

</TabItem>
<TabItem value="others" label="Other ad platforms">
要为其他广告平台创建追踪链接：

1. 在 Adapty UA 看板中，从侧边栏菜单前往 **Tracking links**，然后点击 **Create link**。

    <ZoomImage id="new-tracking-link.webp" width="700px" />

2. 从列表中选择您的应用，然后点击 **Next**。

    <ZoomImage id="ua-choose-app.webp" width="700px" />

3. 填写链接参数，以便与您想要追踪的广告活动和广告匹配。
4. 默认情况下，您创建的是一个 One Link。它会自动检测用户的平台，并在追踪点击后将其重定向到 App Store 或 Google Play。

    如果您希望为每个平台使用单独的重定向 URL，请取消选中 **One Link** 复选框，并手动提供平台专属的应用商店链接。

    <ZoomImage id="one-link.webp" width="700px" />

5. 点击 **Create**。
6. 打开您的追踪链接页面，从以下某个部分复制 **Click link**：
    - **One link** – 使用此链接追踪点击并自动将用户重定向到正确的应用商店。
    - **iOS link** 或 **Android link** — 可选的平台专属版本，适用于希望为每个应用商店使用单独链接的情况。
7. 前往您的广告平台，将该链接粘贴为广告目标 URL。

</TabItem>
</Tabs>

## 第三步：启动网页转应用广告活动并查看结果 \{#step-3-launch-your-web-to-app-campaign-and-view-results\}

一旦您的广告活动上线且用户开始安装应用，Adapty 便会开始将安装和收入归因到您的广告活动。

在 [Adapty UA 数据分析看板](ua-analytics)中，您将看到广告活动级别的数据图表，例如：
- 安装量和转化量
- 订阅和购买收入
- 按广告平台、广告活动、广告组和创意素材细分的效果数据

一旦从您的应用收到安装和收入事件，数据图表便会出现。广告支出数据仅适用于具有原生集成的平台。
