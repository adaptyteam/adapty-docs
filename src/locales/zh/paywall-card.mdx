卡片是一种付费墙元素，它将多个其他元素组合成一个单独的块。如果不需要，卡片本身可以不可见。若要使其可见，可以为其添加背景或背景图片、边框等。

:::warning

付费墙卡片仅在[新版付费墙编辑工具](adapty-paywall-builder)中可用，该工具适用于 iOS、Android 和 React Native SDK 3.0 或更高版本，以及 Flutter 和 Unity SDK 3.3.0 或更高版本。使用 Adapty SDK v2.x 或更早版本的旧版付费墙编辑工具不支持付费墙卡片功能。

:::

1. 将卡片作为独立元素添加到付费墙中，或添加到其他付费墙元素中（例如轮播图）。
2. 在卡片中添加所需的元素。
3. 配置卡片的外观：背景、形状、边框等。
