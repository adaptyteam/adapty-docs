在 Adapty 中，**付费墙是通过应用向用户提供产品的唯一方式**。这样您就可以轻松追踪不同产品组合在各用户群中的表现，并管理产品的视觉呈现方式。

为了充分发挥 Adapty 的优势，本指南将帮助您创建一个付费墙，用于销售上一步添加的产品。

工作原理：
- **付费墙**：付费墙是一个或多个产品的容器。它可以包含在付费墙构建器中创建的可视化付费墙、产品信息，或供代码使用的 JSON 配置。了解更多关于[付费墙](paywalls.md)的内容。
- **版位**：版位是应用中展示付费墙、引导页或 A/B 测试的特定位置。版位允许您向特定[受众](audience.md)展示付费墙。了解更多关于[版位](placements.md)的内容。

:::note
即使您不使用 Adapty 设计付费墙，也仍然需要完成本指南并创建一个付费墙。这样，您才能将产品放入其中并追踪货币化分析数据。
:::

## 1. 构建付费墙 \{#1-build-paywall\}

创建付费墙只需几次点击：

1. 在 Adapty 主菜单中进入 [**Paywalls**](https://app.adapty.io/paywalls)。
2. 点击 **Create paywall**。
3. 输入 **Paywall name**。这是 Adapty 控制台中的内部标识符。
4. 点击 **Add product**，选择要在付费墙上展示的产品。
5. 点击 **Create as a draft**。

### 设计付费墙 \{#design-paywall\}

设计付费墙最简单的方式是使用 Adapty 无代码构建器，无需任何设计或编程技能。您可以从大量专业设计模板中选择，或构建完全自定义的付费墙，以适配您的应用。

:::note
如果您不想使用付费墙构建器，可以通过[远程配置](customize-paywall-with-remote-config.md)和自定义 JSON 数据手动实现付费墙。了解更多关于<InlineTooltip tooltip="implementing paywalls manually">Follow the guide for your platform: [iOS](ios-implement-paywalls-manually.md), [Android](android-implement-paywalls-manually.md), [React Native](react-native-implement-paywalls-manually.md), [Flutter](flutter-implement-paywalls-manually.md), [Unity](unity-implement-paywalls-manually.md).</InlineTooltip>的内容。
:::

让我们来设计您的第一个付费墙。您可以轻松打造引人入胜的付费墙：

1. 在付费墙页面打开 **Builder & Generator**。
2. 点击 **Build no-code paywall**。
3. 选择一个模板并确认您的选择。
4. 根据需要添加和自定义元素。
5. 点击 **Save**。

了解更多内容，请查看关于[付费墙构建器](adapty-paywall-builder.md#paywall-elements)的详细文章。

## 2. 将付费墙添加到版位 \{#2-add-paywall-to-placement\}

现在您需要创建一个<InlineTooltip tooltip="placement">A placement is a specific point in your app where you show a paywall, onboarding flow, or A/B test. Placements let you target specific [audiences](audience.md) with your paywall. Learn more about [placements](placements.md).</InlineTooltip>，并将刚刚创建的付费墙添加进去。

我们从最核心的引导页版位开始。之后，您可以在用户旅程中添加更多[有意义的版位](choose-meaningful-placements.md)。

1. 在 Adapty 主菜单中进入 [**Placements**](https://app.adapty.io/placements/paywalls)。
2. 点击 **Create placement**。
3. 输入 **Placement name**（例如 `main` 或 `onboarding`）。这是 Adapty 控制台中的内部标识符。
4. 输入 **Placement ID**。您将在 Adapty SDK 中使用此 ID 来加载该版位的付费墙。
5. 点击 **Run Paywall**，选择您想要展示的付费墙。
6. 点击 **Save & publish**。

在应用代码中，您只需硬编码版位 ID。其他所有内容均在 Adapty 控制台中配置，随时可以更改，无需更新应用。

:::tip
Adapty 为您提供灵活性，可以向不同用户群展示不同付费墙并分析其表现。了解更多关于[受众](audience.md)和 [A/B 测试](ab-tests.md)的内容。
:::
