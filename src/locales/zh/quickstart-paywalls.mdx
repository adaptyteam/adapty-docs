在 Adapty 中，**付费墙是通过应用向用户提供产品的唯一方式**。这样您可以轻松追踪不同产品组合在各用户群体中的表现，并管理产品的视觉呈现方式。

为了充分发挥 Adapty 的优势，本指南将引导您创建一个付费墙，以销售您在上一步骤中添加的产品。

工作原理：
- **付费墙**：付费墙是一个或多个产品的容器。它可以包含在付费墙编辑工具中创建的可视化付费墙、产品信息，或供代码使用的 JSON 配置。详细了解[付费墙](paywalls)。
- **版位**：版位是应用中展示付费墙、用户引导流程或 A/B 测试的特定位置。版位允许您向特定[目标受众](audience)展示付费墙。详细了解[版位](placements)。

:::note
即使您不使用 Adapty 设计付费墙，仍然需要完成本指南并创建一个付费墙。这样，您才能将产品放入其中并追踪变现分析数据。
:::

## 1. 创建付费墙 \{#1-build-paywall\}

创建付费墙只需几次点击：

1. 在 Adapty 主菜单中前往 [**Paywalls**](https://app.adapty.io/paywalls)。
2. 点击 **Create paywall**。
3. 输入**付费墙名称**（**Paywall name**）。这是 Adapty 控制台中的内部标识符。
4. 点击 **Add product**，选择要在付费墙上展示的产品。
5. 点击 **Create as a draft**。

### 设计付费墙 \{#design-paywall\}

设计付费墙最简单的方式是使用 Adapty 无代码编辑工具，无需任何设计或编码技能。您可以从大量专业设计模板中进行选择，或构建完全自定义的付费墙以适配您的应用。

:::note
如果您不想使用付费墙编辑工具，可以使用带有自定义 JSON 有效负载的[远程配置](customize-paywall-with-remote-config)手动实现付费墙。详细了解<InlineTooltip tooltip="implementing paywalls manually">请参考您所使用平台的指南：[iOS](ios-implement-paywalls-manually)、[Android](android-implement-paywalls-manually)、[React Native](react-native-implement-paywalls-manually)、[Flutter](flutter-implement-paywalls-manually)、[Unity](unity-implement-paywalls-manually)。</InlineTooltip>。
:::

:::tip
如果您的应用已发布在 App Store 上，您可以在几秒钟内创建一个专为您的应用量身定制的高转化率付费墙。请使用 **Builder & Generator** 标签页中的 AI 生成器。
:::

让我们来设计您的第一个付费墙。您可以轻松创建引人入胜的付费墙：

1. 在付费墙页面打开 **Builder & Generator**。
2. 点击 **Build no-code paywall**。
3. 选择一个模板并确认您的选择。
4. 根据需要添加和自定义元素。
5. 点击 **Save**。

如需了解更多，请参阅关于[付费墙编辑工具](adapty-paywall-builder#paywall-elements)的详细文章。

## 2. 将付费墙添加到版位 \{#2-add-paywall-to-placement\}

现在您需要为刚刚创建的付费墙创建一个<InlineTooltip tooltip="placement">版位是应用中展示付费墙、用户引导流程或 A/B 测试的特定位置。版位允许您向特定[目标受众](audience)展示付费墙。详细了解[版位](placements)。</InlineTooltip>。

我们先从最核心的版位开始——用户引导版位。之后，您可以在用户旅程中添加更多[有意义的版位](choose-meaningful-placements)。

1. 在 Adapty 主菜单中前往 [**Placements**](https://app.adapty.io/placements/paywalls)。
2. 点击 **Create placement**。
3. 输入**版位名称**（**Placement name**）（例如 `main` 或 `onboarding`）。这是 Adapty 控制台中的内部标识符。
4. 输入**版位 ID**（**Placement ID**）。您将在 Adapty SDK 中使用此 ID 来加载版位对应的付费墙。
5. 点击 **Run Paywall**，选择您想要展示的付费墙。
6. 点击 **Save & publish**。

在您的应用代码中，只需硬编码版位 ID。其他所有内容均在 Adapty 控制台中配置，随时可以修改，无需更新应用。

:::tip
Adapty 提供灵活性，可向不同用户群体展示不同的付费墙并分析表现。详细了解[目标受众](audience)和 [A/B 测试](ab-tests)。
:::
