在付费墙中添加视频片段是吸引用户、提升转化率的绝佳方式。视频能够更清晰地展示您的产品，突出核心功能，并打造更具动感和视觉吸引力的体验，从而抓住用户注意力。

:::warning

主视频在以下版本的 Adapty SDK 中受支持：

- iOS：从 v3.1.0 起
- Android：从 v3.1.1 起
- React Native：从 v3.1.0 起
- Flutter：从 v3.2.0 起
- Unity：从 v3.3.0 起

如果不支持视频或在备用情况下，将显示视频的第一帧。

:::

在 **Hero image** 元素的位置添加 **Hero video**：

1. 首先，切换到视频模式。

## 支持的格式 \{#supported-formats\}

| 规格     | 详情          |
|----------|---------------|
| 扩展名   | MP4 和 WEBM   |
| 最小尺寸 | 640×480       |
| 最大时长 | 30 秒         |
| 音频     | 不支持        |
