Adapty User Acquisition 提供全面的**数据图表**，用于衡量广告系列效果和用户行为。这些数据图表以标准值形式提供，其中部分数据图表还以**同期群数据图表**的形式提供，用于对用户群体进行基于时间的分析。

| **数据图表**                        | 描述                                                                                                                                                                                                                                                                                                                                                                                                                                                     | 同期群 |
|-----------------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------|
| **Spend**                         | 用户每次点击您广告的费用总和。                                                                                                                                                                                                                                                                                                                                                                                                            | 否     |
| **Impressions**                   | 在所选时间段内，您的广告被展示的次数。                                                                                                                                                                                                                                                                                                                                                                                           | 否     |
| **Clicks**                        | 在报告期内，用户点击您广告的次数。                                                                                                                                                                                                                                                                                                                                                                                       | 否     |
| **CPI**                           | **CPI（每次安装费用）** 是您为每次安装所支付的金额。<br/>**公式**：`Spend / Installs`                                                                                                                                                                                                                                                                                                                                         | 否     |
| **CPC**                           | **CPC（每次点击费用）** 是您为每次广告点击所支付的金额。<br/>**公式**：`Spend / Clicks`                                                                                                                                                                                                                                                                                                                                                    | 否     |
| **CPM**                           | **CPM（每千次展示费用）** 是您为每一千次广告展示所支付的金额。<br/>**公式**：`Spend / (Impressions / 1000)`                                                                                                                                                                                                                                                                                                                                | 否     |
| **ICR**                           | **ICR（安装转化率）** 是广告点击中产生安装的百分比。<br/>**公式**：`(Installs / Clicks) × 100%`                                                                                                                                                                                                                                                                                                                                      | 否     |
| **IPM**                           | **IPM（每千次展示安装数）** 表示每一千次广告展示的安装数量。<br/>**公式**：`(Installs / Impressions) × 1000`                                                                                                                                                                                                                                                                                                                             | 否     |
| **CTR**                           | **CTR（点击率）** 是展示次数中产生点击的百分比。<br/>**公式**：`(Clicks / Impressions) × 100%`                                                                                                                                                                                                                                                                                                                                        | 否     |
| **Inline link clicks**            | 用户点击广告素材或应用页面中内联链接的次数。                                                                                                                                                                                                                                                                                                                                                                                | 否     |
| **Cost per inline link click**    | 每次内联链接点击的平均费用。<br/>**公式**：`Spend / Inline Link Clicks`                                                                                                                                                                                                                                                                                                                                                | 否     |
| **Inline link click CTR**         | 展示次数中产生内联链接点击的百分比。<br/>**公式**：`(Inline Link Clicks / Impressions) × 100%`                                                                                                                                                                                                                                                                                                                                                | 否     |
| **Installs**                      | 在报告期内安装您应用的用户总数（包括重新安装）。                                                                                                                                                                                                                                                                                                                                                                           | 否     |
| **Revenue**                       | 在所选时间段内，与该广告系列关联的购买所产生的总收入（扣除应用商店佣金前）。                                                                                                                                                                                                                                                                                                                                                             | 是    |
| **ROAS**                          | **ROAS（广告支出回报率）** 是广告收入除以广告支出，以百分比表示。<br/>**公式**：`(Revenue / Spend) × 100%（若 Spend > 0，否则为 0%）`                                                                                                                                                                                                                                                                                                                            | 是    |
| **ARPU**                          | **ARPU（每用户平均收入）** 是同期群中每位用户的平均收入。<br/>**公式**：`Revenue / Users`                                                                                                                                                                                                                                                                                                                                          | 是    |
| **LTV**                           | **LTV（生命周期价值）** 是归因于每位用户在其生命周期内的平均收入。<br/>**公式**：`Revenue / Installs`                                                                                                                                                                                                                                                                                                                                | 否     |
| **Cost per trial**                | 每次试用开始的平均费用。<br/>**公式**：`Spend / Count trial started`                                                                                                                                                                                                                                                                                                                                                   | 否     |
| **Cost per subscription**         | 每次订阅产品购买的平均费用。<br/>**公式**：`Spend / Count subscription started`                                                                                                                                                                                                                                                                                                                                           | 否     |
| **Count subscription events**     | 用于统计报告期内订阅相关事件的一组数据图表，包括：<br/>- Count subscription started<br/>- Count subscription renewed<br/>- Count subscription renewal cancelled<br/>- Count subscription renewal reactivated<br/>- Count subscription expired<br/>- Count [subscription deferred](https://adapty.io/glossary/subscription-purchase-deferral/)<br/>- Count subscription refunded               | 是    |
| **Count trial events**            | 用于统计报告期内试用相关事件的一组数据图表，包括：<br/>- Count trial started<br/>- Count trial converted<br/>- Count trial expired<br/>- Count trial renewal reactivated                                                                                                                                                                                                                                                       | 是    |
| **Count billing issue detected**  | 在报告期内检测到的账单问题数量。                                                                                                                                                                                                                                                                                                                                                                               | 是    |
| **Count entered grace period**    | 因账单问题而进入宽限期的订阅数量。                                                                                                                                                                                                                                                                                                                                                                                  | 是    |
| **Count non-subscription events** | 用于统计报告期内非订阅相关事件的一组数据图表，包括：<br/>- Count non-subscription purchased<br/>- Count non-subscription refunded                                                                                                                                                                                                                                                                                      | 是    |
| **Subscription events rate**      | 显示报告期内订阅相关事件相对于应用安装数的比率的数据图表，包括：<br/>- Rate subscription started<br/>- Rate subscription renewed<br/>- Rate subscription renewal cancelled<br/>- Rate subscription renewal reactivated<br/>- Rate subscription expired<br/>- Rate [subscription deferred](https://adapty.io/glossary/subscription-purchase-deferral/)<br/>- Rate subscription refunded | 是    |
| **Trial events rate**             | 显示报告期内试用相关事件相对于应用安装数的比率的数据图表，包括：<br/>- Rate trial started<br/>- Rate trial converted<br/>- Rate trial expired<br/>- Rate trial renewal reactivated                                                                                                                                                                                                                                      | 是    |
| **Rate billing issue detected**   | 报告期内账单问题相对于应用安装数的比率。                                                                                                                                                                                                                                                                                                                                                              | 是    |
| **Rate entered grace period**     | 报告期内进入宽限期的订阅相对于应用安装数的比率。                                                                                                                                                                                                                                                                                                                                                                     | 是    |
| **Non-subscription events rate**  | 显示报告期内非订阅相关事件相对于应用安装数的比率的数据图表，包括：<br/>- Rate non-subscription purchased<br/>- Rate non-subscription refunded                                                                                                                                                                                                                                                                   | 是    |
