付费墙远程配置是一个功能强大的工具，提供灵活的配置选项。它支持使用自定义 JSON 数据精确定制您的付费墙。通过它，您可以定义标题、图片、字体、颜色等各种参数。

<details>
   <summary>开始自定义付费墙之前（点击展开）</summary>

   1. [创建产品](create-product)。
2. [创建付费墙并添加产品](create-paywall)。
</details>

要使用远程配置开始自定义付费墙：

1. 在 Adapty 主菜单中打开 [**Paywalls**](https://app.adapty.io/paywalls) 部分。

2. 点击付费墙以打开它。

远程配置有两种视图：

- [表格视图](customize-paywall-with-remote-config#table-view-of-the-remote-config)
- [JSON 视图](customize-paywall-with-remote-config#json-view-of-the-remote-config)

**Table** 和 **JSON** 视图包含相同的配置元素，区别仅在于个人偏好，唯一不同之处在于表格视图提供了上下文菜单，这对于修正本地化错误非常有帮助。  
您可以随时点击 **Table** 或 **JSON** 标签页在两种视图之间切换。

无论您选择哪种视图来自定义付费墙，都可以通过 SDK 使用 `AdaptyPaywall` 的 `remoteConfig` 或 `remoteConfigString` 属性访问这些数据，并对付费墙进行调整。您也可以通过[服务端 API](api-adapty/operations/updatePaywall) 以编程方式更新远程配置值，从而无需手动更新看板即可动态修改付费墙配置。以下是一些使用远程配置的示例。
