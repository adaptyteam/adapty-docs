用户引导远程配置是一个提供灵活配置选项的工具。它允许使用自定义 JSON 载荷来精确定制您的用户引导。借助它，您可以定义标题、图片、字体、颜色等各种参数。例如，您可以使用远程配置发送额外的元数据。

在开始自定义用户引导之前，请先[创建用户引导](create-onboarding)。

要使用远程配置开始自定义用户引导：

1. 从 **Onboardings** 打开用户引导，然后点击 **Edit onboarding**。

2. 切换到 **Remote config** 标签页。

远程配置有两种视图：

- [表格视图](customize-paywall-with-remote-config#table-view-of-the-remote-config)
- [JSON 视图](customize-paywall-with-remote-config#json-view-of-the-remote-config)

**Table** 和 **JSON** 视图包含相同的配置元素。两者的区别仅在于使用偏好，唯一的不同之处在于表格视图提供了上下文菜单，这对纠正本地化错误很有帮助。  
您可以随时点击 **Table** 或 **JSON** 标签页在两种视图之间切换。

无论您选择哪种视图来自定义用户引导，您都可以在之后通过 SDK 使用 `AdaptyOnboarding` 的 `remoteConfig` 属性访问这些数据，并对您的用户引导进行一些调整。

您可以组合不同的选项并创建自己的配置。
