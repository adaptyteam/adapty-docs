您可以将付费墙连接到用户引导，使用户在用户引导流程中或完成后立即看到订阅优惠。

有两种方式可以实现：

- **在用户引导结束后显示付费墙**：当用户关闭用户引导时，付费墙自动打开。如果您希望每个用户在流程结束时都能看到付费墙，无需任何操作，请选择此选项。
- **在用户引导内部显示付费墙**：用户引导中的按钮触发付费墙。如果您希望用户能够在流程中的特定时刻订阅——例如，在他们在某个屏幕上看到价值主张之后——请选择此选项。

在开始之前，请先创建[付费墙](paywalls)和[用户引导](onboardings)，并将它们添加到版位中。

:::important
您需要两个不同的版位：一个用于付费墙，另一个用于用户引导。在代码中获取用户引导和付费墙时，请确保使用正确的版位 ID。
:::

## 在用户引导结束后显示付费墙 \{#show-paywall-after-onboarding\}

要在用户引导结束后显示付费墙，您只需处理每次用户关闭用户引导时生成的事件。

用户关闭用户引导后，会触发[事件](ios-handling-onboarding-events#closing-onboarding)。因此，如果您希望在用户引导结束后立即显示付费墙，可以将 <InlineTooltip tooltip="getting and displaying a paywall">[iOS](ios-present-paywalls)、[Android](android-present-paywalls)、[React Native](react-native-present-paywalls)、[Flutter](flutter-present-paywalls) 和 [Unity](unity-present-paywalls)</InlineTooltip> 中的实现作为对该事件的响应。

## 在用户引导内部显示付费墙 \{#show-paywall-inside-onboarding\}

1. 在用户引导编辑工具中，创建一个将用户重定向到付费墙的按钮。选择 **Open paywall** 作为其操作。

2. 您可以为该按钮指定任意操作 ID，并用它来标识需要打开的付费墙。不过，从用户引导中打开付费墙最简便的方式是将操作 ID 设置为版位 ID。这样，您无需在代码中硬编码版位 ID，即可直接获取并显示付费墙。

   为此，请前往 Adapty 看板，找到付费墙的版位 ID。在编辑工具中，将付费墙版位 ID 粘贴到 ID 字段中。
