您还可以为特定元素添加条件可见性。条件元素仅对给出特定测验答案的用户可见。

例如，对"您的经验水平如何"回答"初学者"的用户将在下一步中看到额外的详细信息。

要使元素具有条件性：
1. 点击右侧的 **Element** 选项卡。
2. 在 **Visible** 部分选择 **Conditional**。
3. 通过选择以下内容来设置条件：
- 测验 ID
- 运算符
- 测验答案
4. （可选）点击 **Advanced condition** 以添加多个条件。

例如，如果您选择了 'goal' 作为测验 ID，'Has' 作为运算符，'Education' 作为测验答案，则该元素仅对该测验答案中包含 'Education' 选项的用户可见。
