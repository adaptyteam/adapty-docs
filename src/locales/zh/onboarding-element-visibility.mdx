你可以通过为元素添加条件来控制它是否显示。带条件的元素只对满足指定条件的用户可见。

:::important
如果你使用 **Show** 或 **Hide** [操作](onboarding-actions)来显示或隐藏某个元素，该操作会覆盖该元素上设置的 **Visibility** 条件。
对于应始终根据固定条件显示或隐藏的元素，请使用**可见性**条件。当可见性需要根据用户交互动态变化时（例如，用户回答测验问题后显示某个按钮），请使用操作。
:::

为元素添加条件：

1. 在画布或图层面板中选择该元素。
2. 在右侧面板的**可见性**部分，选择**条件式**。

<ZoomImage id="flow-conditional-visibility.webp" width="500px" alt="条件可见性设置" />
3. 通过从以下三个标签页中选择属性类型来设置条件：
   - **Custom**：由你创建和管理的变量，其值可通过用户交互更新。详情请参阅[变量](onboarding-variables)。
   - **Products**：流程中产品的属性，例如价格或名称。
   - **Elements**：流程中其他元素的状态，例如试用开关是否已激活。
4. 输入要匹配的 **Value**。
5. 点击运算符可根据需要进行更改。
6. （可选）点击 **Add condition** 添加更多条件。使用选择器来要求所有条件都匹配，或任意一个条件匹配即可。
