轮播是一组可左右滑动的动态卡片，能带来引人入胜的视觉体验。它是打造付费墙的绝佳工具，不仅能吸引用户注意，还能通过互动内容提升用户参与度。

:::warning

轮播功能仅在[新版付费墙编辑工具](adapty-paywall-builder)中可用，需配合 iOS、Android 和 React Native SDK 3.0 或更高版本，以及 Flutter 和 Unity SDK 3.3.0 或更高版本使用。基于 Adapty SDK v2.x 或更早版本的旧版付费墙编辑工具不支持付费墙轮播功能。

:::

借助轮播的强大功能，提升付费墙的吸引力：

1. **添加卡片**：首先向轮播中添加卡片，每张卡片都是展示创意的画布。
2. **自定义卡片**：在卡片中填充各种元素——图片、文字、按钮等。轮播的高度设置确保所有卡片尺寸统一，呈现简洁一致的视觉效果。

充分发挥轮播的灵活性，展示精选产品、突出专属优惠或讲述引人入胜的故事。借助这一互动元素，您的付费墙不仅会脱颖而出，还能为用户带来流畅而沉浸式的体验。
