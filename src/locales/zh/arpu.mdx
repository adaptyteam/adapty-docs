ARPU（每用户平均收入）数据图表显示给定时期内每位用户产生的平均收入。该数据图表通过将一组客户产生的总收入除以该组用户数量来计算得出。

### 计算方式 \{#calculation\}

:::warning
以下计算器**不考虑**[应用商店佣金和税费](how-adapty-analytics-works#tax-and-commission-logic)。请将计算结果与您的**总收入**进行比对。
:::

Adapty 计算 ARPU 的方式是：将特定时间段内的总收入除以同期应用的用户总数。

在交互式计算器中输入您的销售额、退款及用户数，即可计算出毛 ARPU：
<CompoundCalculator client:load
    heading="ARPU"
    formuLatex="\frac{\sum P_i \times Q_i - D}{U_p}"
    variables={[
        {
            nameInTheFormula: "P",
            variableName: "price",
            variableDescription: "产品价格",
            variableValue: 10
        },
        {
            nameInTheFormula: "Q",
            variableName: "qty",
            variableDescription: "已购产品数量",
            variableValue: 1,
            isInteger: true
        },
        {
            nameInTheFormula: "D",
            variableName: "refunds",
            variableDescription: "退款金额",
            variableValue: 35,
            global: true
        },
        {
            nameInTheFormula: "Up",
            variableName: "users",
            variableDescription: "总用户数",
            variableValue: 160,
            global: true,
            isInteger: true
        }
    ]}
    rowFormula="price * qty"
    resultFormula="(_sum - refunds) / users"
    defaultRows={[
        { price: 10, qty: 5 },
        { price: 50, qty: 10 },
        { price: 100, qty: 1 }
    ]}
/>
举个例子，如果你的应用在一周内产生了 10,000 美元的收入，同期共有 2,000 名非唯一用户，那么该周的 ARPU 为 5 美元（10,000 / 2,000）。
计算在扣除应用商店费用之前进行，且退款金额不计入收入。

### ARPU 数据图表的用途 \{#arpu-chart-usage\}

ARPU 数据图表对企业追踪整体收入产生情况以及了解每位用户带来的收入非常有价值。通过分析 ARPU 数据图表，企业可以识别趋势、规律和改进领域，从而优化收入产生策略。这有助于企业做出数据驱动的决策，以提升用户参与度、精准定向营销工作并改善整体变现策略。
