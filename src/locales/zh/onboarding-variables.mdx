变量是根据用户输入或环境数据设置的值。它们对于创建个性化且引人入胜的用户引导体验至关重要。

## 变量的用途 \{#what-variables-are-for\}

变量允许您将动态数据（如测验回答或用户文本输入）直接插入到用户引导屏幕中。每位用户都能看到个性化内容，无需任何编码。例如，使用文本输入以用户姓名问候用户，或将测验答题者引导至自定义的后续屏幕。

使用变量时，需将数据源的元素 ID 放置在双大括号之间，格式如下：`{{element_id}}`。

作为变量，您可以使用在前序屏幕上收集的数据：
- **输入框**：变量包含用户输入的数据。
- [**测验**](onboarding-quizzes)：变量包含所选选项的标签数据。如果允许多选，变量将包含所有已选选项，以逗号和空格分隔。

:::note
元素 ID 与用于自定义操作逻辑的[操作 ID](onboarding-actions#action-id) 不同，请注意不要混淆。
:::
