## 授权 \{#authorization\}

您需要使用您的私密 API 密钥作为 Authorization 请求头来验证 API 请求。您可以在 [App Settings](https://app.adapty.io/settings/general) 中找到它。格式为 `Api-Key {YOUR_SECRET_API_KEY}`，例如：`Api-Key secret_live_...`。

## 请求格式 \{#request-format\}

**请求头**

服务端 API 请求需要特定的请求头和 JSON 请求体。请使用以下详细信息来构建您的请求：

| 请求头       | 描述                                                         |
| ------------ | ------------------------------------------------------------ |
| Content-Type | （必填）设置为 `application/json` 以便 API 处理请求。        |
| Adapty-Tz    | （可选）设置时区以定义数据的分组和显示方式。请使用 [IANA 时区数据库格式](https://en.wikipedia.org/wiki/List_of_tz_database_time_zones)（例如 `Europe/Berlin`）。 |

## 请求体 \{#body\}

API 需要一个 JSON 格式的请求体，其中包含请求所需的数据。
