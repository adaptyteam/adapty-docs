Apple Ads Manager 与 Apple Ads 实现了双向集成：您可以获取近实时的效果数据，并且可以直接在 Adapty 看板中以比原生界面更便捷的方式创建和编辑关键词。

如果您在原生 Apple Ads 看板中创建了关键词，它将在 24 小时内自动出现在 Apple Ads Manager 中。

除了[探索全面的数据分析](adapty-ads-manager-analytics)之外，您还可以管理所有关键词设置：
- 向广告组添加关键词
- 添加否定关键词
- 将关键词添加为 SKAG（单关键词广告组）
- 直接在表格中编辑关键词
- 对多个关键词执行批量操作
- 启动和暂停关键词

## 什么是关键词 \{#what-are-keywords\}

关键词是触发您的广告出现在 App Store 搜索结果中的搜索词。它们组织在[广告组](ads-manager-create-ad-group)内，而广告组隶属于[广告活动](ads-manager-create-campaign)。这种层级结构使您能够有效地组织和管理广告策略。

:::important
关键词仅适用于**搜索结果**版位类型的广告活动。对于其他版位类型（搜索标签页或产品页面）的广告活动，不使用关键词。
:::

### 标准关键词 \{#standard-keywords\}

标准关键词是您出价以触发广告的主要词语。当用户在 App Store 中搜索这些词语时，您的广告可能会出现在搜索结果中。

### 否定关键词 \{#negative-keywords\}

否定关键词可防止您的广告出现在与您的应用不相关的搜索中。通过添加否定关键词，您可以减少在无关搜索上的无效支出。

否定关键词可以在广告组级别添加，也可以作为跨组否定关键词同时应用于多个广告活动。

### 关键词作为 SKAG（单关键词广告组）\{#keywords-as-skag-single-keyword-ad-group\}

SKAG（单关键词广告组）是一种策略，即为每个关键词创建单独的广告组。这种方式可以让您：
- 精确控制高价值关键词的出价
- 更好地在关键词级别分析效果

SKAG 对于识别表现最佳的关键词并通过专属广告组最大化其潜力特别有用。

## 添加关键词 \{#add-keywords\}

要向广告组添加关键词：

1. 从侧边栏菜单进入 **Ads Manager**。在任意标签页上，点击表格上方的 **+**，然后从下拉菜单中选择 **Add keywords**。

<ZoomImage id="add-keywords-action.webp" width="700px" /> 

2. 在弹窗中，选择您要添加关键词的广告活动和广告组。在一个广告活动中选择广告组后，您可以再选择另一个广告活动并向列表中添加更多广告组。

<ZoomImage id="select-ad-groups-keywords.webp" width="700px" />

3. 点击 **Select** 继续。
4. 在 **Add keywords** 对话框中，在 **Keywords list** 字段中输入关键词。如果您有包含关键词的逗号分隔文件，可以粘贴其内容，Apple Ads Manager 将批量上传所有关键词。
5. 对表格中的每个关键词进行配置：
   - **Match type**：选择 **Exact**（精确）或 **Broad**（广泛）匹配类型
   - **CPT bid**：为该关键词设置最高每次点击费用出价，或留空以使用广告组的默认最高 CPT 出价
6. 查看您的关键词，然后点击 **Add X keywords**（X 为您要添加的关键词数量）。

:::important
关键词保存后，其匹配类型无法更改。如需更改匹配类型，请删除该关键词并以所需匹配类型重新添加。
:::

## 添加否定关键词 \{#add-negative-keywords\}

要添加否定关键词：

1. 从侧边栏菜单进入 **Ads Manager**。在任意标签页上，点击表格上方的 **+**，然后从下拉菜单中选择 **Add negative keywords**。
   <ZoomImage id="add-negative-keywords.webp" width="700px" />
2. 在 **Add negative keywords to** 弹窗中，选择您要在哪个级别添加否定关键词：
   - **Selected campaigns**：添加广告活动级别的否定关键词。
   - **Selected ad groups**：添加广告组级别的否定关键词。
   - **All ad groups in selected campaigns**：向所选广告活动中的所有广告组添加广告组级别的否定关键词。
   :::note
   请注意以下事项：
   - 广告组级别的否定关键词优先级高于广告活动级别的否定关键词。
   - 如果您向所选广告活动的所有广告组添加了否定关键词，后续若向这些广告活动添加新广告组，则需要手动添加否定关键词。
   :::

  <ZoomImage id="select-campaigns.gif" width="700px" /> 

3. 在 **Keywords list** 字段中输入否定关键词。如果您有包含关键词的逗号分隔文件，可以粘贴其内容，Apple Ads Manager 将批量上传所有关键词。
4. 对表格中的每个关键词，选择 **Match type**：
   - **Exact**：仅排除精确关键词或非常接近的变体。
   - **Broad**：排除该关键词及相关搜索词。
      或选中其旁边的复选框，批量更改匹配类型。
5. 查看您的否定关键词，然后点击 **Add X keywords**（X 为您要添加的关键词数量）。

:::note
跨组否定关键词在您希望一次性排除多个广告活动中的特定搜索词时特别有用，可节省时间并确保广告策略的一致性。
:::
   <ZoomImage id="negative-keywords.gif" width="700px" />

## 将关键词添加为 SKAG \{#add-keywords-as-skag\}

要将关键词添加为 SKAG（单关键词广告组）：

1. 从侧边栏菜单进入 **Ads Manager**。在任意标签页上，点击表格上方的 **+**，然后从下拉菜单中选择 **Add keywords as SKAG**。
   <ZoomImage id="add-skag-keywords.webp" width="700px" />
2. 选择您要创建 SKAG 广告组的广告活动，可以选择多个广告活动。
3. 默认情况下，新广告组将以针对所有用户的默认设置创建。如果您想更改，请选择 **Copy settings from ad group** 并选择一个现有广告组以复制其设置。
   <ZoomImage id="copy-settings-from-ad-group.webp" width="700px" />
4. 配置新广告组的设置：
   - **Ad group name prefix**：可选前缀，添加到每个广告组名称前（例如，"SKAG_" 将创建 "SKAG_keyword1"、"SKAG_keyword2" 等）。您可以点击 **Tag** 动态添加关键词、广告活动名称和国家/地区到组名中。
   - **CPT bid** 和 **CPA cap**：一次性为所有关键词设置出价，或选择 **Set CPT bid and CPA cap for each word manually** 并为每个关键词单独设置。
5. 在 **Keywords list** 字段中输入关键词。如果您有包含关键词的逗号分隔文件，可以粘贴其内容，Apple Ads Manager 将批量上传所有关键词。
6. 对表格中的每个关键词，选择 **Match type**：
   - **Exact**：仅匹配精确关键词或非常接近的变体
   - **Broad**：匹配该关键词及相关搜索词
     或选中其旁边的复选框，批量更改匹配类型。
7. 选择 **Check for duplicates in target campaign** 以确保目标广告活动中没有相同的关键词。
8. 点击 **Create** 创建 SKAG 广告组。

每个关键词将在每个所选广告活动中被放置在其专属广告组中，使您能够独立管理和优化它们。

<ZoomImage id="new-skag.webp" width="700px" />

## 编辑关键词 \{#edit-keywords\}

要编辑现有关键词：

1. 进入 **Ads Manager > Keywords** 或 **Ads Manager > Negative keywords**，在表格中找到您要编辑的关键词，或导航到广告活动页面，再进入广告组页面找到该关键词。
2. 直接在表格中编辑值：
   - **CPT bid**：点击出价值并输入新的最高每次点击费用出价
   - **Status**：使用切换开关暂停或激活关键词

:::note
直接在 Apple Ads 中对关键词所做的编辑会自动同步到 Apple Ads Manager，但可能需要一些时间才能在 Apple Ads Manager 中显示。
:::

<ZoomImage id="edit-keywords.webp" width="500px" />

## 批量操作 \{#bulk-actions\}

您可以对多个关键词执行批量操作，以节省时间并更高效地管理关键词。

要执行批量操作：

1. 进入 **Ads Manager > Keywords** 或 **Ads Manager > Negative keywords** 标签页。
2. 勾选您要管理的关键词旁边的复选框以选择多个关键词。
3. 点击 **Actions** 下拉菜单，选择以下选项之一：
   - **Add as keywords**：将所选关键词添加为标准关键词
   - **Add as negative keywords**：将所选关键词添加为否定关键词
   - **Add as SKAG**：为所选关键词创建单关键词广告组
   - **Activate**：激活所选关键词
   - **Pause**：暂停所选关键词
   - **Edit CPT bids**：编辑所选关键词的 CPT 出价。您可以通过不同方式进行编辑：
     - **Set to**：将多个出价设置为特定金额。
     - **Increase by/decrease by**：按特定美元金额或出价百分比增加或减少出价。可选择设置出价上限，以避免意外超支。
     - **Set to average CPT**：使用 CPT（每次点击费用）数据图表将出价与之对齐。设置乘数系数。例如，当效果低于预期时将乘数设为 0.9，当效果超出预期时设为 1.1。
     - **Set to average CPA**：使用 CPA（每次获取费用）数据图表将出价与之对齐。设置乘数系数。

:::tip
批量操作特别适用于：
- 在不同类型之间转换关键词（标准、否定、SKAG）
- 快速为多个关键词添加具有其他匹配类型的关键词
- 筛选表现最佳的关键词并调整其出价
- 识别表现不佳的关键词并将其暂停
:::

## 探索关键词级别数据图表 \{#explore-keyword-level-charts\}

您可以直接从 **Ads Manager > Keywords** 表格中打开任意关键词的数据图表，从而对每个关键词进行精确的逐日效果分析。

要显示数据图表，请点击表格中关键词旁边的图表图标。

<ZoomImage id="keyword-chart.webp" width="500px" />

默认情况下，数据图表将显示所选关键词的 **Spend**（支出）数据图表。

您可以同时显示多个数据图表以发现相关性和随时间的变化。点击 **+** 添加新的数据图表。

点击 **Reset** 重新开始，或直接取消勾选数据图表复选框以隐藏它们。

<ZoomImage id="keyword-chart.gif" width="700px" />
