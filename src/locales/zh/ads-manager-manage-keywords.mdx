Apple Ads Manager 与 Apple Ads 实现了双向集成：您可以获得近实时的效果数据，并且可以直接从 Adapty 看板创建和编辑关键词，比在原生界面中操作更加便捷。
如果您在原生 Apple Ads 看板中创建了关键词，它将在 24 小时内自动出现在 Apple Ads Manager 中。

除了[查看全面的数据分析](adapty-ads-manager-analytics)之外，您还可以管理所有关键词设置：
- 将关键词添加到广告组
- 添加否定关键词
- 将关键词添加为 SKAG（单关键词广告组）
- 直接在表格中编辑关键词
- 对多个关键词执行批量操作
- 启动和暂停关键词
## 什么是关键词 \{#what-are-keywords\}

关键词是触发您的广告出现在 App Store 搜索结果中的搜索词。它们在[广告组](ads-manager-create-ad-group)中进行组织，而广告组属于[广告系列](ads-manager-create-campaign)。这种层级结构可以让您有效地组织和管理广告策略。

:::important
关键词仅适用于版位类型为 **Search results** 的广告系列。对于其他版位类型（Search tab 或 Product pages）的广告系列，不使用关键词。
:::
### 标准关键词 \{#standard-keywords\}

标准关键词是您竞价的主要词条，用于触发广告展示。当用户在 App Store 中搜索这些词条时，您的广告可能会出现在搜索结果中。
### 否定关键词 \{#negative-keywords\}

否定关键词可防止您的广告出现在与您的应用无关的搜索结果中。通过添加否定关键词，您可以减少因无关搜索而造成的无效消耗。

否定关键词可以在广告组层级添加，也可以作为跨组否定关键词，同时应用于多个广告系列。
### 关键词作为 SKAG（单关键词广告组）\{#keywords-as-skag-single-keyword-ad-group\}

SKAG（单关键词广告组）是一种为每个广告组仅设置单个关键词的策略。这种方法允许您：
- 对高价值关键词的出价进行精确控制
- 在关键词层面更好地分析效果

SKAG 特别适用于识别表现最佳的关键词，并通过专属广告组最大化其潜力。
## 添加关键词 \{#add-keywords\}

要向广告组添加关键词：

1. 从侧边栏菜单进入 **Ads Manager**。在任意标签页中，点击表格上方的 **+**，然后从下拉菜单中选择 **Add keywords**。

<ZoomImage id="add-keywords-action.webp" width="700px" /> 

2. 在弹窗中，选择您想要添加关键词的广告活动和广告组。在一个广告活动中选择广告组后，您还可以选择另一个广告活动，并将更多广告组添加到列表中。

<ZoomImage id="select-ad-groups-keywords.webp" width="700px" />
3. 点击 **Select** 继续。
4. 在 **Add keywords** 对话框中，在 **Keywords list** 字段中输入关键词。如果您有一个以逗号分隔的关键词文件，可以粘贴其内容，以便 Apple Ads Manager 批量上传所有关键词。
5. 对表格中的每个关键词进行配置：
   - **Match type**：选择 **Exact** 或 **Broad** 匹配类型
   - **CPT bid**：为此关键词设置每次点击的最高出价，或留空以使用广告组的默认最高 CPT 出价
6. 检查您的关键词，然后点击 **Add X keywords**（其中 X 为您正在添加的关键词数量）。
:::important
关键词保存后，其匹配类型将无法更改。如需更改匹配类型，请删除该关键词并以所需匹配类型重新添加。
:::

<ZoomImage id="add-keywords-dialog.gif" width="700px" />
## 添加排除关键词 \{#add-negative-keywords\}

要添加排除关键词：
1. 从侧边栏菜单进入 **Ads Manager**。在任意标签页中，点击表格上方的 **+**，然后从下拉菜单中选择 **Add negative keywords**。
   <ZoomImage id="add-negative-keywords.webp" width="700px" />
2. 在 **Add negative keywords to** 弹窗中，选择要添加否定关键词的层级：
   - **Selected campaigns**：添加广告系列级别的否定关键词。
   - **Selected ad groups**：添加广告组级别的否定关键词。
   - **All ad groups in selected campaigns**：将广告组级别的否定关键词添加到所选广告系列的所有广告组中。
   :::note
   请注意以下事项：
   - 广告组级别的否定关键词优先级高于广告系列级别的否定关键词。
   - 如果您将否定关键词添加到所选广告系列的所有广告组中，后续若需向这些广告系列添加新广告组，则需要手动为新广告组添加否定关键词。
   :::
<ZoomImage id="select-campaigns.gif" width="700px" />
3. 在 **Keywords list** 字段中输入否定关键词。如果您有一个以逗号分隔的关键词文件，可以粘贴其内容，Apple Ads Manager 将批量上传所有关键词。
4. 对于表格中的每个关键词，选择 **Match type**：
   - **Exact**：仅排除完全相同的关键词或非常接近的变体。
   - **Broad**：排除该关键词及相关搜索词。
      或勾选关键词旁边的复选框，批量更改匹配类型。
5. 检查您的否定关键词，然后点击 **Add X keywords**（其中 X 是您要添加的关键词数量）。
:::note
跨组否定关键词在您希望同时在多个广告活动中排除某些搜索词时特别有用，既节省时间，又能确保整个广告策略的一致性。
:::
   <ZoomImage id="negative-keywords.gif" width="700px" />
## 将关键词添加为 SKAG \{#add-keywords-as-skag\}

要将关键词添加为 SKAG（单关键词广告组）：
1. 从侧边栏菜单进入 **Ads Manager**。在任意标签页中，点击表格上方的 **+**，然后从下拉菜单中选择 **Add keywords as SKAG**。
   <ZoomImage id="add-skag-keywords.webp" width="700px" />
2. 选择要创建 SKAG 广告组的广告活动，可以同时选择多个广告活动。
3. 默认情况下，新广告组将使用默认设置创建，目标受众为所有用户。如需修改，请选择 **Copy settings from ad group** 并选择一个现有广告组以复制其设置。
   <ZoomImage id="copy-settings-from-ad-group.webp" width="700px" />
4. 配置新广告组的设置：
   - **Ad group name prefix**：可选的前缀，用于添加到每个广告组名称前（例如，"SKAG_" 将创建 "SKAG_keyword1"、"SKAG_keyword2" 等）。您可以点击 **Tag** 动态地将关键词、广告活动名称和国家/地区添加到组名称中。
   - **CPT bid** 和 **CPA cap**：为所有关键词统一设置出价，或选择 **Set CPT bid and CPA cap for each word manually** 分别为每个关键词单独设置。
5. 在 **Keywords list** 字段中输入关键词。如果您有包含关键词的逗号分隔文件，可以直接粘贴其内容，Apple Ads Manager 将批量上传所有关键词。
6. 在表格中为每个关键词选择 **Match type**：
   - **Exact**：仅匹配完全相同的关键词或非常相近的变体
   - **Broad**：匹配该关键词及相关搜索词
     或勾选关键词旁边的复选框，批量更改匹配类型。
7. 选择 **Check for duplicates in target campaign**，确保目标广告活动中没有重复的关键词。
8. 点击 **Create** 创建 SKAG 广告组。
每个关键词将被放置在每个所选广告系列中各自独立的广告组中，使您能够独立管理和优化它们。

<ZoomImage id="new-skag.webp" width="700px" />
## 编辑关键词 \{#edit-keywords\}

要编辑现有关键词：

1. 前往 **Ads Manager > Keywords** 或 **Ads Manager > Negative keywords**，在表格中找到要编辑的关键词，或者导航到广告系列页面，再进入广告组页面，找到该关键词。
2. 直接在表格中编辑以下值：
   - **CPT bid**：点击出价值并输入新的每次点击最高费用出价
   - **Status**：使用切换开关来暂停或激活关键词

:::note
直接在 Apple Ads 中对关键词所做的编辑会自动同步到 Apple Ads Manager，但可能需要一些时间才能在 Apple Ads Manager 中显示。
:::

## 批量操作 \{#bulk-actions\}

您可以对多个关键词执行批量操作，以节省时间并更高效地管理关键词。

执行批量操作的步骤：
1. 前往 **Ads Manager > Keywords** 或 **Ads Manager > Negative keywords** 标签页。
2. 勾选关键词旁边的复选框，选择多个您想管理的关键词。
3. 点击 **Actions** 下拉菜单，选择以下选项之一：
   - **Add as keywords**：将所选关键词添加为标准关键词
   - **Add as negative keywords**：将所选关键词添加为否定关键词
   - **Add as SKAG**：为所选关键词创建单关键词广告组
   - **Activate**：激活所选关键词
   - **Pause**：暂停所选关键词
   - **Edit CPT bids**：编辑所选关键词的 CPT 出价。您可以通过以下方式进行编辑：
     - **Set to**：将多个出价设置为指定金额。
     - **Increase by/decrease by**：按指定美元金额或出价百分比增加或降低出价。可选择设置出价上限，以避免意外超额消费。
     - **Set to average CPT**：使用 CPT（每次点击费用）指标对齐出价。设置一个乘数系数。例如，当效果低于预期时将乘数设为 0.9，效果超出预期时设为 1.1。
     - **Set to average CPA**：使用 CPA（每次获客费用）指标对齐出价。设置一个乘数系数。
:::tip
批量操作在以下场景中特别有用：
- 在不同类型（标准、否定、SKAG）之间转换关键词
- 快速为多个关键词添加其他匹配类型
- 筛选表现最佳的关键词并调整其出价
- 识别表现不佳的关键词并将其暂停
:::

## 导出关键词 \{#export-keywords\}

如需将关键词表格导出为 CSV 文件，请点击表格上方的下载图标，然后选择 **Export current page** 或 **Export all pages**。

**Export all pages** 会将所有页面的关键词下载至单个文件。下载进度弹窗会实时显示进度，您可随时取消。此外，还提供两个可选筛选条件：

- **Enabled only**：仅包含已启用的关键词。
- **With spend ≥**：仅包含消耗金额高于指定阈值的关键词。
- **Group by country**：将每条关键词数据按国家/地区进行细分。
表格将按照您在看板上的显示方式导出，包含您已选择显示的列。

<ZoomImage id="export-keywords.webp" width="700px" alt="导出关键词" />
## 探索关键词级别数据图表 \{#explore-keyword-level-charts\}

您可以直接从 **Ads Manager > Keywords** 表格中打开任意关键词的数据图表，从而对每个关键词进行精确的逐日效果分析。

要显示数据图表，请点击表格中关键词旁边的图表图标。

<ZoomImage id="keyword-chart.webp" width="500px" />

默认情况下，数据图表将显示所选关键词的 **Spend** 数据图表。

您可以同时显示多个数据图表，以发现相关性和随时间发生的变化。点击 **+** 添加新的数据图表。
点击 **Reset** 重新开始，或直接取消勾选数据图表复选框以隐藏它们。

<ZoomImage id="keyword-chart.gif" width="700px" />
## 竞价历史 \{#bid-history\}

要查看某个关键词的竞价历史，请点击表格中该关键词旁边的 **Bid History** 按钮。此时会打开一个包含两个标签页的面板。
- **Metrics** 标签页显示一个包含随时间变化的各项指标的数据图表。您可以用与关键词级别数据图表相同的方式添加或移除指标——点击 **+** 添加，或取消勾选复选框来隐藏指标。每个出价发生变化的时间点都会显示一个标记——将鼠标悬停在标记上可查看该日期的具体出价金额。利用此功能可将出价变化与效果波动关联起来：如果某项指标在变化后出现下滑或上涨，标记会精确显示发生的时间点。
- **Bid History** 标签页以倒序时间顺序显示所有出价变化的记录表格，包含前一次出价值、新出价值、每次变化的日期以及触发原因。
