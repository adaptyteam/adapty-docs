Adapty Ads Manager 与 Apple Ads 实现了双向集成：你可以获取近实时的效果数据，还能直接在 Adapty 看板中创建和编辑关键词，操作体验远比原生界面便捷。
如果您在原生 Apple Ads 看板中创建了关键词，该关键词将在 24 小时内自动出现在 Adapty Ads Manager 中。

除了[查看全面的分析数据](adapty-ads-manager-analytics)，您还可以管理所有关键词设置：
- 向广告组添加关键词
- 添加否定关键词
- 将关键词添加为 SKAG（单关键词广告组）
- 直接在表格中编辑关键词
- 对多个关键词执行批量操作
- 启动和暂停关键词
## 什么是关键词 \{#what-are-keywords\}

关键词是触发您的广告出现在 App Store 搜索结果中的搜索词。它们组织在[广告组](ads-manager-create-ad-group)内，而广告组隶属于[广告活动](ads-manager-create-campaign)。这种层级结构使您能够有效地组织和管理广告策略。

:::important
关键词仅适用于**搜索结果**版位类型的广告活动。对于其他版位类型（搜索标签页或产品页面）的广告活动，不使用关键词。
:::

### 标准关键词 \{#standard-keywords\}

标准关键词是您出价以触发广告的主要词语。当用户在 App Store 中搜索这些词语时，您的广告可能会出现在搜索结果中。

### 否定关键词 \{#negative-keywords\}

否定关键词可防止您的广告出现在与您的应用不相关的搜索中。通过添加否定关键词，您可以减少在无关搜索上的无效支出。

否定关键词可以在广告组级别添加，也可以作为跨组否定关键词同时应用于多个广告活动。

### 关键词作为 SKAG（单关键词广告组）\{#keywords-as-skag-single-keyword-ad-group\}

SKAG（单关键词广告组）是一种策略，即为每个关键词创建单独的广告组。这种方式可以让您：
- 精确控制高价值关键词的出价
- 更好地在关键词级别分析效果

SKAG 对于识别表现最佳的关键词并通过专属广告组最大化其潜力特别有用。

## 添加关键词 \{#add-keywords\}

要向广告组添加关键词：

1. 从侧边栏菜单进入 **Ads Manager**。在任意标签页中，点击表格上方的 **+**，然后从下拉菜单中选择 **Add keywords**。

<ZoomImage id="add-keywords-action.webp" width="700px" /> 

2. 在弹窗中，选择要添加关键词的广告系列和广告组。选择某个广告系列中的广告组后，你还可以继续选择其他广告系列，并将更多广告组加入列表。

<ZoomImage id="select-ad-groups-keywords.webp" width="700px" />
3. 点击 **Select** 继续。
4. 在 **Add keywords** 对话框中，在 **Keywords list** 字段输入关键词。如果你有一个以逗号分隔的关键词文件，可以直接粘贴其内容，Adapty Ads Manager 会批量上传所有关键词。
5. 针对表格中的每个关键词进行配置：
   - **Match type**：选择 **Exact**（精确匹配）或 **Broad**（广泛匹配）
   - **CPT bid**：为该关键词设置每次点击的最高出价，或留空以使用广告组的默认最高 CPT 出价
6. 确认关键词无误后，点击 **Add X keywords**（X 为你要添加的关键词数量）。
:::important
关键词保存后，其匹配类型将无法修改。如需更改匹配类型，请删除该关键词并以所需匹配类型重新添加。
:::

<ZoomImage id="add-keywords-dialog.gif" width="700px" />
## 添加否定关键词 \{#add-negative-keywords\}

要添加否定关键词：
1. 从侧边栏菜单进入 **Ads Manager**。在任意标签页中，点击表格上方的 **+**，然后从下拉菜单中选择 **Add negative keywords**。
   <ZoomImage id="add-negative-keywords.webp" width="700px" />
2. 在 **Add negative keywords to** 弹窗中，选择要在哪个层级添加否定关键词：
   - **Selected campaigns**：添加广告系列级别的否定关键词。
   - **Selected ad groups**：添加广告组级别的否定关键词。
   - **All ad groups in selected campaigns**：向所选广告系列中的所有广告组添加广告组级别的否定关键词。
   :::note
   请注意以下几点：
   - 广告组级别的否定关键词优先级高于广告系列级别的否定关键词。
   - 如果您将否定关键词添加到所选广告系列的所有广告组，那么后续若向这些广告系列新增广告组，则需要手动添加否定关键词。
   :::
 <ZoomImage id="select-campaigns.gif" width="700px" /> 
3. 在 **Keywords list** 字段中输入否定关键词。如果你有一个以逗号分隔的关键词文件，可以直接粘贴其内容，Adapty Ads Manager 将批量上传所有关键词。
4. 对表格中的每个关键词，选择 **Match type**：
   - **Exact**：仅排除完全匹配的关键词或非常接近的变体。
   - **Broad**：排除该关键词及相关搜索词。
      或勾选关键词旁边的复选框，批量更改匹配类型。
5. 检查你的否定关键词，然后点击 **Add X keywords**（其中 X 为你要添加的关键词数量）。
:::note
跨组否定关键词在您希望一次性跨多个广告活动排除特定搜索词时特别有用，既节省时间，又能确保整体广告策略的一致性。
:::

## 将关键词添加为 SKAG \{#add-keywords-as-skag\}

要将关键词添加为 SKAG（单关键词广告组）：
1. 在侧边栏菜单中进入 **Ads Manager**。在任意标签页中，点击表格上方的 **+**，然后在下拉菜单中选择 **Add keywords as SKAG**。
   <ZoomImage id="add-skag-keywords.webp" width="700px" />
2. 选择要在其中创建 SKAG 广告组的广告活动，可以同时选择多个。
3. 默认情况下，新广告组将使用默认设置创建，面向所有用户。如需修改，请选择 **Copy settings from ad group**，然后选择一个现有广告组以复制其设置。
   <ZoomImage id="copy-settings-from-ad-group.webp" width="700px" />
4. 配置新广告组的设置：
   - **Ad group name prefix**：可选前缀，添加到每个广告组名称前（例如，"SKAG_" 会生成 "SKAG_keyword1"、"SKAG_keyword2" 等）。你可以点击 **Tag**，将关键词、广告活动名称和国家/地区动态添加到组名中。
   - **CPT bid** 和 **CPA cap**：为所有关键词统一设置出价，或选择 **Set CPT bid and CPA cap for each word manually**，为每个关键词单独设置。
5. 在 **Keywords list** 字段中输入关键词。如果你有逗号分隔格式的关键词文件，可以直接粘贴其内容，Adapty Ads Manager 将批量上传所有关键词。
6. 在表格中为每个关键词选择 **Match type**（匹配类型）：
   - **Exact**：仅匹配完全相同的关键词或高度相似的变体
   - **Broad**：匹配该关键词及相关搜索词
     也可以勾选旁边的复选框，批量修改匹配类型。
7. 选择 **Check for duplicates in target campaign**，确保目标广告活动中没有重复的关键词。
8. 点击 **Create** 创建 SKAG 广告组。
每个关键词将被放置在所选广告系列中各自独立的广告组内，让您可以对其进行独立管理和优化。

<ZoomImage id="new-skag.webp" width="700px" />
## 编辑关键词 \{#edit-keywords\}

编辑已有关键词的步骤如下：

1. 前往 **Ads Manager > Keywords** 或 **Ads Manager > Negative keywords**，在表格中找到想要编辑的关键词；或者进入某个广告系列页面，再进入广告组页面，找到对应关键词。
2. 直接在表格中修改对应值：
   - **CPT bid**：点击出价值，输入新的最高单次点击费用出价
   - **Status**：使用开关切换暂停或激活该关键词

:::note
直接在 Apple Ads 中对关键词所做的修改会自动同步到 Adapty Ads Manager，但可能需要一些时间才能在 Adapty Ads Manager 中显示。
:::

## 批量操作 \{#bulk-actions\}

你可以对多个关键词执行批量操作，从而节省时间、更高效地管理关键词。

执行批量操作的步骤：
1. 前往 **Ads Manager > Keywords** 或 **Ads Manager > Negative keywords** 标签页。
2. 勾选关键词旁边的复选框，选中多个你想管理的关键词。
3. 点击 **Actions** 下拉菜单，选择以下操作之一：
   - **Add as keywords**：将所选关键词添加为标准关键词
   - **Add as negative keywords**：将所选关键词添加为否定关键词
   - **Add as SKAG**：为所选关键词创建单关键词广告组
   - **Activate**：激活所选关键词
   - **Pause**：暂停所选关键词
   - **Edit CPT bids**：编辑所选关键词的 CPT 出价。你可以通过以下几种方式进行编辑：
     - **Set to**：将多个出价统一设置为指定金额。
     - **Increase by/decrease by**：按指定美元金额或出价百分比增加或减少出价。可选择设置出价上限，防止意外超支。
     - **Set to average CPT**：使用 CPT（每次点击成本）指标对齐出价，并设置一个乘数系数。例如，当效果低于预期时将乘数设为 0.9，表现超预期时设为 1.1。
     - **Set to average CPA**：使用 CPA（每次转化成本）指标对齐出价，并设置一个乘数系数。
:::tip
批量操作特别适合用于：
- 在不同类型（标准、否定、SKAG）之间转换关键词
- 为多个关键词快速添加其他匹配类型
- 筛选表现最佳的关键词并调整出价
- 识别表现不佳的关键词并暂停投放
:::

## 导出关键词 \{#export-keywords\}

如需将关键词表导出为 CSV 文件，点击表格上方的下载图标，然后选择 **Export current page** 或 **Export all pages**。

**Export all pages** 会将所有页面的关键词合并到一个文件中下载。下载过程中会显示进度弹窗，您可以随时取消。此外，还提供两个可选筛选项：

- **Enabled only**：仅包含已启用的关键词。
- **With spend ≥**：仅包含消耗金额高于指定阈值的关键词。
- **Group by country**：按国家/地区细分每条关键词数据。
导出的表格将与看板上显示的内容保持一致，包含您所选择显示的列。

<ZoomImage id="export-keywords.webp" width="700px" alt="Export keywords" />
## 查看关键词级别的数据图表 \{#explore-keyword-level-charts\}

你可以直接从 **Ads Manager > Keywords** 表格中打开任意关键词的数据图表，从而对每个关键词进行精确的逐日效果分析。

点击表格中关键词旁边的图表图标，即可显示对应的数据图表。

<ZoomImage id="keyword-chart.webp" width="500px" />

默认情况下，数据图表将显示所选关键词的 **Spend** 数据图表。

你可以同时显示多个数据图表，以发现各指标之间的相关性及随时间的变化趋势。点击 **+** 可添加新的数据图表。
点击 **Reset** 重新开始，或直接取消勾选数据图表复选框以隐藏它们。

<ZoomImage id="keyword-chart.gif" width="700px" />
## 出价历史 \{#bid-history\}

要查看某个关键词的出价历史，点击表格中该关键词旁边的 **Bid History** 按钮。随即会打开一个面板，包含两个选项卡。
- **指标**标签页以数据图表的形式展示各项指标随时间的变化趋势。添加或移除指标的方式与关键词级别的数据图表相同——点击 **+** 添加指标，或取消勾选复选框来隐藏指标。每次出价变更的时间点都会显示一个标记——将鼠标悬停在标记上，可查看该日期的具体出价金额。通过这些标记，您可以将出价变更与指标波动关联起来分析：若某项指标在变更后出现下滑或突增，标记会精确定位到变化发生的时间。
- **出价历史**标签页以表格形式展示所有出价变更记录，按时间倒序排列，包含变更前后的出价金额、每次变更的日期以及触发原因。
