## 新版付费墙编辑工具的主要功能 \{#key-features-of-the-new-paywall-builder\}

- **扩展模板选择**：从大量专业设计的模板中选择，快速启动付费墙创建。这些模板提供多种风格和布局，以满足不同需求和偏好。
- **增强灵活性**：通过使用设计图层以及轮播图、卡片、产品列表和页脚等新元素，享受更大的灵活性。这些增强功能赋予您创意自由，构建任何您想象的付费墙类型。
- **改进现有元素**：现有元素得到了显著改进，提供更多选项和功能，让您的付费墙创意得以实现。
- **可扩展性**：如果您管理多个应用，只需点击一下即可[迁移](migrate-paywalls)可视化付费墙配置，而无需从头构建。

:::warning
本节介绍新版付费墙编辑工具，该工具适用于 iOS、Android 和 React Native SDK 3.0 或更高版本，以及 Flutter 和 Unity SDK 3.3.0 或更高版本。有关兼容 Adapty SDK v2.x 或更早版本的旧版付费墙编辑工具的信息，请参阅[使用旧版付费墙编辑工具设计付费墙](adapty-paywall-builder-legacy)。
:::

## 付费墙的结构 \{#structure-of-a-paywall\}

在新版 Adapty 付费墙编辑工具中，付费墙由以下部分组成：

- [**布局**](paywall-layout-and-products)：付费墙的基础层，用于设置背景颜色并定义产品的显示方式以及顶部按钮的位置。
- [**主图**](paywall-head-picture)：付费墙的主要图片。
- **主区域**：您可以在此放置各种元素，如产品块、轮播图、图片、卡片、文本、按钮和列表。
- **页脚**：与主区域类似，但它是一个始终固定在付费墙底部、叠加在主区域上方的容器。您可以添加任意数量的元素，它们将按照左侧面板中显示的顺序从上到下排列在页脚内。
- [**元素**](adapty-paywall-builder#paywall-elements)：放置在主区域或页脚中用于构建付费墙的基本构件。它们按照在左侧面板中出现的顺序从上到下堆叠。您可以将元素嵌套在彼此内部，将它们组合成卡片，或在轮播图中显示。

## 如何使用新版付费墙编辑工具开始设计付费墙 \{#how-to-start-designing-a-paywall-with-new-paywall-builder\}

:::warning
新版付费墙编辑工具仅适用于 iOS 和 Android，且需要 Adapty SDK v3.0 或更高版本。请确保您已参阅 <InlineTooltip tooltip="升级至 Adapty SDK v.3.x">[iOS](migration-to-ios-sdk-v3)、[Android](migration-to-android-sdk-v3)、[Flutter](migration-to-flutter-sdk-v3)、[React Native](migration-to-react-native-sdk-v3) 和 [Unity](migration-to-unity-sdk-v3) 迁移指南</InlineTooltip>，以确保您的新付费墙正常运行！
:::

使用 Adapty 付费墙编辑工具：

1. 在 Adapty 主菜单中打开 [**Paywalls**](https://app.adapty.io/paywalls) 部分。

2. 打开现有付费墙或[创建新付费墙](create-paywall)。

3. 转到 **Builder & Generator** 选项卡。

4. 如果尚未添加产品，请向付费墙添加产品。否则，点击 **Build no-code paywall**。

5. 选择模板并点击 **Choose**，或从其他应用[迁移付费墙](migrate-paywalls)。

6. 点击 **Choose** 确认您的选择。

7. [添加](#paywall-elements)和[自定义](#customization-options)付费墙元素。

:::important
要显示付费墙，您必须在付费墙编辑工具中打开 **Show on device** 开关。
:::

## 迁移旧版付费墙 \{#migrate-your-legacy-paywalls\}

目前，Adapty 中有两个版本的付费墙编辑工具并行运行：

- **新版本**：位于 Adapty 看板付费墙功能的 **Builder & Generator** 选项卡中。这是最新、最灵活的版本，提供了许多设计功能，帮助您构建完美的付费墙。

:::note
使用此版本设计的付费墙需要 Adapty SDK v3.0 或更高版本。
:::

- **旧版本**：位于 Adapty 看板付费墙功能的 **Legacy Builder** 选项卡中。此版本已过时，仅应用于支持已安装 SDK 低于 v.3.х.х 的应用版本。我们不建议将其用于新付费墙，因为它很快将被弃用。

将付费墙从旧版付费墙编辑工具迁移到新版意味着将在 **Builder & Generator** 选项卡中创建付费墙的新版本。

此版本可以使用新版付费墙编辑工具进行编辑，并将显示在已安装 Adapty SDK v3.0 或更高版本的应用中。请参阅 <InlineTooltip tooltip="迁移至 Adapty SDK v.3.x 指南">[iOS](migration-to-ios-sdk-v3)、[Android](migration-to-android-sdk-v3)、[Flutter](migration-to-flutter-sdk-v3)、[React Native](migration-to-react-native-sdk-v3) 和 [Unity](migration-to-unity-sdk-v3) 迁移指南</InlineTooltip>，获取有关升级 Adapty SDK 的详细参考。

您现有的付费墙版本将保留在旧版编辑工具选项卡中。您可以继续使用旧版付费墙编辑工具对其进行调整，它将显示在已安装 Adapty SDK 2.x 或更早版本的应用中。

您的付费墙将以两种付费墙编辑工具格式并行且独立存在，直到您不再需要为止。针对一个版本的付费墙配置所做的更改不会影响为另一个版本构建的配置。

将付费墙迁移到新版付费墙编辑工具：

1. 打开要迁移的付费墙。
2. 打开 **Builder & Generator** 选项卡。
3. 点击 **Migrate paywall**。

## 自定义选项 \{#customization-options\}

您可以灵活设置每个元素：

- **Style** 选项卡：调整元素的大小、外观、背景颜色或图片、边框和透明度。某些元素（如轮播图）还提供页面控制和幻灯片设置等附加选项。
- **Layout** 选项卡：使用偏移（移动元素而不改变其大小或父元素大小）或内边距（在可能调整父元素大小以适应子元素大小和位置的情况下移动元素）来设置元素及其子元素的位置。
- **Contents** 选项卡：配置复合元素的内容。
