Adapty UA 与自定义 S3 兼容存储的集成允许您将用户获取活动数据安全地存储在您自己的 S3 兼容存储解决方案中。您将能够以 .csv 文件格式将活动效果数据、归因数据和用户获取事件保存到您的自定义 S3 存储桶中。  
要设置此集成，您需要在 S3 兼容存储控制台和 Adapty UA 看板中执行几个简单步骤。

:::note
Adapty UA 每 **24 小时**在 UTC 时间 4:00 发送一次数据。

每个文件将包含以 UTC 时间计算的整个前一日历日内创建的事件数据。例如，UTC 时间 3 月 8 日 4:00 自动导出的数据将包含 UTC 时间 3 月 7 日 00:00:00 至 23:59:59 期间创建的所有事件。
:::

## 设置自定义 S3 集成 \{#set-up-custom-s3-integration\}

要开始接收数据，请在 Adapty UA 中配置集成：

1. 前往 [**Integrations** -> **Custom S3**](https://app.adapty.io/ua/integrations/custom-s3)
2. 开启 **Export install events to custom S3** 开关。
3. 填写必填字段，以在您的自定义 S3 存储和 Adapty UA 用户画像之间建立连接。

| 字段                                    | 说明                                                                                                                                                                                                                               |
|:----------------------------------------|:-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| **Access Key ID**                       | 用于验证用户或应用程序访问您的 S3 兼容存储服务的唯一标识符。请在您的存储提供商控制台中查找此 ID。                                                                                                                                  |
| **Secret Access Key**                   | 与 Access Key ID 配合使用的私钥，用于验证用户或应用程序访问您的 S3 兼容存储服务。请在您的存储提供商控制台中查找此密钥。                                                                                                            |
| **S3 Bucket Name**                      | 用于标识存储环境中特定 S3 存储桶的全局唯一名称。S3 存储桶是一种简单的存储服务，允许用户在云端存储和检索数据对象（如文件和图片）。                                                                                                  |
| **Region**（可选）                      | 从管理控制台获取您的区域信息。                                                                                                                                                                                                     |
| **Folder Inside the Bucket**（可选）    | 您希望在所选 S3 存储桶中创建的文件夹名称。请注意，S3 使用对象键前缀模拟文件夹，这些前缀本质上就是文件夹名称。                                                                                                                     |
| **Custom Endpoint URL**                 | 您的 S3 兼容存储服务的端点 URL。该 URL 应由您的存储提供商提供（例如 MinIO、DigitalOcean Spaces、Wasabi 等）。                                                                                                                      |

:::note
您也可以在 S3 存储桶名称字段中指定嵌套目录，例如 `adapty-ua-events/com.sample-app`
:::

## 手动数据导出 \{#manual-data-export\}

除了自动将事件数据导出到您的自定义 S3 存储外，Adapty UA 还提供手动文件导出功能。通过此功能，您可以选择用户获取数据的日期并手动将其导出到您的 S3 存储桶。这使您能够更好地控制导出的数据内容和导出时机。

## 表结构 \{#table-structure\}

在自定义 S3 集成中，Adapty UA 提供了一张用于存储安装事件历史数据的表。该表包含用户画像、收入与收益、来源应用商店等数据点的信息。

:::warning
请注意，随着我们或第三方合作伙伴引入新数据，此结构可能会随时间增长。请确保您处理数据的代码足够健壮，依赖特定字段而非整体结构。
:::

以下是事件的表结构：

| 列名                     | 说明                                      |
