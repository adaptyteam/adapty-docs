年度经常性收入数据图表展示了所有有效自动续费订阅的收入，并将其折算为年度数值。该数据图表将所有已付费且未到期的订阅视为有效订阅。ARR 是追踪订阅业务增长、预测未来收入的核心指标。

## 计算方式 \{#calculation\}

:::warning
以下计算器**不考虑**[商店佣金和税费](how-adapty-analytics-works#commissions-and-taxes)。请将结果与您的**总收入**数据进行对比。
:::

ARR 是将周期性订阅收入换算为年度金额后的指标。当年度订阅是您的主要产品时，ARR 最具参考价值——如果您的业务以月订阅或周订阅为主，[MRR](mrr) 则更具参考意义。
ARR 是所有订阅类型的（价格 × 活跃订阅者 ÷ 以年为单位的计费周期）之和。月付使用 1/12，周付使用 1/52。

        }

    formulaCalculation="(price * subs ) / periods"

        { price: 240, subs: 2, periods: "1" },
        { price: 30, subs: 10, periods: "1/12" },
        { price: 10, subs: 20, periods: "1/52" }
    ]}
/>
ARR 不包含已退款的订阅——当某笔订阅发生退款时，Adapty 会将其从该订阅有效期内所有周期的计算中移除。
