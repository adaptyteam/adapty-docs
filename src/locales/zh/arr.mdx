年度经常性收入数据图表显示所有活跃自动续费订阅的收入，并将其归一化为一年。该数据图表将任何已付费且未到期的订阅视为活跃订阅。ARR 是跟踪订阅业务增长和预测未来收入的重要数据图表。

### 计算方式 \{#calculation\}

:::warning
以下计算器**不考虑**[商店佣金和税费](how-adapty-analytics-works#tax-and-commission-logic)。请将结果与您的**总收入**计算进行比较。
:::

ARR 表示您订阅业务可预测的、持续的年度收入。仅当年度订阅占销售额的最大份额时，此数据图表才具有实际参考价值。
对于每种订阅类型，将其价格乘以订阅用户数，再除以计费周期的年数，然后将所有结果相加。在下方的交互式计算器中输入您的订阅数据，即可计算出您的总 ARR：
<SimpleCalculator client:load
    heading="ARR"
    formuLatex="\sum \frac{P_s \times U_s}{D_y}"
    variables={[
        {
            nameInTheFormula: "P_s",
            variableName: "price",
            variableDescription: "订阅价格",
            variableValue: 240
        },
        {
            nameInTheFormula: "U_s",
            variableName: "subs",
            variableDescription: "活跃付费订阅数",
            variableValue: 2,
            isInteger: true
        },
        {
            nameInTheFormula: "D_y",
            variableName: "periods",
            variableDescription: "订阅周期",
            variableValue: 1,
            options: [
                { label: "每周", value: "1/52" },
                { label: "每月", value: "1/12" },
                { label: "2个月", value: "2/12" },
                { label: "3个月", value: "3/12" },
                { label: "6个月", value: "6/12" },
                { label: "每年", value: 1 }
            ]
        }
    ]}
    formulaCalculation="(price * subs ) / periods"
    isSum={true}
    defaultRows={[
        { price: 240, subs: 2, periods: "1" },
        { price: 30, subs: 10, periods: "1/12" },
        { price: 10, subs: 20, periods: "1/52" }
    ]}
/>
例如，有 2 个有效年度订阅，价格为 $240；10 个月度订阅，价格为 $30；20 个周订阅，价格为 $10，
ARR = (2 _ $240 / 1) + (10 _ $30 / (1/12)) + (20\*$10 / (1/52)) = $14480

:::note
ARR 不含已退款的订阅。当某笔订阅发生退款时，该订阅在其所有有效期内的金额都会从 MRR 计算中移除，从而确保 MRR 仅反映仍然有效的订阅所产生的经常性收入。
:::
### 可用筛选条件与分组方式 \{#available-filters-and-grouping\}

- ✅ 筛选条件：归因、国家/地区、付费墙、商店、产品和时长。
- ✅ 分组方式：续费状态、周期、产品、国家/地区、商店、付费墙、时长、归因状态、归因渠道、归因活动、归因广告组、归因广告集和归因创意。

您可以在[此文档](controls-filters-grouping-compare-proceeds)中找到有关可用控件、筛选条件、分组选项、税收和佣金控件及其使用方法的更多信息。

### ARR 数据图表的使用 \{#arr-chart-usage\}

年度经常性收入（ARR）数据图表是衡量订阅业务增长和规模的重要工具。它是一种被广泛使用的数据图表，提供了 12 个月周期内经常性收入的归一化视图。为了更好地了解推动 ARR 增长的因素，您可以按关键订阅者细分维度（如商店或产品时长）对数据进行市场细分。通过这种方式，您可以识别哪些细分群体带来了最多收入，并相应地优化业务策略。
