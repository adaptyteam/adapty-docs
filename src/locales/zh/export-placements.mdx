当您同时管理多个付费墙和用户引导时，追踪哪些内容展示给哪些用户非常重要。您可以将所有[版位](placements)设置导出为 CSV 文件，以查看每个目标受众对应的付费墙/用户引导内容，并在进行更改或运行实验后审查您的配置。

:::tip
如果您觉得更方便，也可以[通过服务端 API 导出版位](api-export-analytics/operations/retrievePlacementInfo)。
:::

要导出付费墙/用户引导版位：

1. 在主菜单中前往 **[Placements](https://app.adapty.io/placements)**。切换到 **Paywalls** 或 **Onboardings** 标签页，因为它们的版位需要分别导出。
2. 点击 **Export to CSV**。

导出的 CSV 文件包含以下版位相关信息：

- 版位 ID
- 版位名称
- 目标受众名称
- 市场细分名称
- 跨版位 A/B 测试名称
- A/B 测试名称
- 付费墙名称
