关键词规则根据完整漏斗表现（从安装到试用、订阅和收入）自动对关键词执行操作。使用支出、CPA、ROAS 和同期群数据等数据图表定义条件，然后选择满足条件时规则所执行的操作。

规则按照您设定的计划运行，无需人工干预即可响应效果变化。
## 可用操作 \{#available-actions\}

每条关键词规则在满足条件时执行一个操作：

| 操作 | 说明 |
|--------|-------------|
| **Change bid** | 提高、降低或设置 CPT 出价 |
| **Enable keyword** | 重新启用已暂停的关键词 |
| **Pause keyword** | 暂停活跃的关键词 |
| **Add as keyword to…** | 将关键词复制到另一个广告组，并指定出价和匹配类型 |
| **Add as negative keyword to…** | 将关键词作为否定关键词添加到指定广告组或广告系列 |
## 创建关键词规则 \{#create-a-keyword-rule\}

您可以从模板创建关键词规则，也可以手动从头创建。
### 从模板创建 \{#from-a-template\}

Adapty 提供了针对常见优化场景的即用型模板。常用模板包括：

- **削减无转化关键词的浪费**：在支出 > X 且安装量或试用量 = 0 的情况下降低出价。
- **扩大表现优异的关键词**：在 ROAS > 目标值或 CPA < 目标值的情况下提高出价。

从模板创建规则：
1. 在左侧边栏中，进入 **Automations** 并点击 **Templates**。
   <ZoomImage id="asa-rule-templates.webp" width="700px" alt="Automation rule templates" />
2. 选择一个模板并点击 **Next**。
   <ZoomImage id="asa-choose-template.webp" width="700px" alt="Choose a template" />
3. 查看并调整预填设置：
   - **Rule name**：自动设置为模板名称和当前日期（例如，"Scale Winning Keywords - [2025-11-12]"）。
   - **Apply to**：选择规则应应用的广告活动组、应用、广告活动或广告组。
   - **Conditions**：如有需要，修改预配置的条件。
   - **Action**：如有需要，修改预设的操作。
   - **Schedule**：设置规则的运行频率。
   <ZoomImage id="asa-templated-rule.webp" width="700px" alt="Templated rule settings" />
4. 点击 **Save** 以激活规则。
### 手动操作 \{#manually\}

要从头创建自定义关键词规则：

1. 在左侧边栏中，进入 **Automations**，点击 **Create rule**，并选择 **Keywords** 作为规则类型。
   <ZoomImage id="asa-create-rule.webp" width="700px" alt="Create rule button" />
2. 输入一个描述性的 **Rule name**。
3. 在 **Apply to** 部分，选择该规则应适用的广告系列组、应用、广告系列或广告组。
   <ZoomImage id="asa-rule-apply-to.gif" width="700px" alt="Apply to selection" />
4. 点击 **Add condition**，并从列表中选择一个[数据图表](adapty-ads-manager-metrics)。
数据图表根据您账户货币计算所选时间范围内的数据。数据近乎实时更新，因此规则始终使用最新的效果数据。
   <ZoomImage id="asa-conditions.webp" width="700px" alt="添加条件" />
5. 设置时间段（例如，前 3 天或前 7 天），选择比较运算符，并输入阈值。
   <ZoomImage id="asa-condition.webp" width="700px" alt="条件设置" />
6. 若要添加更多条件，请点击 **Add condition**，并在左侧选择 **And** 或 **Or** 运算符。
   <ZoomImage id="combine-conditions.gif" width="700px" alt="组合条件" />
7. 在 **Action** 部分，选择满足条件时执行的操作：
**更改出价**
   - **操作类型**：选择 **Increase by**、**Decrease by** 或 **Set to**。
   - **值类型**：在 **$**（绝对值）和 **%**（相对于规则运行时的当前出价）之间切换。
   - **出价上限**（可选）：最高出价上限，防止在强信号重复触发规则时出现过度竞价。

   **启用关键词**
   - 无需额外配置。该规则会重新启用满足条件的已暂停关键词。

   **暂停关键词**
   - 无需额外配置。该规则会暂停满足条件的活跃关键词。
**添加为关键词到…**
   - **目标广告组**：选择接收复制关键词的广告组。
   - **CPT 出价**：为复制的关键词设置初始出价。
   - **匹配类型**：选择**精确**或**广泛**。
   - **如果关键词已存在则跳过**：启用后，跳过目标广告组中已存在的词条。

   **添加为否定关键词到…**
   - **范围**：选择添加否定关键词的广告组或推广活动。
   - **匹配类型**：选择**精确**或**广泛**。
<ZoomImage id="asa-action.gif" width="700px" alt="Action settings" />
8. 在 **Schedule** 部分：
   - 选择频率：**Every day**、**Every 2 days**、**Every week** 等。
   - 选择运行时间（所有时间均为 UTC）。

   规则将在 UTC 计划时间运行。执行通常在几分钟内完成，之后您可以在 Logs 和主看板中查看变更。
   <ZoomImage id="asa-schedule.webp" width="700px" alt="Schedule settings" />
9. 点击 **Save** 以创建规则。
## 最佳实践 \{#best-practices\}
- **从小范围开始**：先将新规则应用于少数广告活动或广告组，验证行为后再扩大范围。
- **对活跃广告活动使用较短的回溯时间窗口**：对于快速变化的广告活动，过去 3–7 天通常比 30 天更有效。
- **结合花费与转化数据**：避免使用单一指标规则。将花费与安装量、试用次数或 ROAS 结合使用，以获得更可靠的信号。
- **在"修改出价"规则中设置出价上限**：当强信号多次触发规则时，出价上限可防止出价失控。
- **将"启用关键词"与同期群数据结合使用**：某个关键词可能因早期 CPA 表现不佳而被暂停，但随着同期群数据成熟，其 D31 或 D61 ROAS 可能表现强劲。针对同期群 ROAS 设置条件，当其达到目标值时自动重新启用。
- **使用"添加为关键词"构建测试到规模化的流程**：当测试广告活动中的某个关键词达到 CPA 目标时，自动将其复制到规模化广告活动。
- **使用"添加为否定关键词"保持探索广告活动的纯净度**：当某个关键词被确认为精确匹配关键词后，在探索或搜索匹配广告活动中将其否定，以防止对同一查询产生竞争。
- **对新晋升的关键词，在出价规则生效前留出等待时间**：如果您使用[搜索词自动化](ads-manager-automations-search-terms)将词语晋升至关键词广告活动，请先给这些关键词一两天时间积累数据。
