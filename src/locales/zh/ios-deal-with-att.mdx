如果您的应用程序使用了 AppTrackingTransparency 框架，并向用户呈现了应用追踪授权请求，则您应将[授权状态](https://developer.apple.com/documentation/apptrackingtransparency/attrackingmanager/authorizationstatus/)发送给 Adapty。

<Tabs groupId="current-os" queryString>

<TabItem value="swift" label="Swift" default>

```swift showLineNumbers
let builder = AdaptyProfileParameters.Builder()
    .with(appTrackingTransparencyStatus: .authorized)

do {
  try await Adapty.updateProfile(params: builder.build())
} catch {
  // handle the error
}
```
</TabItem>
<TabItem value="swift-callback" label="Swift-Callback" default>

```swift showLineNumbers
if #available(iOS 14, macOS 11.0, *) {
    let builder = AdaptyProfileParameters.Builder()
        .with(appTrackingTransparencyStatus: .authorized)

    Adapty.updateProfile(params: builder.build()) { [weak self] error in
        if error != nil {
            // handle the error
        }
    }
}
```
</TabItem>
</Tabs>

:::warning
我们强烈建议您在该值发生变化时尽早发送，只有这样，数据才能及时传送到您已配置的集成渠道。
:::
