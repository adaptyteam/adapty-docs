Adapty 的用户引导功能让您的非技术团队无需编写代码，即可创建美观且可自定义的用户引导流程。我们的无代码编辑工具帮助您设计一系列屏幕，引导用户体验应用的首次使用。

:::important
用户引导功能适用于使用 Adapty SDK v3.8.0+（iOS、Android、React Native、Flutter）、v3.14.0+（Unity）或 v3.15.0+（Kotlin Multiplatform、Capacitor）的应用。
:::

良好的用户引导通过以下方式向用户介绍您的应用：
- 展示应用的核心价值
- 说明关键功能与特性
- 提供必要的使用技巧

Adapty 的用户引导解决方案具有以下优势：
- 无代码[用户引导编辑工具](design-onboarding)，赋能非技术团队
- 支持[通过互动问题和变量实现个性化体验](onboarding-user-engagement)
- 支持 A/B 测试，以确定哪种用户引导流程效果最佳

## 工作原理 \{#how-it-works\}

启动您的用户引导：

1. [在无代码编辑器中设计用户引导。](design-onboarding)
2. [为用户引导创建一个版位。](create-onboarding#step-2-create-a-placement-for-your-onboarding)
3. 使用 Adapty SDK 将用户引导集成到您的项目中：
   - [iOS](ios-onboardings)
   - [Android](android-onboardings)
   - [React Native](react-native-onboardings)
   - [Flutter](flutter-onboardings)
   - [Unity](unity-onboardings)
   - [Kotlin Multiplatform](kmp-onboardings)
   - [Capacitor](capacitor-onboardings)
4. 测试用户引导并向您的用户发布。

如需进一步提升效果，您还可以尝试更高级的用户引导使用方式：

- 为版位添加更多目标受众，向不同用户群体展示不同的用户引导。
- 对用户引导进行 A/B 测试，找到最有效的方案。
- 将用户引导与付费墙关联，提升转化率。
