Adapty SDK v.3.0 新增了对全新[付费墙编辑工具](adapty-paywall-builder)的支持——这是一款无需编写代码、操作简便的付费墙创建工具的全新版本。凭借其极高的灵活性和丰富的设计功能，您的付费墙将变得更加高效且盈利能力更强。

## 升级至 3.0.1 版本 \{#upgrade-to-version-301\}

1. 按常规方式升级至 3.0.1 版本。
2. 替换备用付费墙文件：
   1. 从 Adapty 看板[下载最新版本](fallback-paywalls)。
   2. 将文件存储在用户设备上，并按照[此处](react-native-use-fallback-paywalls)的说明将其传递给 `.setFallbackPaywalls` 方法。
