Xuất dữ liệu analytics sang CSV giúp bạn linh hoạt hơn trong việc phân tích sâu các chỉ số hiệu suất của ứng dụng, tùy chỉnh báo cáo và theo dõi xu hướng theo thời gian. Với Adapty API, bạn có thể dễ dàng lấy dữ liệu analytics chi tiết dưới dạng CSV, thuận tiện để theo dõi, chia sẻ và tinh chỉnh các insight dữ liệu khi cần.

## Bộ sưu tập và môi trường Postman \{#postman-collection-and-environment\}

Để đơn giản hóa việc sử dụng API xuất dữ liệu analytics, chúng tôi đã chuẩn bị một bộ sưu tập Postman và file môi trường mà bạn có thể tải xuống và nhập vào Postman.

- **Bộ sưu tập request**: Bao gồm tất cả các request có trong API xuất analytics của Adapty. Lưu ý rằng nó sử dụng các biến mà bạn có thể định nghĩa trong môi trường.
- **Môi trường**: Chứa danh sách các biến để bạn định nghĩa giá trị một lần. Chúng tôi đã chuẩn bị một môi trường thống nhất cho server-side API, web API và API xuất analytics để giúp mọi thứ thuận tiện hơn. Sau khi kích hoạt môi trường này, Postman sẽ tự động thay thế các giá trị biến đã định nghĩa vào các request của bạn.

:::tip

[Tải xuống bộ sưu tập và môi trường](https://raw.githubusercontent.com/adaptyteam/adapty-docs/refs/heads/main/Downloads/Adapty_export_analytics_API_postman_collection.zip)

:::

Để biết cách nhập bộ sưu tập và môi trường vào Postman, vui lòng tham khảo [tài liệu Postman](https://learning.postman.com/docs/getting-started/importing-and-exporting/importing-data/).
