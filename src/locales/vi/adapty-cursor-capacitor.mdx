Hướng dẫn này giúp bạn tích hợp Adapty vào ứng dụng Capacitor theo từng bước với sự hỗ trợ của công cụ lập trình AI — bạn chỉ cần cung cấp đúng tài liệu Adapty theo đúng thứ tự.

<SkillPromo />
## Trước khi bắt đầu: thiết lập dashboard \{#before-you-start-dashboard-setup\}

Adapty yêu cầu một số cấu hình trên dashboard trước khi bạn viết bất kỳ dòng code SDK nào. Bạn có thể thực hiện điều này bằng một LLM skill tương tác, hoặc thủ công qua Dashboard.

### Cách dùng Skill (khuyến nghị) \{#skill-approach-recommended\}

Adapty CLI skill cho phép LLM của bạn thiết lập ứng dụng, sản phẩm, mức độ truy cập, paywall và placement trực tiếp — mà không cần mở Dashboard cho từng bước. Bạn chỉ cần [kết nối cửa hàng](integrate-payments) trong Dashboard.

```
npx skills add adaptyteam/adapty-cli --skill adapty-cli
```

Sau khi thêm skill, chạy `/adapty-cli` trong agent của bạn. Nó sẽ hướng dẫn bạn qua từng bước — kể cả khi nào cần mở Dashboard để kết nối cửa hàng.

### Cách tiếp cận qua Dashboard \{#dashboard-approach\}

Nếu bạn muốn tự cấu hình mọi thứ theo cách thủ công, đây là những gì bạn cần chuẩn bị trước khi viết bất kỳ dòng code nào. LLM của bạn không thể tra cứu các giá trị trên dashboard thay bạn — bạn cần tự cung cấp chúng.

1. **Kết nối cửa hàng ứng dụng của bạn**: Trong Adapty Dashboard, vào **App settings → General**. Kết nối cả App Store và Google Play nếu ứng dụng Capacitor của bạn hướng đến cả hai nền tảng. Đây là điều kiện bắt buộc để tính năng mua hàng hoạt động.
   [Kết nối cửa hàng ứng dụng](integrate-payments)
2. **Sao chép Public SDK key của bạn**: Trong Adapty Dashboard, vào **App settings → General**, rồi tìm phần **API keys**. Trong code, đây là chuỗi bạn truyền vào `adapty.activate()`.

3. **Tạo ít nhất một sản phẩm**: Trong Adapty Dashboard, vào trang **Products**. Bạn không cần tham chiếu trực tiếp đến sản phẩm trong code — Adapty sẽ cung cấp chúng thông qua các paywall.
   [Thêm sản phẩm](quickstart-products)
4. **Tạo paywall và placement**: Trong Adapty Dashboard, tạo một paywall trên trang **Paywalls**, sau đó gán nó vào một placement trên trang **Placements**. Trong code, placement ID là chuỗi bạn truyền vào `adapty.getFlow()`.
   [Tạo paywall](quickstart-paywalls)
5. **Thiết lập mức độ truy cập**: Trong Adapty Dashboard, cấu hình theo từng sản phẩm trên trang **Products**. Trong code, chuỗi được kiểm tra là `profile.accessLevels['premium']?.isActive`. Mức độ truy cập `premium` mặc định phù hợp với hầu hết các ứng dụng. Nếu người dùng trả phí được truy cập các tính năng khác nhau tùy theo sản phẩm (ví dụ: gói `basic` so với gói `pro`), hãy [tạo thêm mức độ truy cập](assigning-access-level-to-a-product) trước khi bắt đầu viết code.
:::tip
Khi đã có đủ năm thông tin trên, bạn đã sẵn sàng để viết code. Hãy nói với LLM của bạn: "Public SDK key của tôi là X, placement ID của tôi là Y" để nó có thể tạo ra code khởi tạo và lấy flow chính xác.
:::
### Thiết lập khi sẵn sàng \{#set-up-when-ready\}

Những bước này không bắt buộc để bắt đầu code, nhưng bạn sẽ cần chúng khi tích hợp trở nên hoàn chỉnh hơn:

- **A/B test**: Cấu hình trên trang **Placements**. Không cần thay đổi code.
  [A/B test](ab-tests)
- **Thêm paywall và placement**: Thêm các lệnh gọi `getPaywall` với các placement ID khác nhau.
- **Tích hợp analytics**: Cấu hình trên trang **Integrations**. Cách thiết lập khác nhau tùy theo tích hợp. Xem [tích hợp analytics](analytics-integration) và [tích hợp attribution](attribution-integration).

## Cung cấp tài liệu Adapty cho LLM \{#feed-adapty-docs-to-your-llm\}

### Sử dụng Context7 (khuyến nghị) \{#use-context7-recommended\}

[Context7](https://context7.com) là một MCP server giúp LLM của bạn truy cập trực tiếp vào tài liệu Adapty mới nhất. LLM tự động lấy đúng tài liệu cần thiết dựa trên câu hỏi của bạn — không cần dán URL thủ công.

Context7 hoạt động với **Cursor**, **Claude Code**, **Windsurf** và các công cụ tương thích MCP khác. Để cài đặt, chạy lệnh:

```
npx ctx7 setup
```

Lệnh này sẽ tự động nhận diện trình soạn thảo của bạn và cấu hình Context7 server. Để cài đặt thủ công, xem [Context7 GitHub repository](https://github.com/upstash/context7).
Sau khi cấu hình xong, hãy tham chiếu thư viện Adapty trong các prompt của bạn:

```
Use the adaptyteam/adapty-docs library to look up how to install the Capacitor SDK
```

:::warning
Mặc dù Context7 giúp bạn không cần dán thủ công các link tài liệu, nhưng thứ tự triển khai vẫn rất quan trọng. Hãy làm theo từng bước trong [hướng dẫn triển khai](#implementation-walkthrough) bên dưới để đảm bảo mọi thứ hoạt động đúng.
:::
### Dùng tài liệu dạng văn bản thuần \{#use-plain-text-docs\}

Bạn có thể truy cập bất kỳ tài liệu Adapty nào dưới dạng Markdown thuần. Thêm `.md` vào cuối URL, hoặc nhấn **Copy for LLM** bên dưới tiêu đề bài viết. Ví dụ: [adapty-cursor-capacitor.md](https://adapty.io/docs/vi/adapty-cursor-capacitor.md).

Mỗi giai đoạn trong [hướng dẫn triển khai](#implementation-walkthrough) bên dưới đều có khối "Gửi cho LLM của bạn" với các link `.md` để dán vào.

Để lấy nhiều tài liệu hơn cùng một lúc, xem [file index và các tập con theo nền tảng](#plain-text-doc-index-files) bên dưới.

## Hướng dẫn triển khai \{#implementation-walkthrough\}

Phần còn lại của hướng dẫn này sẽ dẫn bạn qua tích hợp Adapty theo đúng thứ tự triển khai. Mỗi giai đoạn bao gồm tài liệu cần gửi cho LLM, kết quả mong đợi khi hoàn thành và các lỗi thường gặp.

### Lên kế hoạch tích hợp \{#plan-your-integration\}

Trước khi bắt đầu viết code, hãy nhờ LLM phân tích dự án của bạn và tạo một kế hoạch triển khai. Nếu công cụ AI của bạn hỗ trợ chế độ lập kế hoạch (như chế độ plan của Cursor hoặc Claude Code), hãy dùng nó để LLM có thể đọc cả cấu trúc dự án lẫn tài liệu Adapty trước khi viết code.

Hãy cho LLM biết bạn đang dùng cách tiếp cận nào cho các giao dịch mua — điều này ảnh hưởng đến các hướng dẫn mà LLM cần tuân theo:
- [**Adapty Flow Builder**](adapty-flow-builder): Bạn tạo flow trong trình tạo không cần code của Adapty, và SDK sẽ tự động render chúng.
- [**Paywalls tự tạo thủ công**](capacitor-making-purchases): Bạn tự xây dựng giao diện paywall trong code nhưng vẫn dùng Adapty để lấy sản phẩm và xử lý thanh toán.
- [**Observer mode**](observer-vs-full-mode): Bạn giữ nguyên hệ thống thanh toán hiện có và chỉ dùng Adapty cho phân tích và tích hợp.

Chưa biết chọn cái nào? Đọc [bảng so sánh trong phần quickstart](capacitor-quickstart-paywalls).
### Cài đặt và cấu hình SDK \{#install-and-configure-the-sdk\}

Thêm dependency Adapty SDK bằng npm và kích hoạt nó với Public SDK key của bạn. Đây là bước nền tảng — mọi thứ khác đều không hoạt động nếu thiếu bước này.

**Hướng dẫn:** [Cài đặt & cấu hình Adapty SDK](sdk-installation-capacitor)
:::info
Hướng dẫn này dành cho Adapty Capacitor SDK v4 (beta) — API được giới thiệu trong [quickstart](capacitor-quickstart-paywalls). v4 là bản pre-release, vì vậy hãy đảm bảo LLM của bạn ghim đúng phiên bản (`npm install @adapty/capacitor@4.0.0-beta.1`) thay vì cài đặt phiên bản stable 3.x mới nhất. Xem [phần cài đặt SDK 4.0](sdk-installation-capacitor#adapty-sdk-40-beta) và [hướng dẫn migration](migration-to-capacitor-sdk-v4).
:::

Gửi nội dung này cho LLM của bạn:
```
Read these Adapty docs before writing code:
- https://adapty.io/docs/vi/sdk-installation-capacitor.md
```
:::tip[Checkpoint]
- **Mong đợi:** Ứng dụng build và chạy được trên cả iOS và Android. Console hiển thị log kích hoạt Adapty.
- **Lưu ý:** "Public API key is missing" → kiểm tra xem bạn đã thay thế placeholder bằng key thực từ **App settings** chưa.
:::

## Tệp chỉ mục tài liệu dạng văn bản thuần \{#plain-text-doc-index-files\}

Nếu bạn cần cung cấp cho LLM của mình ngữ cảnh rộng hơn ngoài các trang riêng lẻ, chúng tôi lưu trữ các tệp chỉ mục liệt kê hoặc tổng hợp toàn bộ tài liệu Adapty:
- [`llms.txt`](https://adapty.io/docs/vi/llms.txt): Liệt kê tất cả các trang với liên kết `.md`. Đây là [tiêu chuẩn đang nổi lên](https://llmstxt.org/) giúp các LLM có thể truy cập website. Lưu ý rằng với một số AI agent (ví dụ: ChatGPT), bạn cần tải `llms.txt` xuống và tải lên chat dưới dạng file.
- [`llms-full.txt`](https://adapty.io/docs/vi/llms-full.txt): Toàn bộ tài liệu Adapty được gộp thành một file duy nhất. Rất lớn — chỉ dùng khi bạn cần xem toàn bộ nội dung.
- Các file dành riêng cho Capacitor: [`capacitor-llms.txt`](https://adapty.io/docs/vi/capacitor-llms.txt) và [`capacitor-llms-full.txt`](https://adapty.io/docs/vi/capacitor-llms-full.txt): Các tập hợp con theo nền tảng giúp tiết kiệm token so với file đầy đủ.
