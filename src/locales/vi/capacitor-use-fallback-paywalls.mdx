## Cấu hình \{#configuration\}

### Android

1. Thêm tệp cấu hình dự phòng vào ứng dụng của bạn. Chọn một trong các thư mục sau:
   * **android/app/src/main/assets/**
   * **android/app/src/main/res/raw/**
:::note
Thư mục `res/raw` có quy tắc đặt tên file đặc biệt (phải bắt đầu bằng chữ cái, không dùng chữ hoa, không dùng ký tự đặc biệt ngoại trừ dấu gạch dưới, và không có khoảng trắng trong tên file).
:::
2. Cập nhật thuộc tính `android` của hằng số `FileLocation`:
   * Nếu file nằm trong thư mục `assets`, truyền vào đường dẫn của file tương đối so với thư mục đó.
   * Nếu file nằm trong thư mục `res/raw`, truyền vào tên file không bao gồm phần mở rộng.
### iOS \{#ios\}

1. Thêm file JSON dự phòng vào bundle dự án của bạn: mở menu **File** trong XCode và chọn tùy chọn **Add Files to "YourProjectName"**.
2. Truyền tên file cấu hình của bạn vào thuộc tính `ios` của hằng số `FileLocation`.

## Ví dụ \{#example\}

```typescript showLineNumbers

const fileLocation = {
  ios: {
    fileName: 'ios_fallback.json'
  },
  android: {
    //if the file is located in 'android/app/src/main/assets/'
    relativeAssetPath: 'android_fallback.json'
  }
};

await adapty.setFallback({ fileLocation });
```

:::important
`setFallback` phải chạy trước khi SDK tải flow, paywall hoặc onboarding mục tiêu.
:::

Tham số:
| Parameter            | Description                                              |
| :------------------- | :------------------------------------------------------- |
| **fileLocation** | Đối tượng biểu thị vị trí của tệp cấu hình paywall dự phòng. |
