:::tip
**Starting from SDK v4 (beta)**, you can build [flows](android-get-pb-paywalls) as a more powerful alternative to onboardings. Unlike onboardings which run inside a WebView, flows render natively on the device — giving you smoother animations, a consistent Android look and feel, faster load times, and no WebView runtime dependency. See [Get paywalls & flows](android-get-pb-paywalls) and [Display flows & paywalls](android-present-paywalls) to get started.
:::
