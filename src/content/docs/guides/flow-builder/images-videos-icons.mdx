### Sizing

Sizing Logic (Native):

Width: Typically set to Fixed (for icons) or Fill Container (for hero images).

Height:

Auto (Aspect Ratio): Recommended. The height is automatically calculated based on the width to preserve the original proportions of the image.

Fixed: Forces a specific height. If the image aspect ratio differs, use Content Mode to control cropping.

Aspect Fit: The entire image is visible, leaving empty space if the aspect ratio doesn't match the frame (great for logos).

Aspect Fill: The image fills the entire frame, cropping edges if necessary (great for photos).

## Inline images (icons)

### Icon selector

The input field for choosing the icon symbol. It combines a "Quick Access" row for speed and a full Modal Library for variety.

UI Behavior:

Quick Access Row (Default State):
Instead of a simple dropdown, the panel displays a horizontal row of the most frequently used icons (e.g., Checkmark, Arrow Right, Chevron, Close ‘X’, Info).

This allows users to switch between standard navigation symbols with a single click, without opening the modal.

Full Library:
Clicking the More (⋮) button or the current icon preview opens the full Icon Library popover.

Here, users can search, filter by style (Stroke/Filled), or upload custom SVG icons.

Dynamic Selection Logic:

The Rule: The currently selected icon must always be visible in the Quick Access row.

The Behavior: If the user opens the library and selects an icon that is NOT present in the default "Top Frequent" list (e.g., a "Rocket"), that specific icon temporarily replaces the first slot of the Quick Access row.
