## Screens

### Screen settings
Controls the interaction between the app layout and the physical device hardware (Status Bar, Notch, Home Indicator).

Safe Area:
Label: Features a dotted underline indicating a tooltip on hover (e.g., "Keeps content from being covered by the notch or system bars").
Toggle (On/Off):
ON (Default): Applies dynamic padding to the content container (text, buttons) to avoid the device's physical cutouts.
OFF: Content extends to the absolute edge of the physical screen. Used for immersive full-screen media where text placement is manually handled.
Note: The screen background always ignores this setting and fills the entire view.

Status Bar:
Toggle shows or hides the system status bar (time, battery, signal icons). Often disabled for immersive paywalls.

Status Bar Theme:
☀️ Light Mode (Sun): Forces dark text/icons in the status bar (intended for Light Backgrounds).
🌙 Dark Mode (Moon): Forces light (white) text/icons in the status bar (intended for Dark Backgrounds).
📱 Device Default (Phone): Inherits the user’s system-wide preference.


## Layers

<ZoomImage id="flow-builder/flow-builder-layer-sidebar.webp" width="300px" alt="Flow builder UI" />

The Layers Panel represents the visual structure of your screen. It allows you to visualize nesting, reorder elements via drag-and-drop, and manage the hierarchy of your layout.

Each row in the panel represents a single element on the canvas.

### Layer actions

<ZoomImage id="flow-builder/flow-builder-layer-context-menu.webp" width="300px" alt="Flow builder UI" />

#### Name and rename

Auto-Naming: By default, layers are named based on their content or component name (e.g., a text block "Subscribe" is named "Subscribe", a button “Button”).

#### Hide and show

Visibility: Hovering over a layer reveals the 👁 (eye) icon on the right. Clicking it hides/shows the layer on the canvas.

#### Fix the layer's position

Pin: Position =Pin to page (Fixed). It means, layer pined to Screen and it’s not scrollable
Position = Absolute - It means, layer pined to its parent
