## Tab settings (global)

Layout & Spacing: Defines the direction, alignment, and internal Gap between tabs.

Sizing: Supports Fixed or Fill width to distribute tabs evenly, and Hug height to wrap content automatically.

Global Style: Sets the background Fill, Border, and Corner radius for the entire container track.

## Tab settings (individual tab)

Selection Logic: Includes a Selected by default toggle to set the initial active state.

Shared Tab Settings: Features a state-switcher (Default / Selected) to independently style the tab's appearance for both active and inactive states.

Typography & Content: Controls font styles, colors, and alignment for labels.
