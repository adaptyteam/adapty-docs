## Lists

Do not confuse with a text list.

A flexible vertical stack component designed for displaying repeating content rows. Essential for Paywall features, Onboarding steps, and Settings menus. The component supports rapid layout switching between Lists, Grids, and Timelines while maintaining consistent styling across all items.

Includes presets with icons of different styles:

* IconList
* Timeline
* Image List
* Icon cards
* Image cards
* Comparison Table
