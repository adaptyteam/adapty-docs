Text elements help you create clear, personalized conversations with your users. Add titles, paragraphs, or lists with a single click, style them to match your brand, and use [dynamic variables](onboarding-variables) to personalize content for each user.

## Set up text styles

The **Styles** panel includes pre-configured text styles: H1, H2, H3, Button Label, Body, Caption, and Small Label. Click any style to edit its font family, weight, size, alignment, decoration, and other properties.

:::tip
We recommend you to configure the text styles before you start adding text elements. This way, your configurations will be applied globally and consistently.
:::

To create a new style:

1. Click <Inline id="Styles.webp" alt="Palette"/> to open the **Styles** panel.
2. On the **Text** tab, click **Create style**.

<ZoomImage id="new-text-style.webp" width="500px" alt="New text style" />

3. Enter a name and configure the typography:
- **Font family**: Choose a typeface. For details on working with custom fonts, see the [guide](using-custom-fonts-in-paywall-builder).
- **Weight** and **size**: Set the font weight and size.
- **Color**: Set the text color.
- **Line height**: Set the spacing between lines, or leave it on **Auto**.
- **Alignment**: Set horizontal (left, center, right) and vertical (top, middle, bottom) alignment.
- **Decoration**: Apply underline or strikethrough.
- **Truncate**: Set it to **true** to set the maximum line number. When it is exceeded, the rest of the lines will be truncated. Useful when the content length can be affected by variables.
4. Click **Create**.

:::note
You can't use the SF Pro font, because it's not suitable for cross-platform applications, but we recommend you to use Inter instead, since they look quite similar.
:::

<ZoomImage id="text-style-details.webp" width="500px" alt="New text style" />

## Add text

You can add various text elements to your onboarding screens. To add text elements:
1. Click **+** at the top left. Select **Text** and choose one of the [pre-configured styles](#set-up-text-styles).

<ZoomImage id="new-text.webp" width="500px" alt="New text" />

2. Click the new element and edit its content in the **Content** section in the **Design** tab on the right.
3. If needed, adjust the typography in the **Design** tab. Or, select the text in the preview to open a tooltip for quick style customization.
4. Optionally, in the **Design** and **Interaction** tabs, you also can apply any other configurations available for components in flow.

:::tip
If you need to use the same text element on multiple screens, you can copy and paste it: select the element and press Ctrl+C (or ⌘+C on Mac), navigate to another screen, select the element you want to paste after, and press Ctrl+V (or ⌘+V on Mac).
:::

<ZoomImage id="text-settings.webp" width="500px" alt="New text" />


## Add lists

You can add lists to screens:

1. Click **+** at the top left. Select **List** and choose one of the list templates.

2. Go to the **Design** panel on the right to edit list items or upload an image as an item marker.

To edit an existing list element, click on it and make changes in the **Design** panel.
