The Billing issue chart displays the number of subscriptions that have entered the Billing Issue state. This state is typically triggered when the store, such as Apple or Google, is unable to receive payment from the subscriber for some reason. This could happen due to reasons such as an expired credit card or insufficient funds.

<ZoomImage id="8749d28-CleanShot_2023-07-11_at_15.21.262x.webp" width="700px" alt="Billing issue chart" />

## Calculation

The billing issue metric counts subscriptions that entered the billing issue state during the period. A subscription enters this state when the store (Apple or Google) can't process the renewal payment — typically because of an expired credit card or insufficient funds. During the billing issue state, the subscription is not active.

If the [grace period](grace-period) feature is enabled, the subscription enters the billing issue state only after the grace period expires without payment.
