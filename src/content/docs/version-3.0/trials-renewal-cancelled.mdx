The Trials renewal cancelled chart displays the number of trials with cancelled renewal (cancelled by user). When the renewal for the trial is disabled, this means that this trial won't be automatically converted to a paid subscription, yet the user still has premium features of the app until the end of the current period.

<ZoomImage id="fcb6948-small-CleanShot_2023-05-05_at_16.13.012x.webp" width="700px" alt="Trials renewal cancelled chart" />

## Calculation

The trials renewal cancelled metric counts trials whose auto-renewal was switched off by the user during the period. The user keeps trial access until it ends, but the trial won't convert to a paid subscription automatically.
