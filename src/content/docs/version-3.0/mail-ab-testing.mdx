Adapty Mail A/B tests work by running two or more separate campaigns against each other. Each campaign is a complete, independent email sequence. An A/B test assigns incoming users randomly across the campaigns and tracks performance per campaign — you then finish the test and select a winner.

## What you can test

Each campaign in the test can differ in:

- Email copy (subjects, headlines, body, CTA)
- Hero images (AI-generated per campaign)
- Tone preset
- Sequence length (number of emails)
- Delay timing between emails

## Subject line testing

Subject line testing happens automatically within every campaign — no setup required and it cannot be disabled.

Every email is generated with **3 subject line variants**. When the campaign runs, Adapty tests all three and continues sending the best-performing variant to the remaining audience.

Subject line results are visible per-email in the Analytics dashboard.

## Create an A/B test

1. Create two or more campaigns independently in **Campaigns**. Each campaign is a full email sequence.
2. Go to **A/B Tests → Create**.
3. Add each campaign as a variation and set its weight. Weights must total 100%.
4. Assign a segment to control which users enter the test.
5. Launch the A/B test.

## How the test runs

1. Each new user who meets the segment criteria is assigned randomly to one campaign according to the configured weights.
2. Each campaign runs independently — same timing, independent metrics.
3. Delivery, engagement, and revenue are tracked per campaign.

## Read results

In **Analytics**, select the A/B test to see per-campaign metrics side by side:

- Open rate per campaign
- Click rate per campaign
- Conversion rate per campaign
- Revenue per campaign

## Deploy a winner

When you've identified the best-performing campaign:

1. Open the A/B test.
2. Click **Finish A/B test**.
3. Select the winning campaign in the dialog.

Adapty routes all new users who enter the segment to the winning campaign. Users already mid-sequence in a losing campaign continue receiving their remaining emails — they are not switched to the winner.
