The Active subscriptions chart displays the number of unique paid subscriptions that have not yet expired at the end of each selected period. It includes regular (unexpired) in-app subscriptions that started and are currently active and excludes both free trials and subscriptions with canceled renewals. It's a proxy for the size and growth of your subscriber base.

<ZoomImage id="a4895aa-small-CleanShot_2023-05-04_at_15.13.262x.webp" width="700px" alt="Active subscriptions chart" />

## Calculation

The active subscriptions metric counts paid, unexpired subscriptions at the end of each period. For subscriptions without a grace period, expiration happens when the next renewal date passes without a successful renewal.

For example: 500 active subscriptions at the end of last month, plus 50 new this month, minus 25 that expired this month = 525 active subscriptions at the end of this month.

## Refund handling

When a subscription is refunded, Adapty removes it from the active count — both currently and retroactively for past dates.

For the full comparison across metrics, see [How metrics handle refunds](refund-events#how-metrics-handle-refunds).
