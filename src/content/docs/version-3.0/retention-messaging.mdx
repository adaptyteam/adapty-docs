Retention Messaging shows subscribers a custom message on the iOS Cancel Subscription screen, before they confirm a cancellation. Use it to persuade churning subscribers to stay.

A default message gives every subscriber a reason to stay. A promo message adds a concrete incentive: a discount, a free trial, or an opportunity to switch to a more fitting plan. Apple shows each subscriber the message that matches their product and language.

You create, localize, and test messages in the Adapty Dashboard. Adapty delivers them to Apple automatically, with no app release.

## Quickstart

Before you start, you need:

- Access to [Apple's Retention Messaging program](https://developer.apple.com/documentation/retentionmessaging).
- An [App Store Connect API key](app-store-connection-configuration) in Adapty.

Then:

1. [Connect Adapty in sandbox mode](retention-messaging-setup).
2. [Create a default message](retention-messaging-create) in sandbox and make a sandbox purchase.
3. Pass Apple's connectivity test.
4. Switch to production mode.
5. Create your production messages — the default first, then any promo — and submit each for Apple's review.
