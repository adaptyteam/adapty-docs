The new trial chart displays the number of activated trials during the selected time period. Use it to track trial volume from ad campaigns and other acquisition efforts.

<ZoomImage id="47820b3-small-CleanShot_2023-05-05_at_15.30.492x.webp" width="700px" alt="New trials chart" />

## Calculation

The new trials metric counts trials started during the period, regardless of whether they're still active by the end of the period.

For example, if 50 users start a trial in May, the May data point shows 50 — even if some have already expired or converted to paid by the time you view the chart.

## Available filters and grouping

:::link
Main article: [Analytics controls](controls-filters-grouping-compare-proceeds)
:::

- ✅ Filter by: Attribution, country, paywall, store, product, and duration.
- ✅ Group by:  Product, country, store, paywall, duration, attribution status, attribution channel, attribution campaign, attribution ad group, attribution ad set, and attribution creative.
