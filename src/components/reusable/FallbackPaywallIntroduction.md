<!--- FallbackPaywallIntroduction--->

To maintain a fluid user experience, it is important that you set up a [fallback version](/fallback-paywalls) for your [paywalls](paywalls) and [onboardings](onboardings). That way, if the user encounters connection issues or server outages, the application will display a complete paywall anyway.

:::important
Before you follow the steps in this guide, [download](/local-fallback-paywalls) the fallback configuration files from Adapty.
:::
